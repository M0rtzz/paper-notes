# Dual Goal Representations

**会议**: ICLR 2026  
**arXiv**: [2510.06714](https://arxiv.org/abs/2510.06714)  
**代码**: 无  
**领域**: 强化学习 / 目标条件RL  
**关键词**: goal-conditioned RL, goal representation, temporal distance, offline RL, OGBench

## 一句话总结

提出 dual goal representation，通过"从所有其他状态到达该状态的时间距离集合"来刻画每个状态，为目标条件强化学习提供了一种理论上可证明最优、实践中可插拔的目标表征学习方法。

## 研究背景与动机

目标条件强化学习（Goal-Conditioned RL, GCRL）要求智能体学会从任意初始状态到达任意目标状态。核心挑战之一是目标的表征问题：在高维观测空间（如像素图像）中，原始状态表示包含大量与目标到达无关的信息（如背景变化、光照等外部噪声），直接用原始状态做目标匹配效率低下且不够鲁棒。

现有的 GCRL 方法通常直接使用原始状态表示或手搜特征来定义目标，缺乏对"什么才是好的目标表征"的系统化理论分析。一个理想的目标表征应当：
1. 只依赖环境的内在动力学，与原始状态表示无关（表示不变性）
2. 包含足够信息来恢复最优的目标到达策略（信息充分性）
3. 能够过滤掉与目标到达无关的外部噪声（噪声鲁棒性）

基于此，作者提出了 dual goal representation 的概念，旨在为 GCRL 提供一个满足上述所有性质的通用目标表征框架。

## 方法详解

### 整体框架

Dual goal representation 的核心思想非常直觉：**用一个状态与所有其他状态之间的时间距离（temporal distance）来定义该状态的表征**。形式化地说，对于状态 s，其 dual representation 是一个函数 φ(s)，编码了"从所有可能的状态 s' 到达 s 需要的最短时间步数"这一信息。

这种表征的命名来源于对偶（dual）的思想：传统表征直接编码状态本身的特征，而 dual representation 则通过状态与其他状态之间的关系来间接描述状态，类似于数学中从原空间到对偶空间的映射。

### 关键设计

1. **时间距离函数（Temporal Distance Function）**: 定义为从状态 s 到达状态 g 所需的最短期望步数。这是 GCRL 中的核心量，dual representation 本质上就是将每个状态嵌入到由时间距离定义的函数空间中。通过学习一个参数化的距离函数网络来近似真实时间距离。

2. **Dual Representation 的形式化定义**: 对于状态 s，其 dual representation φ(s) 被定义为 d(·, s)，即所有状态到 s 的时间距离构成的向量/函数。在有限状态空间中，这等价于距离矩阵的一列；在连续空间中，则需要通过采样或参数化来近似。

3. **理论性质证明**:

    - **不变性**: dual representation 只依赖 MDP 的转移动力学，不依赖具体的状态编码方式。如果两个 MDP 在动力学上是同构的，它们的 dual representation 也是同构的
    - **充分性**: 从 dual representation 可以恢复出最优的目标到达策略，不会丢失关键信息
    - **去噪能力**: 外部噪声（exogenous noise）不影响状态间的时间距离，因此会被 dual representation 自然过滤掉

4. **实际学习算法**: 将 dual representation 学习转化为一个可与现有 GCRL 方法结合的目标编码器。具体来说：

    - 训练一个距离预测网络 d_θ(s, g) 来预估时间距离
    - 利用 d_θ 的中间特征或输出来构造 goal embedding
    - 将学到的 goal embedding 注入到下游的 GCRL 算法中（如 GCSL、Contrastive RL 等），替换原本的原始状态表征

### 损失函数 / 训练策略

训练过程使用离线数据集中的轨迹来学习时间距离。关键损失函数包括：
- 时间距离的回归损失：使用轨迹中状态对之间的实际步数作为监督信号
- 可能辅之以对比学习损失来增强表征的区分性
- 整体训练为两阶段或端到端：先学表征再学策略，或将表征学习与策略学习联合优化

由于是离线（offline）设置，所有训练数据来自预收集的轨迹数据集，不需要在线交互。

## 实验关键数据

### 主实验

实验在 OGBench 任务套件上进行，覆盖 20 个任务，包括：
- 状态空间任务（state-based）：直接使用低维状态向量
- 像素空间任务（pixel-based）：使用图像观测

| 设置 | 说明 | 改进情况 |
|------|------|---------|
| State-based tasks | 多个连续控制环境 | 一致性提升 |
| Pixel-based tasks | 图像观测的导航/操作任务 | 显著提升 |
| 与多种 GCRL 算法结合 | 验证可插拔性 | 跨算法提升 |

### 消融实验

| 配置 | 说明 |
|------|------|
| 有/无 dual representation | 验证表征的有效性 |
| 不同距离函数变体 | 比较时间距离 vs 其他度量 |
| 有无外部噪声 | 验证去噪能力 |

### 关键发现

- Dual goal representation 在 20 个 OGBench 任务中一致提升离线目标到达性能
- 在像素空间任务中改进尤为明显，说明表征在高维空间中的去噪能力特别重要
- 方法具有良好的可插拔性，能够与多种现有 GCRL 算法兼容
- 在存在外部噪声干扰的环境中，性能优势更加显著

## 亮点与洞察

1. **理论基础扎实**: 不仅提出了 dual representation 的概念，还严格证明了其不变性、充分性和去噪性，这在 GCRL 领域比较少见
2. **从对偶视角理解目标表征**: "用关系定义实体"的思想很有哲学深度，类似于图论中节点可以用其到所有其他节点的距离来唯一表征
3. **实用性强**: 作为一个可插拔模块，不需要改变底层 GCRL 算法的架构，降低了使用门槛
4. **来自 Sergey Levine 组**: 延续了 Berkeley 在 GCRL 和表征学习方面的研究传统

## 局限性 / 可改进方向

1. **计算开销**: 学习完整的时间距离函数可能在大规模环境中计算量较大
2. **离线设置限制**: 目前仅在离线 GCRL 上验证，在线学习场景的效果有待探索
3. **连续空间近似**: 在连续状态空间中，dual representation 需要通过采样或参数化来近似，近似精度可能影响性能
4. **环境假设**: 理论分析可能依赖一些环境假设（如确定性转移等），在高度随机的环境中效果可能折扣
5. **缺乏真实机器人实验**: OGBench 虽然多样，但仍是仿真环境，实机验证缺失

## 相关工作与启发

- **GCRL 基础方法**: 如 HER（Hindsight Experience Replay）、GCSL 等，dual representation 可以增强这些方法
- **表征学习 for RL**: 如 contrastive RL、ACRO 等，本文提供了一个有理论保证的表征学习目标
- **时间距离学习**: C-Learning、Quasimetric RL 等也使用了时间距离，但本文将其提升为表征的核心构建块
- 对目标条件任务的表征设计有启发意义，特别是在需要鲁棒性和泛化性的场景中
