# How LLMs Learn to Reason: A Complex Network Perspective

**会议**: ICLR 2026  
**arXiv**: [2509.23629](https://arxiv.org/abs/2509.23629)  
**代码**: [GitHub](https://anonymous.4open.science/r/CoNet-83A4)  
**领域**: LLM推理/理论分析  
**关键词**: RLVR, 概念网络, 稀疏图, 灾难性遗忘, 模拟退火

## 一句话总结
从复杂网络视角统一解释RLVR训练的四大谜题（两阶段学习曲线、V型回复长度、灾难性遗忘、策略坍缩），提出稀疏概念网假说（平均度≈2），并据此设计Annealed-RLVR算法在Minerva和AIME上超越标准RLVR。

## 研究背景与动机

1. **领域现状**：RLVR训练LLM展现出四个令人困惑的现象：(i) 快速提升后长期平台期的两阶段学习曲线；(ii) 正确回复长度先缩短再增长的V型轨迹；(iii) 后续SFT导致的灾难性遗忘；(iv) 解的多样性坍缩。

2. **现有痛点**：现有解释碎片化——平台期归因于熵耗尽、V型归因于奖励驱动裁剪或自反思、遗忘归因于目标不匹配。缺乏统一框架把四个现象联系到共同的结构或动力学机制。

3. **核心矛盾**：直接从LLM高维潜空间构造推理图极其困难，无法直接研究结构根源。

4. **切入角度**：借鉴复杂系统的重整化群(RG)思想——宏观涌现行为独立于微观细节。构建最小代理模型CoNet（概念网络模型），在语义层面粗粒化推理图。

5. **核心假说**：概念网是平均度钉在 $\langle k \rangle \approx 2$ 的稀疏网络，主要为树状结构，高效但脆弱。

## 方法详解

### 整体框架
用1.5B LLM (DeepScaleR) 验证四大RLVR现象→构建CoNet最小模型复现相同现象→利用CoNet透明性分析微观结构→从稀疏网假说推导统一解释→设计Annealed-RLVR算法。

### 关键发现

1. **V型轨迹 = 从局部优化到全局整合的结构演化**:
   - 快速学习阶段（V的下降段）：并行发现短的、独立的"技能岛"，策略快速丢弃低效探索
   - 慢学习阶段（V的上升段）：技能岛融合为统一的概念网。平均度≈2意味着链状/树状结构，连接远距概念需要长路径→回复长度必然增长

2. **灾难性遗忘 = 稀疏网关键桥的拓扑断裂**:
   - 稀疏网依赖少量"主干边"连接大量下游子图
   - SFT在高流量分支点覆盖权重→断开整个子树
   - 但损伤是局部的：断裂子图的知识仍完好，只是不可达→恢复RLVR只需"重新焊接"

3. **策略坍缩 = 前沿节点处的连续相变累积**:
   - 慢学习阶段，前沿叶节点经历连续相变式学习——从广泛探索锁定到单一最优路径
   - 单个技能的坍缩是必要的，但标准RLVR在所有叶节点累积这些局部熵减→全局多样性丧失

4. **最大挫败态 = 退火干预的最佳时机**:
   - 在慢学习开始时，技能岛间竞争达到峰值（最大挫败态），同时探索能力也达到峰值
   - 此时执行短暂SFT"加热"→打破过早承诺→恢复RLVR"冷却"→沉降到更优配置

### Annealed-RLVR算法
- 监控V型曲线底部和奖励曲线拐点→确定最大挫败态时机
- 对低准确率(<0.1)的问题执行短暂SFT"加热"
- 恢复标准RLVR"冷却"

## 实验关键数据

### 主实验
DeepSeek-R1-Distill-Qwen-1.5B：

| 方法 | 训练集best@k | Minerva(OOD) | AIME 2024/25(OOD) |
|------|-------------|-------------|-------------------|
| 标准RLVR | 基线 | 基线 | 基线 |
| Annealed-RLVR | **更好** | **更好** | **更好** |

### CoNet vs LLM对应
| 现象 | CoNet | LLM | 一致性 |
|------|-------|-----|--------|
| 两阶段曲线 | ✓ | ✓ | 高度吻合 |
| V型长度 | ✓ | ✓ | 定性一致 |
| 灾难性遗忘+快速恢复 | ✓ | ✓ | 定性一致 |
| 策略坍缩 | ✓ | ✓ | 定性一致 |

### 关键发现
- CoNet中概念网的平均度确实收敛到≈2，直接验证了核心假说
- SFT"加热"后恢复RLVR的恢复速度远快于从头训练——因为只需重连断裂的桥
- Annealed-RLVR在分布内和分布外基准上都超越标准RLVR
- 最大挫败态时探索能力达峰（best@k最高），之后因策略坍缩而下降

## 亮点与洞察
- **物理学思维解AI**：用重整化群、相变、模拟退火等物理概念构建RLVR的统一理论。四个现象的统一解释来自一个简洁假说（平均度≈2）。
- **CoNet作为"计算显微镜"**：无法直接分析LLM的推理图，但可以用最小代理模型复现同样的宏观现象，然后利用代理模型的透明性分析微观结构。
- **"最大挫败态"的精确干预**：不是随意加SFT，而是在理论指导的最优时机执行精确干预——就像模拟退火中的温度调度。

## 局限性 / 可改进方向
- CoNet是极简模型，可能遗漏LLM的某些重要特性
- 直接从LLM中提取完整推理图仍是开放问题
- 仅在1.5B模型上验证，更大模型的稀疏网特性可能不同
- 退火时机的自动检测方法需要进一步开发

## 相关工作与启发
- **vs 现有RLVR分析**: 碎片化分析各现象的起因，本文提供统一的拓扑结构解释
- **vs 课程学习**: Annealed-RLVR可以看作理论驱动的课程调度，比启发式方法有更强理论支撑
