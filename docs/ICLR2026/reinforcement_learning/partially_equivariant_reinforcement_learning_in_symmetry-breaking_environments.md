# Partially Equivariant Reinforcement Learning in Symmetry-Breaking Environments

**会议**: ICLR 2026  
**arXiv**: [2512.00915](https://arxiv.org/abs/2512.00915)  
**代码**: [项目页面](https://pranaboy72.github.io/perl_page/)  
**领域**: 强化学习/等变性  
**关键词**: 部分等变性, 对称性破缺, 群不变MDP, 门控策略, 误差传播

## 一句话总结
提出部分群不变MDP(PI-MDP)框架解决RL中的对称性破缺问题——分析证明局部对称性违反通过Bellman backup在整个状态-动作空间产生全局值估计误差，PI-MDP在对称区域使用等变更新、在破缺区域回退到标准更新→阻止误差传播，开发PE-DQN(离散)和PE-SAC(连续)算法在Grid-World/运动/操作任务上显著超越严格和近似等变基线。

## 研究背景与动机

1. **领域现状**：群对称为RL提供强大归纳偏置→等变网络实现对称状态泛化→样本效率提升。群不变MDP(奖励和转移函数均群不变)是理论基础。

2. **现有痛点**：
   - (1) 现实环境几乎不完全群不变→动力学/执行器限制/奖励设计打破对称
   - (2) 严格等变RL在对称破缺下→局部误差通过Bellman传播到全局→值估计退化/策略次优
   - (3) 近似等变方法(全局松弛)→丧失严格等变的样本效率+破缺严重时不稳定

3. **切入角度**：选择性应用等变性→对称保持区域用等变、破缺区域用标准→阻止误差传播。

## 方法详解

### 理论分析：局部→全局误差传播

定义逐点对称性偏差：
- $\epsilon_R(s,a) = |R_N(s,a) - R_E(s,a)|$ (奖励偏差)
- $\epsilon_P(s,a) = \frac{1}{2}\int|P_N(s'|s,a) - P_E(s'|s,a)|ds'$ (转移偏差)

**定理**：群不变Bellman backup的一步误差 ≤ $\epsilon_R + \gamma V_{max}\epsilon_P$
→ 但折扣回溯→局部误差$\epsilon$产生全局误差$\frac{\epsilon}{1-\gamma}$

### PI-MDP框架

```
对每个(s,a):
  if 对称性保持(λ=1):  使用等变策略π_E
  if 对称性破缺(λ=0):  使用标准策略π_N
```

### 对称性破缺检测
- 训练等变预测器$f_E$和无约束预测器$f_N$
- 两者预测不一致→该(s,a)处对称性破缺
- 用预测器分歧的异常值检测确定λ

### 实用算法
- **PE-DQN**：离散控制→等变Q网络+标准Q网络+门控λ
- **PE-SAC**：连续控制→等变actor-critic+标准actor-critic+门控λ

## 实验关键数据

### Grid-World(有障碍物破缺旋转对称)
| 方法 | 成功率 | 说明 |
|------|--------|------|
| 标准DQN | 中 | 无等变性 |
| 等变DQN | 低(破缺时) | 误差传播 |
| 近似等变DQN | 中 | 全局松弛 |
| **PE-DQN** | **最高** | 选择性等变 |

### 运动/操作(MuJoCo)
| 方法 | 奖励 | 样本效率 |
|------|------|---------|
| SAC | 基线 | 基线 |
| 等变SAC | 破缺时差 | 高(无破缺) |
| **PE-SAC** | **最好** | **最好** |

### 关键发现
- 对称性破缺程度越大→PE方法相对优势越大
- 门控λ的学习→自动发现破缺区域→与人工标注一致
- 完全对称环境→PE退化为严格等变→无损失
- 误差传播的理论预测与实验吻合

## 亮点与洞察
- **"选择性等变"的哲学**：不是"全等变or全不等变"→而是"哪里对称就哪里等变"→更精细的归纳偏置控制。
- **误差传播的理论分析**：证明了局部破缺→全局误差→解释了之前等变RL在真实环境中失败的根本原因。
- **预测器分歧检测对称性**：不需要先验知识说哪里对称→数据驱动自动检测→实用。
- **连续体：全标准←→PE←→全等变**：PE是两个极端的可控插值→通过λ可以平滑过渡。
