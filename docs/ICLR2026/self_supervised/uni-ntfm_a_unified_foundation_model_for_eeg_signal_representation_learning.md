# Uni-NTFM: A Unified Foundation Model for EEG Signal Representation Learning

**会议**: ICLR 2026  
**arXiv**: [2509.24222](https://arxiv.org/abs/2509.24222)  
**代码**: [https://anonymous.4open.science/r/Uni-NTFM-0924](https://anonymous.4open.science/r/Uni-NTFM-0924)  
**领域**: self_supervised  
**关键词**: EEG, 基础模型, 神经拓扑, 混合专家, 自监督学习, 脑机接口

## 一句话总结

Uni-NTFM 是一个受生物神经机制启发的 EEG 统一基础模型，通过异质特征投影模块解耦时域和频域编码、拓扑嵌入机制将不同传感器配置对齐到统一功能拓扑空间、以及混合专家 Transformer 实现功能模块化和稀疏编码，在 28000 小时 EEG 数据上预训练并达到 19 亿参数规模，在 9 个下游任务上的线性探测和微调设定下均超越现有模型。

## 研究背景与动机

脑电图（EEG）是最广泛使用的神经信号记录方式之一，在脑机接口、临床诊断、认知科学等领域有重要应用。近年来，随着大规模预训练基础模型在 NLP 和 CV 领域的巨大成功，EEG 基础模型也成为活跃研究方向。

然而，现有的 EEG 基础模型存在根本性缺陷：
- **架构移植不匹配**：大多数模型直接借用计算机视觉（将 EEG 视为像素网格）或 NLP（将 EEG 视为 token 序列）的架构，忽略了神经信号的独特特性
- **忽视稀疏编码原理**：大脑的神经活动是通过复杂几何拓扑皮层上的多样稀疏编码激活的，而非密集的像素/文本模式
- **传感器配置不统一**：不同实验和设备使用不同的电极布局（通道数从几个到几百），不同数据集之间难以迁移
- **时频特性解耦不足**：EEG 同时包含时域的非平稳瞬态事件（如棘波、K-complex）和频域的稳态节律（如 alpha、beta 波），需要不同的编码策略

本文的核心思想：模型架构应该与生物神经机制对齐——解耦编码、拓扑感知、模块化稀疏处理。

## 方法详解

### 整体框架

Uni-NTFM 基于三个核心神经科学原理设计三个关键模块：(1) 异质特征投影模块（Heterogeneous Feature Projection Module, HFPM）对应大脑的解耦编码机制；(2) 拓扑嵌入机制（Topological Embedding）重建脑区几何；(3) 混合专家 Transformer（MoE Transformer）实现功能模块化和稀疏编码效率。

### 关键设计

1. **异质特征投影模块（HFPM）**：
   - 核心动机：大脑对时域瞬态事件（如运动执行时的突发放电）和频域稳态节律（如静息态的 alpha 振荡）采用不同的神经编码机制
   - 同时编码时域非平稳瞬态信号和频域稳态节律两个分支
   - 确保波形形态（waveform morphology）和谱节律（spectral rhythms）的高质量表示
   - 这种双分支设计避免了将两种本质不同的信号特征混合编码带来的信息损失

2. **拓扑嵌入机制（Topological Embedding）**：
   - 核心动机：不同数据集使用不同的电极配置（如 10-20 系统 vs 高密度 EEG），需要统一的空间表示
   - 注入结构化空间先验（structured spatial priors）
   - 将不同传感器配置对齐到统一的潜在功能拓扑空间（unified latent functional topography）
   - 有效重建脑区的几何结构——即使不同数据集的电极布局差异很大
   - 这使得跨数据集的知识迁移成为可能

3. **混合专家 Transformer（MoE Transformer）**：
   - 核心动机：生物神经网络通过功能模块化和稀疏激活实现高效处理——视觉皮层 V1 区响应视觉，布罗卡区响应语言，不同功能区并行而不互相干扰
   - 动态路由机制将不同信号模式和任务分配给专门的神经子网络
   - 有效防止任务干扰（task interference）
   - 通过稀疏激活大幅增加模型容量——达到创纪录的 19 亿参数
   - 关键优势：参数量大但计算量可控（每个 token 只激活部分专家）

### 损失函数 / 训练策略

- 预训练数据：28,000 小时的多样化 EEG 语料库，涵盖多种记录条件和任务类型
- 模型规模：19 亿（1.9B）参数——EEG 基础模型的记录规模
- 评估设定：
  - **线性探测（Linear Probing）**：冻结预训练权重，仅训练线性分类头
  - **微调（Fine-tuning）**：在预训练权重基础上端到端训练
- 涵盖 9 个不同的下游任务，全面评估通用表示的质量

## 实验关键数据

### 主实验

论文在 9 个不同的下游 EEG 任务上评估，涵盖了 EEG 研究的主要应用场景。在线性探测和微调两种评估范式下，Uni-NTFM 均超越现有模型。

| 评估方式 | 表现 | 说明 |
|---------|------|------|
| 线性探测 | 9 个任务全面领先 | 说明预训练表示质量高 |
| 微调 | 9 个任务全面领先 | 说明模型可有效适配下游 |

### 消融实验

| 配置 | 关键作用 | 说明 |
|------|---------|------|
| 移除 HFPM | 时频编码质量下降 | 验证解耦编码的必要性 |
| 移除拓扑嵌入 | 跨数据集迁移能力下降 | 验证空间对齐的重要性 |
| 移除 MoE → 标准 Transformer | 容量受限，多任务干扰增加 | 验证稀疏模块化的价值 |
| 减小预训练数据量 | 性能下降 | 验证大规模数据的作用 |

### 关键发现

- **神经机制对齐显著有效**：三个模块分别对应大脑的解耦编码、拓扑组织和功能模块化，每个都对最终性能有独立贡献
- **统一拓扑空间解决了 EEG 领域的核心难题**：不同电极配置的数据集终于可以在同一模型中学习和迁移
- **稀疏 MoE 实现了高效扩展**：19 亿参数的模型相比标准 dense Transformer 有更好的容量-计算权衡
- **通用表示的质量**：线性探测的良好表现说明模型学到了具有通用性的 EEG 特征编码

## 亮点与洞察

- **生物启发的架构设计范式**：不是简单移植 CV/NLP 架构，而是从神经科学原理出发设计架构，这种"first principles"方法在 EEG 语境下非常合理
- **解决了 EEG 基础模型的三个核心瓶颈**：时频混合编码、异构电极配置、多任务干扰
- **19 亿参数是 EEG 领域的里程碑**：MoE 架构使得这一规模在合理计算预算内成为可能
- **28000 小时的预训练数据**：体现了与 NLP/CV 基础模型类似的"大数据 + 大模型"范式在 EEG 领域的可行性
- **9 个下游任务的全面评估**：不仅仅是单一任务的改进，而是通用表示学习的系统验证

## 局限性 / 可改进方向

- ar5iv HTML 转换失败导致无法获取完整实验数据和方法细节，以上分析主要基于摘要和架构描述
- 19 亿参数的模型在实际部署时可能面临计算资源限制
- 拓扑嵌入的"统一功能拓扑空间"的解释性和可视化分析是否充分尚不清楚
- MoE 的路由策略是否能稳定收敛、不同专家是否真正学到了功能性分化需要详细验证
- 预训练数据的 28000 小时是否覆盖了足够的人群多样性（年龄、性别、病理状态等）
- 目前 EEG 基础模型领域发展迅速，需要与 LaBraM、EEGFormer 等近期模型进行详细对比

## 相关工作与启发

- **EEG 基础模型前置工作**：LaBraM、BrainBERT、EEGFormer 等尝试将 Transformer 应用于 EEG，但未考虑解耦编码和拓扑对齐
- **MoE 在 NLP/CV 中的成功**：Switch Transformer、Mixtral 等证明了稀疏 MoE 的扩展效率
- **神经拓扑学**：Brain Network 研究中的功能连通性分析为拓扑嵌入提供了理论基础
- **启发**：
  - "生物机制对齐"的设计思路可推广到其他生物信号（EMG、fNIRS、MEG）
  - MoE 的任务路由机制对多任务 EEG BCI 系统有直接应用价值
  - 统一拓扑空间的思路可能对多中心 EEG 数据的联邦学习有帮助
