# Horizon Imagination: Efficient On-Policy Rollout in Diffusion World Models

**会议**: ICLR 2026  
**arXiv**: [2602.08032](https://arxiv.org/abs/2602.08032)  
**代码**: [GitHub](https://github.com/leor-c/horizon-imagination)  
**领域**: 世界模型/强化学习  
**关键词**: 扩散世界模型, 并行去噪, 想象效率, 采样调度, 离散策略

## 一句话总结
提出Horizon Imagination(HI)——扩散世界模型中的高效在策略想象过程：并行去噪多个未来观测(而非逐帧串行)，引入稳定化机制和新型采样调度(将去噪预算从有效horizon解耦+支持亚帧预算)，在Atari 100K和Craftium上仅用一半去噪步骤即保持控制性能，系统分析串行vs并行生成的质量trade-off。

## 研究背景与动机

**领域现状**：扩散世界模型(如DIAMOND)→高保真生成→但用于RL控制面临效率瓶颈：每帧需多步去噪→想象(训练控制器)时需逐帧串行交替(策略决策→世界模型预测)→极其耗时。

**现有痛点**：
   - (1) 大型扩散世界模型→推理成本高→不适合实时控制
   - (2) 串行想象→每帧的去噪必须完成才能做下一步决策→序列瓶颈
   - (3) 并行去噪→已有探索但仅限连续动作空间(Diffusion Forcing/Rigter等)
   - (4) 缺乏系统分析：并行 vs 串行+去噪预算对生成质量的影响

**切入角度**：离散随机策略+并行去噪多帧→将去噪预算和horizon解耦→更灵活的效率-质量trade-off。

## 方法详解

### 标准串行想象(基线)
```
for each frame:
    action = policy(current_obs)
    next_obs = denoise(noise, action, K steps)  # K步去噪
    current_obs = next_obs
```
总去噪步数 = H × K (H=horizon, K=每帧步数)

### Horizon Imagination(并行)
```
初始化所有H帧为噪声
for step i = 1 to budget:
    对所有帧并行去噪(不同帧不同去噪水平)
    策略在部分去噪的帧上采样动作
```
总去噪步数 = budget (可< H×K)

### 关键机制

1. **策略稳定化**：
   - 部分去噪的帧→策略可能不稳定
   - 引入参考动作分布→与去噪帧的策略加权平均
   - 随去噪进展→逐渐信任当前帧的策略

2. **Horizon采样调度**：
   - 将去噪预算和有效horizon独立控制
   - 支持亚帧预算(budget < K)→每帧分配不到完整去噪
   - 调度函数控制不同帧的去噪优先级(近帧多步/远帧少步)

3. **在策略想象**：
   - 策略决策嵌入去噪过程→不是离线生成
   - 生成的轨迹→直接用于训练轻量独立策略

## 实验关键数据

### Atari 100K
| 方法 | 去噪预算 | 控制性能 | 想象效率 |
|------|---------|---------|---------|
| 串行(baseline) | H×K | 基线 | 基线 |
| HI(全预算) | H×K | ≈基线 | 更好(并行) |
| **HI(半预算)** | **H×K/2** | **≈基线** | **2x快** |
| HI(1/4预算) | H×K/4 | 轻微降 | 4x快 |

### 生成质量分析
| 配置 | FVD↓ | 视觉质量 |
|------|------|---------|
| 纯串行 | 基线 | 好 |
| **并行(Horizon调度)** | **更好** | **更好** |
| 并行(均匀分配) | 较差 | 较差 |

### Craftium验证
- 更复杂的3D环境→HI同样有效
- 半预算性能保持→验证泛化性

### 关键发现
- 并行生成不仅更快→而且可以更好→因为近帧信息反馈到远帧
- Horizon调度是关键→均匀分配不如智能调度
- 亚帧预算可行→每帧不需要完整K步也能工作
- 策略稳定化→必需→否则部分去噪帧产生错误动作→cascading失败

## 亮点与洞察
- **"去噪预算≠horizon"的解耦**：之前budget=H×K→HI允许budget<H×K→突破了线性关系。
- **并行>串行(某些配置)**：反直觉→并行去噪可以提供跨帧一致性→类似视频扩散的时序一致性。
- **训练无关(training-agnostic)**：适用于任何预训练扩散世界模型→不需要修改/重训模型。
- **亚帧预算的意义**：当计算预算极其有限时→HI仍能工作→对edge deployment重要。
