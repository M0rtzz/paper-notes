# Swap-guided Preference Learning for Personalized RLHF (SPL)

**会议**: ICLR 2026  
**arXiv**: [2603.12595](https://arxiv.org/abs/2603.12595)  
**代码**: https://github.com/cobang0111/SPL  
**领域**: 对齐RLHF / 个性化对齐  
**关键词**: 个性化奖励模型, 后验崩坏, 潜变量偏好学习, swap引导正则化, 偏好多样性

## 一句话总结
解决变分偏好学习(VPL)中的后验崩坏问题：提出SPL，通过swap引导基础正则化(强制潜变量编码用户偏好而非被忽略)+Preferential-IAF分解swap可逆/不可逆信号+自适应潜变量调节。在Llama-3.1-8B上达63.71%准确率+97.10%活跃单元，而VPL崩坏到57.14%+0%。

## 研究背景与动机

1. **领域现状**：统一奖励模型假设所有用户偏好一致，但实际中用户偏好存在显著多样性。VPL用潜变量建模用户特定偏好。
2. **现有痛点**：VPL的潜变量在稀疏数据+强解码器组合下会后验崩坏——潜变量被完全忽略，退化为单一奖励模型
3. **核忊idea一句话**：swap引导正则化强制潜变量有用 + IAF分解用户特定信号

## 方法详解

### 关键设计

1. **Swap引导基础正则化**：交换偏好对中chosen/rejected的顺序，强制潜变量的均值符号翻转（偏好反向→潜变量方向反向）且标准差不变，确保潜变量编码偏好信息
2. **Preferential-IAF**：将潜变量分解为swap可逆部分(偏好方向)和swap不可逆部分(過当自信)
3. **自适应潜变量调节**：动态推理时根据当前偏好对调整潜变量

## 实验关键数据

| 模型 | 方法 | 准确率 | 活跃单元率 |
|------|------|--------|------------|
| Llama-3.2-3B | VPL | 62.37% | 88.22% |
| Llama-3.2-3B | **SPL** | **63.28%** | **93.07%** |
| Llama-3.1-8B | VPL | 57.14% | **0%(崩坏!)** |
| Llama-3.1-8B | **SPL** | **63.71%** | **97.10%** |

### 关键发现
- **VPL在8B模型上完全崩坏**：活跃单元从88%降到0%，潜变量被忽略
- **SPL消除崩坏**：即使在8B模型上仍保持97.10%活跃单元

## 亮点与洞察
- Swap引导是解决后验崩坏的优雅方案——通过对称性约束强制潜变量有5e义

## 局限性 / 可改进方向
- 测试场景有限，未验证实际RLHF训练中的个性化效果

## 相关工作与启发
- **vs VPL**：SPL解决了VPL的核心缺陷(后验崩坏)，使个性化奖励模型在较大模型上也可用
