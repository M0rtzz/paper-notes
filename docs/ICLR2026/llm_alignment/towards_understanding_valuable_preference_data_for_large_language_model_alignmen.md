# Towards Understanding Valuable Preference Data for LLM Alignment

**会议**: ICLR 2026  
**arXiv**: [2510.13212](https://arxiv.org/abs/2510.13212)  
**代码**: https://github.com/tmlr-group/TIF_LossDiff-IRM  
**领域**: 对齐RLHF / 数据选择  
**关键词**: 偏好数据质量, 影响函数, 模型相关数据价值, 数据筛选, DPO

## 一句话总结
挑战“数据质量是固有属性”的假设：发现偏好数据价值是模型相关的——对一个模型有益的数据可能伤害另一个。提出截断影响函数(TIF)识别高价值数据，及高效的LossDiff-IRM代理，仅用50-64%数据即提升WinRate 13.58%。

## 研究背景与动机

1. **领域现状**：DPO/RLHF的效果很大程度取决于偏好数据质量。现有方法用强度/分数等固有指标判断数据价值。
2. **现有痛点**：数据价值不是固有的而是模型相关的——一个偏好对对模型A有益但可能对模型B有害
3. **核心idea一句话**：截断影响函数识别模型特定高价值数据 + 高效代理让筛选实用化

## 方法详解

### 关键设计

1. **截断影响函数(TIF)**：使用影响函数分析每个偏好对对模型的实际影响，截断极端值保持稳定
2. **LossDiff-IRM代理**：仅需前向传播的评分函数，与TIF的Pearson相关性达0.77-0.80，计算时间从~10小时降到~5分钟
3. **中等IF最优**：不是高影响值最好，而是中等影响值的数据提供最稳定的对齐信号

## 实验关键数据

| 方法 | 使用数据量 | WinRate提升 | 计算时间 |
|------|----------|-------------|---------|
| 全量数据 | 100% | baseline | - |
| TIF筛选 | 50-64% | **+13.58%** | ~10h |
| LossDiff-IRM代理 | 50-64% | 接近TIF | **~5min** |

### 关键发现
- **中等IF最优**：反直觉地，高IF数据导致过拟合，低IF数据是噪声
- **数据价值是模型相关的**：同一分布的数据对不同模型有完全不同的价值
- **少用数据反而更好**：50-64%的筛选数据 > 100%全量

## 亮点与洞察
- 挑战"更多数据更好"和"数据质量是固有的"两个常见假设
- LossDiff-IRM代理的实用性极强：5分钟完成评分

## 局限性 / 可改进方向
- 需要对每个新模型重新计算影响值
- 中等IF最优的界限未理论化

## 相关工作与启发
- **vs On-Policy DPO分析(同会议)**：那篇从data source角度分析，本文从每个偏好对的细粒度价值分析——两者互补
- **vs Skywork-Reward-V2**：Skywork从Train角度优化数据质量，本文从筛选角度去除低价值数据
