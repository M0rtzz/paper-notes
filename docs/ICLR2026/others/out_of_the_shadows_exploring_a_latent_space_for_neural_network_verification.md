# Out of the Shadows: Exploring a Latent Space for Neural Network Verification

**会议**: ICLR 2026  
**arXiv**: [2505.17854](https://arxiv.org/abs/2505.17854)  
**代码**: 无  
**领域**: 神经网络验证 (Neural Network Verification)  
**关键词**: 神经网络验证, 可达性分析, 潜空间, zonotope, 分支定界

## 一句话总结

提出一种基于潜空间（latent space）的规范驱动输入细化方法，通过在高维潜空间中转移输出约束到输入空间，显著减少分支定界过程中的子问题数量，实现高效GPU加速的神经网络验证工具。

## 研究背景与动机

神经网络在安全关键应用中的部署日益广泛，但其对微小输入扰动的敏感性使得形式化验证成为必要。然而，神经网络的形式化验证是一个公认的困难问题。

### 现有方法的局限性

1. **保守性问题**：当前最先进的验证算法通常使用可达性分析或抽象解释来包围神经网络的输出集合，但由于包围的保守性，验证结果常常不确定（inconclusive）
2. **计算瓶颈**：分支定界（branch-and-bound）方法虽然能提高精度，但子问题数量随分支次数指数增长
3. **效率限制**：许多方法依赖的操作难以在GPU上高效实现

### 核心动机

作者观察到一个关键现象：投影型集合表示（如zonotope）在神经网络传播过程中实际上是高维潜空间中一个集合的"影子"（shadow）。这个潜空间在传播过程中保持不变，因此输入集合和输出包围体是同一潜空间的不同"影子"。这一洞察使得可以通过潜空间在输入空间和输出空间之间传递约束。

## 方法详解

### 整体框架

本文提出一种迭代式的规范驱动输入细化（specification-driven input refinement）方法，核心思路是：

1. 对所有不安全输出计算神经网络的原像（preimage）
2. 利用潜空间将输出规范转移到输入空间
3. 通过迭代减少输入集合的范围，使其仅包围不安全的输入

### 关键设计

1. **潜空间利用**：
   - zonotope 是高维集合在低维空间的线性投影（"影子"）
   - 当 zonotope 通过神经网络的仿射层传播时，高维潜空间不会改变
   - 输入和输出的 zonotope 共享同一潜空间，只是投影方向不同
   - 利用此性质，可以在输出端添加约束，直接在潜空间中反映为输入端的约束

2. **规范驱动输入细化**：
   - 不是从输入到输出的传统正向传播验证
   - 而是从输出规范反向追溯，找到哪些输入可能导致不安全输出
   - 通过迭代收紧输入集合，大幅减少后续分支需要处理的子问题数量

3. **纯矩阵操作实现**：
   - 与许多其他方法不同，本方法可以完全通过矩阵运算实现
   - 这一特性使得可以充分发挥GPU并行加速的优势
   - 避免了需要求解优化问题或进行复杂的符号计算

### 分支定界中的应用

- 传统方法：直接在输入空间进行分支，导致子问题数量快速膨胀
- 本文方法：通过潜空间将输出约束传递到输入，预先排除大量不可能导致不安全输出的输入区域
- 效果：显著减少需要处理的子问题数量

### 损失函数 / 训练策略

本文不涉及训练，而是一种确定性的形式化验证方法。核心计算流程包含：
- zonotope 的仿射变换
- 投影操作
- 区间运算
- 线性规划用于精确界的计算

## 实验关键数据

### 主实验

本文工具在国际神经网络验证竞赛（VNN-COMP）中与顶级工具进行了对比。

| 对比项 | 本文方法 | 现有SOTA |
|--------|---------|---------|
| 竞赛排名 | 具有竞争力 | 顶级工具 |
| GPU加速 | 支持（纯矩阵操作） | 多数不支持 |
| 子问题减少 | 显著（通过输入细化） | 需大量分支 |

### 消融实验

| 配置 | 效果 | 说明 |
|------|------|------|
| 无输入细化 | 子问题数量多 | 基线方法 |
| 有输入细化 | 子问题大幅减少 | 本文核心贡献 |
| 无GPU加速 | 速度较慢 | CPU实现 |
| 有GPU加速 | 显著加速 | 受益于纯矩阵操作 |

### 关键发现

1. 潜空间输入细化能够将分支定界中的子问题数量减少到原来的一小部分
2. 纯矩阵操作的实现方式在GPU上获得了显著的加速效果
3. 该方法在多种不同的验证基准上都展示了一致的性能提升
4. 在VNN-COMP中与顶级工具取得了可比的性能

## 亮点与洞察

1. **优雅的数学洞察**：将 zonotope 视为高维集合的"影子"这一思想非常直观且深刻，为输入-输出约束传递提供了自然的数学框架
2. **方向的新颖性**：不是改进传播方式或分支策略，而是从潜空间的角度重新审视验证问题
3. **实用性强**：纯矩阵操作的实现使得可以直接利用现有的GPU计算基础设施
4. **通用性**：该方法可与各种分支定界框架结合使用
5. **规范驱动**：从规范出发进行输入细化，而非盲目分支，体现了一种"目标导向"的验证思路

## 局限性 / 可改进方向

1. **集合表示限制**：目前主要基于 zonotope 表示，可能不是所有问题类型的最优选择
2. **仿射层限制**：潜空间在通过非线性层（如 ReLU）时会产生近似误差，影响细化效果
3. **可扩展性**：对于非常大的网络或非常高维的输入，潜空间的维度可能成为计算瓶颈
4. **竞赛表现**：虽然具有竞争力，但在某些特定基准上可能不如高度特化的工具
5. **非ReLU激活函数**：对于更复杂的激活函数（如sigmoid），方法的有效性需要进一步验证

## 相关工作与启发

- **可达性分析**：本文在可达性分析框架下工作，与传统的抽象解释方法互补
- **zonotope技术**：扩展了 zonotope 在神经网络验证中的应用范围
- **分支定界**：提出了一种新的预处理策略，可与现有分支定界方法结合
- **VNN-COMP竞赛**：展示了该方法在实际竞赛环境中的竞争力

### 对研究的启发

1. 从几何角度（投影、影子）思考计算问题可以带来全新的算法设计思路
2. 关注计算的可并行性（GPU友好）在实际系统中非常重要
3. 规范驱动（由目标引导搜索）比盲目搜索更高效
