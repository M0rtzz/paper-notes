# PlanetAlign: A Comprehensive Python Library for Benchmarking Network Alignment

**会议**: ICLR 2026
**arXiv**: [2505.21366](https://arxiv.org/abs/2505.21366)
**代码**: [GitHub](https://github.com/yq-leo/PlanetAlign)
**领域**: 其他
**关键词**: 网络对齐, 基准库, 图匹配, 最优传输, 评估框架

## 一句话总结

提出PlanetAlign，一个集成18个数据集、14种方法和标准化评估流程的PyTorch网络对齐基准库，首次覆盖一致性方法、嵌入方法和最优传输方法三大类，支持有效性、可扩展性和鲁棒性的全面评估。

## 研究背景与动机

网络对齐（NA）旨在发现不同网络中节点的对应关系，是跨社交网络用户匹配、蛋白质同源性发现、知识图谱融合等任务的关键基础。尽管NA方法日益增多，但缺乏一个**全面统一的基准库**来系统比较不同方法。

现有基准（如SGAPBSA、CAPABN、ASNets等）存在的问题：
1. 数据集局限于单一领域（生物网络或社交网络）
2. 方法覆盖单一类别（如只有一致性方法或嵌入方法），缺少最新的最优传输（OT）方法
3. 评估维度单一（通常只看有效性），且数据集划分不统一

## 方法详解

### 整体框架

PlanetAlign是一个PyTorch基础的Python库，包含三个核心设计：
1. 18个数据集（6个领域：社交、出版物、生物、知识图谱、基础设施、通信）
2. 14种NA方法（一致性2种、嵌入8种、OT 4种）
3. 标准化评估流程（有效性/可扩展性/鲁棒性/监督敏感性）

### 关键设计

1. **全面的数据集收集**:
   - 做什么：收集11个真实世界数据集，合成7个额外数据集
   - 核心思路：合成方法为向原始网络插入10%噪声边并删除15%已有边
   - 设计动机：覆盖6个不同领域，避免单一领域偏见

2. **统一易用的API设计**:
   - 做什么：提供BaseData和BaseModel基类，支持仅几行代码即可集成新数据集和新方法
   - 核心思路：数据集一行代码下载和划分，模型统一train/test接口
   - 设计动机：降低使用门槛，确保可重现性

3. **多维度评估体系**:
   - 做什么：评估Hits@K、MRR（有效性）、运行时间/内存（可扩展性）、边/属性/监督噪声（鲁棒性）
   - 核心思路：统一数据集划分、随机种子控制、双向对齐评估
   - 设计动机：此前的基准只包含部分维度，无法全面理解方法的优劣

### 损失函数 / 训练策略

作为库/基准工作，不涉及新的训练策略。各内置方法使用其原始设计。

## 实验关键数据

### 主实验：各类方法在不同领域的平均Hits@1（训练比20%）
| 方法类别 | 社交网络 | 出版物网络 | 生物网络 | 知识图谱 | 基础设施 | 通信网络 |
|----------|----------|-----------|----------|----------|----------|----------|
| 一致性(IsoRank) | 4.2 | 18.9 | 21.6 | — | — | — |
| 嵌入(最佳) | 中等 | 较高 | 中等 | 较高 | 中等 | 中等 |
| OT(最佳) | 高 | 高 | 高 | 最高 | 高 | 高 |

### 消融实验：实现效率对比
| 方面 | PlanetAlign实现 | 官方实现 | 说明 |
|------|-----------------|----------|------|
| 运行速度 | 最高3倍加速 | 基准 | 统一优化后更快 |
| 有效性 | 相当 | 基准 | 准确性不牺牲 |

### 关键发现

- OT方法在大多数设定中取得最佳性能，是当前最有前景的NA方向
- 一致性方法虽然经典但性能明显落后，仍有基准意义
- 嵌入方法在有属性的网络上表现更好，在纯拓扑网络上与OT差距更大
- PlanetAlign的实现在保持准确性的同时达到最高3倍加速

## 亮点与洞察

- **首个覆盖OT方法**的NA基准库，填补了重要空白
- 6个领域×3类方法×4个评估维度的全面组合使得实验发现更可靠
- API设计简洁，真正实现"几行代码完成一个实验"的目标
- 合成数据集的方法论（10%噪声边+15%删边）已成为社区标准

## 局限性 / 可改进方向

- 合成数据集的噪声模型较简单，可能无法反映真实世界的不对称性
- 未包含动态网络对齐的场景
- 部分方法在大规模网络上的效率仍有瓶颈
- 文档和教程可以进一步丰富

## 相关工作与启发

与OpenEA（知识图谱对齐基准）、NAB（社交网络对齐基准）等直接对比。PlanetAlign的优势在于领域覆盖广、方法类别全、评估维度多。可作为NA研究者的标准实验平台。
