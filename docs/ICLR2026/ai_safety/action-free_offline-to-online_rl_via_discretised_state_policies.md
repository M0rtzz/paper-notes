# Action-Free Offline-to-Online RL via Discretised State Policies

**会议**: ICLR 2026  
**arXiv**: [2602.00629](https://arxiv.org/abs/2602.00629)  
**代码**: 无  
**领域**: 强化学习  
**关键词**: 离线强化学习, 在线强化学习, 无动作数据, 状态策略, 状态离散化

## 一句话总结
提出首个形式化的"无动作离线到在线 RL"框架，通过学习离散化状态策略（推荐期望的下一状态转移而非动作），利用仅含 (s, r, s') 的离线数据预训练，再通过引导机制加速在线学习。

## 研究背景与动机

离线强化学习（Offline RL）在近年来取得了重大进展，使 agent 能从静态数据集中学习策略而无需与环境交互。然而，现有的离线 RL 方法几乎都假设数据集中包含完整的动作标签。

**现有痛点**：在许多实际场景中，动作信息可能由于隐私保护、存储限制或传感器局限而缺失。例如，医疗决策数据中可能只记录了患者状态和结果（state-reward-next_state），但缺失了具体的治疗决策（action）。自动驾驶的观测数据中也可能只有车辆状态轨迹而没有控制命令。这类 (s, r, s') 形式的数据在实际中广泛存在但尚未被充分利用。

**核心矛盾**：传统 RL 方法依赖动作空间的显式建模，而无动作数据无法直接用于标准的基于值（value-based）或基于策略（policy-based）的算法。简单的反向动力学模型（inverse dynamics model）推断动作的方法在高维场景下不稳定。

**核心 idea**：不学习"执行什么动作"，而是学习"转移到哪个状态"——即状态策略（State Policy），通过状态离散化变换使之可扩展到高维问题。

## 方法详解

### 整体框架
整体分为两个阶段：（1）离线预训练阶段——从无动作数据集 (s, r, s') 中学习状态策略，该策略推荐期望的下一状态转移；（2）在线微调阶段——利用预训练的状态策略引导在线 agent 的学习，加速收敛。

### 关键设计

1. **状态离散化变换（State Discretisation Transformation）**：

    - 核心创新：将连续状态空间离散化为有限的离散标记（tokens），使得状态预测问题转化为分类问题
    - 设计动机：直接在连续状态空间中预测下一状态会面临不稳定性和过拟合问题；离散化后可使用基于值的分解方法（如 DecQN）高效处理
    - 离散化方案：对每个状态维度独立进行分箱（binning），将连续值映射到离散区间
    - 这种方法既保留了足够的状态信息表征，又使得高维状态空间变得可处理

2. **Offline State-Only DecQN (ALGO)**：

    - 基于值的算法，专为从无动作数据中预训练状态策略而设计
    - 使用 Decomposed Q-Network（DecQN）对离散化后的状态策略进行值分解
    - 每个状态维度独立维护一个 Q 值分支，总 Q 值为各维度的加和
    - 设计动机：DecQN 的分解结构天然适合离散化后的多维状态表示，避免了组合爆炸
    - 集成正则化技术防止离线训练中常见的过拟合和值函数过估计

3. **引导式在线学习机制（Guided Online Learning）**：

    - 提出新颖的机制，利用预训练的状态策略加速在线 agent 的学习
    - 状态策略提供"目标方向"——告诉在线 agent 应该追求什么样的状态转移
    - 在线 agent 将状态策略的推荐作为额外的学习信号，同时保留自主探索能力
    - 随着在线学习进展，逐步减弱状态策略的引导强度，让 agent 自主决策
    - 这种渐进式引导避免了离线策略中可能存在的分布偏移问题

### 损失函数 / 训练策略
- 离线阶段：使用标准 TD（Temporal Difference）损失在离散化状态空间上训练 Q 网络
- 正则化策略：采用保守估计或具有悲观性的值函数更新，防止对未见过的状态-动作对过度乐观
- 在线阶段：结合标准在线 RL 目标和状态策略引导信号的加权组合

## 实验关键数据

### 主实验
在多个标准强化学习基准上进行评估，涵盖不同维度的状态空间和任务复杂度：

| 基准/任务 | 指标 | ALGO（本文） | 基线方法 | 改进 |
|-----------|------|-------------|---------|------|
| D4RL 系列 | 收敛速度 | 显著加速 | 标准在线RL | 大幅提前收敛 |
| 高维任务 | 渐近性能 | 改善 | 无预训练 | 高于随机初始化 |
| 跨域任务 | 鲁棒性 | 稳定 | 其他离线方法 | 更稳定的表现 |

### 消融实验
| 配置 | 关键指标 | 说明 |
|------|---------|------|
| 无离散化（连续状态预测） | 性能显著下降 | 连续预测不稳定，离散化是关键 |
| 无正则化 | 过拟合加剧 | 正则化对离线预训练至关重要 |
| 不同离散化粒度 | 存在最优区间 | 过粗丢失信息，过细增加复杂度 |
| 无引导（直接迁移） | 收益有限 | 引导机制优于简单初始化 |

### 关键发现
- 离散化和正则化是该方法成功的两个关键因素
- 状态策略方法在完全缺少动作标签的情况下仍能提供有效的预训练
- 引导式在线学习显著加速收敛，且最终渐近性能也有所提升
- 方法在高维状态空间问题上的扩展性良好，得益于 DecQN 的分解结构
- 与基于反向动力学模型的方法相比，直接学习状态策略避免了动作推断的误差累积

## 亮点与洞察
- 首次形式化"无动作离线到在线 RL"这一实际但被忽视的设置，具有重要的实际意义
- 状态策略的概念很巧妙——绕过动作空间，直接在状态空间中规划
- 离散化变换简单而有效，将不适定的连续预测问题转化为结构化的分类问题
- 两阶段框架（离线预训练 + 引导在线学习）的设计自然且实用
- 方法组件模块化，每个组件的贡献通过消融实验清晰验证

## 局限性 / 可改进方向
- 离散化粒度的选择需要根据任务手动调整，缺乏自适应机制
- 状态策略的"推荐下一状态"假设环境动力学是确定性或低随机性的
- 在极高维或连续动作空间的任务中，状态到动作的映射可能不唯一
- 缺少与最新的动作推断（action inference）方法的系统对比
- 理论分析相对有限，缺少收敛性保证
- 未探索在 image-based observation 等更复杂状态表示上的表现

## 相关工作与启发
- 与 CQL、IQL 等离线 RL 方法相关，但处理无动作数据这一新设定
- 与 Inverse RL 有概念上的联系，但方向不同（本文不推断动作而是直接建模状态转移）
- DecQN 的值分解思想来自多智能体 RL 领域，巧妙应用于离散化状态空间
- 启发：1）在更多实际场景中挖掘无动作数据的价值；2）状态策略 + 在线引导的框架可迁移至其他domain
