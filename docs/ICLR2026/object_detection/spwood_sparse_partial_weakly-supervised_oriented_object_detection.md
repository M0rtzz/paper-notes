# SPWOOD: Sparse Partial Weakly-Supervised Oriented Object Detection

**会议**: ICLR 2026  
**arXiv**: [2602.03634](https://arxiv.org/abs/2602.03634)  
**代码**: [GitHub](https://github.com/VisionXLab/SPWOOD)  
**领域**: 目标检测/遥感  
**关键词**: 旋转目标检测, 弱监督, 稀疏标注, 伪标签, 教师-学生

## 一句话总结
提出首个稀疏部分弱监督旋转目标检测(SPWOOD)框架——用极少量稀疏弱标注(HBox/Point)+大量无标注数据训练，通过SOS-Student分离前景/背景并从弱标注学习角度/尺度、多层级伪标签过滤(MPF)自适应筛选高质量伪标签、全局稀疏分割保持类别分布平衡，在DOTA/DIOR上接近全监督性能。

## 研究背景与动机

**领域现状**：旋转目标检测(OOD)在遥感中至关重要→但密集分布+多类别使得旋转框标注成本极高。现有方法按监督级别：全监督→半监督→弱监督→部分弱监督(PWOOD)→稀疏监督(SAOD)。

**现有痛点**：
   - (1) 全监督需要每个目标的旋转框标注→遥感图像中密集目标标注极耗时
   - (2) 半监督仍需大量全标注数据
   - (3) 弱监督(HBox/Point)省了角度标注但仍需标注所有目标
   - (4) 稀疏标注中未标注目标被视为背景→误导训练

**切入角度**：首次合并稀疏+部分弱监督→同时解决"标注太少"和"标注太粗"的双重挑战。

## 方法详解

### 整体框架：教师-学生范式

**训练阶段1 (Burn-in)**：
- 学生在稀疏弱标注上训练
- EMA复制权重到教师

**训练阶段2 (Self-training)**：
- 教师对无标注数据生成伪标签
- 学生在伪标签+稀疏弱标注上联合训练

### 三个关键创新

1. **SOS-Student (稀疏-角度-尺度感知学生)**：
   - **稀疏标注学习**：受Focal Loss启发→区分"真背景"和"未标注前景"→下调高置信未标注目标的损失
   - **角度学习**：从HBox推断角度(H2RBox思路)
   - **尺度学习**：从Point标注学习尺度(Point2RBox思路)
   - 统一支持RBox/HBox/Point及其组合

2. **MPF (多层级伪标签过滤)**：
   - 不同检测层(FPN多层)的预测不一致→为每层动态调整过滤阈值
   - 利用预测分布(而非静态阈值)→适应训练过程的变化
   - 产生更稳定的伪标签

3. **全局稀疏分割**：
   - 传统：逐图像稀疏化→稀有类别被过度保留(偏差)
   - 本文：全局(跨所有图像)统一采样→每类按相同比例→保持原始分布

## 实验关键数据

### DOTA-v1.0 (5%稀疏+HBox弱标注 vs 全监督)
| 方法 | mAP | 与全监督差距 |
|------|-----|-----------|
| 全监督(RBox所有目标) | ~76 | 0 |
| 半监督(5%全标注) | ~55 | -21 |
| PWOOD(部分弱监督) | ~60 | -16 |
| **SPWOOD(5%稀疏HBox)** | **~68** | **-8** |

### 关键发现
- SPWOOD用5%HBox稀疏标注达到全监督约90%性能→极高的标注效率
- MPF比静态阈值/GMM过滤都稳定
- 全局稀疏分割 vs 逐图稀疏化→长尾类别改善最大(+3-5 AP)
- Point标注作为弱监督也有效→但HBox更好（提供了尺度先验）

## 亮点与洞察
- **"监督层级连续谱"的完善**：本文系统性地梳理了OOD的监督层级(全→半→弱→部分弱→稀疏→SPWOOD)→最低标注成本。
- **实际应用价值极高**：遥感图像动辄包含数千个密集小目标→SPWOOD只需标注其中5%的HBox→节省95%以上标注成本。
- **全局稀疏分割的简单有效**：看似trivial的改变(逐图→全局)→对类别平衡有重大影响→说明数据构建细节对性能攸关。
