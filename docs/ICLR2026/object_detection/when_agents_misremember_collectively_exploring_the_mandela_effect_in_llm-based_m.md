# When Agents "Misremember" Collectively: Exploring the Mandela Effect in LLM-based Multi-Agent Systems

**会议**: ICLR 2026  
**arXiv**: [2602.00428](https://arxiv.org/abs/2602.00428)  
**代码**: [GitHub](https://github.com/bluedream02/Mandela-Effect)  
**领域**: 多Agent安全/认知偏见  
**关键词**: 曼德拉效应, 集体虚假记忆, 多Agent, 认知偏见, 缓解策略

## 一句话总结
首次系统研究LLM多Agent系统中的曼德拉效应(集体虚假记忆)：提出ManBench(4类任务×5种交互协议×4838问题)→评估13个LLM→量化曼德拉效应的存在和影响因素(群组组成/规模/知识域/模型规模/记忆时间尺度)→提出prompt级(认知锚定/来源审查)和模型级(对齐)缓解策略→平均降低74.40%的曼德拉效应。

## 研究背景与动机

1. **领域现状**：LLM多Agent系统→协作解决复杂问题→但集体认知偏见风险未被探索。曼德拉效应→群体共享错误记忆→源于社会强化+内化错误信息。

2. **现有痛点**：
   - (1) 现有研究→关注个体Agent错误/简单从众→忽略曼德拉效应特有的方面(有说服力的虚假证据)
   - (2) 幻觉被视为无状态一次性失败→忽略了记忆相关的本质(巩固和持续召回社会共享谬误)
   - (3) 无系统基准评估此现象

3. **切入角度**：曼德拉效应=集体认知偏见→类比人类→Agent也可能受虚假证据影响形成共享错误记忆。

## 方法详解

### ManBench基准

**4类任务(易受曼德拉效应影响)**
**5种交互协议(不同Agent角色和记忆时间尺度)**

**度量指标**：
- 错误率：Agent产生错误答案的比例
- 现实偏移率：因虚假信息改变原本正确答案的比例
- 最大现实偏移率：最严重的偏移程度

### 缓解策略

**Prompt级**：
- 认知锚定：提醒Agent坚持已验证事实
- 来源审查：要求Agent评估信息来源可靠性

**模型级**：
- 对齐训练：在包含虚假信息场景上微调→增强抗干扰

## 实验关键数据

### 13个LLM的曼德拉效应
| 发现 | 详情 |
|------|------|
| 商业模型也受影响 | GPT-4o/Claude也会被虚假证据改变立场 |
| 大模型不免疫 | 规模增大不能消除效应 |
| 群组越大→效应越强 | 社会强化效果 |
| 知识域影响 | 冷门领域→更容易被误导 |

### 缓解效果
| 策略 | 曼德拉效应降低 |
|------|------------|
| 认知锚定 | ~50% |
| 来源审查 | ~60% |
| 对齐训练 | ~70% |
| **组合** | **74.40%** |

### 关键发现
- 曼德拉效应确实存在于LLM多Agent系统→不是假设→是实证
- 短期记忆→影响弱→长期记忆→影响强→记忆巩固的类比
- 缓解策略有效但不能完全消除→仍有25%+残余

## 亮点与洞察
- **认知科学概念→AI安全**→将人类认知偏见(曼德拉效应)系统研究于AI→跨学科创新。
- **"集体虚假记忆"vs"个体幻觉"**→不同现象→曼德拉效应需要社会交互→单独测不到。
- **74%缓解**→说明问题可解→但25%残余→说明还需更深层解决。
- **对AI安全的实际警示**→多Agent协作→可能传播和固化错误→需要设计防御。
