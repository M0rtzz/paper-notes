# Null-Space Filtering for Data-Free Continual Model Merging: Preserving Stability, Promoting Plasticity

**会议**: ICLR 2026  
**arXiv**: [2509.21413](https://arxiv.org/abs/2509.21413)  
**代码**: [GitHub](https://github.com/zihuanqiu/NUFILT)  
**领域**: 模型压缩  
**关键词**: 模型合并, 持续学习, 零空间投影, 稳定性-可塑性, 无数据

## 一句话总结

提出 NUFILT 框架，通过零空间滤波和投影感知 LoRA 适配，在不访问任何任务数据的条件下实现持续模型合并，同时保持稳定性和可塑性。

## 研究背景与动机

数据无关的持续模型合并（DFCMM）需要将独立微调的模型逐步融合为单一骨干网络，但不能访问任何任务数据。核心挑战在于同时满足两个目标：
- **稳定性**：不干扰已有任务的性能
- **可塑性**：忠实地适应新任务

现有方法的不足：
- 简单平均/算术更新会导致参数干扰（稳定性差）
- 正交投影方法在任务向量相关时无法保持可塑性
- 自适应策略通常依赖辅助数据，违反数据无关约束

本文的关键观察：**任务向量与表示子空间近似对齐**，可以作为数据无关条件下执行稳定性和可塑性约束的结构代理。

## 方法详解

### 整体框架

NUFILT 分三步执行：(1) **滤波**：零空间投影抑制与先前任务重叠的分量；(2) **适配**：轻量 LoRA 适配器恢复任务特定信号；(3) **融合**：线性融合回骨干网络，无额外参数。

### 关键设计

1. **零空间滤波器 (Null-Space Filter)**：
   - 计算累积更新 $\tilde{\tau}_{\leq t-1}^{(l)} = \theta_{t-1}^{\text{merged},(l)} - \theta_0^{(l)}$ 的 top-$r_p$ 右奇异向量
   - 构建投影器：$P_t^{(l)} = I - \hat{V}_{\leq t-1}^{(l)} \hat{V}_{\leq t-1}^{(l)\top}$
   - 对旧任务特征 $x^{(l)} \in \text{span}(\hat{V}_{\leq t-1}^{(l)})$，有 $P_t^{(l)} x^{(l)} = 0$，确保稳定性

2. **投影感知 LoRA 适配 (Projection-Aware LoRA)**：
   - 在滤波子空间中插入低秩适配器：$P_t^{(l)} \to P_t^{(l)} + B_t^{(l)} A_t^{(l)}$
   - 通过数据无关代理损失优化：$\mathcal{L}(A_t^{(l)}, B_t^{(l)}) = \|\mathcal{T} - (M + \tau_t^{(l)} B_t^{(l)} A_t^{(l)}) \hat{V}\|_F^2$
   - 残差项桥接了稳定性（对旧任务一致）和可塑性（追踪新任务）

3. **层级线性融合 (Layer-Wise Fusion)**：
   - 最终更新：$\theta_t^{\text{merged},(l)} = \theta_{t-1}^{\text{merged},(l)} + \tau_t^{(l)}(P_t^{(l)} + B_t^{(l)} A_t^{(l)})$
   - 所有操作是线性的，可吸收进权重矩阵，推理时无额外开销

### 损失函数 / 训练策略

- **理论基础**（Theorem 1）：证明了任务向量与数据表示子空间的近似对齐，子空间亲和度 $\mathcal{A}(V_d^{(l)}, \hat{V}^{(l)}) = \frac{1}{r_d}\|\hat{V}^\top V_d\|_F^2$
- **数据无关上界**（Corollary 1）：将稳定性和可塑性损失的上界转换为纯参数空间中的投影项
- 每个任务仅需 50 步 Adam 优化，学习率 $10^{-3}$

## 实验关键数据

### 主实验（视觉任务 - ViT-B/32）

| 配置 | ACC (%) ↑ | 与 NUFILT 差距 |
|------|----------|---------------|
| Pre-Trained | 48.1 (8任务) | - |
| Individual Fine-Tuned | 90.4 | 上界 |
| Weight Averaging | 66.3 | -11.2 |
| Task Arithmetic | 67.5 | -10.0 |
| OPCM | 75.5 | -2.0 |
| WUDI-Merging | 74.7 | -2.8 |
| **NUFILT** | **77.5** | **最优** |

### 消融实验

| 配置 | 关键指标 | 说明 |
|------|---------|------|
| 仅 Null-Space Filter | 较好稳定性但可塑性不足 | 滤波去除了过多新任务信号 |
| 仅 LoRA（无 Filter） | 可塑性好但遗忘严重 | 缺少与先前任务的隔离 |
| NUFILT (Filter + LoRA) | 最优平衡 | 两者互补 |
| 不同 $r_p$（秩参数） | $r_p=128$ 最优 | 秩过高会过度限制新任务更新 |

### 关键发现

- NUFILT 在 8/14/20 任务设置上均超过 OPCM 和 WUDI-Merging，平均提升 4-7%
- 在 NLP (GLUE) 和多模态 (LLaVA) 任务上同样有效
- 不需要额外参数或数据访问，推理开销与单模型相同

## 亮点与洞察

- 理论贡献扎实：证明了任务向量与表示子空间的近似对齐定理
- 方法设计优雅：零空间投影天然保证了对旧任务的零干扰
- LoRA 适配器的投影感知损失巧妙地将稳定性和可塑性统一到一个目标中
- 所有操作最终可线性融合回权重，不增加推理成本

## 局限性 / 可改进方向

- 零空间的维度随任务增多逐渐减小，可能限制后续任务的可塑性
- 假设任务向量的子空间对齐在所有层均成立，但某些层可能对齐度较低
- 未探索与参数高效微调方法（如 LoRA fine-tuning）的结合
- 秩参数 $r_p, r_l, r_v$ 需要一定的经验设置

## 相关工作与启发

- 与 OPCM 的区别：NUFILT 额外引入 LoRA 适配来恢复可塑性，而非仅靠正交投影
- 与 AdaMerging 的区别：完全无数据，不依赖测试集信号
- 零空间投影思想来自持续学习（OGD, OWM），但本文首次在数据无关模型合并中应用
