# Fusing Pixels and Genes: Spatially-Aware Learning in Computational Pathology

**会议**: ICLR 2026  
**arXiv**: [2602.13944](https://arxiv.org/abs/2602.13944)  
**代码**: [GitHub](https://github.com/Hanminghao/STAMP)  
**领域**: 计算病理学/多模态学习  
**关键词**: 空间转录组, 病理图像, 基因引导, 多尺度对齐, 基础模型

## 一句话总结
提出Stamp框架，构建SpaVis-6M（最大10X Visium空间转录组数据集，575万条数据）训练空间感知基因编码器，再通过层次多尺度对比对齐将病理图像与空间基因表达谱联合预训练，在6个数据集4个下游任务上达到SOTA。

## 研究背景与动机

1. **领域现状**：计算病理学基础模型从纯视觉(UNI/Virchow)发展到视觉-语言(CONCH/PLIP)。但语言缺乏分子特异性，对癌症机制理解有限。空间转录组(ST)技术同时捕捉病理图像和基因表达的空间分布。

2. **现有痛点**：(1) TANGLE用bulk RNA引导WSI表示学习但只有患者级信息，忽略样本内部异质性；(2) 现有ST方法(BLEEP/OmiCLIP)数据规模小、特征交互简单；(3) 直接套用视觉-语言预训练忽略了ST数据的空间结构和多尺度特性。

3. **核心矛盾**：基因表达提供比文本更精确的分子级监督信号，但ST数据昂贵且与视觉表示的对齐方法不成熟。

4. **切入角度**：(1) 构建大规模ST数据集(SpaVis-6M)训练强基因编码器；(2) 设计空间感知预训练目标建模邻域关系；(3) 层次多尺度对比对齐融合patch/region/基因三模态。

## 方法详解

### 两阶段预训练

1. **Stage 1: 空间感知基因编码器预训练**:
   - 数据：SpaVis-6M (575万ST条目，35器官，1982切片)
   - 基因分词：异常表达排序策略（按偏离均值程度降序排列基因ID）
   - 两个训练目标：
     - IGR（内在基因重建）：掩码15%基因token→同spot内重建→学习基因共表达
     - CGR（上下文基因重建）：用邻居spot的信息重建中心spot→学习空间依赖
   - 空间一致mini-batch：基于KNN图采样空间连续的邻域

2. **Stage 2: 层次多尺度对比对齐**:
   - 跨尺度定位(CSP)：patch作为3×3网格中的子区域→预测位置→模拟病理医生缩放
   - 跨模态对齐：InfoNCE对比学习对齐patch-基因和region-基因
   - 模态内对齐：patch-region对比→扩展视觉编码器感受野
   - 统一损失：$\mathcal{L}_{Align} = \mathcal{L}_{CSP} + \mathcal{L}_{P-S} + \mathcal{L}_{R-S} + \mathcal{L}_{P-R}$

### 关键设计

- **基因分词创新**：不直接用表达值（受批次效应影响），而用排序后的基因ID→对批次效应鲁棒
- **Pretext Token**：共享视觉编码器处理patch和region时用pretext token区分→统一架构
- **SpaVis-6M**：比之前最大ST数据集大一个量级

## 实验关键数据

### 主实验（6数据集4任务）
| 任务 | Stamp vs SOTA | 数据集 |
|------|-------------|--------|
| 无监督聚类 | **ARI/NMI/Silhouette全面超越** | DLPFC |
| 基因表达预测 | **PCC显著提升** | HBC/Her2ST |
| 线性探测分类 | **准确率提升** | 多数据集 |
| 零样本检索 | **优于CONCH/PLIP** | — |

### 消融实验
| 配置 | 性能 | 说明 |
|------|------|------|
| Full Stamp | 最优 | 完整方案 |
| 无CGR (仅IGR) | 下降 | 空间上下文重要 |
| 无CSP | 下降 | 多尺度定位有帮助 |
| 无模态内对齐 | 下降 | patch-region对比有价值 |

### 关键发现
- 基因引导微调使CONCH/PLIP的聚类指标显著提升(Figure 1)→分子监督比文本更有效
- 空间感知预训练(CGR)对聚类影响最大→邻域关系编码了组织结构
- Stamp在跨数据集泛化上表现好→SpaVis-6M提供了足够的多样性

## 亮点与洞察
- **"基因>文本"作为监督信号**：文本描述缺乏分子特异性，基因表达直接编码疾病机制。实验证实基因引导产生更好的生物学结构识别。
- **空间感知的独特价值**：不同于单细胞数据（无空间信息），ST数据的空间结构是额外信号维度。CGR目标有效利用了这一信息。
- **SpaVis-6M的数据贡献**：575万条10X Visium数据，覆盖35器官——为ST基础模型提供了前所未有的训练资源。

## 局限性 / 可改进方向
- 仅基于10X Visium平台，更高分辨率ST技术（如MERFISH）未考虑
- 基因编码器的context length限制为1500→可能遗漏重要的低表达基因
- 697K配对数据相比视觉-语言的百万级仍然偏少
- UNI作为视觉backbone→受限于其预训练偏差
