# Few-Shot Learner Generalizes Across AI-Generated Image Detection

**会议**: ICML2025  
**arXiv**: [2501.08763](https://arxiv.org/abs/2501.08763)  
**代码**: [GitHub - FSD](https://github.com/teheperinko541/Few-Shot-AIGI-Detector)  
**领域**: object_detection  
**关键词**: AI生成图像检测, 少样本学习, 原型网络, 域泛化, Deepfake检测

## 一句话总结
首次将AI生成图像检测重新定义为少样本分类任务，提出FSD(Few-Shot Detector)基于原型网络学习度量空间，仅用10个未见域样本即在GenImage基准上超越SOTA 11.6%。

## 研究背景与动机

### 已有检测器的核心困境
训练在已知生成模型上的检测器对未见模型性能急剧下降。且从闭源模型(DALL-E/Midjourney)收集大量训练数据昂贵或不可行。

### 域泛化视角的转换
传统方法追求"万能指标"——但忽略了不同生成模型的数据特征差异巨大。关键洞察：在很多实际场景中，可以获得少量未见域的样本。

### 从"域泛化"到"少样本分类"
将不同生成模型视为不同类别，利用少量样本提取域特异性特征，比追求通用特征更practical。

## 方法详解

### FSD框架（基于原型网络）
1. 为每个类别（真实/各生成模型）计算原型表示
2. 测试图像映射到同一度量空间
3. 按最近邻原则分类
4. 无需对未见类做额外训练或微调

### 训练流程
- Episode训练：每episode随机采样N-way K-shot任务
- 用已见生成模型构建训练集
- 学习跨域可迁移的度量空间

### 推理流程
- 从未见生成模型获取少量(如10张)支持集样本
- 计算每类原型
- 新测试图按距离分类

## 实验关键数据

### GenImage基准（10-shot）

| 方法 | 平均准确率(%) | 提升 |
|------|-------------|------|
| UnivFD | 72.3 | - |
| LASTED | 75.1 | - |
| NPR | 78.6 | - |
| **FSD (10-shot)** | **90.2** | **+11.6%** |

### 不同shot数的影响

| Shot数 | 准确率(%) |
|--------|----------|
| 0 (零样本) | 68.5 |
| 5 | 85.3 |
| 10 | 90.2 |
| 20 | 92.1 |
| 50 | 93.8 |

### 关键发现
1. 仅10个样本就能显著提升检测性能
2. FSD对新生成模型无需重新训练
3. 将不同生成模型视为不同类显著优于二分类
4. 域特异性特征比通用特征更有效

## 亮点与洞察

1. 问题重定义非常精准——从不可能的"万能检测"转为practical的"少样本适配"。
2. 方法极简(原型网络)但效果极好(+11.6%)。
3. 10-shot设置在实际中完全可行。
4. 分源分类的策略比二分类更能捕捉域内共性。
5. 与生成模型更新速度完美匹配——新模型出现只需收集少量样本。

## 局限性 / 可改进方向

1. 需要知道测试图来自哪个生成模型类别（否则退化为零样本）。
2. 原型网络对支持集内异常样本敏感。
3. 视频生成检测未涉及。
4. 对抗性扰动（故意逃避检测）的鲁棒性未评估。

## 相关工作与启发

- 与使用CLIP做语义分析的方法互补。
- 与扩散模型重建的检测方法方向不同。
- 启发：可用元学习做特征提取器pretraining以进一步提升少样本性能。

## 补充分析

### 原型网络为何适合AIGI检测
不同生成模型产生的伪影是系统性的（频率特征、纹理模式等），少量样本就能捕捉类内共性。原型网络通过平均嵌入自然抑制样本级噪声。

### 域特异性 vs 通用特征
实验证明，利用域特异性特征（每个生成模型单独建原型）效果远优于追求通用二分类特征。不同生成模型的伪影确实存在本质差异。

### 与闭源模型的实际对接
Midjourney/DALL-E等只需收集10张生成图即可构建支持集，成本极低，使FSD在实际部署中比大规模数据收集方案更可行。
