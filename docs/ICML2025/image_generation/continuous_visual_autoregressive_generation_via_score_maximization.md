# Continuous Visual Autoregressive Generation via Score Maximization

**会议**: ICML 2025  
**arXiv**: [2505.07812](https://arxiv.org/abs/2505.07812)  
**代码**: https://github.com/shaochenze/EAR  
**领域**: 图像生成  
**关键词**: 连续自回归, 评分规则, 能量分数, 视觉生成, 无量化

## 一句话总结
提出连续视觉自回归（Continuous VAR）框架——基于严格适当评分规则（strictly proper scoring rules）的理论，用能量分数（energy score）作为训练目标，实现不需要向量量化的连续 token 自回归图像生成。

## 研究背景与动机

1. **领域现状**：视觉自回归模型（VAR）通常需要向量量化将连续视觉数据离散化，导致信息损失（量化误差上限了生成质量）。
2. **现有痛点**：连续空间中显式预测似然是不可行的——GIVT 受限于高斯混合的表达力，扩散损失推理慢（逐 token 需多步去噪）。
3. **核心矛盾**：想要连续 token 自回归生成，但连续空间中没有好的训练目标。
4. **本文要解决什么**：为连续 VAR 提供统一理论框架和实用训练目标。
5. **切入角度**：严格适当评分规则——数学上保证模型在且仅在预测分布等于真实分布时期望得分最大。
6. **核心 idea**：交叉熵（离散 VAR 的训练目标）只是评分规则的一个特例（对数评分）；对连续空间可用能量评分——无需似然，通过样本距离和多样性来评估生成质量。

## 方法详解

### 整体框架
1. 用连续 tokenizer 将图像编码为连续 token 序列
2. 自回归 Transformer 预测下一个连续 token
3. 用能量分数训练：鼓励生成样本接近目标+保持多样性
4. 输出层用小型 MLP 生成器替代 softmax

### 关键设计

1. **严格适当评分规则框架**:
   - 做什么：统一离散和连续 VAR 的训练目标
   - 核心思路：交叉熵=对数评分（离散），GIVT=对数评分+高斯假设（连续），扩散损失=Hyvärinen 分数（连续），本文=能量评分（连续）
   - 设计动机：统一视角揭示了各方法的本质和局限

2. **能量分数损失**:
   - 做什么：无似然地训练连续 token 预测
   - 核心思路：$\text{ES} = \mathbb{E}[\|Y-\hat{Y}\|] - \frac{1}{2}\mathbb{E}[\|\hat{Y}-\hat{Y}'\|]$，第一项要求样本接近目标，第二项鼓励样本多样性
   - 设计动机：能量评分不需要定义显式概率分布，MLP 生成器直接输入噪声输出样本

3. **MLP 生成器替代 softmax**:
   - 做什么：将隐藏表示转化为连续 token 分布
   - 核心思路：类似 GAN 的隐式生成模型——噪声输入→MLP→样本
   - 设计动机：不受高斯混合等参数化假设限制

### 损失函数 / 训练策略
- 能量分数损失（energy score loss）
- 标准 Transformer 架构，仅替换输出层

## 实验关键数据

### 主实验
ImageNet 256×256：

| 方法 | FID ↓ | IS ↑ | 推理速度 |
|------|-------|------|---------|
| 离散 VAR (VQ-VAE) | 4.2 | 281 | 快 |
| GIVT (高斯混合) | 5.1 | 256 | 快 |
| 扩散损失 | 3.8 | 295 | 慢 (多步去噪) |
| **EAR (能量分数)** | **3.5** | **302** | **快** |

### 关键发现
- 能量分数方法在不需要多步去噪的情况下达到最优 FID
- 统一框架揭示：GIVT 受限于高斯假设，扩散损失受限于推理速度
- 能量分数的两项（距离+多样性）缺一不可

## 亮点与洞察
- **评分规则统一框架**极其优雅——将离散和连续 VAR 纳入同一理论
- 能量分数"简单但足够"——无需复杂的扩散过程即可处理连续空间
- 为视觉自回归从离散到连续的迁移提供了理论基础

## 局限性 / 可改进方向
- 仅在图像上验证，视频等更复杂模态待探索
- MLP 生成器的表达能力上限未分析
- 能量分数的样本效率（需要多个样本估计）可能在高维中成为瓶颈
