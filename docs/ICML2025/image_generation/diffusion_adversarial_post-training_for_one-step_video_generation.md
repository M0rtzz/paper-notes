# Diffusion Adversarial Post-Training for One-Step Video Generation

**会议**: ICML 2025
**arXiv**: [2501.08316](https://arxiv.org/abs/2501.08316)
**代码**: https://seaweed-apt.com (Demo 页面)
**领域**: 扩散模型 / 图像与视频生成加速
**关键词**: 对抗后训练, 单步生成, 视频生成, GAN-扩散混合, R1 正则化

## 一句话总结
提出对抗后训练（Adversarial Post-Training, APT）方法，将预训练的扩散模型直接用 GAN 对抗训练在真实数据上进行微调，实现单步生成 1024px 图像和 1280×720 24fps 视频，在视觉保真度上甚至超越原始 25 步扩散模型。

## 研究背景与动机

1. **领域现状**：扩散模型（DiT、FLUX、SD3.5 等）已成为图像和视频生成的标准方法，但其迭代去噪过程推理成本极高——在 H100 上生成一段 2 秒 1280×720 视频需要数分钟。

2. **现有痛点**：
   - **蒸馏方法的局限**：确定性蒸馏（progressive distillation、consistency distillation）在单步生成时结果模糊；分布式蒸馏（DMD、ADD）需要预计算教师模型样本，计算成本高，且质量受限于教师模型
   - **视频加速几乎空白**：现有方法仅在低分辨率（512px）、短帧数（16 帧）的小模型上验证，无人实现过高分辨率视频的单步生成

3. **核心矛盾**：蒸馏方法的质量上限被教师模型约束——因为学生只模仿教师的输出；而真实数据的分布信息比教师生成的伪数据更丰富。

4. **本文要解决什么**：能否绕过教师模型，直接在真实数据上对抗训练，让单步生成模型不受教师质量约束？

5. **切入角度**：借鉴 GAN 的对抗训练范式，将预训练扩散模型作为生成器初始化，在真实数据上直接做 GAN 训练。关键挑战是如何稳定 16B 参数量的超大规模对抗训练。

6. **核心 idea**：Adversarial Post-Training（APT）= 预训练扩散模型初始化 + 直接对抗真实数据训练 + 近似 R1 正则化稳定训练。

## 方法详解

### 整体框架
- **输入**：随机高斯噪声 $z \sim \mathcal{N}(0, I)$ + 文本条件 $c$
- **生成器**：8B 参数的 MMDiT（36 层 Transformer），从一致性蒸馏模型初始化，单步预测 $\hat{x} = z - G(z, c, T)$
- **判别器**：同样 8B 参数的 MMDiT（36 层），从原始扩散模型初始化，加入多层特征提取头输出标量 logit
- **输出**：潜空间中的图像/视频 latent，经 VAE 解码为像素

### 关键设计

1. **生成器初始化策略**:
   - 不能直接从扩散模型开始对抗训练（会崩溃）
   - 先做 **一致性蒸馏**（consistency distillation）获得模糊但稳定的单步模型 $\hat{G}$：$\hat{v} = \hat{G}(z, c, T)$, $\hat{x} = z - \hat{v}$
   - 蒸馏使用 CFG scale=7.5 的固定引导
   - 然后以此为初始化，开始对抗训练
   - **为什么这样设计**：直接对抗训练不稳定，一致性蒸馏提供了一个"暖启动"点——虽然模糊但已学会了噪声到图像的粗略映射

2. **Transformer 判别器设计**:
   - 骨干网络：完整 36 层 DiT，从**原始扩散模型**权重初始化（不是蒸馏模型）
   - **多层特征提取**：在第 16、26、36 层各插入一个交叉注意力块，使用单个可学习 token 作为 query 对所有视觉 token 做注意力，提取特征后拼接、归一化、投影为标量 logit
   - **时间步集成**：不在 $t=0$ 输入判别器（会崩溃），而是在 $[0,T]$ 上均匀采样 $t$ 并做 shifting，使用集成 $D(x,c) = \mathbb{E}_{t}[\hat{D}(x, t, c)]$
   - **设计动机**：
     - 多层特征避免仅用最后层（过于语义化）导致的结构失真
     - 时间步集成利用扩散预训练的多尺度表示能力
     - 全深度判别器（不冻结参数）比浅判别器效果好得多

3. **近似 R1 正则化**:
   - R1 正则化 $\|\nabla_x D(x,c)\|^2$ 需要二阶梯度，但 PyTorch FSDP、FlashAttention 等不支持
   - 提出近似版本：$\mathcal{L}_{aR1} = \|D(x,c) - D(\mathcal{N}(x, \sigma I), c)\|^2$
   - 对真实数据加小高斯扰动，要求判别器对原始数据和扰动数据给出相近的预测
   - **为什么关键**：没有 R1，判别器 loss 迅速趋近 0，生成器输出纯色板——训练彻底崩溃

### 损失函数 / 训练策略

$$\mathcal{L}_D = \mathbb{E}[\log\sigma(D(x,c))] + \mathbb{E}[\log(1-\sigma(D(G(z,c),c)))] + \lambda\|D(x,c) - D(\mathcal{N}(x,\sigma I),c)\|^2$$

$$\mathcal{L}_G = \mathbb{E}[\log\sigma(D(G(z,c),c))]$$

- 使用 non-saturating GAN 目标
- 图像阶段：128~256 H100 GPU，batch size 9062，学习率 5e-6，EMA decay 0.995，350 步 generator 更新
- 视频阶段：1024 H100 GPU，batch size 2048，学习率 3e-6，300 步
- 优化器：RMSProp（等价 Adam $\beta_1=0, \beta_2=0.9$）
- $\lambda=100$，图像 $\sigma=0.01$，视频 $\sigma=0.1$

## 实验关键数据

### 主实验：图像用户研究（1步 vs 25步原始扩散模型）

| 方法 (1步 vs 自身25步) | 视觉保真度 | 结构完整性 | 文本对齐 |
|----------------------|-----------|-----------|---------|
| FLUX-Schnell | -36.6% | -24.4% | -2.8% |
| SD3.5-Large-Turbo | -94.4% | -30.1% | -20.4% |
| SDXL-DMD2 | -9.3% | -16.8% | -4.6% |
| SDXL-Hyper | -8.8% | -12.3% | -2.1% |
| SDXL-Lightning | -7.7% | -15.1% | -17.4% |
| **APT (Ours)** | **+37.2%** | -13.1% | -8.1% |

APT 是唯一在视觉保真度上**超越**原始 25 步扩散模型的方法。

### 消融实验

| 配置 | 效果 | 说明 |
|------|------|------|
| 无近似 R1 正则化 | 训练崩溃 | 判别器 loss→0，生成彩色色板 |
| 半深度判别器(18层) | 质量下降 | 判别器学习能力不足 |
| 仅最后层特征 | 结构畸变 | 缺少低层结构信息 |
| batch size 256 (视频) | 模式崩溃 | 不同 prompt/seed 生成相同内容 |
| 无 EMA | 质量下降 | 峰值和稳定性不如 EMA |
| 训练超过 350 步 | 结构退化 | 过训练，结构完整性下降 |

### 关键发现
- 单步 APT 视频可生成 1280×720 24fps 2秒视频，H100 单卡 6 秒，8卡并行可实时
- 视觉保真度改善源于直接对抗真实数据——绕开了 CFG 导致的过曝和不自然问题
- 结构完整性退化的根因：生成器在噪声空间的模式转换区域有限容量
- 模型内部可视化显示浅层处理粗结构、深层处理细节，类似扩散迭代过程被压缩到单次前向传播

## 亮点与洞察
- **范式转变**：不用教师蒸馏，直接对抗训练真实数据，质量上限不受教师模型限制
- **可能是迄今最大的 GAN**：~16B 参数的生成器和判别器
- **近似 R1 的工程价值**：解决了大规模 Transformer 对抗训练的关键稳定性难题，兼容 FSDP/FlashAttention
- **洞察**：扩散模型的视觉质量限制可能源于 MSE 损失而非模型架构——对抗训练作为"可学习感知损失"充当了更好的学习信号

## 局限性 / 可改进方向
- 文本对齐有退化（-8.1%），源于对抗训练让分布趋向真实数据（真实数据本身文本对齐不完美）
- 视频结构完整性明显退化（-38.5% 单步），运动连贯性是主要挑战
- 训练资源需求极高（1024 H100 GPU），不易复现
- 仅验证了 2 秒视频，更长视频未探索

## 相关工作与启发
- **UFO-Gen**：最接近的工作，也做对抗训练真实数据，但用 DiffusionGAN 判别器且规模小得多 (<1B)
- **Consistency Models / Lightning / DMD2**：对比基线，都用教师蒸馏范式
- **启发**：APT 的思路可推广到音频、3D 等其他扩散模型加速场景；近似 R1 对其他大规模 GAN 训练也有价值
