# GaussMarker: Robust Dual-Domain Watermark for Diffusion Models

**会议**: ICML2025  
**arXiv**: [2506.11444](https://arxiv.org/abs/2506.11444)  
**代码**: 待确认  
**领域**: image_generation  
**关键词**: 扩散模型水印, 双域水印, 高斯噪声修复器, 无需微调, 鲁棒性

## 一句话总结
提出GaussMarker——首个双域(空间+频率)扩散模型水印方法，通过流水线注入器在两个域一致嵌入水印，配合可学习高斯噪声修复器(GNR)增强对旋转/裁剪等攻击的鲁棒性，无需微调扩散模型。

## 研究背景与动机

### 现有水印方案的脆弱性
单域水印（仅空间或仅频率）在面对简单图像编辑（如3度旋转）时检测准确率从100%降到64%。

### 双域水印的动机
传统图像水印中双域方法已证明更鲁棒，但扩散模型水印领域尚未探索。

### 高斯噪声修复的创新
利用DDIM反转从编辑后图像估计初始噪声→修复噪声→增强检测。

## 方法详解

### 流水线注入器
在初始高斯噪声中先后嵌入空间域和频率域水印——两次注入保持一致性。

### 高斯噪声修复器(GNR)
- 可学习网络，独立于扩散模型
- 输入：从编辑后图像DDIM反转得到的噪声
- 输出：修复后接近原始水印噪声
- 不依赖特定扩散模型（模型无关）

### 检测融合
融合两个域的检测分数做最终判断，提高鲁棒性。

## 实验关键数据

### 八种图像扭曲下的鲁棒性

| 攻击类型 | Gaussian Shading | Tree-Ring | **GaussMarker** |
|---------|-----------------|----------|---------------|
| JPEG压缩 | 高 | 中 | **最高** |
| 高斯模糊 | 高 | 高 | **最高** |
| 旋转3度 | **64%** | 中 | **99%+** |
| 裁剪25% | 低 | 低 | **高** |
| 亮度调整 | 高 | 中 | **最高** |

### 四种高级攻击

| 攻击 | GaussMarker TPR |
|------|---------------|
| DiffPure | 高 |
| VAE重编码 | 高 |
| 图像修复 | 中-高 |
| 文本引导编辑 | 高 |

### 跨SD版本验证
在SD 1.4/1.5/2.0上都一致有效。

### 关键发现
1. 双域水印在旋转攻击下远优于单域（99% vs 64%）
2. GNR的模型无关性使其可复用
3. 低误报率(FPR<0.01)符合实际应用需求
4. 检测分数融合显著优于单域检测

## 亮点与洞察

1. 首个双域扩散模型水印——从传统水印理论获得灵感。
2. GNR的模型无关设计极具工程价值。
3. 旋转鲁棒性（64%→99%）解决了最大痛点。
4. 无需微调扩散模型——完全插件式。
5. 在真实应用关注的低FPR下表现优异。

## 局限性 / 可改进方向

1. GNR 需要训练一个额外的恢复网络，增加了部署成本和复杂度。
2. 对极强组合攻击（旋转+裁剪+压缩同时施加）的鲁棒性待验证——实际场景可能多种攻击叠加。
3. 视频扩散模型的适配未讨论——视频帧间的水印一致性是新挑战。
4. 水印容量（bit 数）的上限分析缺失——更高容量对版权保护更有价值。
5. 对非 SD 架构（如 DiT、Flux 等新一代扩散模型）的泛化性待测试。
6. 双域嵌入的计算开销相比单域方法更高，对实时生成场景的适用性需评估。

## 相关工作与启发

- **vs Gaussian Shading**：GS 是单域（仅在初始噪声的频率域嵌入），GaussMarker 是双域 + GNR，鲁棒性显著更强
- **vs Tree-Ring**：Tree-Ring 在频率域用同心圆环模式嵌入，仅单域且缺乏恢复机制；GaussMarker 双域嵌入 + GNR 恢复
- **vs Stable Signature**：在解码器中嵌入水印，需修改模型权重；GaussMarker 不需修改模型，即插即用
- **vs DwtDctSvd 等传统方法**：不适配扩散模型的生成流程；GaussMarker 专为扩散模型设计
- **启发**：GNR 的噪声恢复思路可推广到其他需要从退化信号中恢复原始模式的安全任务（如音频水印、3D 模型水印）

## 补充设计细节

### 流水线注入器的工作流
先在空间域嵌入水印(修改高斯噪声的空间分布)，再在频率域嵌入水印(修改DCT系数)。两级注入保持一致性使跌幅最小。

### GNR的训练方式
GNR独立于扩散模型训练：用实变换对(水印噪声, 编辑后反转噪声)做监督学习。这使得GNR可以在SD1.4上训练但用于SD2.0的检测。
