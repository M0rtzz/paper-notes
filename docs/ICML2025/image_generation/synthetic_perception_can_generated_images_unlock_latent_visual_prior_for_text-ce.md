# Synthetic Perception: Can Generated Images Unlock Latent Visual Prior for Text-Centric Reasoning?

**会议**: ICML2025  
**arXiv**: [2506.17623](https://arxiv.org/abs/2506.17623)  
**领域**: image_generation  
**关键词**: 合成视觉, T2I增强, 文本分类, 跨模态探测, 多模态融合

## 一句话总结
系统研究T2I模型生成的图像能否作为"合成感知"增强纯文本推理：在文本分类框架中评估了T2I模型质量/提示策略/融合架构的影响，发现即使增强Llama-3/Qwen-2.5等强LLM也可获得显著提升，但效果高度依赖语义对齐和任务的视觉可接地性。

## 研究背景与动机

### 文本与多模态的"模态鸿沟"
文本数据丰富但纯文本模型缺乏视觉先验。多模态模型需要真实图像——是否可用生成图像替代？

### "合成感知"假说
即时生成的图像可以将文本投影到视觉语义空间，解锁纯文本训练中缺失的感官信息。

## 方法详解

### 评估框架
1. 给定文本输入→用T2I模型即时生成对应图像
2. 将文本嵌入和图像嵌入融合做分类
3. 系统变化：T2I模型(Flux.1/SDXL)、提示策略、融合方式

### 跨模态探测
将生成的图像视为对文本的"跨模态探测"——测试哪些文本任务从视觉增强中受益。

## 实验关键数据

### 文本分类提升

| 基座模型 | 纯文本 | +合成图像 | 改善 |
|---------|--------|---------|------|
| Llama-3 | 基线 | **提升** | 显著 |
| Qwen-2.5 | 基线 | **提升** | 显著 |

### T2I模型质量影响

| T2I模型 | 增强效果 |
|---------|---------|
| Flux.1 | **最好** |
| SDXL | 好 |
| 低质量T2I | 可能反而降低 |

### 条件依赖性分析

| 条件 | 效果 |
|------|------|
| 文本-图像语义对齐高 | 显著提升 |
| 任务视觉可接地性高 | 显著提升 |
| T2I保真度高 | 提升 |
| 上述条件不满足 | 可能无效/有害 |

### 关键发现
1. 合成感知在满足条件时可显著增强纯文本推理
2. T2I质量是关键——低质量生成可能有害
3. 不是所有文本任务都受益——需要"视觉可接地性"
4. 提示策略显著影响生成图像的相关性
5. 即使增强强LLM(Llama-3)也有效

## 亮点与洞察

1. "合成感知"概念的系统化提出。
2. 严格的条件分析避免了过度乐观的结论。
3. 对无图像场景（如纯文本NLP任务）有实用价值。
4. 跨模态探测视角揭示了LLM的"感官剥夺"问题。
5. 建立了该方向的基准框架。

## 局限性 / 可改进方向
1. 仅在文本分类上验证，生成/QA任务待扩展。
2. T2I生成增加推理延迟。
3. 对抽象文本（如法律/数学）的视觉化效果有限。

## 评分
- 新颖性: 4.0/5 — 概念不完全新但系统化研究有价值
- 实验充分度: 4.5/5 — 多T2I×多LLM×条件分析
- 写作质量: 4.0/5
- 价值: 4.0/5 — 对文本增强有新思路

## 补充分析

### "视觉可接地性"的判断
情感分类（表情图像有用）>主题分类（物体图像有用）>抽象推理（图像帮助有限）。这可作为决定"是否值得生成图像"的先验。

### 成本效益分析
T2I推理成本vs文本分类精度提升的权衡。对于计算资源充足的批量处理场景更值得，实时场景需要权衡延迟。

### 与RAG的类比
RAG从外部知识库检索文本补充，"合成感知"从生成模型检索视觉补充。两者在"为模型补充缺失信息"上思路一致。

### 对文本NLP社区的启示
"感官剥夺"是纯文本模型的根本局限。合成感知提供了一种低成本的解决路径，尤其在无真实图像的场景中。

### T2I模型质量的非线性影响
低质量T2I可能引入误导性视觉信号反而降低性能——T2I质量有一个"有效阈值"，低于此生成不如不生成。
