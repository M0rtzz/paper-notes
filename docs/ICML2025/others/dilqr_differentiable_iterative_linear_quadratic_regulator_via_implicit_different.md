# DiLQR: Differentiable Iterative LQR via Implicit Differentiation

**会议**: ICML2025
**arXiv**: [2506.17473](https://arxiv.org/abs/2506.17473)
**代码**: [项目页](https://sites.google.com/view/dilqr/)
**领域**: others
**关键词**: 可微控制, iLQR, 隐式微分, 轨迹优化, 端到端学习

## 一句话总结
提出DiLQR——通过隐式微分在iLQR控制器的不动点处计算梯度，使反向传播成本恒定(不随迭代次数增加)，实现128倍加速和10^6倍学习性能提升。

## 研究背景与动机

### iLQR的差异化难题
iLQR是强大的轨迹优化器，但使其可微需要反向传播穿过所有迭代步骤——内存和计算随迭代数线性增长。

### DiffMPC的启发
Amos et al.开创了LQR的解析梯度，但仅限单步LQR。iLQR的迭代性质使解析梯度更具挑战。

## 方法详解

### 隐式微分策略
不反向传播通过所有迭代，而是在iLQR收敛的不动点处利用隐式函数定理直接求解梯度→O(1)复杂度。

### 解析轨迹导数
为带可调加性代价函数和受约束动力学的最优控制问题推导解析轨迹导数。

### 稀疏性利用
利用张量表达式的稀疏性使某些导数计算与轨迹长度线性缩放。

## 实验关键数据

### 计算效率

| 方法 | 反向传播成本 | 相对加速 |
|------|-----------|---------|
| AutoDiff | O(迭代数) | 基线 |
| **DiLQR** | **O(1)** | **21-128x加速** |

### 学习性能

| 方法 | 模仿学习loss |
|------|------------|
| 标准神经网络策略 | 基线 |
| 非精确可微iLQR | 中等 |
| **DiLQR** | **10^6倍改善** |

### 端到端视觉控制
DiLQR可集成到更大网络中处理像素输入的完全端到端任务。

### 关键发现
1. O(1)反向传播成本使长视野任务可行
2. 精确解析梯度远优于近似梯度
3. 128倍计算加速是实际可部署的差异
4. 可与视觉模块组合做端到端学习

## 亮点与洞察

1. 隐式微分在iLQR不动点的应用是核心理论贡献。
2. O(1)复杂度解决了可微控制的根本瓶颈。
3. 10^6倍学习性能提升说明精确梯度至关重要。
4. 端到端视觉控制展示了作为神经网络模块的潜力。

## 局限性 / 可改进方向

1. 仅在倒立摆和cartpole等简单系统上验证。
2. 非凸控制问题中不动点可能不唯一。
3. 与model-free RL的直接对比缺失。
4. 对约束优化的扩展未讨论。

## 补充技术细节

### 隐式微分的核心思想
AutoDiff必须反向传播穿过每一层iLQR迭代，而隐式微分利用不动点条件直接求解——复杂度从“O(迭代数)”降为“O(1)”。

### 与DiffMPC的进化关系
DiffMPC(Amos 2018)开创了单步LQR解析梯度，DiLQR将其推广到迭代iLQR——解决了迭代引入的可扩展性问题。

### 精确解析梯度的重要性
10^6倍学习性能提升说明近似梯度在控制任务中影响巨大——控制场景对梯度质量比NLP/CV更敏感。

### 端到端视觉控制的潜力
DiLQR可作为神经网络的“控制层”——视觉编码器提取状态→DiLQR规划动作→执行，端到端可微。
