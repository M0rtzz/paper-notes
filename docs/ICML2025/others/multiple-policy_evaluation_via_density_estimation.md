# Multiple-Policy Evaluation via Density Estimation

**会议**: ICML2025  
**arXiv**: [2404.00195](https://arxiv.org/abs/2404.00195)  
**代码**: 待确认  
**领域**: others  
**关键词**: 多策略评估, 密度估计, 重要性采样, 粗估计, 样本复杂度

## 一句话总结
提出CAESAR算法解决K个策略同时评估问题：通过两阶段方法（粗估计访问分布→计算最优采样分布→密度比估计）实现实例依赖的样本复杂度，比单策略评估K次的朴素方法高效得多。

## 研究背景与动机

### 多策略评估的实际需求
训练超参不同的K个策略后需要评估哪个最好。朴素方法（独立评估K次）样本复杂度线性于K，忽略了策略间的相似性。

### 已有工作的局限
Dann et al. (2023)提出在线方法利用策略重叠，但需要构建生成模型。单策略离线评估方法依赖覆盖条件，无法直接扩展到多策略。

### 核心创新
设计最优采样分布（而非为每个策略单独采样），用一组共享数据同时评估所有K个策略。关键技术是"粗估计"——只需常数倍精度即可有效指导后续精确估计。

## 方法详解

### 两阶段框架
**Phase I（粗估计）**：
- 对每个策略收集O(1/epsilon)条轨迹（比精确估计的O(1/epsilon^2)少一个量级）
- 得到访问分布的常数倍近似估计

**Phase II（精确评估）**：
- 基于粗估计计算最优混合采样分布
- 从该分布采样共享数据集
- 用IDES算法（基于DualDICE）估计重要性密度比
- 输出所有K个策略的值估计

### IDES算法
通过最小化逐步二次损失函数估计边际重要性比，是DualDICE到有限视野MDP的非平凡扩展。

### 理论保证
样本复杂度：$O(H^4/\epsilon^2 \sum_h \max_k \sum_{s,a} d_h^{\pi_k}(s,a)^2 / \mu_h^*(s,a))$

## 实验关键数据

### 样本复杂度对比

| 方法 | 样本复杂度 | 利用策略相似性 |
|------|----------|-------------|
| 朴素K次评估 | O(K/epsilon^2) | 否 |
| Dann et al. | 依赖轨迹合成 | 是(在线) |
| CAESAR | O(H^4/epsilon^2 * 实例项) | 是(离线) |

### 关键理论贡献

| 贡献 | 描述 |
|------|------|
| 粗估计技术 | O(1/epsilon)代替O(1/epsilon^2)，低阶 |
| IDES | DualDICE到有限视野的扩展 |
| beta-距离 | 确定性策略的coarse estimation |
| MARCH | beta-距离上的高效算法 |

### 关键发现
1. 粗估计的低阶成本使Phase I可忽略
2. 最优采样分布有效利用了策略间的访问重叠
3. 非渐近有界优于已有渐近结果
4. 确定性策略的特殊结构可进一步利用

## 亮点与洞察

1. 粗估计概念简单但强大：常数倍近似足以指导最优采样，大幅降低前期成本。
2. 两阶段设计将探索（Phase I）和利用（Phase II）优雅分离。
3. IDES作为独立贡献可应用于更广泛的离线评估场景。
4. 理论结果是非渐近和实例依赖的，比渐近结果更强。

## 局限性 / 可改进方向

1. 理论重于实验，缺少大规模实证验证。
2. 表格MDP假设限制了应用范围。
3. H^4的视野依赖仍然较大。
4. 函数逼近场景的扩展是重要开放问题。

## 相关工作与启发

- DualDICE思想被巧妙扩展到有限视野多策略场景。
- 与Amortila et al. (2024)的探索目标有联系但技术更简单。
- 启发：粗估计技术可在其他需要两阶段估计的问题中复用。

## 补充分析

### MARCH算法的特殊贡献
针对确定性策略提出beta-距离和MARCH算法，在确定性策略数指数增长时仍可实现多项式复杂度。

### 粗估计的通用性
粗估计技术不限于多策略评估，可推广到任何需要两阶段估计的RL问题：先粗略探索再精确利用。

### Median of Means的作用
将期望结果转化为高概率结果，保证CAESAR的非渐近结果。
