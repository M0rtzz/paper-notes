# Revisiting Instance-Optimal Cluster Recovery in the Labeled Stochastic Block Model

## 元信息
- **会议**: ICML 2025
- **arXiv**: [2306.12968](https://arxiv.org/abs/2306.12968)
- **代码**: 无
- **领域**: 社区检测 / 图聚类
- **关键词**: 标签随机块模型, 实例最优, 谱聚类, 似然改进, 社区检测

## 一句话总结
提出IAC算法，在标签随机块模型(LSBM)中以$O(n\text{polylog}(n))$复杂度首次实现匹配实例最优下界的聚类恢复，同时给出期望和高概率保证。

## 研究背景与动机
- **LSBM**：SBM的泛化，边标签来自任意集合，能捕捉评分、关系类型等丰富交互
- **实例最优**：最小化每个特定LSBM实例的分类错误，而非minimax（最坏情况）最优
- **已有下界**：误分类节点数需至少 $n\exp(-nD(\alpha,p))$，但此前无算法匹配
- **计算效率**：先前匹配方法需对每个节点进行谱聚类，复杂度 $O(n^2)$ 以上

## 方法详解

### 散度度量
实例难度由KL散度定义：
$$D(\alpha,p) = \min_{i \neq j} D_{L+}(\alpha, p(i), p(j))$$
$$D_{L+} = \min_{y} \max\left\{\sum_k \alpha_k \text{kl}(y(k), p(i,k)),\; \sum_k \alpha_k \text{kl}(y(k), p(j,k))\right\}$$

### IAC算法（两阶段）

**阶段1：谱聚类初始化**
1. 估计平均度 $\tilde{p}$，修剪高度数节点
2. 拼接标签邻接矩阵 $\bar{A} = [A_\Gamma^1, \ldots, A_\Gamma^L]$
3. 迭代幂法+奇异值阈值化自动估计簇数 $\hat{K}$
4. k-means聚类，候选中心扩展到 $(\log n)^2$ 个随机节点

误分类保证：$O(1/\bar{p})$ 个错误，以 $1-\exp(-cn\bar{p})$ 概率成立

**阶段2：似然改进**
- 估计参数 $\hat{p}(i,j,\ell)$
- 对每个节点 $v$，分配到最大对数似然簇：$k^* = \arg\max_k \sum_{i,w,\ell} A_{vw}^\ell \log \hat{p}(k,i,\ell)$
- 迭代 $\log n$ 轮

### 主要定理
若 $\liminf_{n\to\infty} \frac{nD(\alpha,p)}{\log(n/s)} \geq 1$，则IAC以高概率和期望误分类至多 $s$ 个节点。

## 实验
论文为理论工作，附录提供数值验证：
| 比较对象 | 结果 |
|---------|------|
| Gao et al. (2017) | IAC经验精度更高，且只需一次谱聚类 |
| Minimax方法 | IAC实例自适应，对简单实例表现更好 |

## 亮点
- 首个匹配LSBM实例最优下界的算法（期望+高概率双保证）
- 复杂度仅 $O(n(\log n)^3)$，远优于先前 $O(n^2)$
- 自动估计簇数 $K$，无需先验知识
- 处理非对称最优决策边界（体现在KL散度定义中）

## 局限性
- 假设 $K$ 固定不随 $n$ 增长，不适用于簇数增长的场景
- 稀疏regime $\bar{p} = O(\log n / n)$，不覆盖稠密图
- 标签概率比条件 (A1) 的 $\eta$ 常数假设可能限制适用范围
- 缺乏大规模真实网络数据验证

## 评分
⭐⭐⭐⭐ 扎实的理论工作，首次实现LSBM的实例最优聚类且保持低计算复杂度，对社区检测理论有重要推进。
