# PAC Learning with Improvements

**会议**: ICML2025  
**arXiv**: [2503.03184](https://arxiv.org/abs/2503.03184)  
**代码**: [GitHub](https://github.com/idanattias/PAC-Learning-with-Improvements)  
**领域**: PAC学习 / 计算学习理论  
**关键词**: PAC学习, 策略性改进, 样本复杂度, VC维, 交闭类, 零误差学习

## 一句话总结

提出"带改进的PAC学习"框架：当agent能真正提升自身特征时，学习器可以通过保守分类实现零误差，且可学性与标准PAC模型和策略性分类模型存在本质分离。

## 研究背景与动机

传统PAC学习的基本下界是：在几乎任何非平凡设置中，学习到误差 $\epsilon$ 至少需要 $1/\epsilon$ 个样本。然而实际场景中，被分类的对象（如贷款申请人）往往会对分类器做出**真实改进**反应——比如还清高息信用卡债务、参加理财课程来提升信用评分。

与策略性分类（strategic classification）中agent仅"伪装"不同，本文关注agent的改进是**真实的**。核心观察是：如果agent能改进自身特征至多 $r$，那么学习器只需学到"足够接近"的分类器，就可能实现**零误差**——这在标准PAC模型中是不可能的。

以阈值学习为例：真实阈值为 $\theta$，若学到 $\hat{\theta}$ 满足 $\theta \le \hat{\theta} \le \theta + r$，则：
- 所有被判为正的agent确实合格（无假阳性）
- 所有真正合格的agent可以通过努力被正确分类（假阴性可通过改进消除）

## 方法详解

### 形式化定义

设实例空间 $\mathcal{X}$，改进函数 $\Delta: \mathcal{X} \to 2^{\mathcal{X}}$ 将每个agent $x$ 映射到其改进集 $\Delta(x)$。agent对分类器 $h$ 的反应集为：

$$\Delta_h(x) = \begin{cases} \{x\} & \text{若 } h(x)=1 \\ \{x\} & \text{若 } \{x' \in \Delta(x) \mid h(x')=1\} = \emptyset \\ \{x' \in \Delta(x) : h(x')=1\} & \text{其他} \end{cases}$$

损失函数采用**对抗性断键**（adversarial tie-breaking）：

$$\text{Loss}(x; h, f^*) = \max_{x' \in \Delta_h(x)} \mathbb{I}[h(x') \neq f^*(x')]$$

这一设计偏好保守分类器——对不确定的点标记为负而非正。

### 闭包算法（Closure Algorithm）

对交闭类（intersection-closed class），算法返回包含所有正样本的最小假设：

$$h_S^c(x) = \begin{cases} 1 & \text{若 } x \in \text{CLOS}_{\mathcal{H}}(\{x_i \in S : y_i = 1\}) \\ 0 & \text{其他} \end{cases}$$

关键性质：闭包算法返回的假设满足 $h_S^c \subseteq f^*$（正区域是真概念正区域的子集），因此**不会产生假阳性**。负区域中的agent若能通过改进到达正区域，则不计为误差。

### 改进区域（Improvement Region）

$$\text{IR}(h; f^*, \Delta) = \{x : h(x) = 0, \exists x' \in \Delta(x) : h(x') = f^*(x') = 1\}$$

改进带来的增益为 $\mathbb{P}_{x \sim \mathcal{D}}[x \in \text{IR}(h; f^*, \Delta)]$。

## 理论结果

### 与标准PAC模型和策略性分类的分离

| 性质 | 标准PAC | 策略性分类 | 带改进的PAC |
|------|---------|-----------|------------|
| 可学性等价条件 | 有限VC维 | 有限SVC维 | 既非充分也非必要于VC维 |
| 零误差可达 | 不可能（非平凡） | 不可能（一般） | 可能（如阈值类） |
| 无限VC维 | 不可学 | 取决于SVC | 可能可学（当 $\Delta(x)=\mathcal{X}$） |
| 两区间并集类 | 可学 | 可学 (SVC≤4) | 不可学（≥1/4误差） |

**定理3.1**：有限VC维既非带改进PAC可学的必要条件，也非充分条件。

- **非必要**：若 $\Delta(x) = \mathcal{X}$（全局改进），任何VC维无限的类也可学——只需找到一个正例并输出单点分类器，样本复杂度 $O(\frac{1}{\epsilon}\ln\frac{1}{\delta})$。
- **非充分**：两区间并集类（VC维4）在特定改进函数下不可学，期望误差 ≥ 25%。

### 几何概念的学习

**定理4.1（阈值，均匀分布）**：改进集为半径 $r$ 的闭球，均匀分布 $[0,1]$，样本复杂度 $M = O(\frac{1}{\epsilon}\log\frac{1}{\delta})$ 下：

$$\text{Loss}_{\mathcal{D}}(h_{S^+}, h_{t^*}) \le (\epsilon - r)_+$$

当 $\epsilon \le r$ 时直接达到**零误差**。

**定理4.6（轴对齐超矩形）**：改进集为 $\ell_\infty$ 半径 $r$ 的闭球，对任意分布：

$$\text{Loss}_{\mathcal{D}}(R_S^c, R^*) \le \left(\epsilon - \mathbb{P}[x \in \text{IR}(R_S^c; R^*, \Delta)]\right)_+$$

均匀分布 $[0,1]^2$ 下改进增益为 $2r(l_1 + l_2) + 4r^2$，随矩形周长增大而增大。

**定理4.7（一般交闭类）**：任何VC维为 $d$ 的交闭类可在 $O(\frac{1}{\epsilon}(d + \log\frac{1}{\delta}))$ 样本下带改进PAC学习，适用于任意改进函数和分布。

**定理4.9（半空间）**：$d$ 维齐次半空间在单位球均匀分布下，agent可按角度 $r$ 改进，样本复杂度 $\tilde{O}(\frac{d + \log(1/\delta)}{r})$ 即可零误差。

### 图模型结果

在图模型中（节点为agent，邻居为改进集），论文建立了近紧的样本复杂度界，并证明：
- 正子图支配集的标签足以教学
- 可同时优化分类误差和公平性目标

## 实验验证

在3个真实数据集和1个合成数据集上验证理论预测：
- 保守模型（对假阳性惩罚更大）初始误差较高，但随改进预算 $r$ 增大，误差**急剧下降**
- 多个场景下误差降至零
- 更严格的假阳性惩罚导致更精准的正分类，从而在agent改进后获得更大收益

## 亮点与洞察

1. **零误差的可能性**：首次在策略性/改进学习文献中证明非平凡概念类可以零误差学习
2. **保守优于激进**：与直觉相反，偏保守的分类器在改进场景中表现更优——假阴性可被agent自行修复，假阳性则无法挽回
3. **与已有模型的精确分离**：证明改进学习与标准PAC和策略性PAC存在双向不蕴含关系
4. **交闭性的关键角色**：交闭性几乎是proper可学的充要条件（在适当的一般性条件下）
5. **理论优美**：改进不仅降低误差上界（从 $\epsilon$ 到 $(\epsilon-r)_+$），还改变了可学性的根本特征

## 局限性 / 可改进方向

1. **可实现性假设**：仅考虑 $f^* \in \mathcal{H}$ 的情况，未扩展到不可知（agnostic）设置
2. **改进函数已知**：假设学习器知道改进函数 $\Delta$，实际中可能需要从数据中估计
3. **对抗性断键**：最坏情况tie-breaking假设可能过于悲观
4. **缺乏计算复杂度分析**：主要关注信息论（样本复杂度），未讨论算法的计算效率
5. **实验规模有限**：仅用4个数据集验证，可进一步扩展到更大规模和更复杂的场景
