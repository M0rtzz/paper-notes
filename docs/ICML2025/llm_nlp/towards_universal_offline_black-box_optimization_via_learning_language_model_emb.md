# Towards Universal Offline Black-Box Optimization via Learning Language Model Embeddings

**会议**: ICML2025
**arXiv**: [2506.07109](https://arxiv.org/abs/2506.07109)
**代码**: [GitHub](https://github.com/lamda-bbo/universal-offline-bbo)
**领域**: llm_nlp
**关键词**: 黑盒优化, 离线优化, 语言模型嵌入, 跨域泛化, 通用优化

## 一句话总结
探索用语言模型嵌入实现通用离线黑盒优化(BBO)：将异构数值参数token化为字符串后用LM编码，提出端到端(next-token prediction)和潜在空间学习两种范式，在多域离线BBO任务上验证了跨域泛化能力。

## 研究背景与动机

### 离线BBO的局限
现有方法假设训练和测试共享相同维度和变量类型——无法利用跨任务关系。每个新问题都需要重新收集数据。

### LM嵌入的机会
LM的字符串嵌入天然统一了不同类型/维度的数据表示——为跨域优化提供了可能。

## 方法详解

### 范式1：端到端next-token prediction
将优化问题的参数和目标值序列化为token→训练LM预测高目标值对应的参数序列。

### 范式2：潜在空间学习
优先学习具有强表示能力的潜在空间→在潜在空间中做优化而非token空间。

### 跨域训练
收集多域开源离线BBO任务的数据联合训练，使模型学到跨域的优化模式。

## 实验关键数据

### 跨域泛化

| 设置 | 单任务方法 | **通用方法** |
|------|----------|-----------|
| 见过的域 | 基线 | **更好** |
| 未见的域 | 不可行 | **可行** |

### 关键发现
1. LM嵌入有效统一了异构搜索空间
2. 潜在空间学习比纯next-token更稳定
3. 多域联合训练确实提升了泛化
4. 避免了过拟合有限历史数据
5. 代码开源便于复现

## 亮点与洞察

1. "LM嵌入统一异构优化空间"是范式级的洞察。
2. 两种范式的系统对比提供了清晰的设计指导。
3. 对离线BBO领域的跨域泛化有开创性意义。
4. 潜在空间的几何性质分析增加了理论深度。
5. 实际应用广泛（蛋白质/分子/工程设计）。

## 局限性 / 可改进方向
1. Token化数值的精度损失未深入讨论。
2. 对极高维优化的可扩展性待验证。
3. 与贝叶斯优化等经典方法的在线版本对比不够。

## 补充分析
### Token化的关键挑战
将浮点数“3.14159”转为token序列后，数值精度取决于token粒度。论文探索了不同token化策略，发现4-5位有效数字通常足够。

### 潜在空间的几何性质
好的潜在空间应使目标函数在其中“光滑”——即相近嵌入对应相近目标值。通过Lipschitz常数分析验证。

### 与贝叶斯优化的关系
BO在固定维度上有效但不能跨域。LM嵌入统一了空间后，类似BO的技术可在潜在空间中跨域使用。

### 实际应用前景
蛋白质序列设计、分子优化、工程参数调优等卻数据稀缺的场景最受益于跨域迁移。
### 潜在空间的几何性质
好的潜在空间应使目标函数在其中"光滑"——即相近的嵌入对应相近的目标值。论文通过lipschitz常数分析验证了这一点。
