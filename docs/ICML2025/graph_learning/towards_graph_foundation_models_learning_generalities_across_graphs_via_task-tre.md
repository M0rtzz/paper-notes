# Towards Graph Foundation Models: Learning Generalities Across Graphs via Task-Trees

**会议**: ICML2025  
**arXiv**: [2412.16441](https://arxiv.org/abs/2412.16441)  
**代码**: [GIT](https://github.com/Zehong-Wang/GIT)  
**领域**: graph_learning  
**关键词**: 图基础模型, 任务树, 跨域泛化, GNN预训练, 零样本学习

## 一句话总结

提出 Task-Tree 作为统一学习实例对齐节点/边/图级任务，理论分析其稳定性/可迁移性/泛化性，构建图基础模型 GIT 在 32 个图上通过微调/上下文学习/零样本展现跨域跨任务泛化能力。

## 研究背景与动机

- **图基础模型**远落后于 NLP/CV：图的领域异质性和任务异质性巨大
- **Graphon 方法**需强生成假设；**子结构方法**受限于 MP-GNN 表达力
- **核心问题**：如何跨越节点/边/图级任务异质性找到可迁移的通用性？

## 方法详解

### Task-Tree 概念

- 引入虚拟任务节点连接所有任务相关节点
- 以此虚拟节点为根的计算树 = Task-Tree
- **优势**：(1) 可学习性（MP-GNN 可编码树） (2) 统一性（跨任务级别） (3) 高效性

### 理论分析

- **稳定性**：Task-Tree 对图扰动稳定
- **可迁移性**：多样 Task-Tree 预训练产生可迁移表示
- **泛化**：微调后泛化到下游任务

### GIT 模型

- 在多域 Task-Tree 上用重构目标预训练 GNN
- 三种使用模式：微调 / 上下文学习（few-shot） / 零样本
- 指令调优（instruction tuning）适配特定域
- Task-Tree 提取高效：操作等价于在原图上附加虚拟节点

### 预训练策略

- 重构损失：预测被掩码的 Task-Tree 节点特征
- 多域数据混合训练（社交/引用/分子/网页/生物）
- 域内指令调优进一步提升表现

## 实验关键数据

- **32 个图，5 个域**（社交/引用/分子/网页/生物）

### 微调结果

| 域 | 无预训练 | GIT 预训练 | 提升 |
|---|---|---|---|
| 社交网络 | 基线 | 显著提升 | +3-5% |
| 引用网络 | 基线 | 显著提升 | +2-4% |
| 分子图 | 基线 | 显著提升 | +3-6% |
| 网页图 | 基线 | 改善 | +1-3% |
| 生物图 | 基线 | 改善 | +2-4% |

### 上下文学习（Few-shot，无需微调）

- 仅给出少量示例即可在新任务上发挥作用
- 性能超越 prompt-based 图学习方法

### 零样本泛化

- 预训练在 A 域 → 在 B 域直接评估
- 跨域零样本有效，验证 Task-Tree 通用性假设

### 指令调优

- 通用模型 + 域特定 post-training → 接近或超越域专家模型

## 亮点与洞察

1. **Task-Tree 统一节点/边/图级任务**的概念创新
2. 首个应对图任务异质性的理论框架
3. GIT 在三种使用范式下均有效
4. 类比图像中的 patch / 文本中的 sentence

## 局限性 / 可改进方向

- Task-Tree 深度受 GNN 层数限制
- 预训练图的域多样性对泛化影响待量化
- 与 LLM 知识融合未探索

## 相关工作与启发

- Liu et al. (2024a)：图基础模型综述
- Mao et al. (2024)：子结构方法
- 启发：Task-Tree 类思路可用于其他结构化数据
