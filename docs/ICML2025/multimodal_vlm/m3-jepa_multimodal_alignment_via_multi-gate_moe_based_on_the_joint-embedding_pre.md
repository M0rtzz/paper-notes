# M3-JEPA: Multimodal Alignment via Multi-gate MoE based on JEPA

**会议**: ICML2025  
**arXiv**: [2409.05929](https://arxiv.org/abs/2409.05929)  
**代码**: [GitHub - M3-JEPA](https://github.com/HongyangLL/M3-JEPA)  
**领域**: optimization  
**关键词**: 多模态对齐, JEPA, MoE, 能量模型, 模态崩溃

## 一句话总结
将JEPA(联合嵌入预测架构)推广到任意模态组合的多模态对齐中，用Multi-gate MoE作为跨模态预测器在潜在空间对齐，通过交替梯度下降解决不同模态任务间的梯度冲突，在视觉-语言/音频-语言等任务上达SOTA。

## 研究背景与动机

### 模态崩溃问题
现有多模态方法在token空间优化，容易出现模态崩溃（一种模态主导另一种），原因包括梯度冲突、模态缺失、数据分布不匹配。

### JEPA的优势
在潜在空间而非token空间做对齐，自然过滤无关信息，更适合处理信息不确定性和冗余。I-JEPA在视觉中已成功，但尚无通用多模态版本。

### M3-JEPA的定位
首个将JEPA推广到任意模态组合(any-to-any)的通用框架。

## 方法详解

### 整体架构
- 输入/输出分别由冻结的单模态编码器编码
- Multi-gate MoE作为轻量跨模态预测器
- 在潜在空间对齐输入和输出嵌入

### Multi-gate MoE预测器
- 门控函数解耦模态特定信息和共享信息
- 两个gate：一个控制对比损失，一个控制正则化损失
- 随机初始化，从头训练

### 损失函数
- 对比损失：拉近匹配的输入-输出对
- 正则化损失：防止表示坍塌
- 两者构成能量函数

### 交替梯度下降(AGD)
不同模态方向的任务轮流优化，每步只更新一个方向，避免梯度冲突。

### 信息论分析
隐式潜变量g驱动最优信息内容选择，最大化互信息、最小化条件熵。

## 实验关键数据

### 视觉-语言检索（Flickr30K/COCO）

| 方法 | R@1(I->T) | R@1(T->I) |
|------|----------|----------|
| CLIP | 基线 | 基线 |
| ImageBind | 较高 | 较高 |
| **M3-JEPA** | **最高** | **最高** |

### 跨模态泛化（未见域/数据集）

| 场景 | M3-JEPA表现 |
|------|-----------|
| 训练域内 | SOTA |
| 未见数据集 | **仍有竞争力** |
| 跨模态迁移 | 良好 |

### 训练/推理效率

| 方面 | M3-JEPA | 全量微调方法 |
|------|---------|-----------|
| 训练参数 | 仅MoE预测器 | 全部 |
| 推理延迟 | 较低 | 较高 |

### 关键发现
1. 潜在空间对齐显著减少模态崩溃
2. AGD有效缓解多方向任务间的梯度冲突
3. 门控解耦模态特定/共享信息提升了对齐质量
4. 在视觉/音频/文本三种模态上一致有效
5. 轻量MoE预测器使训练高效

## 亮点与洞察

1. 首个any-to-any JEPA多模态框架，扩展了JEPA的应用范围。
2. MoE的模态解耦+信息论分析提供了理论支撑。
3. AGD的巧妙设计避免了多任务优化的常见陷阱。
4. 冻结编码器+轻量预测器的设计极其实用。
5. 理论收敛保证和最优超参分析增加了可信度。

## 局限性 / 可改进方向

1. 门控函数的信息解耦程度取决于数据质量。
2. AGD增加了训练复杂度（需要多方向切换）。
3. 冻结编码器限制了细粒度适配能力。
4. 3D/触觉等更多模态的验证缺失。
5. 与最新的多模态LLM(GPT-4V等)的对比不够。

## 相关工作与启发

- 与I-JEPA/V-JEPA的关系：本文是JEPA在多模态上的推广。
- 与CLIP的区别：CLIP在token空间对比，M3-JEPA在潜在空间。
- 启发：JEPA+MoE的范式可成为自监督多模态学习的新基础。
