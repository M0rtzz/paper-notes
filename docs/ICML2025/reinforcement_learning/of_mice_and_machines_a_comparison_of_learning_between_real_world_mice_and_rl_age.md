# Of Mice and Machines: A Comparison of Learning Between Real World Mice and RL Agents

**会议**: ICML2025  
**arXiv**: [2505.12204](https://arxiv.org/abs/2505.12204)  
**代码**: 待确认  
**领域**: reinforcement_learning  
**关键词**: 生物启发RL, 风险规避, 捕食者-猎物, Q值方差, 经验回放

## 一句话总结

系统比较真实小鼠与RL智能体在捕食者-猎物迷宫中的行为差异，发现RL缺乏自我保护本能，提出创伤启发安全缓冲（TISB）和方差惩罚TD学习（VP-TDMPC-2）两种机制，将智能体与小鼠的状态访问重叠率从20.9%提升至86.1%。

## 研究背景与动机

RL近年在复杂决策任务上取得巨大进步，但一个根本性问题始终悬而未决：**RL智能体的决策行为在多大程度上真正捕捉了生物体的特征？**

作者设计了一个受控对比实验——"cellworld"六边形迷宫环境，其中真实小鼠需要在躲避机器人捕食者的同时抵达目标位置获取水奖励。同时构建了对应的Gymnasium仿真环境，用RL算法控制虚拟猎物。通过行为分析，作者识别出几个**关键行为差异**：

1. **自我保护本能缺失**：RL智能体为了微小的效率提升，常选择危险路径，靠近捕食者；小鼠则表现出成熟的风险评估行为
2. **靠墙行为（thigmotaxis）**：小鼠约70%的轨迹沿墙运动（焦虑相关行为指标），RL智能体为0%
3. **等待观察行为**：小鼠在看不到捕食者时会在入口处停留和观察，RL智能体则直接穿越
4. **学习效率差异**：小鼠仅需一次负面经历即产生持久回避，RL需要多次才能适应

这些观察构成了本文的核心动机：如何从生物行为中提取机制，使RL智能体产生更自然的风险规避行为。

## 方法详解

### 实验环境

- **物理环境**：六边形竞技场，331个磁性单元格（间距11cm），含障碍物。8只实验小鼠（4雄4雌），水限制驱动探索。高速追踪系统（90Hz）记录精确位置
- **机器人捕食者**：使用 $A^*$ 路径规划 + PID控制。当小鼠可见时追踪最后见到的位置，不可见时随机移动到盲区
- **仿真环境**："cellworld gymnasium"，复制物理布局和奖励结构。达到目标 +1，被捕获 -1。部分观测：智能体仅在视野内检测捕食者

### 基线方法：TDMPC-2

论文以TDMPC-2（模型-based RL）为基线，包含编码器 $z = h(s)$、潜在动力学 $z' = d(z,a)$、奖励预测器 $r^* = R(z,a)$、值预测器 $q^* = Q(z,a)$ 和策略先验 $\hat{a} = p(z)$。使用MPPI进行动作选择。

### 机制一：创伤启发安全缓冲（TISB）

模仿生物系统从创伤中学习的方式，包含两个组件：

**（1）威胁经验再平衡**：确保每个训练batch中包含指定比例的负面转换。实验发现**50%的负面经验比例**最优（在5000步实验中，5个种子的最佳表现）。这类似于幼年动物发展风险评估行为的过程。

**（2）威胁经验放大**：将负面经验的奖励在采样时放大**200倍**，反映生物系统中创伤的情感强度增强效应。更小的因子效果有限，大于200则减慢训练。

TISB使RL智能体产生了更广泛的探索和沿墙运动模式，但仍缺乏小鼠特有的入口等待行为。

### 机制二：方差惩罚TD学习（VP-TD）

**为何不用SMiRL？** 作者首先尝试了Surprise Minimization（SMiRL），将内在奖励 $r_{\text{SMiRL}}(s_t) = -\log p(s_t | s_{<t})$ 加入TDMPC-2，并用潜在空间MSE近似负对数似然。但SMiRL**均匀惩罚所有状态转换**，在部分可观测环境中反而阻碍了有益的状态转移（如从危险到安全），且抵消了TISB的路径多样化效果。

**VP-TD 核心公式**：基于Q值方差有效指示需要谨慎行为的状态这一观察，提出修改TD目标：

$$TD_{\text{target}} = r_t + \gamma \left( Q(s_{t+1}, \pi(s_{t+1})) - \alpha \text{Var}_{a \in A}(Q(s_{t+1}, a)) \right)$$

其中 $\alpha$ 控制方差惩罚强度。训练时从动作空间均匀采样网格，计算各动作的Q值，再取方差作为惩罚项。**不同动作导致高度不同结果的状态（如捕食者附近）会受到更大惩罚**。

**与SMiRL对比**：

| 维度 | SMiRL | VP-TD（本文） |
|------|-------|---------------|
| 不确定性度量 | 状态转移概率建模 | Q值方差（直接衡量决策不确定性） |
| 惩罚方式 | 通过奖励静态惩罚所有转换 | 在TD目标中动态调整，上下文相关 |
| 目标函数 | $\max_\pi \mathbb{E}[\sum_t \gamma^t (r_t - \alpha_1 \log p(s_t \| s_{<t}))]$ | $\max_\pi \mathbb{E}[\sum_t \gamma^t (r_t - \alpha_2 \text{Var}_a(Q(s_t,a)))]$ |

## 实验设置与主要结果

### 行为对比数据

| 指标 | TDMPC-2 | VP-TDMPC-2 | 真实小鼠 |
|------|---------|------------|---------|
| 成功率 | ~80% | — | 86% |
| 状态空间覆盖率 | 21.3% | — | 77% |
| 与小鼠访问重叠率 | **20.9%** | **86.1%** | 100% |
| 等待行为比例（1000轨迹） | 0% | **32.4%** | 27.7% |
| 靠墙轨迹比例 | 0% | 部分改善 | ~70% |
| 平均回合步数 | 7.09 | **13.89** | 14.04 |
| 捕食者检测前距离变化 | -0.2 ~ -0.5 | -0.1 ~ -0.2 | -0.05 ~ -0.15 |

### 关键发现

1. **VP-TDMPC-2将访问模式重叠率从20.9%提升至86.1%**，是最核心的定量结果
2. 增强版智能体表现出**保守运动模式**，初始进入环境时行进距离减少约45%，匹配小鼠的谨慎行为
3. 等待行为比例（32.4%）甚至略高于真实小鼠（27.7%），说明方差惩罚有效诱导了犹豫/观察行为
4. 平均回合步数（13.89 vs 14.04）几乎完美匹配小鼠
5. **LLM智能体（GPT）**也表现出与传统RL类似的冒险特征，缺乏生物对齐的行为，暗示这是AI系统的普遍问题

### TISB消融

- 威胁经验再平衡：50%负面经验最优，更高或更低均降低性能
- 威胁经验放大：因子200为最佳，产生更广泛的路径多样性和沿墙行为
- 单独TISB可改善路径多样性，但不足以产生等待行为

## 亮点与洞察

1. **跨学科实验设计出色**：真实小鼠实验（8只，高速追踪，水限制范式）与仿真环境的精确对应，使对比有说服力
2. **Q值方差作为风险信号**是一个简洁有力的insight——不需要建模复杂的状态转移概率，直接利用已有的Q网络输出
3. **86.1% vs 20.9%的重叠率提升**是一个令人印象深刻的结果，且等待行为和步数几乎完美匹配小鼠
4. LLM也缺乏生物对齐行为这一发现具有广泛启示意义：无论训练方式如何，当前AI系统在风险感知上存在根本缺陷

## 局限性 / 可改进方向

1. **靠墙行为仍有差距**：VP-TDMPC-2改善了覆盖范围但未完全复现小鼠的强thigmotaxis，中心区域使用仍然偏多
2. **环境局限**：仅在cellworld一个环境验证，泛化性未知。更复杂的连续空间、多捕食者场景需要进一步测试
3. **方差惩罚系数 $\alpha$ 敏感性**：论文未详细分析 $\alpha$ 的鲁棒性和最优调节策略
4. **生物学对应关系**：TISB的"创伤放大200倍"是工程选择而非严格的神经科学建模，生物学合理性存疑
5. **仅使用TDMPC-2为基线**：VP-TD思想能否移植到其他RL算法（SAC、PPO等）未验证
6. **小鼠个体差异**：虽然用了8只小鼠，但性别、性格（大胆/胆小）的个体差异分析不足

## 相关工作与启发

- **Cellworld框架**（Lai et al., 2024）：提供标准化的捕食者-猎物实验范式
- **SMiRL**（Berseth et al., 2019）：惊奇最小化RL，本文证明其在部分可观测环境中的局限
- **PER**（Schaul et al., 2015）：优先经验回放加速学习但不改变行为策略
- **风险敏感RL**：本文的VP-TD可视为一种新型风险敏感目标函数，与CVaR-based方法互补
- **启发**：Q值方差惩罚思路可推广至安全RL、机器人导航、自动驾驶等需要谨慎行为的领域
