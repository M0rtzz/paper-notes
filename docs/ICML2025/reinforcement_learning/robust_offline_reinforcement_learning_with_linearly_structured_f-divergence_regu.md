# Robust Offline Reinforcement Learning with Linearly Structured f-Divergence Regularization

**会议**: ICML 2025  
**arXiv**: [2411.18612](https://arxiv.org/abs/2411.18612)  
**代码**: 无  
**领域**: Reinforcement Learning  
**关键词**: 鲁棒离线RL, f-散度正则化, 线性结构, 函数近似, RRMDP

## 一句话总结
本文提出 d-rectangular linear RRMDP 框架，将潜在线性结构引入转移核和正则化，设计 R2PVI 算法实现离线数据下的鲁棒策略学习，证明了接近最优的样本复杂度。

## 研究背景与动机
1. **领域现状**: Robust Regularized MDP (RRMDP) 通过在值函数中对转移动态添加正则化来学习对环境扰动鲁棒的策略。这对于从离线数据学习特别重要——训练环境与部署环境可能存在动态差异。
2. **现有痛点**: 现有方法大多使用无结构正则化 (对转移核施加整体约束)，可能导致在不现实的转移下过度保守，策略过于悲观。
3. **核心矛盾**: 鲁棒性 vs 保守性——无结构的正则化虽然提升鲁棒性但代价是在正常情况下性能大幅下降。
4. **本文切入**: 引入 d-rectangular (维度分解式) 线性结构，使正则化更符合实际场景中动态变化的模式。
5. **核心 idea**: 在线性 MDP 假设下设计结构化 f-散度正则化，配合悲观值迭代 (PVI) 实现高效鲁棒学习。

## 方法详解

### 整体框架
输入：离线数据集 (名义环境收集) + d-RRMDP 定义 → R2PVI 算法 → 输出鲁棒策略

### 关键设计

1. **d-Rectangular Linear RRMDP**:
   - 将转移核分解为 d 个维度，每个维度独立施加约束
   - 约束使用 f-散度族 (包含 KL 散度、$\chi^2$ 散度等)
   - 设计动机：d-rectangular 结构避免了不现实的联合扰动，减少保守性

2. **R2PVI 算法 (Robust Regularized Pessimistic Value Iteration)**:
   - 使用线性函数近似估计值函数
   - 加入悲观项以处理离线数据分布外的状态-动作对
   - 利用 f-散度的凸对偶形式实现高效计算
   - 设计动机：PVI 框架适合离线 RL，线性近似使得理论分析可行

3. **Instance-Dependent 上界**:
   - 次优性上界取决于数据覆盖最优鲁棒策略下可达状态-动作空间的程度
   - 建立信息论下界证明算法接近最优
   - 设计动机：instance-dependent 分析比最坏情况分析更精细

### 损失函数 / 训练策略
- 基于 Bellman 方程的拟合值迭代
- 悲观惩罚项依赖于数据覆盖程度
- f-散度正则化的对偶形式用于计算

## 实验关键数据

### 主实验
| 环境 | 指标 | R2PVI | 基线方法 | 提升 |
|------|------|-------|----------|------|
| 合成 MDP | 次优性 gap | 最低 | 无结构 RRMDP 更高 | 显著 |
| 计算效率 | 运行时间 | 快 | 基线方法较慢 | 优势明显 |

### 消融实验
| 配置 | 关键指标 | 说明 |
|------|---------|------|
| 不同 f-散度 | 策略性能 | KL 和 χ² 散度表现相近 |
| 不同正则化强度 | 鲁棒性-性能权衡 | 过强正则→过于保守 |
| 有/无结构 | 次优性 | 结构化正则显著优于无结构 |

### 关键发现
- d-rectangular 结构避免了过度保守，在保持鲁棒性同时显著提升性能
- 信息论下界验证了 R2PVI 的接近最优性
- 线性函数近似在鲁棒离线 RL 中的首次严格分析

## 亮点与洞察
- **结构化正则化**: 将问题结构引入鲁棒性约束是关键创新
- **理论完整**: 既有上界也有下界，说明了基本限制
- **实用线性近似**: 使得算法可扩展到大状态空间

## 局限性 / 可改进方向
- 线性 MDP 假设在实际中可能不满足
- 实验规模较小，缺乏高维连续控制任务验证
- 超参数 (正则化强度) 选择的实际指南不足

## 相关工作与启发
- 经典 RMDP (Nilim & El Ghaoui 2005) 和 RRMDP (Panaganti & Kalathil 2022)
- 离线 RL: CQL, IQL, 悲观值迭代系列
- 分布式鲁棒优化与 RL 的结合是增长趋势
