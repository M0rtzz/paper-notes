# A Bayesian Model Selection Criterion for Selecting Pretraining Checkpoints

**会议**: ICML 2025  
**arXiv**: [2410.05612](https://arxiv.org/abs/2410.05612)  
**代码**: 无  
**领域**: 自监督学习  
**关键词**: 贝叶斯模型选择, 预训练检查点, 自由能, 迁移学习, 损失景观

## 一句话总结
引入"下游自由能"作为预训练检查点的贝叶斯模型选择准则——衡量检查点附近参数空间中适合下游任务的参数浓度，并证明可仅用预训练数据近似（预训练自由能），为选择最优检查点提供理论指导。

## 研究背景与动机

1. **领域现状**：预训练-微调范式在 NLP 和 CV 中占主导，但选择哪个训练阶段的检查点进行微调缺乏理论指导。
2. **现有痛点**：实践中主要靠经验启发式——如"使用最终检查点"或"大学习率更好"，缺乏原则性的选择准则。
3. **核心矛盾**：下游任务多种多样且在预训练时未知，如何在不接触下游数据的情况下选择最佳检查点？
4. **本文要解决什么**：提供检查点选择的理论框架和实用准则。
5. **切入角度**：贝叶斯统计的边际似然（受限于检查点邻域）作为可适应性的度量。
6. **核心 idea**：自由能低 = 检查点附近有更多"好参数"密集分布 = 更容易适应下游任务。

## 方法详解

### 整体框架
1. 定义下游自由能：检查点邻域内下游损失的边际似然的负对数
2. 证明预训练自由能是下游自由能的上界（在特定分布偏移条件下）
3. 验证：低预训练自由能 → 大学习率/小 batch/高 momentum → 更好的下游性能

### 关键设计

1. **下游自由能**:
   - 做什么：量化检查点附近参数空间中"好参数"的密度
   - 核心思路：$F_{\text{down}} = -\log \int_{\|w-w_0\|<\delta} p(D_{\text{down}}|w) dw$
   - 设计动机：低自由能 = 检查点周围有大量参数能在下游任务上表现好 = 微调容易成功

2. **预训练自由能作为代理**:
   - 做什么：证明在分布偏移条件下，预训练自由能控制下游自由能
   - 核心思路：$F_{\text{down}} \leq F_{\text{pre}} + \text{shift term}$
   - 设计动机：预训练时不需要知道下游任务/数据

### 损失函数 / 训练策略
- 理论分析为主，实验验证
- 大学习率/小 batch/高 momentum → 低预训练自由能 → 更好的迁移

## 实验关键数据

### 主实验
预训练自由能 vs 迁移准确率的相关性：

| 设置 | 相关系数 (R²) | 说明 |
|------|-------------|------|
| 学习率变化 | 0.91 | 大学习率→低自由能→高迁移准确率 |
| Batch size 变化 | 0.87 | 小 batch→低自由能→高迁移准确率 |
| Momentum 变化 | 0.85 | 高 momentum→低自由能→高迁移准确率 |

### 关键发现
- 预训练自由能比其他候选指标（如预训练损失、Hessian 迹）与下游性能的相关性更强
- 解释了已知的经验规律：大学习率/小 batch 有利于迁移的原因是它们降低了自由能
- 适用于全参数微调（而非仅线性探针）

## 亮点与洞察
- **将贝叶斯模型选择理论联系到实际预训练选择**——旧理论的新应用
- 解释了 SGD 隐式正则化（平坦最小值）对迁移学习有益的原因
- 预训练自由能可作为预训练过程中的在线监控指标

## 局限性 / 可改进方向
- 自由能的精确计算在高维中不可行，需要近似
- 分布偏移条件的假设在现实中可能不严格成立
- 仅在小-中规模模型上验证
