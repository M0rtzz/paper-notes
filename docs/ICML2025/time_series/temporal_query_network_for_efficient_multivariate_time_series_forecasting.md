# TQNet: Temporal Query Network for Efficient Multivariate Time Series Forecasting

**会议**: ICML2025
**arXiv**: [2505.12917](https://arxiv.org/abs/2505.12917)
**代码**: [GitHub - TQNet](https://github.com/ACAT-SCUT/TQNet)
**领域**: time_series
**关键词**: 多变量时序预测, Temporal Query, 变量相关性, 注意力机制, 轻量模型

## 一句话总结
提出Temporal Query(TQ)技术——用周期性移位的可学习向量作为注意力查询来捕捉全局变量间相关性，keys/values来自原始数据保留局部信息，单层注意力+浅MLP的TQNet即达SOTA。

## 研究背景与动机

### 变量相关性建模的挑战
多变量时序预测的核心在于准确建模变量间相关性。但非平稳干扰（极端值/缺失/噪声）导致样本级相关性与全局相关性存在显著差距。

### 三类方法的演进
1. Channel Mixing(CM)：混合变量但关系不可区分
2. Channel Independence(CI)：分离变量但忽视相关性
3. Channel Dependence(CD)：显式建模但需有效机制

### TQ的创新点
用**可学习的周期性查询**桥接全局和局部相关性。

## 方法详解

### Temporal Query技术
- **Query**：周期性移位的可学习向量，编码全局变量间模式
- **Key/Value**：来自原始输入数据，编码局部样本级相关性
- 周期性移位：参数重用，增强鲁棒性和稳定性

### TQNet架构（极简）
- 单层多头注意力（使用TQ）
- 浅层MLP
- 无复杂位置编码或多级结构

### 设计优势
1. 可学习query自适应捕获最优变量关系表示
2. 周期性位移实现参数复用
3. 全局+局部自然融合在一个注意力层中

## 实验关键数据

### 12个数据集SOTA对比

| 方法类型 | 代表方法 | 平均排名 | TQNet相对表现 |
|---------|---------|---------|-------------|
| CI类 | PatchTST, DLinear | 中等 | **显著优于** |
| CD类 | iTransformer, Crossformer | 中高 | **优于或持平** |
| 混合类 | TimeBridge, DUET | 高 | **多数优于** |

### 效率对比

| 方法 | 预测精度 | 推理速度 |
|------|---------|---------|
| DLinear | 中等 | 最快 |
| iTransformer | 高 | 中 |
| **TQNet** | **最高** | **接近DLinear** |

### 关键发现
1. TQ学到的相关性模式与全局训练集相关性高度吻合
2. 周期性移位显著优于随机初始化或固定查询
3. 单层注意力已足够（多层反而过拟合）
4. 在高维数据集上也保持DLinear级效率
5. 12个真实数据集上整体SOTA

## 亮点与洞察

1. "用可学习查询编码全局模式"极其简洁优雅。
2. 周期性移位的设计直觉清晰：时序数据本身就有周期性。
3. 单层注意力+浅MLP的极简架构说明了设计比规模更重要。
4. 效率与DLinear可比是巨大的实用优势。
5. 实验规模(12数据集)非常充分。

## 局限性 / 可改进方向

1. 周期性假设在无周期或弱周期数据上可能不适用。
2. 可学习查询的初始化和周期长度是额外超参。
3. 对极长序列(>10000步)的扩展性未验证。
4. 在线/增量学习场景中查询的更新策略未讨论。
5. 单层限制可能在超高维(数千变量)下不够。

## 相关工作与启发

- 与iTransformer的区别：iTransformer翻转了注意力的输入输出关系。
- 与TimeXer的区别：TimeXer用外部变量作query。
- 启发：TQ技术可插入其他架构作为变量相关性建模的通用模块。

## 补充分析

### TQ的周期性设计直觉
周期性移位使查询向量随时间窗口循环重用，既减少参数量又捕捉周期性模式。实验证明周期移位显著优于随机初始化或固定查询。

### 与iTransformer的核心区别
iTransformer将整个变量时序作为一个token，让注意力在变量间运作。TQNet则用可学习查询主动检索全局模式，同时保留局部信息。

### 单层注意力的充分性
多层注意力反而过拟合，因为TQ已经在一层中融合了全局+局部相关性。
