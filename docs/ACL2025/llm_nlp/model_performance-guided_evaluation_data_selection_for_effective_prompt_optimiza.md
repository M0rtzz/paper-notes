# Model Performance-Guided Evaluation Data Selection for Effective Prompt Optimization

**会议**: ACL 2025  
**arXiv**: [2505.10736](https://arxiv.org/abs/2505.10736)  
**代码**: 无  
**领域**: LLM/NLP  
**关键词**: prompt optimization, data selection, evaluation efficiency, automatic prompting

## 一句话总结
提出基于模型性能引导的评估数据选择策略，用少量但信息量高的评估样本来加速 prompt 优化过程，同时保持优化质量。

## 研究背景与动机
1. **领域现状**: 自动 prompt 优化需要大量评估样本来判断候选 prompt 的好坏。
2. **现有痛点**: 评估成本高，每个候选 prompt 都需在全部验证集上评估。
3. **核心矛盾**: 如何用更少样本实现同等质量的 prompt 优化？
4. **本文解决**: 基于模型性能选择更有信息量的评估样本。
5. **切入角度**: 类似主动学习的思想，选“最有识别度”的样本。
6. **核心idea**: 不是所有评估样本都同等重要，选择性评估大幅提高效率。

## 方法详解
### 整体框架
在 prompt 优化循环中，基于历史评估结果选择最有信息量的子集进行评估。
### 关键设计
1. **性能引导的样本选择**: 选择模型表现差异最大的样本
2. **自适应子集大小**: 根据优化阶段调整子集大小

## 实验关键数据
| 方法 | 评估调用数 | 优化质量 |
|------|----------|----------|
| 全集评估 | 100% | 基线 |
| 随机子集 | 30% | -5% |
| **性能引导选择** | **30%** | **-1%** |

### 关键发现
- 30% 的评估调用达到接近全集的优化质量
- 评估成本减少 70%
