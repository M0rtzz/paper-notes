# Leveraging Large Language Models to Measure Gender Representation Bias in Gendered Language Corpora

**会议**: ACL 2025  
**arXiv**: [2406.13677](https://arxiv.org/abs/2406.13677)  
**代码**: https://github.com/ellisalicante/grb-corpora  
**领域**: LLM/NLP  
**关键词**: gender bias, representation bias, gendered language, corpus analysis, Spanish

## 一句话总结
提出基于 LLM 的方法量化有语法性别语言（西班牙语、巴伦西亚语）语料中的性别表示偏差，发现 4:1 到 6:1 的男性主导失衡，并证明少量反向偏差数据（5000句）可有效缓解模型输出中的性别不平衡。

## 研究背景与动机

1. **领域现状**：LLM 性别偏差研究主要关注英语和刻板印象偏差，对训练语料中不同性别被提及频率的不均（表示偏差）关注不足。
2. **现有痛点**：现有方法（如 gender polarity）依赖预定义词表只适用英语。有语法性别的语言中名词本身带性别，词表方法无法区分指代人的性别化名词与无生命名词的语法性别。
3. **核心矛盾**：全球 38% 人口使用有语法性别语言，但缺乏适用的偏差量化方法。
4. **本文要解决什么？** 利用 LLM 上下文理解能力自动识别和分类有性别化语言中指代人的词汇。
5. **切入角度**：Prompt engineering + few-shot prompting 让 LLM 从语料提取指代人的名词/代词并判断语法性别。
6. **核心idea一句话**：LLM 可以区分指代人的性别化名词和无生命名词的语法性别，准确量化有性别化语言中的性别表示偏差。

## 方法详解

### 整体框架
输入语料 -> LLM 识别指代人的名词代词 -> 分类男/女指代 -> 计算比率 -> continual pretraining 验证偏差传播 -> 用反向偏差数据缓解。

### 关键设计

1. **LLM 驱动的性别词提取**
   - 做什么：Prompt LLM 识别文本中所有指代人的名词和代词并判断其语法性别
   - 核心思路：利用 LLM 区分 profesor（教授，指人）vs libro（书，不指人），虽然两者都是语法阳性
   - 经过大量迭代的 prompt 工程和 few-shot 示例优化
   - 设计动机：传统词表将所有语法阳性名词视为男性，严重高估

2. **双语言验证**
   - 西班牙语（高资源）：4 个 benchmark 语料
   - 巴伦西亚语（低资源）：5 个语料
   - 设计动机：验证跨资源水平的泛化性

3. **偏差传播与缓解实验**
   - 在有偏差语料上做 continual pretraining
   - 用少量反向偏差数据（5000句）做缓解
   - 设计动机：证明语料偏差到模型偏差的因果关系

## 实验关键数据

### 主实验 -- 性别表示偏差比率
| 语料 | 语言 | 男性:女性 |
|------|------|----------|
| OSCAR | 西班牙语 | 6:1 |
| Wikipedia | 西班牙语 | 5:1 |
| mC4 | 西班牙语 | 4:1 |
| 新闻语料 | 巴伦西亚语 | 3:1 |
| 文学语料 | 巴伦西亚语 | 2:1 |

### 偏差缓解实验
| 训练数据 | 模型输出比率 |
|---------|------------|
| 原始偏差语料 | ~4:1 |
| +5000 句女性偏向数据 | ~1.5:1 |
| +5000 句平衡数据 | ~2:1 |

### 关键发现
- 所有语料存在严重男性主导偏差，最高达 6:1
- 语料偏差直接传播到模型输出
- 少量反向偏差数据即可有效缓解：5000 句将 4:1 降至 1.5:1
- LLM 方法远优于传统词表方法

## 亮点与洞察
- 首次系统解决有性别化语言的表示偏差量化，覆盖全球 38% 人口
- 少量反向偏差数据即可缓解具有极高实用价值
- 从语料偏差到模型偏差的完整因果链为偏差研究提供方法论范本

## 局限性 / 可改进方向
- 仅测试两种语言
- LLM 提取可能有遗漏
- 改进方向：扩展到法语/德语/阿拉伯语

## 相关工作与启发
- **vs BOLD**：BOLD 的 gender polarity 只适用英语
- **vs Biesialska et al.**：他们发现表示偏差与刻板印象偏差相关
