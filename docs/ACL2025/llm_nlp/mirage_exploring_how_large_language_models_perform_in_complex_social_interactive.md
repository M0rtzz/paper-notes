# MIRAGE: Exploring How Large Language Models Perform in Complex Social Interactive Environments

**会议**: ACL 2025  
**arXiv**: [2501.01652](https://arxiv.org/abs/2501.01652)  
**代码**: [GitHub](https://github.com/)（论文声明数据集和模拟代码已公开）  
**领域**: LLM/NLP  
**关键词**: 社会交互评估, 角色扮演, 剧本杀, LLM 评测, 信任与欺骗

## 一句话总结

本文提出 MIRAGE 评估框架，通过八个精心设计的剧本杀场景和四项指标（信任倾向 TII、线索调查 CIC、交互能力 ICI、剧本合规 SCI）系统评估 LLM 在复杂社交互动环境中的表现，发现即使 GPT-4 也在这些场景中面临严峻挑战。

## 研究背景与动机

LLM 在环境感知、推理决策和模拟复杂人类行为方面展现了显著潜力，尤其是在角色扮演场景中。然而，现有评估存在几个关键不足：

**聚焦于 Agent 而忽视 LLM 本身**：Sotopia 和 Lyfe Agents 等工作关注工作流增强的 Agent 系统的社交互动能力，但忽略了一个关键事实——基础社交互动能力源于底层 LLM 本身。

**游戏场景过于简单**：Werewolf 和 Avalon 等桌游受限于刚性决策流程和有限场景多样性，无法充分测试 LLM 的高级社交行为（如欺骗、领导力）。

**已有剧本杀模拟不足**：Wu et al. (2023) 的先驱工作受限于游戏脚本范围窄、评估方法简单、数据集缺乏人工检验。

剧本杀（Murder Mystery Game）是理想的评估载体：参与者扮演角色身份，进行半结构化叙事互动，需要广泛的背景知识、社会驱动的决策、以及开放式交互。这些特性使其特别适合评估 LLM 如何应对复杂的人类社交行为。

## 方法详解

### 整体框架

MIRAGE 由三部分组成：（1）八个精心制作的剧本杀场景构成评估环境；（2）三阶段模拟流程驱动游戏进行；（3）四项客观评估指标量化 LLM 表现。

### 关键设计

1. **剧本构建（8 个实验场景）**：每个剧本包含六部分内容：角色故事（背景信息）、角色剧本（事件时间线）、角色关系（初始人际网络）、角色表演（性格和说话风格）、角色目标（任务和目标）、其他能力（游戏规则）。八个剧本涵盖多种维度：

    - 结构维度：单段式 vs 多段式（角色信息一次性或分阶段揭示）
    - 类型维度：正统（现实设定）vs 非正统（超自然/奇幻）
    - 结局维度：封闭（固定结局）vs 开放（结局随玩家行为变化）
    - 规模维度：5-10 个角色，14-82 条线索，3K-62K 字

2. **三阶段模拟流程**：所有角色分为凶手和平民两个阵营。

    - **阶段 A - 开放对话**：玩家按剧本角色进行回合制开放对话。
    - **阶段 B - 环境交互**：玩家选择"询问"（向特定角色提问，对方必须回答）或"调查"（向所有角色公开一条线索）。
    - **阶段 C - 投票**：玩家指控嫌疑人并投票。凶手被票数最高的指控命中则平民胜利。

3. **辅助模块**：

    - **摘要模块**：输入超过 LLM 上下文限制时压缩历史信息
    - **怀疑模块**：每轮对话后记录对其他角色的怀疑分数
    - **信任模块**：每轮对话后记录对其他角色的信任分数
    - **重试模块**：LLM 输出无法解析时重新提交

4. **四项评估指标**：

    - **TII（信任倾向指数）**：综合怀疑和信任分数，衡量 LLM 在社交互动中平衡信任与怀疑的能力。
    - **CIC（线索调查能力）**：已调查线索与总线索的比率，衡量信息收集和问题解决能力。
    - **ICI（交互能力指数）**：由强力中立 LLM 评估推理分析能力、沟通合作能力、观察能力和创新思维能力。
    - **SCI（剧本合规指数）**：通过两项评估的平均值衡量角色扮演忠实度：直接打分 + 基于 Rouge-L 的剧本重建一致性。

### 损失函数 / 训练策略

MIRAGE 是一个评估框架，不涉及模型训练。评估中使用 GPT-4-Turbo 作为中立评估模型（经消融实验验证其稳定性优于 GPT-4 和 GPT-4o）。每个角色参与 5 轮开放对话和交互阶段的交替循环。

## 实验关键数据

### 主实验

| 模型 | Victory (MRR) | TII | CIC | ICI | SCI |
|------|--------------|-----|-----|-----|-----|
| GPT-3.5 | 29.11 | 47.13 | 27.46 | 70.06 | 49.10 |
| GPT-4 | 34.69 | 76.32 | 19.01 | 76.54 | 50.42 |
| GPT-4o | 47.01 | 78.69 | 35.92 | 76.80 | 51.29 |
| Qwen-2-7B | **51.81** | 75.78 | 18.66 | 74.92 | 50.57 |
| GLM-4-9B | 31.89 | 53.85 | 20.07 | 71.60 | 48.13 |

信任偏差实验（凶手自曝身份后的 TII 变化）：

| 模型 | TII（无自曝） | TII（自曝后） | 变化 Δ |
|------|-------------|-------------|--------|
| Qwen-1-7B | 51.02 | 50.69 | -0.33 |
| Qwen-1.5-7B | 73.00 | 69.14 | -3.86 |
| Yi-1.5-9B | 55.73 | 57.57 | **+1.84（唯一正确提高怀疑）** |
| GLM-4-9B | 57.82 | 55.94 | -1.88 |

### 消融实验

评估模型选择消融：

| 评估模型 | ICI 评分稳定性 | 说明 |
|---------|-------------|------|
| GPT-4 | 不稳定，存在强偏差 | 对某些模型评分偏高 |
| GPT-4-Turbo | **最稳定** | 评分最一致，最终采用 |
| GPT-4o | 偏低 | 系统性低估 |

Token 使用统计：

| 模型 | 环境输入 Token | 模型输出 Token | 环境请求数 | 用户完成数 |
|------|-------------|-------------|----------|----------|
| GPT-3.5 | 2,719,895 | 121,378 | 883 | 580 |
| GPT-4o | 6,252,580 | 204,772 | 1,328 | 574 |
| Qwen-2-7B | 2,204,029 | 192,158 | 743 | 588 |

### 关键发现

1. **GPT-4o 综合最强但胜率不最高**：GPT-4o 在 CIC、ICI、SCI 三个指标上均最优，但在实际胜率上被 Qwen-2-7B 超越（51.81% vs 47.01%），说明"更聪明"不等于"赢得更多"。

2. **LLM 普遍存在过度信任偏差**：大多数 LLM 即使在凶手自曝身份的极端场景下仍维持信任。只有 Yi-1.5-9B 会在此情况下提高怀疑度，这解释了其在 Victory 指标上的较好表现。

3. **线索发现模式**：LLM 在早期轮次展现高环境探索热情（CIC 快速上升），但关键线索的发现不稳定，说明模型虽然积极探索但缺乏识别关键信息的能力。

4. **开源模型可媲美部分闭源**：Qwen-2-7B（7B 参数）在 ICI 上接近 GPT-4，在 SCI 上甚至超越 GPT-4，说明模型大小不是社交互动能力的唯一决定因素。

5. **复杂社交场景仍是 LLM 的短板**：所有模型的 SCI 仅在 48-51% 左右，说明 LLM 在忠实执行复杂角色扮演方面距离理想水平还有很大差距。

## 亮点与洞察

- 剧本杀作为 LLM 社交评估工具的选择很有创意——它比 Werewolf/Avalon 更接近真实社交场景，需要综合运用知识理解、推理、欺骗、合作等多种能力。
- 八个场景的设计非常用心，涵盖了单段/多段、正统/非正统、封闭/开放等多维度变化，确保评估的多样性。
- TII 指标揭示的过度信任现象颇具启发性——LLM 可能因训练中对"友好"和"合作"行为的偏好而难以在对抗场景中正确表达怀疑。
- 胜率与能力指标的错位（Qwen-2-7B 胜率最高但非最"聪明"）提供了有趣的洞察：在博弈场景中，理性分析能力未必转化为实际胜利。

## 局限与展望

- **上下文长度限制**：LLM 的上下文限制导致必须使用摘要模块压缩历史信息，这会影响决策质量。
- **场景规模有限**：八个剧本的数据量相对有限，结论的普遍性需要更多场景验证。
- **安全对齐的不公平影响**：重视安全的 LLM 可能拒绝回答涉及谋杀、欺骗等敏感话题，在模拟中处于不利位置。
- **评估主观性**：ICI 和 SCI 依赖 LLM 评估器，不同评估模型可能产生不同结论。
- **缺乏人类基线**：未与人类玩家对比，难以判断 LLM 与人类的真实差距。

## 相关工作与启发

- **SOTOPIA**（Zhou et al., 2023）：评估 LLM Agent 的社交智能，但关注工作流增强而非裸 LLM。MIRAGE 更聚焦 LLM 本身的社交能力。
- **Generative Agents**（Park et al., 2023）：模拟人类行为的生成式代理，证明了 LLM 在社交模拟中的可行性。
- **Werewolf Game 研究**（Xu et al., 2023; Wu et al., 2024）：用狼人杀评估 LLM 的欺骗和推理能力，但场景刚性且多样性不足。
- 本文证明了在更复杂的叙事互动场景中，LLM 的社交能力边界，为后续的社交 AI 研究提供了基准。

## 评分

- 新颖性: ⭐⭐⭐⭐ 剧本杀作为 LLM 评估工具的选择新颖，八场景四指标体系设计系统
- 实验充分度: ⭐⭐⭐ 五个模型的评估规模适中，但缺乏人类基线对比和更多开源模型测试
- 写作质量: ⭐⭐⭐⭐ 框架描述清晰，场景设计细节充足，但指标公式散布在附录中不够直观
- 价值: ⭐⭐⭐⭐ 填补了复杂社交互动场景的评估空白，TII 揭示的信任偏差现象对 LLM 安全研究也有意义
