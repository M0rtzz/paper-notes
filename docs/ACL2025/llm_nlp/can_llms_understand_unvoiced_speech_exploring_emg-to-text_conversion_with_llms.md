# Can LLMs Understand Unvoiced Speech? Exploring EMG-to-Text Conversion with LLMs

**会议**: ACL 2025  
**arXiv**: 无  
**代码**: 无  
**领域**: LLM / NLP  

## 一句话总结

本文提出了一种基于可训练 EMG 适配器模块的方法，将无声肌电图（EMG）信号映射到大语言模型（LLM）的输入嵌入空间，在闭合词汇无声 EMG 转文本任务中实现了 0.49 的词错误率（WER），仅需 6 分钟训练数据即比专用模型提升约 20%。

## 背景与动机

1. **语音障碍需求广泛**：仅美国就有约 400 万人受语音障碍影响，静默语音接口作为替代交流方式需求迫切。
2. **现有方法依赖有声数据**：此前的 EMG 转文本方法多依赖配对的有声 EMG 信号和音频数据来辅助无声信号转换，对无法发声的用户并不实际。
3. **LLM 在语音识别领域表现优异**：近年来 LLM 在语音识别等多模态任务中展现出强大能力，但其能否理解无声生物信号尚未被充分探索。
4. **个性化需求与数据稀缺**：EMG 信号具有高度的个人特异性，需要从有限数据中高效学习，传统大规模数据驱动方法不适用。
5. **已有 LLM 扩展模态的成功经验**：通过适配器模块将语音、视频等模态映射到 LLM 输入空间已有成功先例，但生物信号（如 EMG）的挑战更大。
6. **直接利用 LLM 理解 EMG 的空白**：虽然有工作将 LLM 作为 EMG 模型的后处理步骤，但直接用 LLM 理解无声 EMG 信号的研究几乎为空白。

## 方法详解

### EMG 适配器网络设计

整体流程：输入无声 EMG 信号 $\mathbf{X}^E \in \mathbb{R}^{T \times C}$（C 为通道数，T 为时间步），经过以下模块处理：

1. **时序下采样**：首先使用步长为 6 的 1D 卷积层对高采样率（>800Hz）信号进行下采样。
2. **特征提取**：采用两层残差块（带 1D 卷积层）提取 EMG 特征，沿用 Gaddy and Klein (2021) 的设计。
3. **序列建模**：使用双向 LSTM（BiLSTM）捕捉复杂的时序依赖关系，消融实验验证其优于 Transformer 和纯全连接层。
4. **二次下采样**：再通过步长为 2 的 1D 卷积进一步压缩，得到 $\tilde{E} \in \mathbb{R}^{(T/48) \times \tilde{F}}$。
5. **维度投影**：通过全连接层将 EMG 嵌入投影到 LLM 输入维度（LLaMA 2-7B 为 4096，LLaMA 3.2-3B 为 3072），激活函数为 GeLU。

### LLM 数据准备与训练

- **上下文化嵌入**：在 EMG 嵌入前添加文本标识符 "Unvoiced EMG:"，后附任务提示 "Prompt: Convert unvoiced EMG embeddings to text"。
- **训练目标**：使用带温度参数 $\tau=0.8$ 的交叉熵损失，LLM 参数冻结，仅训练适配器（约 6M 参数）。
- **推理方式**：自回归生成，beam width 为 4。
- **优化器**：AdamW，最大学习率 $5 \times 10^{-5}$，带权重衰减。

## 实验结果

### 主要性能对比

| 模型 | 输入特征 | WER ↓ |
|------|----------|-------|
| App-Specific (Gaddy & Klein) | Raw EMG | 0.75 ± 0.06 |
| EMG-Ad + Llama2-7B | Raw EMG | 0.65 ± 0.01 |
| EMG-Ad + Llama3-3B | Raw EMG | 0.52 ± 0.05 |
| App-Specific (Gaddy & Klein) | 手工特征 | 0.84 ± 0.06 |
| EMG-Ad + Llama2-7B | 手工特征 | **0.49 ± 0.06** |
| EMG-Ad + Llama3-3B | 手工特征 | **0.49 ± 0.04** |

- 使用手工时频特征（112 维）时 LLM 方法平均提升 15%，最佳 WER 达 0.49。
- 仅 6 分钟训练数据时，LLM 方法仍比专用模型优约 26%。

### 消融实验

| 组件变体 | WER ↓ |
|----------|-------|
| 全连接层 | 0.70 |
| ResBlock(2) | 0.64 |
| ResBlock(2) + Transformer | 0.79 |
| ResBlock(2) + LSTM | **0.53** |
| 交叉熵损失 (Llama2) | **0.65** |
| CTC 损失 (Llama2) | 0.70 |

BiLSTM 在短序列闭合词汇场景中优于 Transformer；交叉熵损失优于 CTC 损失。

## 亮点

- **首次探索 LLM 直接理解无声 EMG**：填补了 LLM 在关节生物信号理解方面的空白。
- **极高的数据效率**：仅需 6 分钟数据、6M 可训练参数即可超越 54M 参数的专用模型。
- **发现手工特征对 LLM 有增益**：与专用模型相反，手工特征在 LLM 设置下显著提升性能。
- **个人识别验证**：EMG 信号可以 96% 准确率识别用户身份，验证了个性化建模的必要性。

## 局限性

- **仅限闭合词汇（67 词）**：尚未扩展到开放词汇的实际应用场景。
- **单一说话人、单一数据集**：仅在 Gaddy and Klein (2021) 的 8 通道、单人数据集上验证。
- **数据增强效果有限**：尝试的时间偏移和 Hilbert 变换相位增强均未显著改善性能。
- **依赖 LLM 嵌入层访问**：无法与仅提供 API 接口的语言模型配合使用。
- **仅英语验证**：缺乏多语言和多设备配置的实验。

## 相关工作

- **静默语音 EMG**：Gaddy and Klein (2020, 2021) 提出的有声 EMG-to-text 基线模型和数据集是本领域的重要基础。
- **LLM 多模态扩展**：Tang et al. (2023) 和 Yu et al. (2024) 将语音模态引入 LLM；Ma et al. (2024) 用简单线性投影实现 LLM 语音识别。
- **适配器设计**：BLIP-2 (Li et al., 2023) 和 Kang et al. (2024) 分别在图像和语音模态中使用投影网络连接预训练编码器与 LLM。
- **EMG + LLM**：Benster et al. (2024) 将 LLM 用作多模态 EMG 模型的后处理步骤，但仍依赖有声数据。

## 评分

- ⭐⭐⭐ 新颖性：首次直接用 LLM 理解无声 EMG，但闭合词汇限制了通用性
- ⭐⭐⭐⭐ 实用性：对语音障碍群体有直接应用价值，数据效率高
- ⭐⭐⭐ 实验充分度：消融实验丰富，但数据集单一、仅单人验证
