# Do Language Models Mirror Human Confidence?

**会议**: ACL 2025  
**arXiv**: [2506.00582](https://arxiv.org/abs/2506.00582)  
**代码**: 无  
**领域**: LLM/NLP  
**关键词**: LLM confidence, psychological insights, overconfidence, calibration

## 一句话总结
探索 LLM 的置信度是否呈现与人类类似的心理学模式（如过度自信效应、达克效应），在 MMLU 等基准上发现 LLM 确实存在类人的过度自信偏差，但在某些维度上也有独特模式。

**本文要解决什么？** 提供新的评估视角/方法/基准来推动领域发展。
**切入角度**：从独特的理论框架或方法论出发。
**核心idea一句话**：探索 LLM 的置信度是否呈现与人类类似的心理学模式（如过度自信效应、达克效应）。
