# Mind the (Belief) Gap: Group Identity in the World of LLMs

**会议**: ACL 2025  
**arXiv**: [2503.02016](https://arxiv.org/abs/2503.02016)  
**代码**: [GitHub](https://github.com/MichiganNLP/BeliefCongruenceSim)  
**领域**: LLM/NLP  
**关键词**: 信念一致性, 多智能体, 社会偏见, 错误信息传播, 群体心理学

## 一句话总结

通过多智能体 LLM 框架模拟信念一致性（Belief Congruence）理论，发现 LLM 表现出比人类更强的信念一致性偏向，导致错误信息传播增加和学习能力受损，并提出三种基于社会心理学的缓解策略。

## 研究背景与动机

信念一致性理论由 Rokeach (1960) 提出，指人们倾向于与持有相似信念的人保持一致，这种倾向在塑造群体互动和社会偏见中起关键作用。随着 LLM 在多智能体系统和社会模拟中的广泛应用，理解它们是否也表现出类似的群体心理特征变得至关重要。

然而，现有研究对 LLM 中群体间动态行为的探索非常有限。如果 LLM 像人类一样表现出信念一致性，那么在多智能体交互中可能会产生回音室效应，加剧错误信息传播并阻碍学习。本文提出三个研究问题：(1) LLM 是否表现出信念一致性？(2) 这种行为是否影响下游任务？(3) 能否缓解其负面影响？

这是首个使用 LLM 智能体模拟信念一致性并检验其对下游任务影响的研究。

## 方法详解

### 整体框架

研究分为三个主要部分：(1) 信念一致性模拟实验，复现经典心理学实验；(2) 下游任务影响评估，包括错误信息传播和 LLM 学习；(3) 基于社会心理学理论的缓解策略。使用 llama-3-70b、qwen-2.5-72b 和 gpt-35 三个模型。

### 关键设计

**1. 信念一致性模拟：复现 Rokeach & Mezei (1966) 实验**

- **功能**: 验证 LLM 是否表现出与人类相似（甚至更强）的信念一致性行为
- **核心思路**: 在校园实验中，白人参与者 LLM 与四个联盟者（2 白人 + 2 黑人，各有一个同意/反对者）讨论话题后选择两人共进咖啡；在现场实验中，黑/白参与者 LLM 与四个联盟者讨论精神科患者场景后选择未来同事。每轮随机化参与者顺序
- **设计动机**: 使用经典心理学实验范式确保研究的理论基础扎实，通过与原始人类实验数据的直接对比量化 LLM 的信念一致性强度

**2. 下游任务评估：错误信息传播 + LLM 学习**

- **功能**: 量化信念一致性对实际应用场景的负面影响
- **核心思路**: **错误信息传播**：使用民主党/共和党政治 persona 的同质化/异质化 LLM 组群，讨论新闻真实性；通过正确率变化衡量回音室效应。**LLM 学习**：采用 Marks et al. (2019) 的两阶段框架（学习阶段 + 选择阶段），测试 LLM 是否偏好政治立场相似的信息源即使该信息源准确度更低
- **设计动机**: 错误信息和学习障碍是信念一致性在人类社会中最重要的负面影响，本文将其迁移到 LLM 场景中验证

**3. 三种缓解策略**

- **功能**: 减轻信念一致性带来的负面影响
- **核心思路**: (1) **接触假说**：引入由民主党和共和党智能体组成的异质化群体；(2) **准确性推动**：在交互中提示智能体验证信息准确性（类似自我反思）；(3) **全球政治公民身份**：将 Reysen & Katzarska-Miller 的全球公民框架改编为政治语境，在系统提示中加入强调跨政治包容的调查问题
- **设计动机**: 每种策略针对不同层面——接触假说改变群体组成，准确性推动改变个体认知过程，全球公民改变身份认同框架

### 损失函数 / 训练策略

本文为评估研究，不涉及模型训练。评估指标包括：信念一致性选择频率、错误信息正确率（$\text{correctness rate} = \sum_{i=1}^{N} \mathbb{I}(f(x_i) = y_i) / n$）、信息源选择比例和置信度变化。

## 实验关键数据

### 主实验：错误信息传播

| 模型 | 初始正确率(Dem) | 最终正确率(Dem) | 初始正确率(Rep) | 最终正确率(Rep) |
|------|----------------|----------------|----------------|----------------|
| gpt-35 | ~0.65 | 0.598 | ~0.65 | 0.601 |
| llama-3 | ~0.50 | 0.444 | ~0.50 | 0.377 |
| qwen-2.5 | ~0.45 | 0.400 | ~0.45 | 0.390 |

### 消融实验：缓解策略效果

| 缓解策略 | gpt-35 最终正确率(Dem/Rep) | llama-3 | qwen-2.5 |
|---------|--------------------------|---------|----------|
| 无缓解 | 0.598 / 0.601 | 0.444 / 0.377 | 0.400 / 0.390 |
| 准确性推动 | 0.664 / 0.632 | 0.501 / 0.450 | 0.520 / 0.460 |
| 全球政治公民 | **0.678 / 0.661** | **0.581 / 0.550** | **0.525 / 0.516** |
| 接触假说(异质化) | 0.674 | 0.544 | 0.493 |

### 关键发现

1. **LLM 信念一致性被放大**: 所有模型在校园和现场实验中都表现出比人类更强的信念一致性（人类选择频率 0.2-0.6，LLM 均 ≥0.5，gpt-35 平均 0.93）
2. **同质化群体加剧错误信息**: 交互后正确率普遍下降，共和党 persona 表现更差
3. **全球政治公民身份是最有效的错误信息缓解策略**: 正确率最高提升 37%
4. **准确性推动对学习任务最有效**: 减少了智能体选择政治相似但准确度低的信息源的倾向（↓11%）
5. **信念一致性在所有人口统计维度上普遍存在**: 替换为年龄、性别、最小群体等维度后效果一致

## 亮点与洞察

- 首次将经典群体心理学理论（信念一致性）引入 LLM 多智能体系统并进行系统验证
- 实验设计严谨，直接复现心理学经典实验并与人类数据对比
- 缓解策略具有实际指导价值，特别是全球公民身份框架的跨领域应用令人印象深刻
- 揭示了 LLM 多智能体系统中的群体行为风险，对 AI 安全具有重要启示

## 局限与展望

- 模拟基于单一研究范式（Rokeach 1960），需要更多心理学实验验证
- LLM 对信念系统的模拟可能过度简化人类复杂的社会文化和情感因素
- 主要基于美国政治语境，跨文化适用性有待验证
- 仅探索了两个下游任务，信念一致性在心理健康等领域可能具有正面影响
- 未来可探索社会认同理论、现实冲突理论等其他群体心理学理论

## 相关工作与启发

- **多智能体 LLM 交互**: Park et al. (2023) 的社会模拟、ChatEval 等工作展示了 LLM 智能体的涌现行为，但群体心理层面的系统研究缺失
- **启发**: LLM 内化的偏见不仅体现在单个输出上，更在多智能体交互中被放大和强化。设计多智能体系统时应考虑群体多样性和自我反思机制

## 评分

- **新颖性**: ⭐⭐⭐⭐⭐ 首次将信念一致性理论系统引入 LLM 领域，问题重要且跨学科
