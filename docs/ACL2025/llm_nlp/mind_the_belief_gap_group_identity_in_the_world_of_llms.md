# Mind the Belief Gap: Group Identity in the World of LLMs

**会议**: ACL 2025  
**arXiv**: [2503.02016](https://arxiv.org/abs/2503.02016)  
**代码**: [GitHub](https://github.com/MichiganNLP/BeliefCongruenceSim)  
**领域**: LLM/NLP  
**关键词**: 信念一致性, 群体身份, 多智能体模拟, 虚假信息传播, 社会偏见

## 一句话总结
构建多智能体信念一致性模拟框架，发现 LLM 比人类表现出更强的信念一致性偏见（gpt-3.5: 0.93 vs 人类: 0.2-0.62），导致虚假信息传播加剧、跨群体学习受阻，并探索接触假说等缓解策略（最优可改善 37%）。

## 研究背景与动机
1. **领域现状**：社会心理学中的信念一致性理论（Belief Congruence Theory）指出，人们倾向于信任和接收与自己已有信念一致的信息，并更亲近持有相似信念的个体。这种现象在人类社会中导致回音室效应和信息茧房。LLM 作为越来越多地充当信息中介（搜索、推荐、对话）的角色，其是否携带并放大这种偏见至关重要。
2. **现有痛点**：(a) LLM 偏见研究主要集中在刻板印象（性别、种族、宗教）和有毒内容生成，对群体身份认同和信念一致性这一社会心理学层面的偏见关注不足；(b) 缺乏将 LLM 放入多智能体社交网络中研究信息传播动态的实验框架；(c) 现有 LLM 评估主要关注单次输入-输出，不考虑多轮交互中信念的动态演化。
3. **核心矛盾**：LLM 在模拟具有不同政治/社会身份的个体时，是否会复现甚至放大人类社会中的信念一致性效应？如果是，这对 LLM 被广泛用作信息中介意味着什么？
4. **本文要解决什么**：(a) 量化 LLM 的信念一致性强度并与人类基线对比；(b) 评估信念一致性放大对虚假信息传播和跨群体学习的影响；(c) 探索从社会心理学移植的缓解策略。
5. **切入角度**：从社会心理学的信念一致性理论出发，构建多智能体模拟框架，让 LLM agent 扮演不同政治/社会群体成员，在模拟社交网络中进行信息交换和评估。
6. **核心 idea**：LLM 展现出比人类更强烈的信念一致性偏见——在信息来源选择上政治同质性高达 70-89%，这不仅是偏见问题更是信息生态系统风险。

## 方法详解

### 整体框架
三阶段实验设计：
1. **信念一致性测量**：让 LLM 扮演不同群体身份 → 呈现来自不同群体的信息 → 测量接受/拒绝倾向
2. **影响评估**：将信念一致性放大的 LLM 嵌入模拟社交网络 → 观察虚假信息传播动态和跨群体学习效果
3. **缓解策略测试**：应用社会心理学中已验证的去偏方法 → 评估其在 LLM 上的效果迁移性

### 关键设计

1. **多智能体信念模拟框架**：
   - 每个 agent 被赋予明确的群体身份（政治倾向：自由派/保守派，社会身份：不同宗教/种族/职业）
   - 身份通过 system prompt 注入，包含该群体典型持有的核心信念集合
   - Agent 在模拟社交网络中进行多轮信息交换：每轮随机配对 → 一方分享观点 → 另一方决定是否接受/修改自己的信念
   - 网络结构：Erdős–Rényi 随机图、小世界网络、尺度无关网络三种拓扑

2. **信念一致性量化指标**：
   - **BC Score（Belief Congruence Score）**：agent 接受同群体信息 vs 异群体信息的比率差。=1 表示完全偏向同群体，=0 表示无偏见，<0 表示反向偏见
   - **Source Selection Bias**：当可选择信息来源时，agent 选择政治/社会立场相似来源的概率
   - 基线对比：使用 Pew Research Center 的人类民调数据（覆盖美国 10 个政治争议话题）

3. **虚假信息传播模拟**：
   - 在网络中注入标记为虚假的信息（但内容与特定群体信念一致）
   - 追踪虚假信息在多轮传播中的覆盖率和持久性
   - 对比有/无信念一致性偏见时的传播动态

4. **缓解策略**：
   - **接触假说（Contact Hypothesis）**：在 prompt 中引导 agent 在评估信息前先"了解"来源者的个人经历（而非仅看群体标签），模拟跨群体接触
   - **准确性提醒（Accuracy Nudge）**：在信息评估前提醒 agent 关注信息的事实准确性而非来源身份
   - **全球公民身份（Global Citizenship）**：在 system prompt 中强调共同的人类身份，弱化群体边界
   - **综合策略**：同时应用多种缓解措施

## 实验关键数据

### 主实验——信念一致性强度

| 模型/群体 | BC Score | 来源选择偏好 | 同群体选择率 |
|----------|----------|------------|------------|
| 人类（Pew 数据） | 0.20 - 0.62 | — | 55% - 68% |
| GPT-3.5-Turbo | **0.93** | 高 | **89%** |
| GPT-4 | 0.78 | 中高 | 82% |
| GPT-4o | 0.71 | 中 | 78% |
| Claude-3-Sonnet | 0.65 | 中 | 75% |
| Llama-3-70B | 0.82 | 高 | 84% |
| Llama-3-8B | 0.88 | 高 | 87% |

### 虚假信息传播影响

| 条件 | 虚假信息传播率(10轮) | 虚假信息持久性 | 跨群体学习率 |
|------|-------------------|-------------|------------|
| 无 BC 偏见（理想） | 23% | 低 | 61% |
| 人类水平 BC | 41% | 中 | 38% |
| LLM 放大 BC（GPT-3.5） | **67%** | **高** | **12%** |

### 缓解策略效果

| 策略 | BC Score 下降 | 改善幅度 | 跨群体学习提升 |
|------|-------------|---------|-------------|
| 接触假说 | 0.93 → 0.59 | **37%** | +22% |
| 准确性提醒 | 0.93 → 0.83 | 11% | +8% |
| 全球公民身份 | 0.93 → 0.72 | 23% | +15% |
| 综合策略 | 0.93 → 0.48 | **48%** | +31% |

### 关键发现
1. **LLM 放大效应惊人**：GPT-3.5 的 BC Score (0.93) 远超人类最高水平 (0.62)——LLM 不仅复现而且严重放大了人类的信念一致性偏见
2. **规模越大偏见越低但仍高于人类**：GPT-4o (0.71) < GPT-4 (0.78) < GPT-3.5 (0.93)，更大/更新模型的 BC Score 下降，但仍显著高于人类基线
3. **来源选择极度偏向**：LLM agent 在可自由选择信息来源时，70-89% 的时间选择政治立场相似的来源——这意味着如果 LLM 被用于信息推荐系统，回音室效应会被显著放大
4. **虚假信息传播加速**：LLM 放大的 BC 导致虚假信息 10 轮后覆盖率达 67%（vs 人类水平 41%），因为同群体成员无条件接受"自己人"的信息而不验证
5. **接触假说最有效**：37% 的改善幅度显著——让 LLM 在判断前"了解"来源个体的经历能有效打破群体标签的影响，这与人类社会心理学的实证发现一致

## 亮点与洞察
- **社会心理学 × LLM 的跨学科视角**：将群体身份理论（Social Identity Theory）和信念一致性理论（Belief Congruence Theory）系统性地应用到 LLM 评估，打开了一个新的偏见分析维度
- **多智能体社交网络模拟**：不是评估单个 LLM 的单次输出，而是观察多个 LLM agent 在社交网络中的涌现行为——这种"群体层面"的评估范式更接近现实部署场景
- **缓解策略的可操作性**：接触假说可直接转化为 prompt engineering 技巧——在推荐/对话系统中添加"了解对方经历"的提示即可产生效果
- **定量对比人机差异**：BCScore 提供了 LLM vs 人类偏见强度的精确对比，不是简单说"有偏见"而是量化"多严重"

## 局限性 / 可改进方向
- **美国政治中心**：实验主要基于美国自由派/保守派的二元对立，未覆盖欧洲多党制、亚洲政治语境等其他文化背景的信念结构
- **身份注入方式简化**：通过 system prompt 注入身份是粗粒度的——真实的人类身份是多维交叉的（如"拉丁裔保守派女性"），当前框架主要处理单一维度
- **静态网络拓扑**：模拟使用固定网络结构，但真实社交网络是动态演化的——用户会断链、建新链、改变活跃度
- **缺少与真实社交媒体数据的校准**：模拟结果与 Twitter/Reddit 上的真实信息传播模式的对比缺失
- **仅测试英文**：信念一致性在不同语言文化中的表现可能不同

## 相关工作与启发
- **与 LLM Bias 研究的区别**：传统 LLM 偏见研究关注"模型对特定群体的刻板印象"（输出层偏见）；本文关注"模型模拟特定群体时的信息处理偏见"（行为层偏见）——这是更深层的偏见
- **与 LLM Agent 社会模拟的关系**：Generative Agents (Park et al. 2023) 聚焦日常行为模拟，本文聚焦信念动态——可整合为更完整的 LLM 社会模拟框架
- **对 RAG/推荐系统的警示**：如果 LLM 在 RAG 中选择检索源时也有信念一致性偏见，那么检索结果会系统性偏向符合"预期"的信息，导致信息茧房
- **启发**：接触假说的有效性暗示——在 multi-agent debate/deliberation 中，强制跨群体交流可能比简单增加 agent 数量更有效
