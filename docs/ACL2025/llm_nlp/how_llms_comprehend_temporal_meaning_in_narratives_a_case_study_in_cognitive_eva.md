# How LLMs Comprehend Temporal Meaning in Narratives: A Case Study in Cognitive Evaluation of LLMs

**会议**: ACL 2025  
**arXiv**: [2507.14307](https://arxiv.org/abs/2507.14307)  
**代码**: 无  
**领域**: LLM/NLP  
**关键词**: 时态理解, 语法体貌, 认知评估, 叙事理解, 因果推理

## 一句话总结

通过认知语言学中的"体貌"（aspect）实验范式，系统评估LLM是否能像人类一样理解叙事中的时间意义，发现LLM过度依赖原型搭配、体貌判断不一致、缺乏远距因果推理能力，揭示其叙事理解与人类存在本质性差异。

## 研究背景与动机

大型语言模型展现出越来越强的语言能力，但这些行为究竟反映了类人认知还是高级模式匹配，仍是一个开放问题。本文选择语言学中的"体貌"（aspect）作为探针，因为体貌是一个微妙且复杂的语言现象，能同时考察语义理解和语用推理。

**体貌的核心概念**：
- **完成体（perfective）**：如"washed"——表示事件已完成
- **未完成体（imperfective）**：如"was washing"——表示事件正在进行
- 在叙事中，未完成体使事件保持"开放状态"，更容易被保留在工作记忆中，并与后续事件建立因果关联

这种机制在人类认知中已被大量研究验证。本文的核心问题是：LLM是否也具备类似的体貌驱动的认知处理能力？

## 方法详解

### 整体框架

作者开发了一个 **Expert-in-the-Loop探测管线**，包含三个核心阶段：
1. **Prompting**：基于人类认知实验材料构建LLM提示
2. **Prompt Paraphrasing**：引入受控变异保证鲁棒性
3. **Models and Inference**：跨多个LLM运行实验

### 关键设计

1. **叙事刺激材料设计**（来自Schramm 1998的语言学实验）：

    - 每个故事包含两个潜在原因（C1和C2）和一个惊喜结果（Effect）
    - C1的动词在完成体和未完成体之间切换（如"Rob washed the dishes" vs "Rob was washing the dishes"）
    - 当C1为未完成体时，人类更倾向于将C1推断为最终结果的原因
    - 共16个故事，每个有两个版本

2. **Prompt扰动策略**：

    - **通用指令扰动**：按照基于话语的句法变化和语义变化，对指令进行3种改写
    - **数据格式扰动**：基于FormatSpread协议，引入10种格式变体（改变空格、大小写、排序、标点）
    - 共30种prompt变体（3×10），避免结论依赖于特定prompt措辞

3. **三组探测实验**：

    - **实验1：真值判断**——测试语义层面的体貌理解（事件完成状态判断）
    - **实验2：词语补全**——测试认知加工层面的体貌影响（工作记忆中的概念激活）
    - **实验3：开放式因果提问**——测试语用层面的因果推理能力

### 损失函数 / 训练策略

本文不涉及模型训练，而是一项评估研究。使用7个LLM进行推理：
- 开源模型：Gemma2 (9B/27B)、Llama3.1 (8B/70B)、Qwen2 (7B/72B)
- 闭源模型：GPT-4o
- 使用4-bit量化处理大于9B参数的模型
- 统计分析采用多层模型分析（R语言lme4包）

## 实验关键数据

### 主实验

实验1a——真值判断准确率：

| 体貌 | 极性 | 正确答案 | LLM | 人类 |
|------|------|---------|-----|------|
| 完成体 | 正面 | True | 88% | 88% |
| 未完成体 | 负面 | True | **18%** | **71%** |
| 完成体 | 负面 | False | 89% | 93% |
| 未完成体 | 正面 | False | **35%** | **61%** |

实验3——因果推理（选择C1作为原因的比率）：

| 模型 | 未完成体条件 | 完成体条件 |
|------|------------|-----------|
| 人类 | ~68% | ~33% |
| Qwen2-72B | ~60% | ~10% |
| Llama3.1-70B | ~55% | ~8% |
| GPT-4o | ~50% | ~12% |
| 小模型（平均） | ~35% | ~8% |

### 消融实验

| 配置 | 关键指标 | 说明 |
|------|---------|------|
| 有叙事上下文 vs 无叙事上下文 | 未完成体准确率变化不一致 | 不同模型表现方向不同，排除了仅是叙事结构干扰的可能 |
| 词语补全Near C1 vs Near Effect | 匹配频率下降33% | LLM缺乏远距工作记忆保持能力 |
| 大模型 vs 小模型 | 因果推理更接近人类 | 仅在实验3中观察到明显的scaling效果 |

### 关键发现

1. **LLM在原型条件下表现良好，但在非原型条件下严重失败**：完成体（原型搭配）的准确率88-89%与人类持平，但未完成体（非原型搭配）的准确率仅18%，远低于人类的71%
2. **LLM过度依赖原型性（prototypicality）**：因为完成体+完成类事件是叙事中最常见的搭配，LLM倾向于默认事件已完成
3. **LLM缺乏远距因果推理能力**：词语补全实验中，当探测位置从C1附近移到Effect附近时，LLM的激活大幅下降，说明无法像人类那样在工作记忆中保持事件
4. **声明式知识vs隐式理解的分离**：LLM能准确阐述体貌的学术定义，却无法在实际判断中正确应用
5. **没有一个模型在所有实验中维持人类水平的表现**，模型在不同任务上的表现不一致
6. 完成体条件下，人类有约1/3会选C1为原因，但LLM几乎不会——说明人类有更灵活的叙事解读能力

## 亮点与洞察

- **研究设计极为精巧**：直接复用认知语言学已验证的实验材料，确保与人类数据的可比性
- **Expert-in-the-Loop管线是重要的方法论贡献**：30种prompt变体避免了prompt依赖的伪发现，可复用于其他认知评估研究
- **三层递进的实验设计**：从语义（真值判断）到认知加工（词语补全/工作记忆）到语用（因果推理），层层深入
- **"声明式知识vs隐式理解"的洞察深刻**：类比于人类语言学习者（能背规则但不能正确使用），揭示了LLM能力的本质局限
- **对prototypicality依赖的发现**：与Second Language Acquisition领域的发现高度一致，LLM类似于语言学习中级阶段的表现

## 局限性 / 可改进方向

1. 依赖LLM的"自我报告"式响应，无法直接观察内部处理机制
2. 在受控实验条件下评估，实际对话中可能表现不同
3. 仅测试英语叙事中的体貌现象，其他语言的体貌系统差异很大
4. 16个故事的规模较小，某些统计结论的power可能不足
5. 虽然使用了prompt扰动，但不同模型的最优交互方式可能截然不同

## 相关工作与启发

- 本文是"用认知科学方法评估LLM"这一新兴方向的高质量范例
- 与Ivanova (2025)提出的LLM认知评估框架一致
- Roberts et al. (2024)研究LLM中的Fan Effect，本文研究的是更微妙的体貌效应
- 对LLM的"理解之墙"提供了新的证据：即使统计上学到了大量语言模式，某些需要概念性理解的任务仍然失败
- 启示：未来需要在预训练或对齐阶段引入更丰富的时间/因果推理信号
