# Psycholinguistic Word Features: A New Approach for the Evaluation of LLMs Alignment with Humans

**会议**: ACL 2025  
**arXiv**: [2506.22439](https://arxiv.org/abs/2506.22439)  
**代码**: https://zenodo.org/records/14866800  
**领域**: LLM NLP / LLM评估  
**关键词**: 心理语言学, LLM评估, 人类对齐, 词汇特征, 具身认知

## 一句话总结
首次系统提出使用心理语言学词汇规范（Glasgow和Lancaster数据集共13种词汇特征）来评估LLM与人类的对齐程度，发现LLM在情感/概念特征上对齐较好，但在感知觉特征（味觉、嗅觉、触觉等）上对齐极差，揭示了LLM缺乏具身认知的根本局限。

## 研究背景与动机
LLM评估目前主要聚焦于任务性能——数学、推理、问答等可客观度量的能力。但LLM与人类交互、生成被人类阅读的文本，因此我们更希望LLM能在情感、感知和偏好上与人类对齐。

**现有评估方法的痛点**：
- 人工评估不可扩展（每个模型需上万条评估）
- LLM-as-Judge受限于评判模型自身的偏差
- Chatbot Arena依赖社区众包，无法控制实验
- 现有基准测试侧重任务完成，忽略语言的深层语义属性

**核心洞察**：心理语言学已经积累了数十年的人类评分数据——对成千上万个单词在arousal（唤醒度）、valence（效价）、concreteness（具体性）、感知觉强度等维度的评分。这些数据集现成可用、规模大、经过验证，完全可以复用来评估LLM与人类的语义对齐。

**更深层动机**：语言的意义不仅仅来自词语的共现（LLM的学习方式），还源于感知经验、身体交互、情感体验等——即具身认知理论所强调的。因此，评估LLM在感知维度上的对齐程度，可以定量揭示"纯文本学习"的根本局限。

## 方法详解

### 整体框架
使用现有的心理语言学词汇评分数据集作为benchmark，让LLM对相同词汇进行评分，然后计算LLM评分与人类评分的相关性来衡量对齐程度。

### 关键设计

1. **数据集选择**：
   - **Glasgow规范**：5,553个英语单词在7个特征上的人类评分——arousal（唤醒度）、valence（效价）、dominance（支配度）、concreteness（具体性）、imageability（可想象性）、familiarity（熟悉度）、gender（性别关联）
   - **Lancaster规范**：39,707个英语单词在6种感知觉模态上的评分——touch（触觉）、hearing（听觉）、smell（嗅觉）、taste（味觉）、vision（视觉）、interoception（内感受）
   - 选择原理：Glasgow涵盖已知对齐较好的特征（验证方法有效性），Lancaster聚焦感知特征（预期对齐较差，检验具身认知假设）

2. **LLM评分获取**：
   - 使用与人类实验**完全相同的指示语**作为prompt，末尾加一句"请只回答数字"
   - 温度设为0保证可复现
   - 两种估计方式：(a) 直接取LLM输出数字；(b) 提取评分量表上每个值的概率，计算加权平均——后者效果更好，因为利用了概率分布信息
   - 设计动机：复用原始人类实验的prompt可以最大程度保证对齐评估的公平性

3. **评估指标体系**：
   - Pearson相关系数：衡量线性关系，更重视远离均值的观测
   - Spearman相关系数：衡量单调关系，对整个分布等权重处理
   - 对原始数据和四舍五入后的整数数据分别计算两种系数（共4个指标）
   - 设计动机：不同相关系数各有偏重（如感知觉特征分布偏态严重时差异显著），同时计算可避免误导性结论

### 损失函数 / 训练策略
本文不涉及模型训练，是纯评估工作。创新点在于评估方法论的设计——如何系统地将心理语言学数据集转化为LLM对齐评估工具。

## 实验关键数据

### 主实验（Glasgow规范 - 7种特征对齐）
| 特征 | GPT-4o | GPT-4o-mini | LLama-3.2-3B | 整体趋势 |
|------|--------|-------------|--------------|---------|
| Arousal | 高相关 | 高相关 | 中等相关 | 整体对齐较好 |
| Valence | 高相关 | 高相关 | 中等相关 | 整体对齐较好 |
| Concreteness | 高相关 | 高相关 | 中等相关 | 整体对齐较好 |
| Imageability | 高相关 | 高相关 | 中等相关 | 对齐较好 |
| Familiarity | 高相关 | 高相关 | 较低 | 模型差异大 |
| Gender | 中等 | 中等 | Gemma-2最好 | 对齐一般 |

### Lancaster规范（感知觉特征 - 核心发现）
| 特征 | GPT-4o | 其他模型 | 说明 |
|------|--------|---------|------|
| Olfactory（嗅觉） | 略好但仍低 | 很低 | 相对最好但绝对值差 |
| Gustatory（味觉） | 低 | 极低 | Gemma对"Lemon"只给0.01 |
| Haptic（触觉） | 低 | 低 | 感知模态普遍差 |
| Auditory（听觉） | 低 | 低 | — |
| Visual（视觉） | 低 | 低 | 多模态模型也没优势 |
| Interoceptive（内感受） | 低 | 低 | — |

### 消融实验
| 分析维度 | 关键发现 |
|---------|---------|
| 模型规模 | LLama系列：3B < 8B < 11B，规模有帮助但非决定性 |
| 多模态 | LLama3.2-11B（多模态）在视觉特征上并不比纯文本模型更好 |
| Pearson vs Spearman | 在偏态分布（味觉/嗅觉）上差异大，Pearson更关注高值区 |
| 不同模型家族 | GPT-4o系列整体最优，开源模型在部分特征上也有竞争力 |

### 关键发现
- Glasgow规范（情感/概念特征）对齐尚可，但距理想区间（0.8-1.0）仍有明显差距
- Lancaster规范（感知觉特征）对齐极差，证实了LLM在缺乏具身认知下的根本局限
- 多模态模型在视觉相关特征上并未展现出预期优势
- 同一个词（如"bicycle"），LLama-3.2-3B给出的具体性评分几乎无区分（4.73 vs 4.50），而GPT-4o能给出比人类更极端的区分（7 vs 2.96）

## 亮点与洞察
- **方法论创新**：不是做新任务或新模型，而是提出了一种全新的LLM评估视角——复用心理语言学几十年的积累，这是跨学科融合的典范
- **理论深度**：将实验结果与符号锚定问题（symbol grounding problem）、具身认知理论紧密关联，不只报分数，还解释"为什么"
- **实用价值**：提供了一种可自动化、可扩展的对齐评估方法，可以纳入标准LLM领导力排行榜
- **关于加权概率估计**：从logprobs计算加权平均比直接取输出数字更准确，这个技巧对任何需要LLM做Likert量表评分的场景都有启发

## 局限性 / 可改进方向
- 仅使用两个英语数据集，缺乏多语言覆盖（翻译可能引入偏差）
- LLM数量有限，应扩展到更多模型（包括最新的GPT-5等）
- 评估指标仅使用相关系数，可探索更精细的对齐度量
- 未分析对齐差异的原因（如训练数据中感知觉相关文本的比例）
- 未探索改善对齐的方法（如用合成感知觉描述做后训练）

## 相关工作与启发
- **Trott (2024); Martínez et al. (2025)** 已证明GPT-4在valence和concreteness上与人类高度相关，但那些工作是"用LLM生成心理语言学数据"；本文反过来"用心理语言学数据评估LLM"
- **具身认知与LLM**：Borghi et al. (2024) 从认知科学角度讨论了LLM时代语言作为认知工具的角色，本文提供了定量证据
- **启发方向**：如果感知觉对齐差距来源于缺乏身体经验，那么多模态训练（视频、音频、触觉传感器数据）是否能弥补？这为未来的多模态LLM训练提供了可测量的目标
