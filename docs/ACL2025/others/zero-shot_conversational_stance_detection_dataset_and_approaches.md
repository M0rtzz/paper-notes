# Zero-Shot Conversational Stance Detection: Dataset and Approaches

**会议**: ACL 2025  
**arXiv**: [2506.17693](https://arxiv.org/abs/2506.17693)  
**代码**: 有 (https://github.com/whu-yzding/ZS-CSD)  
**领域**: NLP / 立场检测  
**关键词**: 立场检测, 对话理解, 零样本, 说话者交互, 原型对比学习

## 一句话总结

构建了首个零样本多轮多方对话立场检测数据集 ZS-CSD（280 个目标、17k+ 样本），并提出 SITPCL 模型，通过说话者交互建模和目标感知原型对比学习实现了 SOTA 性能（F1-macro 43.81%）。

## 研究背景与动机

立场检测旨在识别文本对特定目标的观点倾向（支持/反对/中立）。现有研究主要分为目标内（in-target）、跨目标（cross-target）和零样本三类。然而，现有工作存在三个关键局限：

1. **目标数量有限**：最大的对话立场检测数据集 MT-CSD 仅包含 5 个目标，难以覆盖真实世界中大量未见过的目标
2. **忽略说话者信息**：此前的标注和建模过程仅考虑回复关系和历史上下文，未充分利用说话者上下文和说话者间交互信息
3. **缺乏零样本设置**：对话立场检测研究局限于目标内和跨目标两种任务，未探索更贴近实际的零样本场景

## 方法详解

### 整体框架

SITPCL（Speaker Interaction and Target-aware Prototypical Contrastive Learning）包含四个核心模块：
1. 预训练语言模型 + GRU 获取上下文表示
2. 说话者交互编码器捕捉说话者内/间依赖
3. 目标感知原型对比学习增强表示
4. 分类器进行立场预测

### 关键设计

1. **数据集构建（ZS-CSD）**：
   - 数据源：微博（中国最大社交平台），通过 API 收集 200 万帖子及评论
   - 目标选择：三名专家标注者为每段对话确定 1-2 个目标，包含名词短语型目标（113 个）和声明型目标（167 个）
   - 立场标注：五名标注者结合对话历史、说话者上下文和说话者交互信息标注立场（支持/反对/中立）
   - Cohen's Kappa = 0.83，标注质量高
   - 零样本划分：训练/验证/测试集的目标完全不重叠

2. **话语编码器**：使用中文 RoBERTa 分别编码每条话语-目标对，添加模板 "s_i expresses [mask] towards target" 隐式表达立场，经 GRU 层获取上下文表示。

3. **说话者交互编码器**：
   - **说话者内依赖建模**：利用同一说话者的历史增强表示和当前话语表示构建查询向量，通过注意力机制聚合该说话者的先前上下文，捕捉说话者观点的连贯性
   - **说话者间依赖建模**：利用其他说话者的先前状态信息作为键向量，通过注意力机制捕捉说话者之间的互动关系
   - 将两种状态向量拼接后映射为最终说话者增强表示

4. **目标感知原型对比学习**：
   - 计算每个目标的原型表示 p_t（该目标下所有话语表示的平均）
   - 对比损失将话语表示拉近其对应目标原型，推远其他目标原型
   - 使不同目标的表示空间逐步分离，提升零样本泛化能力

### 损失函数 / 训练策略

总损失 = 交叉熵分类损失 L_CE + γ × 目标感知原型对比损失 L_TPC

- L_CE：标准交叉熵，用于立场分类
- L_TPC：InfoNCE 风格的对比损失，以目标原型为锚点
- 优化器：AdamW，学习率 1e-5，权重衰减 1e-6
- 训练 20 epochs，batch size 16，5 个随机种子取平均

## 实验关键数据

### 主实验（零样本对话立场检测 F1-macro）

| 方法 | 混合目标 | 名词短语目标 | 声明目标 |
|------|---------|------------|---------|
| Llama3-8B | 35.91 | 39.37 | 33.79 |
| GPT4o-mini | 38.16 | 40.21 | 36.57 |
| Llama3-70B | 41.07 | 48.72 | 36.56 |
| Qwen2.5-14B | 42.78 | 44.60 | 41.34 |
| RoBERTa | 40.27 | 42.79 | 38.71 |
| GLAN | 41.79 | 45.04 | 39.83 |
| **SITPCL (ours)** | **43.81** | **47.54** | **41.47** |

### 消融实验

| 方法 | 混合目标 F1-macro |
|------|-----------------|
| SITPCL (完整) | **43.81** |
| W/o 说话者交互编码器 | 42.91 (-0.90) |
| W/o 目标原型对比学习 | 42.59 (-1.22) |
| W/o 两者 | 41.40 (-2.41) |

### 关键发现

1. SITPCL 在所有设置上超越所有基线，包括 LLM 零样本方法和微调方法
2. 但 43.81% 的 F1-macro 仍然较低，凸显零样本对话立场检测的巨大挑战
3. 名词短语目标比声明目标更容易检测（47.54 vs 41.47），可能因为名词短语语义更明确
4. LLM 在零样本提示下的表现不如微调模型，说明对话立场检测需要专门优化
5. 消融实验证明说话者交互和目标对比学习两个模块都有贡献，其中后者贡献更大

## 亮点与洞察

- 同时覆盖名词短语和声明两种目标类型，比现有数据集更全面
- 标注过程中显式考虑说话者上下文和说话者交互信息，是方法论上的重要改进
- 目标感知原型对比学习通过目标级别的表示分离来增强零样本泛化，思路简洁有效
- 低 F1-macro（43.81%）诚实地揭示了任务的难度，为后续研究指明了方向

## 局限性 / 可改进方向

1. 仅覆盖中文微博数据，缺乏多语言和跨平台验证
2. 43.81% 的 F1-macro 说明模型仍有很大提升空间
3. 说话者交互编码器的设计相对简单（单层注意力），可探索更复杂的图神经网络
4. 未考虑多模态信息（如表情包、图片），而微博中这类信息很常见
5. 目标原型的计算基于 mini-batch，可能受批次采样影响

## 相关工作与启发

- VAST (Allaway & McKeown, 2020) 是句子级零样本立场检测的代表，本工作将零样本扩展到对话级
- MT-CSD (Niu et al., 2024) 和 MmMtCSD 关注目标内和跨目标设置，本工作首次定义对话级零样本任务
- 原型对比学习的灵感可能来自 few-shot 学习中的原型网络
