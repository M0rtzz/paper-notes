# VAQUUM: Are Vague Quantifiers Grounded in Visual Data?

**会议**: ACL 2025  
**arXiv**: [2502.11874](https://arxiv.org/abs/2502.11874)  
**代码**: 有 (https://github.com/hughmee/vaquum)  
**领域**: NLP / 视觉语言模型  
**关键词**: 模糊量词, 视觉语言模型, 人类判断, 视觉基础, 数据集

## 一句话总结

构建 VAQUUM 数据集（20300 条人类评分），系统评估视觉语言模型在视觉场景中对模糊量词（few, many 等）理解和使用的能力，发现 VLM 的生成概率与人类判断存在一定对齐但在不同评估范式间表现不一致。

## 研究背景与动机

模糊量词（如 "a few"、"many"）在日常对话中无处不在，其使用受多种上下文因素影响（物体数量、大小、说话者预期等）。尽管在形式语义学和心理语言学中被广泛研究，但在 NLP 特别是多模态视觉场景中几乎未被关注。

已有工作对 VLM 量词能力的评估主要集中在精确量词（none, all），或将模糊量词用固定比例阈值定义（如 "few" = <17%），这消除了其本质的模糊性。本文通过收集真人连续尺度评分并用多种评估方法考察 VLM，提供更全面的分析。

## 方法详解

### 整体框架

三层评估设计：（1）收集人类对量化陈述的适当性评分；（2）提取 VLM 对同一陈述的生成概率（Exp1）；（3）提示 VLM 生成评分（Exp2）；（4）多选 QA 设置中提取标签概率（Exp3）。

### 关键设计

1. **VAQUUM 数据集构建**：从 FSC-133（物体计数数据集，count 7-100）和 TallyQA（count 2-15）采样 1089 张图片，均匀分布在 33 个计数区间。记录三种物体特征：数量（count）、分割面积（CLIPSeg 估计）、尺寸规范（THINGSplus 数据库的真实世界大小）。

2. **人类评分收集**：203 名英语母语参与者通过 Prolific 招募，每人评 100 题。对 "There are [QUANT] [OBJECT] in the image" 格式的陈述用连续滑块评分（完全不准确→完全准确），量词包括 few / a few / some / many / a lot of + 无量词对照。

3. **线性混合效应模型（LMM）分析**：预测变量为量词类型、数量、分割面积、尺寸规范，随机效应为参与者和物体类别，分析各因素对人类判断和模型预测的影响。

4. **三种 VLM 评估范式**：

    - Exp1 (生成概率)：提示 "How would you describe the amount of..."，提取各陈述的条件对数概率
    - Exp2 (生成评分)：直接提示模型在 0-100 范围打分
    - Exp3 (多选 QA)：所有陈述作为选项，提取标签概率

## 实验关键数据

### 人类评分 — LMM 固定效应估计

| 量词 | Count 交互 | 分割面积交互 | 尺寸规范交互 |
|------|-----------|------------|------------|
| few | -0.37 | -0.07 | -0.13 |
| a few | -0.38 | -0.10 | -0.11 |
| many | +0.38 | +0.08 | +0.14 |
| a lot of | +0.42 | +0.06 | +0.17 |

### VLM 与人类评分的 Pearson 相关（Exp1 生成概率）

| 模型 | few | a few | many | a lot of |
|------|-----|-------|------|----------|
| BLIP-2 | -0.18 | -0.19 | 0.14 | 0.13 |
| InstructBLIP | 0.06 | 0.04 | -0.01 | -0.04 |
| LLaVA-NeXT | **0.34** | **0.39** | **0.43** | **0.52** |
| LLaVA-OneVision | **0.30** | **0.40** | **0.52** | **0.54** |
| Molmo | 0.16 | 0.20 | -0.17 | -0.21 |

### 关键发现

1. **数量是最强预测因子**：无论人类还是 VLM，物体数量对量词判断的影响远大于分割面积和尺寸规范。
2. **LLaVA 系列与人类最对齐**：LLaVA-NeXT 和 LLaVA-OneVision 在生成概率层面与人类判断相关性最强。
3. **评估范式间不一致**：VLM 在 Exp1（生成概率）和 Exp2（生成评分）中表现不一致——生成评分时模型倾向于极端值（0 或 100），且 "some" 被视为存在量词而非模糊量词。
4. **InstructBLIP 和 Molmo 无法区分量词**：这两个模型对所有量词给出近乎相同的概率分布。
5. **多选设置（Exp3）下 Molmo 表现改善**：说明评估方式对结论影响巨大。

## 亮点与洞察

- 巧妙连接了心理语言学中的"近似数系统"理论与 VLM 的计数能力，VLM 的模糊量词使用可能也依赖类似的近似数量估计。
- 三种范式的系统比较揭示了：VLM 的生成行为（token 概率）与其元语言判断能力（评分）是两个不同的过程，可能受不同训练阶段的影响。
- 数据集可用于 NLP 透视主义（perspectivism）研究——标注者之间的分歧不是噪声而是模糊性的本质反映。

## 局限性 / 可改进方向

- 仅测试了 5 种 VLM，未覆盖最新的 GPT-4V、Gemini 等闭源模型。
- 分割面积和尺寸规范的分布可能不够均匀，导致对这两个因素的分析说服力不足。
- 使用 CLIPSeg 估计分割面积可能引入噪声。
- 未考虑场景语义（如"20个人在会议上" vs "20个幼儿在会议上"的量词差异）。

## 相关工作与启发

- 与 Testoni et al. (2024) 的工作互补：后者用合成图像+选择题评估，本文用真实图像+连续评分。
- 可结合常识知识和场景理解进一步研究上下文对量词使用的影响。

## 补充细节

### 近似数系统的类比

认知科学中的近似数系统（ANS）理论认为人类对数量有快速但粗略的感知能力（subitizing 阈值约 4）。而 VLM 虽然在精确计数上表现差，却在模糊量词使用上呈现合理模式——暗示它们可能也依赖某种"近似"数量表征。

### Prompt 的关键影响

将提示从 "How would you describe the amount of [OBJECT]?" 改为 "How would you describe the image?" 后，模型的量词区分能力基本消失。说明 VLM 的量词行为高度依赖 prompt 是否明确指向"数量"属性。

### some 的存在量词解读

在 Exp2 中 some 被模型视为始终适用——可能因为 "There are some apples" 被解读为存在性断言而非模糊数量量词，揭示了语义歧义对评估的影响。

### LMM 统计分析的深度

人类判断的 LMM 解释了 50.3% 方差——剩余 49.7% 由本文未考虑的因素决定（如场景语义、物品功能、说话者期望等），为后续研究明确了方向。
