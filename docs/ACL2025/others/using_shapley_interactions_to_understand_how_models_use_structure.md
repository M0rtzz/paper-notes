# Using Shapley Interactions to Understand How Models Use Structure

**会议**: ACL 2025  
**arXiv**: [2403.13106](https://arxiv.org/abs/2403.13106)  
**代码**: 无  
**领域**: NLP / 可解释性  
**关键词**: Shapley交互, 句法结构, 多词表达, 语音模型, 非线性表征

## 一句话总结

利用 Shapley Taylor 交互指数（STII）分析语言模型和语音模型如何在内部表征中编码输入的结构关系，发现自回归模型在句法邻近度、非组合语义和语音协同发音方面均展现出有意义的非线性交互模式。

## 研究背景与动机

特征归因方法（如 Shapley 值）是理解神经网络决策的重要工具，但其假设特征独立线性可加，忽略了特征之间的非线性交互。对于高度结构化的语言数据来说，这意味着丢失了模型如何编码句法依赖、语义组合等关键信息。

已有工作在 LSTM 等旧架构上初步探索了 Shapley 交互，但仅限于简单分类任务。本文将这一方法扩展到现代 Transformer（GPT-2、BERT），并首次系统性地将 Shapley 交互与三种不同层面的语言结构（句法、语义、语音）关联分析。

## 方法详解

### 整体框架

使用 Shapley Taylor 交互指数（STII）测量成对输入特征的非线性交互强度，然后将交互模式与已知的语言结构特征进行相关性分析。

### 关键设计

1. **STII 计算**：对特征集 A 和 B，计算二阶离散导数：STII(A,B) = ||φ(∅) - φ(A) - φ(B) + φ(A,B)||₂ / ||φ(∅)||₂。用 Monte Carlo 排列采样近似 Shapley 值（避免 O(2^N) 的精确计算）。

2. **位置基线控制**：定义两种距离——交互对距离 d_i（两交互 token 间距）和预测距离 d_p（交互对到预测 token 的距离）。在后续实验中按这两种距离分层，以消除位置效应的混淆。

3. **句法结构实验**：用 spaCy 解析依存树，计算句法距离（树上路径边数），按位置距离分层后检验句法距离与 STII 的 Spearman 相关性。

4. **多词表达实验**：使用 AMALGrAM 标注器识别强/弱多词表达（MWE），比较 MWE 内 token 对与任意 token 对的 STII 差异。

5. **语音模型实验**：用 Wav2Vec 2.0，以蒙特利尔自动对齐器标注音素边界，比较辅音-元音边界和辅音-辅音边界的特征交互差异。

## 实验关键数据

### 文本模型结果

| 实验 | GPT-2 (自回归) | BERT (遮蔽) |
|------|--------------|-----------|
| 位置效应 | STII 随距离单调递减 ✓ | STII 随距离单调递减 ✓ |
| 句法距离相关性 | 显著负相关（句法越近，交互越强）| 正负相关混合，不一致 |
| MWE 交互增强 | 强 MWE > 弱 MWE > 一般对 ✓ | 强 MWE > 弱 MWE > 一般对 ✓ |

### 语音模型结果

| 比较类型 | 平均 STII |
|---------|----------|
| 辅音-元音边界 | **显著更高** |
| 辅音-辅音边界 | 较低 |
| 高响度辅音（近似音等） | 更高（类似元音） |
| 低响度辅音（塞音等） | 较低 |

### 关键发现

1. **自回归模型编码句法的方式不同于遮蔽模型**：GPT-2 中句法距离与 STII 一致负相关，而 BERT 不一致，说明自回归目标更倾向于学习句法关系。
2. **语义组合性反映在非线性交互中**：非组合语义的强 MWE（如 kick the bucket）比弱 MWE 的交互更强，且该效应在两种模型中都成立。
3. **语音模型捕获了协同发音现象**：辅音-元音交互强于辅音-辅音交互，符合语音学中元音受邻近辅音影响更大的规律。
4. **辅音的发音方式影响交互强度**：近似音等类元音辅音的交互强度更高，印证了响度层级的语音学预测。

## 亮点与洞察

- 跨模态（文本+语音）、跨任务（生成+识别）的统一分析框架，使 Shapley 交互成为理解模型内部结构编码的通用工具。
- 揭示了自回归与遮蔽模型在句法编码上的本质差异，提供了可能的可解释性新维度。
- 语音实验以优雅的方式连接了语音学知识和模型内部表征。

## 局限性 / 可改进方向

- 仅分析了 GPT-2 和 BERT-base 等较小规模模型，结论可能不适用于当前生产级大模型。
- 仅考虑成对交互，未探索更高阶交互（如三元组）对应的层级结构。
- 相关性分析而非因果分析，无法确定 Shapley 交互是否因果性地影响模型行为。

## 相关工作与启发

- 与线性探针（如 structural probe）互补：线性探针检测可线性提取的结构信息，STII 检测非线性编码的结构信息。
- 可与因果干预方法（如 causal abstraction）结合以获得更强的因果性结论。

## 补充细节

### 预测距离的 bag-of-words 效应

STII 随预测距离 d_p 增大急剧下降，意味着模型在预测某个 token 时，对远距上下文几乎以 bag-of-words 方式处理——不再编码复杂句法关系。这与 Khandelwal et al. 2018 的 "Sharp Nearby, Fuzzy Far Away" 一致。

### 自回归 vs 遮蔽模型的深层差异

BERT 在句法实验中正负相关混合并非噪声，可能反映遮蔽模型用位置信息补偿句法距离信号，导致两者纠缠难以分离。这与 Ahuja et al. (2024) 的发现吻合：自回归模型更倾向句法风格的泛化。

### 辅音图谱分析

用国际音标辅音图作为热力图 layout，按发音方式（行）和发音位置（列）组织 STII 强度，使语音学规律直观呈现。浊音-清音对中大多数浊音 STII 更高，验证了声带振动增加交互强度的假说。

### 方法论意义

本文呼应了 Bitter Lesson 的一个推论：研究者应利用领域知识指导可解释性分析（而非避免它），数据的结构决定了有意义分析的方向。
