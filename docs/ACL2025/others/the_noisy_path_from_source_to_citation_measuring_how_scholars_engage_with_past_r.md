# The Noisy Path from Source to Citation: Measuring How Scholars Engage with Past Research

**会议**: ACL 2025  
**arXiv**: [2502.20581](https://arxiv.org/abs/2502.20581)  
**代码**: 有 (https://github.com/hongcchen/citation)  
**领域**: NLP / 科学计量学  
**关键词**: 引用保真度, 知识传播, 电话效应, 引用行为, 计量分析

## 一句话总结

本文构建了一个大规模计算管道来量化引用保真度（citation fidelity），基于 1300 万引用句对分析发现：引用保真度与时间/学科接近度、开放获取和团队规模等因素显著相关，并通过准实验证实了中介引用导致信息失真的"电话效应"。

## 研究背景与动机

学术引用被广泛用于评估研究和追踪知识流动，但这种使用通常依赖原始引用计数，**隐含假设所有引用是同质的**。实际上，引用可以在忠实度上有很大差异——原始知识可能被改述、概括或重新解读，甚至被错误引用。

一个经典的负面案例：1980 年发表的一封简短信件（Porter and Jick）记录了住院患者中的阿片类药物使用情况，但后续被不加批判地引用为阿片类处方安全的证据——到 2017 年被引用超过 600 次，不到 20% 的引用提及了原始研究的关键背景限制。这些引用共同构成了一个误导性的低成瘾风险叙事，在阿片类危机中扮演了角色。

**难以大规模研究引用保真度**是核心障碍：之前的研究通常依赖劳动密集的人工验证，规模有限。本文的目标是构建自动化管道解决这一瓶颈。

## 方法详解

### 整体框架

管道包含四个核心步骤：
1. 从施引论文中识别**报告性引用句**（reporting citations）
2. 从被引论文中识别**对应声明句**（corresponding claims）
3. 进行句子级**匹配**
4. 用科学信息变化度量评估**保真度**

### 关键设计

1. **报告性引用提取**
   - 限制为单句引用（83% 的引用上下文出现在单句内）
   - 仅选择括号类引用（方括号或圆括号在句末），排除叙述性引用
   - 使用微调的 SciBERT 模型进行背景引用分类（F1=0.81），筛选报告性引用
   - 训练数据：SciCite + MultiCite 联合（27,052 实例，43% 背景类）

2. **被引论文声明提取**
   - 使用微调的 RoBERTa 句子分类器标注被引论文全文中的"结果"和"结论"句子
   - 训练数据：PubMed 20 万篇摘要的自标注分类（F1=0.92）
   - 筛选：约 30% 的句子被标记为结果/结论

3. **保真度度量**
   - 使用 Wright et al. (2022) 的**科学信息变化度量**（基于 MPNet 微调）
   - 评分 1-5：1=完全不同的声明，5=完全相同的声明
   - 该度量专为科学文本设计，捕捉抽象层级、不确定性和上下文适当性
   - MSE=0.489, Pearson=76.48

4. **上界方法（Upper-bound approach）**
   - 问题：引用通常指向整篇论文而非特定句子
   - 解决：取引用句与被引论文所有声明句中**最高保真度分数**作为该引用对的保真度
   - 动机：近似"最佳情况"保真度，确保捕获最相关的声明

5. **"电话效应"准实验**
   - 设计：找到约 5 万对论文 (C, D)，二者引用同一原始论文 A 的同一声明
   - 处理组：论文 C 同时引用 A 和中介论文 B（B 也引用 A）
   - 对照组：论文 D 仅引用 A
   - 匹配：C 和 D 按出版年份和领域精确匹配
   - 衡量：处理组和对照组对 A 的引用保真度差异

## 实验关键数据

### 数据规模

| 项目 | 数值 |
|------|------|
| 总论文数 | ~4200 万（S2ORC） |
| 过滤后英文全文论文 | ~4200 万 |
| 最终引用句对 | ~1300 万 |
| 保真度分布 | 近似正态，峰值在 ~3.5 |

### 引用保真度影响因素分析

| 因素 | 方向 | 解释 |
|------|------|------|
| 同领域 vs 跨领域 | 同领域**更高** | 熟悉方法论和术语有助于准确转述 |
| 自引 vs 非自引 | 自引**更高** | 作者对自己的工作最为了解 |
| 开放获取 vs 非开放 | OA**更高** | 全文可及促进直接参与 |
| 发表时间间隔 | 间隔**越短越高** | 近期工作更具上下文相关性 |
| 第一作者资历 | H-index**越高越低** | 资深作者可能更依赖经验而非直接阅读 |
| 末位作者资历 | 无显著关系 | 末位作者通常为指导角色，不负责引用细节 |
| 团队规模 | **中等规模最高** | 小团队缺乏多样性，大团队存在协调成本 |

### "电话效应"核心结果

| 组别 | 对 A 的引用保真度 |
|------|------------------|
| 对照组（仅引A） | 基准 |
| 处理组（引A+B） | **低 0.06** |
| 处理组-B高保真 | 较接近对照组 |
| 处理组-B低保真 | 显著低于对照组 |

→ **中介引用的保真度与后续引用的保真度正相关**，证实了信息失真的"级联效应"。

### 关键发现

1. **引用保真度分布集中在 3.5 左右**，多数引用具有合理的保真度，但存在显著的两端变异
2. **所有假设均得到验证**：时间/学科接近度、可及性、自引均正向影响保真度
3. **电话效应实证确立**：中介引用导致对原始声明的保真度降低 0.06（在 5 分制上是有意义的差异）
4. **第一作者 vs 末位作者的不对称性**：反映了学术团队中不同角色的不同责任
5. **中等规模团队最优**——4-6 人团队在引用保真度上表现最好

## 亮点与洞察

1. **管道设计精巧且可扩展**：将全文级引用分析降维为句子级匹配+评分，实现了 1300 万级别的规模
2. **准实验设计严谨**：通过年份和领域的精确匹配控制混杂因素，使因果推断更可信
3. **电话效应的实证意义深远**：首次以数据驱动的方式证明了引用链中的信息损失
4. **实际启示丰富**：对学术评估、知识图谱构建和科学信息素养教育均有价值

## 局限性 / 可改进方向

- **单句限制**：仅考虑单句引用，实际中引用可跨多句；仅匹配单句对，忽略了跨句对应关系
- **上界近似**：最高分数近似可能无法准确定位作者真正参考的句子
- **保真度度量的聚合性**：分数综合了主题对齐、泛化程度和事实准确性，难以区分不同失真类型
- **因果性有限**："引用中介论文"≠"受中介论文影响"，引用行为复杂
- **英语中心**：仅分析英文论文，不同语言和学科的引用文化可能不同
- 未区分"可接受的改述"与"实质性错误引用"

## 相关工作与启发

- Simkin and Roychowdhury (2005) 通过参考文献错误传播估计 70-90% 的引用是从他人参考列表复制的
- Greenberg (2009) 展示了假设如何通过肯定性引用链转变为"accepted facts"
- Cobb et al. (2024) 发现心理学顶刊约 19% 的引用遗漏关键信息或完全错误引用
- Wright et al. (2022) 的科学信息变化度量是本文保真度评估的基础
