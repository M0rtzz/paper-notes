# Mitigating Confounding in Speech-Based Dementia Detection through Weight Masking

**会议**: ACL 2025  
**arXiv**: [2506.05610](https://arxiv.org/abs/2506.05610)  
**代码**: [有 (GitHub)](https://github.com/LinguisticAnomalies/DualFilter.git)  
**领域**: NLP / 医疗AI / 公平性  
**关键词**: 痴呆检测, 混淆偏差, 权重掩码, Transformer去偏, 性别偏差

## 一句话总结

本文针对基于语音转录文本的痴呆检测任务中的**性别混淆偏差**问题，提出两种权重掩码方法（Extended Confounding Filter 和 Dual Filter），通过识别并消除 Transformer 网络中与性别关联的权重来实现去混淆，在保持痴呆检测性能的同时显著降低性别间的公平性差距。

## 研究背景与动机

Transformer 模型在痴呆检测等临床任务中表现良好，但隐藏着**混淆偏差**（confounding bias）问题。在语音数据集中，性别既影响语言模式（男女完成同一任务时的语言风格不同），也与痴呆发病率相关联。模型可能学到性别特异性语言线索而非真正的认知衰退标志，导致对不同性别的预测表现不均。

作者的实验首先验证了这一偏差的存在：在 DementiaBank 和 CCC 两个数据集上，微调后的 BERT 模型在男性和女性样本间展现出显著的性能差异（Mann-Whitney-Wilcoxon 检验 p < 0.001），且这种差异在平衡性别分布后依然存在。

此前的 Confounding Filter 方法仅在分类头上操作，未充分探索 Transformer 架构中混淆信息的分布。这激发了本文对整个网络范围内混淆权重定位和消除的研究。

## 方法详解

### 整体框架

两阶段流程：**阶段1** 正常微调模型用于痴呆检测；**阶段2** 定位与性别关联的权重，生成掩码矩阵将其置零。

### 关键设计

1. **Extended Confounding Filter (ECF)**：扩展原始 Confounding Filter 至完整 Transformer 架构。在阶段2中，从分类头开始逐层解冻网络层，训练模型预测性别标签。跟踪每层内权重的变化幅度，选取每层中变化最大的 top-15% 权重进行掩码。这比原方法仅在分类头操作更灵活，因为语义信息分布在 Transformer 的各层中。

2. **Dual Filter (DF)**：进一步放松 ECF 的逐层局部掩码限制。从相同预训练检查点初始化两个模型——一个微调用于痴呆检测（primary model f），一个微调用于性别分类（confounder model g）。跟踪两个模型在整个网络中的权重变化，选取变化最大的 top-k% 权重，然后通过集合运算获得三种掩码策略：
   - **交集掩码 M_I**：同时在两个任务中变化较大的权重（可能编码了混淆的性别-痴呆信息）
   - **差集掩码 M_D**：仅在性别模型中变化大但不在痴呆模型中变化大的权重（纯性别信息）
   - **联合掩码 M_I ∪ M_D**：等价于性别模型中变化最大的 top-k% 权重

3. **关键细节**：阶段2训练性别分类时仅使用非痴呆样本（健康对照组），避免混入痴呆信号。分类头不参与跟踪（因为两个任务的分类头必然差异最大）。

### 评估框架：混淆偏移（Confounding Shift）

采用 α 参数控制训练/测试集中性别与痴呆的条件分布：α = P(dementia|female) / P(dementia|male)。训练在 α_train 上，测试在 1/α_train 上，模拟极端分布偏移。固定 P(gender=1) = P(dementia=1) = 0.5 确保公平比较。

## 实验关键数据

### 主实验：性别混淆偏差验证

| 数据集 | 设置 | 男女差异均值 | p值 |
|--------|------|------------|-----|
| DementiaBank | 原始分布 | 0.055 | <0.001 |
| DementiaBank | 平衡分布 | 0.068 | <0.001 |
| CCC | 原始分布 | 0.152 | 0.002 |
| CCC | 平衡分布 | 0.102 | 0.007 |

### 去偏效果示例（DB 数据集，α=0.2）

| 方法 | 掩码比例 | AUPRC | ΔFPR |
|------|---------|-------|------|
| 原始模型 | 0% | 0.83 | 0.23 |
| DF (M_I) | 10% | 0.80 | 0.03 |

### 方法对比（α_train=3, α_test=1/3）

在 AUPRC-ΔFPR 权衡曲线上：
- **CCC 数据集**：ECF 取得最佳权衡
- **DB 数据集**：DF (M_D) 优于其他方法
- 两种方法一致性地优于 adapter 基线（ConGater, ModDiffy）和原生 Confounding Filter

### 关键发现

1. **简单的分类头掩码不够**：原始 Confounding Filter 仅在分类层操作不足以缓解混淆，需要向更深层扩展。
2. **上层权重消除性别信息后模型仍具弹性**：从上层开始移除性别关联权重不会显著损害痴呆检测性能，直到波及底层才出现退化。
3. **token embedding 层至关重要**：移除该层少量权重就会剧烈改变模型性能。
4. **权重掩码比损失优化方法提供更细粒度的权衡轨迹**：可以连续调节掩码比例精细控制公平性与性能的平衡。
5. **统计均等性（ΔSP）也显著改善**：在平衡测试集上，两种方法均大幅减小男女间的预测概率差异。

## 亮点与洞察

- **方法与模型无关（model-agnostic）**：ECF 和 DF 可应用于任何 Transformer 架构，不引入额外训练目标或模块。
- **DF 的"双模型比较"思路简洁有效**：通过比较两个独立微调模型的权重变化定位混淆权重，无需联合训练或对抗目标。
- **权重掩码视角**：揭示了性别信息和痴呆信息在 Transformer 权重中存在纠缠，尤其在各层变化最大的权重中。
- **临床启示**：不处理混淆偏移即使在标签和性别分布平衡的情况下也会导致模型性能退化。

## 局限性 / 可改进方向

- 仅在两个较小数据集上验证，泛化性需进一步考察
- 由于数据量小，不同 α 配置需要重复采样，结果中包含大量重复
- ECF 的逐层解冻-重训策略可扩展性差，DF 更高效（仅需两次微调）
- 仅用 BERT-base 验证，未涉及其他编码器模型
- 仅处理二元混淆变量（性别），多类别混淆变量的场景未探索
