# Activating Distributed Visual Region within LLMs for Efficient and Effective Vision-Language Training and Inference

**会议**: ACL 2025  
**arXiv**: [2412.12785](https://arxiv.org/abs/2412.12785)  
**代码**: 无  
**领域**: 多模态VLM  
**关键词**: 视觉区域, 高效训练, 层选择, 视觉-语言模型, 层剪枝

## 一句话总结

本文发现 LLM 中存在类似人脑视觉皮层的"视觉区域"——稀疏且均匀分布的层集合，仅更新 25% 的层即可保留 99% 的视觉性能，同时保持甚至提升语言能力，并据此提出了高效的视觉区域定向训练和剪枝范式。

## 研究背景与动机

**领域现状**：大视觉-语言模型（LVLM）如 LLaVA、Bunny 等通常通过视觉指令微调来获取视觉能力，训练过程中需要更新投影层和 LLM 骨干中所有层的参数。即使使用 LoRA 等高效微调方法，全层更新仍然消耗大量计算资源。

**现有痛点**：(1) 全层视觉指令微调计算成本高，尤其对于 13B+ 规模模型；(2) 多模态训练会损害 LLM 预训练的语言知识和推理能力——LLaVA 在文本任务上的困惑度明显高于其 LLM 骨干 Vicuna；(3) 现有的层级冻结技术（如只微调最后几层）是为语言下游任务设计的，不能直接适用于视觉对齐任务，因为视觉感知需要超越文本理解和推理的能力。

**核心矛盾**：视觉对齐需要修改 LLM 参数来适应新模态，但过度修改会破坏已有的语言能力。如何在"充分学习视觉能力"和"最小化语言能力损失"之间取得平衡，是一个关键且未解决的问题。

**本文目标**：(1) LLM 内部是否存在专门负责视觉学习的"视觉区域"？(2) 这些区域的位置在哪里、最小规模是多少？(3) 能否基于视觉区域进行更高效的训练和推理？

**切入角度**：受神经科学中人脑视觉皮层（visual cortex）概念的启发——人脑中有专门处理视觉信息的特定区域，而非全脑参与。类比地，LLM 中可能也存在专门对视觉对齐起关键作用的层子集。作者通过一个初步实验验证了这一假设：将 LLaVA 某些层的参数恢复为 Vicuna 原始参数后，视觉任务性能几乎不变甚至下降，说明这些层对视觉学习是冗余的。

**核心 idea**：通过实验系统地识别 LLM 中稀疏且均匀分布的"视觉区域"层，仅更新这些层即可实现接近全层训练的视觉性能，同时更好地保留语言能力，并基于此提出视觉区域剪枝范式。

## 方法详解

### 整体框架

方法的核心流程为三步：(1) 通过系统的层选择实验确定"视觉区域"的最优位置——即哪些层对视觉学习最重要；(2) 进行视觉区域定向训练——仅微调选中层的参数（结合 LoRA），冻结其余层；(3) 训练完成后，基于视觉区域进行层剪枝——移除视觉区域外的低重要性层以加速推理。

### 关键设计

1. **启发式稀疏均匀层选择策略**:

    - 功能：确定 LLM 中视觉区域的最优层位置
    - 核心思路：对于 32 层的 LLM（如 Llama-3-8B），选择 8 个均匀间隔分布的层（0, 4, 8, 12, 18, 22, 26, 30）作为视觉区域。与之对比了多种选择策略：连续块选择（底层 0-7、中下层 8-15、中上层 16-23、顶层 24-31）、混合选择（底层+顶层）以及基于重要性指标的后验选择（图像注意力分数、参数变化比率、Block Influence 分数、多模态 BI 分数、角距离分数）。实验表明稀疏均匀分布策略在 10 个视觉任务上平均表现最优（61.82%），甚至超过了所有需要训练后才能计算的重要性指标方法
    - 设计动机：连续层的表征相似度高，同时更新它们提供的多样性有限，不利于适应多样化的视觉任务。稀疏均匀分布让视觉学习信号渗透到 LLM 的每个深度级别，保持了 LLM 预有的层间知识分工

2. **视觉区域规模探索（25% 层即够）**:

    - 功能：确定实现高效训练所需的最小视觉区域规模
    - 核心思路：系统地测试了更新 1、2、4、6、8、16、32 层时的性能表现。结果发现：更新 25%（8 层）可保留 99.36% 的完整训练性能；20%（6 层）保留 98.99%；12.5%（4 层）保留 98.01%；但低于 4 层（如 2 层、1 层）会导致显著性能下降。在不同数据规模（100%、25%、10% 训练数据）下，25% 层的策略均一致有效
    - 设计动机：确定最小可用规模有双重价值——既可直接用于节省训练开销，也为后续剪枝提供了安全边界（视觉区域内的层不能被剪除）

3. **视觉区域导向的层剪枝范式**:

    - 功能：在训练完成后通过移除冗余层加速推理
    - 核心思路：传统的 LLM 层剪枝（如基于角距离的策略）在 LVLM 上效果极差——即使只移除 1 层也会导致视觉性能严重下降。本文提出的范式是：先用视觉区域定向训练微调模型，然后仅在视觉区域之外的层中按角距离选择重要性最低的 0-4 层进行剪枝。实验表明，在 LLaVA-1.5-7B 上移除 3-4 层时，该范式的性能保持显著优于直接在全层训练的模型上剪枝
    - 设计动机：视觉区域定向训练本身就将视觉知识集中在特定层中，使得区域外的层更加"干净"——它们主要保留语言能力，部分冗余层可以被安全移除

### 训练策略

训练使用 LoRA 进行参数高效微调，仅对选中的视觉区域层应用 LoRA adapter。预训练阶段 LLM 冻结不变；监督微调阶段仅更新选中层（通过 LoRA）和投影层。训练数据为 695K（Bunny）和 665K（LLaVA）视觉指令跟随数据。实验在 A800 GPU 上进行。

### 关键发现

- **稀疏均匀分布是最优策略**：在所有启发式和基于重要性的层选择方法中，简单的稀疏均匀分布表现最好。这意味着视觉学习需要模型各深度级别的参与，而非集中在某些特定层
- **25% 层是性价比最优点**：保留 99.36% 性能的同时，LLaVA-1.5-7B 和 LLaVA-1.5-13B 训练时间减少 23%，Bunny 减少 13%
- **顶层微调最差**：传统 NLP 任务中常用的"只微调最后几层"策略在视觉对齐任务上效果最差，这是一个反直觉但重要的发现
- **部分训练反而提升语言性能**：视觉区域定向训练的模型在文本任务（MMLU、BIG-Bench-H 等）上一致优于全层训练模型，有时甚至超过 LLM 骨干本身。这证实了全层多模态训练确实会损害语言能力
- **跨模型一致有效**：在 LLaVA-1.5-7B、LLaVA-1.5-13B、Bunny-LLaMA-3-8B-V、Bunny-Phi3-mini-4B-V 四个不同模型上均验证了 25% 层策略的有效性

## 亮点与洞察

- **类脑视觉区域假说**：将神经科学中的视觉皮层概念引入 LLM 层分析是一个优雅的类比，且实验结果有力地支持了这一假说。这提示我们 LLM 的不同层可能具有功能专业化的倾向，值得进一步探索"听觉区域"、"推理区域"等
- **简单方法胜过复杂方法**：无需训练的启发式稀疏均匀选择超过了所有需要完整训练模型才能计算的后验重要性指标。这说明问题本身的结构（视觉学习需要各层参与）比精确的层重要性估计更重要
- **多模态训练的语言能力保护**：部分训练不仅高效，还能作为防止多模态训练破坏语言能力的正则化手段。这个副发现对 LVLM 的实际部署很有价值
- **视觉区域剪枝范式**：传统剪枝在 LVLM 上失败，而视觉区域先训练再剪枝的范式成功，这说明"训练方式决定了模型的可压缩性"

## 局限与展望

- 主要在 LLaVA 系列和 Bunny 系列上验证，更新一代的模型（如 Qwen-VL、InternVL）尚未测试
- 当前方法在层级粒度上操作（冻结/更新整层），未探索更细粒度的参数级别视觉区域定位
- 视觉区域剪枝在移除 3-4 层后仍有明显性能下降，更激进的剪枝策略需要新方法
- 探索将视觉区域思路与 MoE（混合专家）架构结合——让不同专家负责不同模态的处理
- 该方法与 LoRA 互补但未探索与 QLoRA、全参数微调的组合效果

## 相关工作与启发

- **vs LoRA/QLoRA**: LoRA 在所有层上高效微调，而本文进一步减少了需要微调的层数。两者正交互补——可以在视觉区域层上用 LoRA
- **vs ShortGPT (Men et al., 2024)**: ShortGPT 基于 Block Influence 分数剪枝 LLM 层。本文发现 BI 分数在 LVLM 场景下的层选择不如简单的稀疏均匀策略
- **vs LISA (Pan et al., 2024)**: LISA 提出层级重要性采样训练策略，动态选择层更新。与本文固定层选择不同，但都认为不需要全层更新
- **vs 大脑功能区域研究 (Zhao et al., 2023)**: Zhao 等人发现了 LLM 中的核心语言区域（约 1% 参数），本文从视觉角度发现了互补的视觉区域

## 评分

- 新颖性: ⭐⭐⭐⭐ 视觉区域假说新颖且有启发性，从神经科学到 AI 的类比优雅
- 实验充分度: ⭐⭐⭐⭐⭐ 10 个视觉任务 + 4 个文本任务，4 个不同模型，多种层选择策略对比，实验非常系统全面
- 写作质量: ⭐⭐⭐⭐ 论述清晰，实验设计层层递进，从"在哪里"到"要多少"到"能否剪枝"逻辑流畅
- 价值: ⭐⭐⭐⭐⭐ 发现具有直接实用价值——任何使用 LVLM 的团队都可以立即用 25% 层策略节省训练开销
