# Comparing LLM-generated and human-authored news text using formal syntactic theory

**会议**: ACL 2025  
**arXiv**: [2506.01407](https://arxiv.org/abs/2506.01407)  
**代码**: https://github.com/olzama/llm-syntax/releases/tag/1.0.0  
**作者**: Olga Zamaraeva, Dan Flickinger, Francis Bond, Carlos Gómez-Rodríguez（Universidade da Coruña 等）

## 一句话总结

利用英语资源语法（ERG）实现的 HPSG 形式句法理论，首次从句法构式、词汇类型和形态规则三个层级系统分析 LLM 生成文本与人类新闻写作的语法差异，发现 LLM 在句法构式使用上趋于"扁平化均值"。

## 研究背景与动机

- **已有工作的局限**：LLM 文本 vs 人类文本的对比研究主要集中在分类器训练、词汇分布和依存句法（UD），但 UD 等标注体系是为 NLP 任务设计的，粒度有限且不独立于任务本身
- **形式理论的优势**：HPSG 作为独立于 NLP 的语言学形式理论，具有完全显式的形式化、精细的词汇类型层级和一致的解析，可提供更深层的语法分析
- **关键空白**：没有研究从形式句法角度全面比较 LLM 和人类的写作模式，特别是在句法构式长尾分布和个体作者差异方面
- **核心问题**：LLM 文本在精细语法结构上与人类文本有何系统性差异？

## 方法详解

### 整体框架

```
NYT 人类文本 / LLM 生成文本
        ↓
    ERG (HPSG) 解析
        ↓
    三层级类型频率统计
    ├── 句法构式 (298 种)
    ├── 词汇类型 (1398 种)
    └── 词法规则 (100 种)
        ↓
    余弦相似度 + Shannon 多样性指数分析
        ↓
    人类 vs LLM 语法差异诊断
```

### 关键设计 1：基于 ERG 的形式句法解析

- 使用 **English Resource Grammar**（ERG 2025 版），这是目前覆盖率最高的形式语法实现，可解析 94% 的编辑良好英语文本
- ERG 将句子映射为完整的 HPSG 结构，包含**句法构式**（298 种短语级规则）、**词汇类型**（1398 种，编码词的句法行为）和**词法规则**（100 种屈折/派生规则）
- 关键优势：词汇类型区分同一词的不同句法用法（如 "law" 既可做普通名词 law_n1，也可带从句补语 law_n2——"There is a law that..."），这是 UD 无法做到的

### 关键设计 2：多数据集交叉验证

数据来源：
- **NYT 人类文本**：2023.10–2024.01 的纽约时报文章引导段（26,102 句）
- **LLM 生成文本**：6 个模型（LLaMA-7B/13B/30B/65B, Falcon-7B, Mistral-7B）基于 NYT 标题生成（共 ~214K 句）
- **Redwoods 树库**：WSJ 和 Wikipedia 部分，用于跨体裁/风格验证

使用余弦相似度和 PCA 投影可视化类型分布差异，采用 Shannon 熵和 Gini-Simpson 指数量化多样性。

### 关键设计 3：个体作者 vs LLM 的配对比较

选取 12 位发表 >100 篇文章的 NYT 记者，逐对计算其 HPSG 类型分布的余弦相似度，并与 LLM 做交叉比较——首次揭示人类作者间的语法差异结构。

## 实验结果

### 句法构式分布差异

| 比较维度 | 人类 NYT 特征 | LLM 特征 |
|----------|---------------|----------|
| 高频构式 | 更少使用 head-complement | 更多使用 head-complement（最通用结构） |
| 修辞手法 | 分词修饰、双同位语、绝对 VP、裸 NP 并列 | 相对从句、标记子句、提取主语从句 |
| 标点句法 | 较少使用 | 更多使用子句片段连接（"But the observation suits him."） |
| Shannon 熵 | H=3.342（最高） | H=3.221–3.284（均低于人类） |
| 长尾构式 | 偶尔使用风格性结构 | 过度使用数字序列、括号修饰等 |

### 词汇类型与词条差异

| 维度 | 人类 | 单个 LLM | 全部 LLM 合并 |
|------|------|----------|---------------|
| 唯一词条数 | 基准 | ~50% | >人类 |
| 词汇类型多样性 Shannon | H=4.727 | H=4.700–4.877 | 高于人类 |
| 人类独有词条特征 | 非正式语言（haven't, a couple dozen）、祈使句、直接强烈表达 | — |
| LLM 独有词条特征 | — | 数字/标点类型、说服性词汇（fact, clear）、正式表达 |

### 个体作者 vs LLM

| 发现 | 说明 |
|------|------|
| 人类间差异 > 人类-LLM 差异 | 人类作者之间的句法分布差异大于任何人类与 LLM 的差异 |
| LLM 间差异极小 | 6 个 LLM 的句法类型分布高度聚集 |
| 词汇类型方差最大 | 人类在词汇类型使用上个体差异尤为显著 |
| 词法规则差异最小 | 屈折/派生形态在 NYT 体裁下，人类与 LLM 高度一致 |

## 论文亮点

1. **方法论独特性**：首次使用独立于 NLP 的形式句法理论（HPSG/ERG）分析 LLM 文本，提供比 UD 更精细和一致的语法分析
2. **"LLM 是语法均值"**：人类作者间的语法差异大于人类与 LLM 的差异——LLM 生成文本可视为人类语法特征的"平均化"，这是一个深刻的发现
3. **三层级解耦分析**：将句法构式、词汇类型和词法规则分开分析，发现词法规则层面人类与 LLM 几乎无差异，揭示了差异的层级性
4. **长尾构式分析**：利用 ERG 的全面类型层级发现 LLM 偏好的低频结构（数字序列、括号修饰等）

## 局限性

1. **仅限英语 NYT 体裁**：ERG 仅覆盖英语，且主要分析结果限于新闻风格，跨语言/跨体裁通用性待验证
2. **LLM 采样偏差**：LLM 每次生成不同文本，结果受 prompt 和采样策略影响，当前仅用一组 prompt
3. **模型版本较老**：评测 LLM 为 LLaMA-1/Falcon/Mistral（2023 年前），未包含 GPT-4、Claude 等新一代模型
4. **统计显著性有限**：多重比较校正后（FDR），高频构式的差异均不显著（仅 9 个数据集）

## 相关工作

- **词汇/UD 维度比较**：Muñoz-Ortiz et al. 2024 使用 UD 分析同一数据集，发现人类文本更短小、依存距离更优化——本文确认并深化了这些结论
- **Biber 修辞特征**：Reinhart et al. 2024, Sardinha 2024 使用 Biber 特征分析 LLM 偏好的语法结构
- **文本检测**：多数工作训练分类器区分 LLM/人类文本，但缺乏语言学深层分析
- **POS 模板分析**：Shaib et al. 2024 发现 LLM 更倾向重复 POS 序列模板
