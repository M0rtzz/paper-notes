# Seeing the Wind from a Falling Leaf

**会议**: NeurIPS 2025  
**arXiv**: [2512.00762](https://arxiv.org/abs/2512.00762)  
**代码**: https://chaoren2357.github.io/seeingthewind/ (项目页面)  
**领域**: 视频理解 / 物理理解  
**关键词**: 力场恢复, 可微物理模拟, 逆图形学, 3D高斯, 基于物理的视频生成

## 一句话总结

本文提出端到端可微逆图形学框架，通过联合建模物体几何、物理属性和力表示，从视频中恢复不可见的力场（如风场），并支持基于物理的视频生成与编辑。

## 研究背景与动机

人类可以通过观察树叶在风中飘落来"感知"不可见的风——这种从可见运动推断不可见物理交互的能力（直觉物理），一直是计算机视觉中未被充分探索的领域。

现有工作主要集中在**系统辨识**（即从视频中估计少量物理参数，如质量、摩擦系数、杨氏模量），这类问题的参数空间较小、相对可控。然而，**力是全方位的向量，存在于整个3D空间中**，估计如此密集复杂的力表示面临极大挑战。

此外，可微物理模拟器中的时间积分容易导致梯度爆炸或消失，使得通过反向传播优化力场变得非常不稳定。

本文的核心insight是：如果我们能建立一个从力→物体运动→视频渲染的完全可微pipeline，就可以通过反向传播从观察到的运动中恢复力。关键创新在于提出稀疏跟踪目标来稳定这一优化过程。

## 方法详解

### 整体框架

框架包含四个核心组件：
1. **物体建模**：用3D高斯表示物体的几何和外观，用VLM赋予物理属性
2. **力表示**：用因果三平面（causal tri-plane）表示时空变化的力场
3. **物理过程**：用可微MPM（物质点法）模拟物体运动
4. **力优化**：通过4D稀疏跟踪目标和反向传播恢复力场

### 关键设计

1. **物体建模（3D高斯 + VLM物理属性）**:

    - 每个高斯核心参数化为：位置x、速度v、协方差Σ、不透明度σ、球谐函数SH、形变梯度D、质量m、杨氏模量E、泊松比ν
    - 使用预训练的度量深度模型（UniDepth）从首帧初始化高斯位置
    - **创新点**：利用GPT-4o-Vision的常识知识推断物体类型和物理属性（密度、杨氏模量、泊松比），再通过Grounded SAM将属性分配给对应的高斯。实验表明VLM对密度的MAPE仅4.65%，对杨氏模量的log-MAPE仅7.17%

2. **因果三平面力表示**:

    - 采用欧拉视角，定义力f(x,t) = D(γ(x) + φ(t; φ(t-1)))
    - γ(·)为三平面特征图，将3D空间高效编码
    - φ(·)为时间编码MLP，关键创新是φ(t)递归依赖于φ(t-1)的权重初始化，建模力的时间因果性
    - 相比K-Planes、HexPlane等4D表示，因果三平面将空间和时间解耦，计算效率更高

3. **可微物理模拟器（MPM）**:

    - 实现了基于物质点法的可微模拟器，核心更新步骤包括：
      - 形变梯度更新：D^t = (I + ∇v^{t-δt} · δt) · D^{t-δt}
      - 速度更新：v^t = v^{t-δt} + δt·f_ext/m + δt·f_int/m（外力+内力）
      - 位置更新：x^t = x^{t-δt} + δt·v^t
      - 协方差更新：Σ^t = D^t · Σ^0 · (D^t)^T
    - 高斯作为拉格朗日粒子、力场作为网格，恰好符合MPM的粒子-网格混合框架

4. **4D稀疏跟踪目标**:

    - 使用CoTracker获取稀疏关键点的像素级运动轨迹p^t → p^{t+1}
    - 通过度量深度反投影得到首帧3D关键点P^0
    - 通过最小化重投影误差+ARAP（尽可能刚性）约束，逐帧求解3D关键点运动P^t → P^{t+1}
    - 用重心坐标插值将稀疏关键点运动传递给每个高斯粒子
    - 最终损失：L = L_motion + λ1·L_space + λ2·L_time

### 关键发现

- 因果三平面在力恢复精度上远超K-Planes：幅度误差从18%降至5%，方向误差从40°降至4°
- 稀疏跟踪损失相比图像损失和稠密流深度损失，力恢复精度提升一个数量级
- VLM推断的物理属性足够准确（密度MAPE 4.65%），无需精确系统辨识即可实现鲁棒力估计
- 方法在弹性、弹塑性、粘塑性三种材料类型上均有效

## 亮点与洞察

- 问题定义极具创新性——从"看风"这一诗意的类比出发，将不可见力场的恢复形式化为可求解的优化问题
- 整个pipeline设计精妙：3D高斯作为拉格朗日粒子、三平面作为欧拉网格，完美契合MPM框架
- VLM赋予物理属性的方法简洁高效——避免了复杂的系统辨识，利用了大模型的常识知识
- 稀疏跟踪目标的提出解决了可微物理中长期存在的梯度不稳定问题
- 恢复的力场可以直接用于物理驱动的视频生成和编辑，具有很强的应用价值

## 局限性 / 可改进方向

- 仅适用于小变形或纯弯曲变形的物体，无法处理流体等需要不同物理过程的对象
- 使用静态背景合成，前景物理与背景分离的假设限制了适用场景
- 从单帧初始化的点云可能不完整（遮挡问题），虽然稀疏跟踪目标提供了一定鲁棒性
- 力场的空间分辨率受限于三平面的分辨率
- 未与接触力建模（如机器人操作中的力估计）方法进行定量比较

## 相关工作与启发

- 将GradSim、DiffPD等可微物理工作从参数估计扩展到了力场估计，是一个重要的泛化
- 与PhysGaussian、PhysDreamer等基于物理的视频生成工作不同，本文不需要手动指定力，而是从视频中自动恢复力
- 启发：力场估计可能成为连接视觉理解和物理推理的重要桥梁
- 3D高斯 + MPM的组合为后续可微物理模拟工作提供了优雅的范式
