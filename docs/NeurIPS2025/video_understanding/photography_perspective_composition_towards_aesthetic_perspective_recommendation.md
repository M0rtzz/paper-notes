# Photography Perspective Composition: Towards Aesthetic Perspective Recommendation

**会议**: NeurIPS 2025  
**arXiv**: [2505.20655](https://arxiv.org/abs/2505.20655)  
**代码**: https://vivocameraresearch.github.io/ppc  
**领域**: 视频理解 / 摄影构图  
**关键词**: 摄影构图, 视角变换, 视频生成, 美学评估, RLHF

## 一句话总结

首次提出摄影视角构图（PPC），超越传统 2D 裁剪，通过 3D 透视变换生成"差→优"的构图过程视频，并基于人类评估训练视角质量评估模型，帮助普通用户提升摄影构图水平。

## 研究背景与动机

1. **领域现状**：传统构图方法基于裁剪（显著性、学习裁剪、强化学习裁剪），仅在 2D 平面操作，无法改变主体间的投影空间关系。
2. **现有痛点**：当场景混乱、主体排列不当时，任何裁剪都无法改善构图；专业摄影师通过改变拍摄位置和角度来调整透视关系。
3. **核心矛盾**：三大挑战——(1) 透视变换视频数据极稀缺；(2) 构图美学遵循偏序关系而非全序；(3) 缺乏统一的构图评估标准。
4. **切入角度**：不是裁剪，而是虚拟相机运动；不是静态推荐，而是变换过程视频。
5. **核心 idea**：用 3D 重建生成反向视频数据 + Rectified Flow + DPO 优化 + VLM 美学评估。

## 方法详解

### 整体框架

1. 自动构建透视变换训练数据（3D 重建 + 反向视频）
2. 基于 Rectified Flow 的视角变换视频生成
3. 视角质量评估模型（PQA）提供奖励信号和数据过滤

### 关键设计

1. **数据集自动构建（Section 3.2）**
   - 做什么：从优秀摄影作品出发，通过 ViewCrafter 进行单图 3D 重建，指定相机轨迹生成正向序列（优秀→不良），反向播放得到训练数据（不良→优秀）
   - 核心思路：$\text{优秀照片} \xrightarrow{\text{3D重建}} \text{3D场景} \xrightarrow{\text{相机轨迹}} \text{正向序列} \xrightarrow{\text{反向}} \text{训练数据}$
   - 设计动机：3D 重建质量有限，数据包含失真和模糊，用 PQA 模型自动过滤

2. **Flow-DPO 优化（Section 3.3）**
   - 做什么：某些偏离 GT 轨迹的视频美学更优，不能单纯约束 GT，需允许探索更佳轨迹
   - 核心思路：$\max_{p_\theta} \mathbb{E}_{s,v \sim p_\theta(v|s)} [r(v,s)] - \beta \mathbb{D}_{KL}[p_\theta(v|s) \| p_{ref}(v|s)]$，其中 $\beta_t = \beta(1-t)^2$ 为时间衰减
   - 设计动机：监督学习仅能模仿 GT，DPO 允许模型发现比 GT 更好的轨迹

3. **视角质量评估模型 PQA（Section 3.4）**
   - 做什么：自动过滤生成的训练数据 + 为 PPC 的 RLHF 提供奖励信号
   - 核心思路：基于 Qwen2-VL，两阶段训练（非配对视频阶段 + 配对视频阶段），三维评估（VQ 视觉质量、MQ 运动质量、CA 构图美学）
   - 设计动机：构图美学是偏序关系，Bradley-Terry with ties 比普通回归更合适

### 损失函数 / 训练策略

- 监督基础损失：Rectified Flow 的速度场回归
- Flow-DPO 损失：对偏好对 $(v_h, v_l)$ 最大化偏好概率
- PQA 损失：Bradley-Terry loss with ties 扩展

### 主实验

| 模型 | 透视准确性(CMM↑) | FVD↓ | VQ↑ | MQ↑ | CA↑ |

| **WAN 2.1** | **0.5989** | 345 | 0.7195 | 0.7454 | 0.7072 |

| 配置 | VQ 准确度 | MQ 准确度 | CA 准确度 | 说明 |

### 关键发现

- 20° 为最优旋转角度平衡点，超 30° 视觉差异过大，模型泛化失效
- 在单主体、多主体、风景、无人机摄影等多个场景中均有效
- PQA 模型在构图美学评估上达到 81% 准确度，可替代人工评估

## 亮点与洞察

- **范式转换**：首次引入 3D 透视变换替代 2D 裁剪，是构图 AI 的根本性创新。这个思路可迁移到视频编辑、虚拟现实等领域。
- **自动数据构建**：反向视频生成的思路非常巧妙——优秀照片是已知的，差照片可以通过破坏性变换生成。
- **多维评估**：VQ/MQ/CA 三维度评估超越了单一 FID/FVD 指标，更贴近人类对构图的多维度判断。

## 局限性 / 可改进方向

- 20° 角度限制了激进的视角变换，可能遗漏创意视角
- 3D 重建依赖 ViewCrafter 等模型，复杂场景可能失效
- 视频生成需要计算资源，部署在手机上仍有困难
- 未涉及动态场景或视频输入的处理

## 相关工作与启发

- **vs 传统裁剪（GAICD 等）**：裁剪只能在已有画面中选取区域，PPC 可以虚拟改变拍摄角度
- **vs I2V 模型（Sora 等）**：PPC 有明确的摄影美学目标，不只是随机生成视频
