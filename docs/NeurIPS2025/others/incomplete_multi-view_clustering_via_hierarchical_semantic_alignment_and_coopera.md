# Incomplete Multi-view Clustering via Hierarchical Semantic Alignment and Cooperative Completion

**会议**: NeurIPS 2025  
**arXiv**: [2510.13887](https://arxiv.org/abs/2510.13887)  
**代码**: [GitHub](https://github.com/XiaojianDing/2025-NeurIPS-HSACC)  
**领域**: 多视图学习 / 聚类  
**关键词**: 不完整多视图聚类, 层次语义对齐, 动态加权融合, 分布对齐, 缺失视图补全

## 一句话总结

提出 HSACC 框架，通过双层语义空间设计（低层互信息对齐 + 高层自适应加权融合）和联合优化的缺失视图隐式恢复策略，解决不完整多视图聚类中静态融合和两阶段流水线（先补全后聚类）的误差传播问题，在 5 个基准数据集上全面超越 SOTA。

## 研究背景与动机

不完整多视图聚类 (IMVC) 需要处理部分视图完全缺失的样本。现有方法存在两个关键问题：

1. **静态融合策略的局限性**：传统方法采用均匀加权等静态策略融合不同视图，无法适应视图间的分布差异。虽然一些方法引入了动态融合（如基于方差计算权重或跨视图共享权重），但它们**缺乏层次化的语义分离**——没有清晰区分低层一致性对齐和高层语义融合，导致多粒度信息丢失。

2. **两阶段流水线的误差传播**：传统"先补全后聚类"的流程中，补全和聚类目标独立优化，补全阶段的误差会传播到聚类阶段。虽然一些研究尝试统一框架，但忽略了不同视图在融合表示中的**差异化重要性**，使补全过程无法充分利用高质量视图的信息。

## 方法详解

### 整体框架

HSACC 由三个模块组成：(a) 视图重建模块——自编码器提取潜在表示并重建输入；(b) 多视图表示学习模块——通过互信息最大化对齐视图、自适应加权融合互补信息、最小化分布差异；(c) 数据恢复与聚类模块——补全缺失视图并基于补全后的表示执行聚类。

### 关键设计

#### 1. 低层语义空间：跨视图互信息最大化

**做什么**：确保不同视图的潜在表示在低层语义空间中保持一致性。

**核心思路**：通过特征级相似度估计互信息。对第 $i$ 个样本在两个视图中的潜在表示 $\mathbf{z}_i^1$ 和 $\mathbf{z}_i^2$，计算特征间的点积相似度 $p_{m,n}^{(i)} = z_{i,m}^1 \cdot z_{i,n}^2$，聚合得到联合概率分布 $\mathbf{P}$，进而计算互信息损失：

$$L_{MMI} = -I(\mathbf{Z}^1; \mathbf{Z}^2) = -\sum_{m,n} P_{(m,n)} \ln \frac{P_{(m,n)}}{P_{(m)} \cdot P_{(n)}}$$

**设计动机**：通过特征粒度的互信息最大化建立视图间的底层共享模式，为后续高层融合提供对齐基础。

#### 2. 高层语义空间：自适应加权融合

**做什么**：根据各视图与初始融合表示的分布亲和度，动态分配视图权重，生成统一的全局表示。

**核心思路**：两步走——

- **Step 1: 估计视图权重**：首先计算初始融合表示 $\mathbf{R} = (\mathbf{Z}^1 + \mathbf{Z}^2) / 2$，然后基于线性核 MMD 度量每个视图 $\mathbf{Z}^v$ 与 $\mathbf{R}$ 的分布差异 $D(\mathbf{Z}^v, \mathbf{R})$，通过 softmax 将差异转换为权重：$W^v = \frac{\exp(-D(\mathbf{Z}^v, \mathbf{R}))}{\sum_v \exp(-D(\mathbf{Z}^v, \mathbf{R}))}$。与初始融合表示越一致的视图获得越高权重。

- **Step 2: 分布对齐**：使用加权融合得到高层共同表示 $\mathbf{H} = \sum_v W^v \cdot \mathbf{Z}^v$，然后通过 MMD 最小化 $\mathbf{H}$ 与每个 $\mathbf{Z}^v$ 之间的分布差异，增强跨视图一致性。

**设计动机**：不同视图包含的信息质量不一，静态等权融合会被低质量视图拖累。自适应权重让高质量视图主导融合过程。

#### 3. 缺失视图隐式恢复

**做什么**：通过跨视图推理模块隐式恢复缺失视图，避免显式的数据生成。

**核心思路**：将已对齐的潜在表示 $\mathbf{Z}^v$ 通过 MLP 投影到高维语义空间，得到 $\mathbf{Q}^{v_1 \to v_2} = f_{MLP}(\mathbf{Z}^{v_1})$，利用已学到的判别性特征推断缺失视图的语义。推理一致性损失确保推断结果与真实潜在表示一致：

$$L_{INF} = \frac{1}{N} \sum_i \|\mathbf{z}_i^2 - \mathbf{q}_i^1\|_2^2 + \frac{1}{N} \sum_i \|\mathbf{z}_i^1 - \mathbf{q}_i^2\|_2^2$$

**设计动机**：与显式生成（如 GAN 生成缺失数据）相比，隐式恢复基于已学到的高质量判别特征，更加稳定且避免了模式坍塌。

### 损失函数 / 训练策略

总体目标函数：

$$L = \lambda_1 L_{REC} + \lambda_2 L_{INF} + \lambda_3 L_{MMI} + \lambda_4 L_{MMD}$$

其中 $L_{REC}$ 为重建损失，$L_{INF}$ 为推理一致性损失，$L_{MMI}$ 为互信息损失，$L_{MMD}$ 为分布对齐损失。推理损失从第 $E_1$ 个 epoch 起才引入（如 $E_1 = 100$），给自编码器足够的预热时间。

训练完成后，补全的多视图表示被拼接，使用 $k$-means 聚类得到最终结果。

**实现细节**：全连接自编码器架构 Input-1024-1024-1024-Output（编码器），镜像解码器。MLP 隐层 256-128-256。学习率 0.0001，batch size 256，训练 500 epoch。

## 实验关键数据

### 主实验

在 5 个数据集、3 种 missing rate（0.3, 0.5, 0.7）下，与 9 种 SOTA 方法对比，指标为 ACC / NMI / ARI。

| 数据集 (MR=0.5) | 指标 | HSACC (本文) | DCP | COMP | SURE | DSIMVC |
|----------|------|------------|-----|------|------|--------|
| Caltech101-20 | ACC | **76.81** | 71.51 | 70.13 | 48.78 | 28.18 |
| Caltech101-20 | NMI | **73.15** | 71.60 | 69.04 | 58.52 | 48.94 |
| Caltech101-20 | ARI | **87.13** | 78.56 | 76.10 | 42.04 | 21.00 |
| Noisy MNIST | ACC | **92.55** | 86.50 | 78.68 | 92.34 | 83.93 |
| Hdigit | ACC | **95.57** | 94.10 | 91.36 | 66.59 | 94.40 |
| 100leaves | ACC | **52.17** | 35.50 | 28.40 | 30.52 | 27.29 |
| LandUse_21 | ACC | **28.11** | 27.34 | 25.24 | 25.68 | 21.53 |

**关键观察**：在 Caltech101-20 (MR=0.5) 上，HSACC 的 ACC 和 ARI 分别比次优方法高 5.3% 和 8.57%。在 Noisy MNIST 上，missing rate 从 0.3 升到 0.7 时，HSACC 的准确率仅下降 6.92%，而 ICMVC 下降 35.19%，展现出优异的鲁棒性。

### 消融实验

在 Caltech101-20 数据集上评估各损失组件的贡献：

| 模型 | $L_{REC}$ | $L_{MMI}$ | $L_{MMD}$ | $L_{INF}$ | ACC | NMI | ARI |
|------|-----------|-----------|-----------|-----------|------|------|------|
| M-1 | ✓ | | | | 42.07 | 41.06 | 28.38 |
| M-9 | ✓ | ✓ | | ✓ | 68.69 | 67.63 | 73.68 |
| M-12 | ✓ | ✓ | ✓ | | 71.29 | 68.48 | 78.27 |
| M-15 (完整) | ✓ | ✓ | ✓ | ✓ | **76.81** | **73.15** | **87.13** |

从 M-1 到 M-15 的逐步引入清晰展示了每个组件的贡献：互信息对齐 ($L_{MMI}$) 带来最大的单步提升（ACC +26.6%, ARI +45.3%），而分布对齐和推理一致性损失进一步稳步提升。

### 关键发现

1. **层次化语义设计的有效性**：低层互信息提供共享模式对齐基础，高层自适应加权利用分布亲和度融合互补信息，两层协作实现了比单层方法更好的表示学习
2. **高 missing rate 下的鲁棒性**：在 MR=0.7 的极端缺失下，HSACC 在 Noisy MNIST (ACC 88.89) 和 Caltech101-20 (ACC 72.02) 上仍保持强劲表现
3. **超参数鲁棒性**：在 Caltech101-20 上，对 $\lambda_1 \sim \lambda_4$ 在 $\{0.01, 0.1, 1, 10, 100\}$ 范围内扫描，性能波动小，表明模型对超参数不敏感
4. **收敛性**：在多个数据集上损失值稳步下降、聚类指标稳步上升，t-SNE 可视化显示随训练推进聚类结构越来越清晰

## 亮点与洞察

- **层次语义分离**的设计思路清晰有力：低层对齐保证基本一致性，高层融合追求最优互补——这种分层思想可迁移到其他多模态学习任务
- **自适应权重机制**基于 MMD 度量分布亲和度来分配视图权重，比基于方差或共享权重的方案更有理论基础
- **隐式恢复**避免了 GAN 生成的不稳定性和显式插值的信息假设，理念简洁且有效
- **联合优化消除误差传播**：补全和聚类目标同时优化，补全质量因聚类反馈而提升，反之亦然

## 局限性 / 可改进方向

1. **两视图假设**：虽然方法理论上可扩展到多视图（3+），但所有实验仅在两视图设定下进行，$V(V-1)/2$ 次视图推理在视图数量多时代价大
2. **自编码器架构简单**：使用基本的全连接自编码器，对图像等结构化数据未利用 CNN/ViT 等专用架构
3. **线性核 MMD 近似**：分布对齐使用线性核简化计算，可能在复杂分布下精度不足
4. **聚类方法固定为 k-means**：最终聚类依赖 k-means，受初始化敏感且不适合非凸簇
5. **缺乏对大规模数据的验证**：最大数据集仅 10,000 个样本，工业级多视图数据的表现未知
6. **推理延迟对 epoch $E_1$ 的依赖**：推理损失的引入时机影响性能，但缺乏自适应调度策略

## 相关工作与启发

- **与 COMP 的对比**：COMP 也结合对比学习和双预测机制实现数据恢复和视图一致性学习，但不区分视图质量差异，融合策略不如 HSACC 精细
- **与 DCP 的对比**：DCP 使用双对比预测和信息论框架，在 Caltech101-20 和 Hdigit 上仅次于 HSACC，但在高缺失率下性能下降更快
- **与域自适应的联系**：分布对齐损失源于域自适应文献中的 MMD 方法，将其引入多视图融合是有益的跨领域借鉴
- **对多模态学习的启示**：层次语义对齐 + 自适应加权的范式可直接迁移到视觉-语言模型中的多模态融合

## 评分
- 新颖性: ⭐⭐⭐⭐ — 层次语义分离和联合优化是有意义的贡献，但各组件（互信息、MMD、MLP推理）均非全新
- 实验充分度: ⭐⭐⭐⭐ — 5 数据集 × 3 missing rate × 9 基线，消融和参数分析完整
- 写作质量: ⭐⭐⭐⭐ — 方法描述清晰，公式推导完整，但部分实验分析可更深入
