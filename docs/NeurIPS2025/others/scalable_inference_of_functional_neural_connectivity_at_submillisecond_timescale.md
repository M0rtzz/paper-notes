# Scalable Inference of Functional Neural Connectivity at Submillisecond Timescales

**会议**: NeurIPS 2025
**arXiv**: [2510.20966](https://arxiv.org/abs/2510.20966)
**代码**: [有](https://github.com/macari216/poisson-process-glm.git)
**领域**: 计算神经科学
**关键词**: 功能连接推断, Poisson点过程, GLM, 蒙特卡洛采样, 突触动力学

## 一句话总结
开发连续时间Poisson过程GLM的蒙特卡洛（MC）和多项式近似（PA）方法，支持亚毫秒精度的神经功能连接推断，在大规模神经记录上实现分钟级训练并揭示与已知海马解剖结构一致的连接模式。

## 研究背景与动机
Poisson广义线性模型（GLM）是分析神经放电序列的基础工具，但标准实现需要将放电时间离散化为分箱计数数据。离散化分辨率（通常1-10ms）远粗于突触动力学的亚毫秒时间尺度。更细的分箱会使设计矩阵$\mathbf{X}$爆炸式增长（200神经元×0.1ms分箱×100分钟需$10^{12}$比特），批量梯度的高方差导致拟合不稳定。本文转向无限小分箱的极限——连续时间Poisson点过程。

## 方法详解

### 整体框架
连续时间GLM对数似然：$\log p(\mathbf{y}|\mathbf{X},\mathbf{w}) = \sum_{k=1}^K \log\lambda(y_k) - \int_0^T\lambda(t)dt$。第一项在放电时刻精确计算，第二项（累积强度函数CIF）需要数值近似。提出MC和PA两种方法。

### 关键设计

**MC采样方法**：对CIF积分使用分层（stratified）蒙特卡洛估计，将$[0,T]$分为$M$等分子区间，各区间内均匀采样。每次迭代重新采样$\tau$，损失为$f(\mathbf{w},\tau) = \frac{T}{M}\sum_{m=1}^M \lambda(\tau_m) - \sum_{k=1}^K \log\lambda(y_k)$。关键优势：放电项始终对所有放电精确计算，且分层采样保证均匀覆盖。

**PA多项式近似**：用二阶Chebyshev多项式近似非线性$\exp(x)\Delta \approx a_2x^2 + a_1x + a_0$，使对数似然关于参数$\mathbf{w}$为二次型$\approx \mathbf{w}^\top(\mathbf{k}-a_1\mathbf{m}) - a_2\mathbf{w}^\top\mathbf{M}\mathbf{w}$，承认闭式解。线性充分统计量$\mathbf{m}$和二次充分统计量$\mathbf{M}$可预计算。

**广义Laguerre基**：提出带指数缩放的广义Laguerre多项式作为正交时间基函数，替代传统的raised cosine（RC）基：
- 正交性使滤波器表示更高效（3个GL基 ≈ 4个RC基）
- 近似Gamma函数包络与突触动力学的快升慢降特性匹配
- 承认单基和双基积分的解析闭式解

**混合方法**：PA初始化 + MC微调，兼顾速度和精度。

### 损失函数 / 训练策略
- MC：标准梯度下降 + 分层采样
- PA-c：直接闭式解或L-BFGS
- 混合：PA闭式解初始化 → MC少轮优化
- 岭正则化$\beta=1000$鼓励连接稀疏性

|------|--------|-----------|---------|-------------|

### 关键发现
- CA3表现出最高密度的循环兴奋性连接（~4%），与解剖估计一致
- 跨区域连接延迟（~2ms）长于区域内延迟（~1ms），符合轴突传导时间
- 批量离散GLM即使运行时间远长于其他方法，仍因梯度方差过大而收敛差
- MC方法的随机梯度误差比离散批量方法低若干个数量级
- GL基在5折交叉验证中持续优于RC基

## 亮点与洞察
1. **尺度化突破**：首次实现数百神经元×数千秒记录的亚毫秒GLM推断
2. **生物物理验证**：推断的连接模式与已知海马解剖结构高度一致
3. **正交基函数**：GL多项式兼具物理合理性和计算效率
4. **梯度方差分析**：理论解释为何MC优于离散批量（放电项精确计算+分层采样）

## 局限性 / 可改进方向
- PA精度受多项式近似范围限制
- 难以区分单突触连接和共同输入导致的相关放电
- Poisson分布假设可能不够灵活（方差等于均值），负二项分布是替代方案
- 未纳入潜在群体动力学建模

## 相关工作与启发
- 与Hawkes过程互补（Hawkes仅建模兴奋性连接）
- 连续时间点过程潜变量模型关注潜在结构而非精细连接
- 方法可推广至包含外部刺激和其他非线性函数
