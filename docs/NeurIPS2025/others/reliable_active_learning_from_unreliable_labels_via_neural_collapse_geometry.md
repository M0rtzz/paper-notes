# Reliable Active Learning from Unreliable Labels via Neural Collapse Geometry

**会议**: NeurIPS 2025 (Workshop)
**arXiv**: [2510.09740](https://arxiv.org/abs/2510.09740)
**代码**: [有](https://github.com/Vision-IIITD/NCAL)
**领域**: 主动学习 / 可靠机器学习
**关键词**: 主动学习, 神经坍缩, 特征几何, OOD泛化, 噪声标签

## 一句话总结
提出NCAL-R框架，利用神经坍缩（Neural Collapse）的几何正则性指导主动学习的样本选择，通过类均值对齐扰动（CMAP）和特征波动（FF）两个互补信号，在标签噪声和分布偏移条件下实现可靠的主动学习。

## 研究背景与动机
主动学习（AL）通过优先选择信息量大的样本减少标注成本，但其可靠性在标签噪声或数据分布偏移时受损。传统AL策略（不确定性、多样性）可能通过反复选择错标或冗余样本来放大这些错误。Neural Collapse理论指出深度网络训练后期特征集中于类均值并形成单纯形ETF，这些几何规律即使在不完美监督下也提供稳定性。本文将NC动态引入样本选择，以同时改善效率和鲁棒性。

## 方法详解

### 整体框架
池基主动学习设定：小标注集$\mathcal{L}$和大未标注集$\mathcal{U}$，每步选取批次$\mathcal{B}_t$标注。NCAL-R结合两个互补评分对候选样本排序，选择结构上具有影响力且预测不稳定的样本。

### 关键设计

**CMAP（类均值对齐扰动）**：衡量候选样本加入后对类间几何结构的影响。定义类均值对齐CMA为单位归一化类均值的平均余弦相似度，CMAP为候选样本加入前后CMA的差值，简化为点积形式：$\text{CMAP}(x) = (\bar{\tilde{\mu}}_t^c - \bar{\mu}_t^c)^\top(M_t - \bar{\mu}_t^c)$。高CMAP表示对特征几何的高扰动，训练这类样本可减少广义化误差。

**FF（特征波动）**：在训练终态阶段（TPT）跨多个检查点测量预测标签的变化次数$\text{FF}(x) = \sum_{t=T_i+1}^{T_f} \mathbf{1}[\hat{y}_t(x) \neq \hat{y}_{t-1}(x)]$。高FF标识出即使大部分特征已稳定仍持续不确定的样本。

**组合策略**：分别标准化CMAP和FF后取平均$\text{Score}(x) = \frac{\text{CMAP}(x) + \text{FF}(x)}{2}$，不需要辅助网络、伪标签或任务特定调参。

### 损失函数 / 训练策略
- 使用ResNet-18骨干，5%逐周期采集
- 标准分类交叉熵训练
- NCAL可应用于任何可提取特征嵌入的骨干和模态

### 消融实验
- 长尾分布场景：NCAL-R达45.15% vs Random 42.30%（+3%改进，仅16k图像）
- ActiveOOD策略（SISOMe）在闭集AL中表现远差于NCAL
- 类间距离分析：NCAL平均类间距15.944 vs Random 15.114，类分离度更大

### 关键发现
- NCAL-R在分类精度、OOD检测、OOD泛化和广义类别发现（GCD）上一致优于基线
- GCD设定下新类准确率提升+2.1点，已知类+1.6点
- 低标注预算时改进尤为明显，说明NC引导的选择促进更结构化的特征学习

## 亮点与洞察
1. **理论驱动实践**：将Neural Collapse的理论洞见转化为实用的主动学习策略
2. **统一框架**：单一方法同时处理OOD泛化和类别发现，无需OOD过滤或额外监督
3. **计算轻量**：不需要辅助网络或伪标签，可直接应用于任何骨干

## 局限性 / 可改进方向
- 仅在图像分类任务上验证，其他模态和任务有待探索
- FF评分需要多个训练检查点，增加了存储和计算成本
- 理论分析局限于TPT阶段，早期训练阶段的NC性质未充分利用

## 相关工作与启发
- Neural Collapse理论（Papyan等2020）为主动学习提供了新的几何视角
- 与BADGE等基于梯度的方法互补，NCAL关注特征空间的结构信息
- 可推广到自监督学习中的数据选择策略
