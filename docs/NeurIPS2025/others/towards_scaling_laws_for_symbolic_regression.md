# Towards Scaling Laws for Symbolic Regression

**会议**: NeurIPS 2025 (Workshop: MATH-AI)
**arXiv**: [2510.26064](https://arxiv.org/abs/2510.26064)
**代码**: 无
**领域**: 符号回归, 缩放定律
**关键词**: 符号回归, Scaling Laws, Transformer, 端到端, 计算最优

## 一句话总结

本文首次系统研究了符号回归中的缩放定律，发现验证损失和求解率随计算量呈幂律关系，最优token-参数比约为15，最优学习率和batch size随模型规模增长。

## 研究背景与动机

- 符号回归旨在发现解释数据的数学表达式，兼具科学洞察力和可解释性
- 基于深度学习的符号回归已可与遗传编程竞争，但**规模的作用**尚未被探索
- 受LLM缩放定律的启发，提出核心问题：符号回归是否存在类似的缩放定律？
- 此前工作主要关注训练过程的调优，模型规模基本固定在~100M参数以下

## 方法详解

### 整体框架

提出可扩展的端到端Transformer流水线，包含两大改进：
1. **系统化的合成数据生成**
2. **针对性的架构改进**

### 关键设计

**数据生成**（两步法）：
1. 递归生成基础表达式集：从变量出发，迭代应用一元/二元运算符，筛去等价表达式并统一表示
2. 对每个基础表达式采样k=3600个(表达式,数据集)对：随机插入整数常数，从高斯混合中采样数据点

**分词**：
- 表格数据用十进制浮点记法（尾数+指数）
- 目标表达式用LaTeX字符串，常数按位分词

**架构**：
- 编码器：每个数据单元独立嵌入（尾数和指数分别上投影后求和），借鉴表格基础模型的行/列交叉注意力
- 解码器：标准seq2seq Transformer，仅交叉注意力到目标单元的更新嵌入
- 训练使用交叉熵损失

### 实验配置

- 5个模型规模：6.5M-93M参数
- Token-参数比范围：5-80
- 训练数据：|E|=100,000表达式（最多2个变量），各表达式最多3600个数据对
- 独立验证/测试集：各1000个表达式，新常数和数据
- 评估：采样128个表达式保留最高R²的一个，报告完美求解率和R²>0.99比率

## 实验关键数据

### 主实验：缩放定律

| 模型大小 | 最高计算量 | Acc_solved | Acc_{R²>0.99} | 验证损失 |
|---------|----------|-----------|--------------|---------|
| 6.5M | 7.20e+16 | 0.149 | 0.526 | 0.424 |
| 13.5M | 2.88e+17 | 0.271 | 0.667 | 0.312 |
| 24M | 9.81e+17 | 0.378 | 0.762 | 0.240 |
| 45.5M | 8.81e+17 | 0.429 | 0.799 | 0.207 |
| 93M | (最高) | ~0.6 | ~0.85 | ~0.18 |

### 缩放趋势

| 发现 | 描述 |
|------|------|
| 求解率vs计算量 | 幂律关系，跨3个数量级 |
| 验证损失vs计算量 | 幂律关系，无饱和迹象 |
| R²>0.99比率 | 改善速度快于完美求解率 |
| 外推预测 | 3.8×10²¹ FLOPs可达Acc_solved≈0.8 |
| 最优token-参数比 | ≈15，随计算量略有上升 |
| 最优batch size | 随模型规模增长 |
| 最优学习率 | 随模型规模增长（与LLM发现相反！） |

### 关键发现

- 求解率和损失**严格遵循幂律**关系，与LLM中观察到的缩放定律一致
- 最优学习率的**上升趋势**与LLM研究结论相反，展示了任务间训练特性的差异
- 精确匹配（Acc_solved）比R²>0.99的改善慢得多，凸显精确表达式匹配的难度
- 最大模型测试时未见性能饱和迹象

## 亮点与洞察

- **首次**在符号回归中建立缩放定律，为未来模型设计提供可预测性
- 合成数据的系统化生成避免了表达式重复和偏差
- 编码器中行/列交叉注意力的引入是从表格基础模型到符号回归的有效迁移
- 结果表明：通过简单增加计算量就能显著提升符号回归性能

## 局限性 / 可改进方向

- 限制在最多2变量+整数常数的表达式，现实任务通常涉及更多变量和浮点常数
- 计算范围有限（最大93M参数），外推到更大规模的预测未经验证
- 每个配置仅单次运行（无多种子统计）
- 未与其他符号回归方法比较（因为自定义setup下比较困难）
- 数据生成中的规范化和去重策略可能引入隐含偏差

## 相关工作与启发

- 继承Kamienny等人的端到端范式，但焦点从损失工程转向规模研究
- 方法论上参考Hoffmann等人（Chinchilla）和Kaplan等人的LLM缩放定律研究
- 为未来结合更好的数据生成+架构+缩放来超越所有现有方法指明方向

## 评分

⭐⭐⭐ — 研究问题新颖，实验设计清晰，首次建立符号回归缩放定律有探索价值，但规模和适用范围有限。
