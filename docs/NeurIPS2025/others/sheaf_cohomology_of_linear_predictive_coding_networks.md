# Sheaf Cohomology of Linear Predictive Coding Networks

**会议**: NeurIPS 2025 (Workshop: Symmetry and Geometry in Neural Representations)
**arXiv**: [2511.11092](https://arxiv.org/abs/2511.11092)
**代码**: 无
**领域**: 理论深度学习, 代数拓扑, 预测编码
**关键词**: 预测编码, 细胞层(Sheaf), 上同调, Hodge理论, 层Laplacian

## 一句话总结
将线性预测编码(PC)网络形式化为细胞层(cellular sheaf)结构，利用层上同调和Hodge分解分析循环拓扑中的"内部矛盾"如何阻碍学习，揭示权重初始化的全局接线模式决定网络可学习性。

## 研究背景与动机
- 预测编码(PC)将神经网络训练重新定义为对权重和激活的优化，用局部损失替代全局反向传播
- PC可处理任意循环拓扑而无需展开计算图，但引入了根本性歧义：深层循环节点从所有连接接收误差信号——监督信号和矛盾反馈循环混杂，无法区分
- 本文利用细胞层(cellular sheaf)理论系统分析这一问题，提供诊断工具和设计原则

## 方法详解

### 整体框架
**预测编码层(Predictive Coding Sheaf)**：
- 将激活组织为0-上链(节点)，预测误差为1-上链(边)
- 余边界算子$\delta^0$同时计算所有预测误差
- PC推理 = 层Laplacian下的扩散

### 关键设计

#### 层结构定义
- 顶点$v$赋予向量空间$\mathcal{F}(v) = \mathbb{R}^{n_v}$（神经元激活）
- 边$e$赋予向量空间$\mathcal{F}(e) = \mathbb{R}^{m_e}$（预测误差）
- 限制映射编码网络权重：$\rho_{e\leftarrow u} = W_e$, $\rho_{e\leftarrow v} = I$

#### 上同调群
- $H^0$：零预测误差的激活模式（全局一致的解）
- $H^1$：无法由任何激活选择产生的误差模式（**不可约误差**）

#### 相对系统与钳位网络
- 输入/输出节点钳位到数据/标签 → 离散Dirichlet问题
- Hodge分解：目标$b = (-Dz^*) + r^*$
  - $-Dz^*$：推理可消除的部分
  - $r^*$：推理无法消除的残差（$\in \ker D^\top$）

#### 谐波-扩散算子
- 谐波投影$\mathcal{H} = I - DD^\dagger$：分布监督信号到边的谐波子空间
- 扩散算子$\mathcal{G} = D^\dagger$：将激励扩散到内部顶点
- **学习需要谐波-扩散重叠**：边$e$的梯度 = 谐波负载(边) × 扩散激活(顶点)

### 损失函数 / 训练策略
- PC能量：$E_{PC}(s) = \frac{1}{2}\|\delta^0 s\|^2$
- 推理：固定$W$，最小化$E_{PC}$（关于激活$h$）
- 学习：固定$h$，更新$W$（单步梯度下降）
- 边$e$的权重梯度：$\partial E / \partial W_e = -r_e s_u^\top$

## 实验关键数据

### 主实验：10节点循环网络

| 初始化角度$\theta$ | 含义 | 1000步后验证MSE |
|---|---|---|
| $\theta = 0$（共振） | 反馈循环自我强化 | ≤ 0.001 ✓ |
| $\theta < 0.4$ | 适度张力 | ≤ 0.001 ✓ |
| $\theta > 0.4$ | 强内部矛盾 | 极慢收敛 ✗ |
| $\theta = \pi$（矛盾） | 反馈翻转信号 | 学习停滞 ✗ |

### 谐波-扩散分析

| $\theta$ | 谐波负载分布 | 扩散激活分布 | 权重更新 |
|---|---|---|---|
| 0 (共振) | 均匀分布于所有边 | 均匀分布于所有顶点 | 处处强更新 |
| $\pi$ (矛盾) | 集中于$h_1$/$h_{10}$的内部边 | $h_1$/$h_{10}$被阻断 | 相邻边饿死 |

### 关键发现
- 可学习性取决于权重的**全局接线模式**而非局部权重尺度（所有初始化都是正交归一的）
- 矛盾反馈($\theta = \pi$)使谐波负载集中在扩散无法到达的位置→梯度饿死
- 全连接变体(all-to-all)中只有≤5节点的网络能在1000步内收敛
- Laplacian的条件数$\kappa$可预测推理动力学的困难程度

## 亮点与洞察
1. **优雅的数学对应**：PC推理 = 层扩散，预测误差 = 上链，全局一致性 = 上同调
2. **诊断工具**：层上同调可判断网络配置是否存在"不可约内部矛盾"
3. **设计原则**：权重初始化应保证反馈循环的单值映射(monodromy)接近恒等
4. **关键洞察**：学习的瓶颈不在于局部权重大小，而在于循环的全局方向性一致性

## 局限性 / 可改进方向
- 仅分析线性网络——非线性网络需用Jacobian替代权重矩阵
- 当前关注收敛后的推理最优解——实际中需考虑推理动力学的条件数
- 未讨论精度(precision)加权的推广
- 需扩展到超图上的层理论以处理神经元级误差聚合
- 10节点实验规模较小，未在更大/更深的网络上验证
- 缺乏与实际PC训练算法的集成验证（如结合提出的初始化原则训练真实任务）

## 相关工作与启发
- 层神经网络(Hansen 2020, Bodnar 2022)用层扩散替代GNN消息传递——但作用于数据图
- PC网络的深度缩放(Ishikawa, Innocenti)——处理前馈架构
- 本文首次系统地将层理论应用于PC的**计算图**，揭示循环拓扑的学习障碍
