# Sensorium Arc: AI Agent System for Oceanic Data Exploration and Interactive Eco-Art

**会议**: NeurIPS 2025
**arXiv**: [2511.15997](https://arxiv.org/abs/2511.15997)
**代码**: 无
**领域**: 人机交互, 生态艺术, 多智能体系统
**关键词**: 海洋数据可视化, RAG, 多智能体LLM, 交互式生态艺术, 对话式AI

## 一句话总结
构建了一个多模态交互式AI智能体系统"Sensorium Arc"，将海洋拟人化为诗意的讲述者，通过多智能体RAG架构将NASA海洋数据转化为沉浸式的视听体验，引导用户探索复杂海洋环境数据。

## 研究背景与动机
- 海洋拥有海量公开环境数据集（如NASA EarthData），但缺乏直觉、情感化的访问方式
- 传统数据可视化将海洋数据呈现为抽象数据集，而非"活的叙事"
- 受生态美学家Newton Harrison"赋予世界海洋声音"理念启发
- 核心问题："如果你能问海洋一个问题，它会说什么？"
- 早期沉浸式版本（VR、装置）反馈：观众渴望直接与海洋对话

## 方法详解

### 整体框架
模块化四层架构：输入处理 → LLM管线 → 响应处理 → 视听层

### 关键设计

#### 输入处理
- 鹦鹉螺壳形交互模块(Arduino控制)，含距离传感器+LED+麦克风
- 用户接近50cm范围内，蓝光波纹闪烁表示就绪
- 语音录制后通过Whisper-tiny进行语音转文字

#### 多智能体LLM管线
1. **可视化决策智能体**：小型无状态决策器(Llama 3.2 3B)，根据用户查询选择匹配的数据可视化。使用GBNF语法约束输出格式
2. **检索+查询重写智能体**：Qwen 8B重写用户查询→all-MiniLM-L12-v2嵌入→ANN语义搜索(k=2)Harrison档案文本库。采用Chain-of-Thought提示
3. **响应者智能体**：最终LLM整合可视化描述+检索段落+用户查询，以"海洋人格"生成诗意回复

#### RAG知识库
- Harrison夫妇的生态美学文本库：《The Time of the Force Majeure》、"Apologia Mediterraneo"等
- 非结构化文档→句子分割→384维向量空间(all-MiniLM-L12-v2)→本地向量数据库

#### 视听层
- NASA EarthData科学数据集：大气CO₂、叶绿素浓度、海表温度、洋流等
- 关键词触发：根据对话中的地点/时间/主题线索动态触发可视化和视频播放
- 语音合成：Jets TTS模型(Unity Sentis)

### 训练策略
- 全部基于预训练模型，无需额外训练
- 硬件：i9-13900HX + 64GB RAM + RTX 4090 Laptop，CUDA加速
- 平均响应延迟 < 4秒
- Unity 6000.0.24f1引擎整合所有模块

## 实验关键数据

### 系统性能指标

| 模块 | 技术栈 | 性能 |
|-----|-------|------|
| 语音识别 | Whisper-tiny | 实时转写 |
| 可视化决策 | Llama 3.2 3B + GBNF约束 | Few-shot推理 |
| 查询重写 | Qwen 8B + CoT | 强规则遵循 |
| RAG检索 | all-MiniLM-L12-v2 + ANN | k=2，避免上下文窗口溢出 |
| 响应生成 | 大型LLM | 零样本提示 |
| TTS | Jets (Unity Sentis) | 实时 |
| 端到端延迟 | - | < 4秒 |

### 展览反馈
| 维度 | 观察结果 |
|-----|---------|
| 感官参与度 | 高——用户被沉浸式体验吸引 |
| 对话需求 | 强——观众主动寻求直接与海洋对话 |
| 情感共鸣 | 科学准确性与诗意叙事的结合增强环境共情 |

### 关键发现
- 多智能体架构相比单一大模型提供更好的可调试性、可适应性和细粒度控制
- RAG的k=2效果最佳——更多段落会填满上下文窗口导致响应质量下降
- 体化交互（鹦鹉螺壳界面）将被动观察转化为主动共创的生态对话
- 系统可在Windows和macOS上部署，适合学校和公共机构的教育安装

## 亮点与洞察
1. **拟人化数据访问**：将海洋数据从抽象数据集转化为"活的叙事"，重新定义人机生态交互
2. **模块化多智能体设计**：分离增强灵活性——可独立更新数据集、语言支持、上下文
3. **RAG用于诗意基础**: 首次将RAG从事实检索扩展到"诗意语调基础"——通过Harrison档案塑造海洋人格
4. **体化设计**: 鹦鹉螺壳界面将数字交互与物理手势结合

## 局限性 / 可改进方向
- RAG可靠性：查询扩展(生成多个语义不同的重写)可增加检索多样性
- LLM推理与Unity实时渲染竞争GPU资源——未来可将推理卸载到云端
- 当前视频为预渲染——计划替换为实时渲染以支持用户交互
- 需要进行控制实验验证知识保留率、共情和行为改变
- 缺乏多语言支持

## 相关工作与启发
- 沉浸式气候参与工具：Rising Together, Sea Level Rise Explorer等
- 对话式AI扩展到创意和体化应用：ChatCam, USER-LLM等
- RAG在环境数据中的应用：ClimateQA, NASA EarthData
- 与传统数据可视化的"被动呈现"模式形成对比
