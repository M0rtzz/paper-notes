# UGM2N: An Unsupervised and Generalizable Mesh Movement Network via M-Uniform Loss

**会议**: NeurIPS 2025  
**arXiv**: [2508.08615](https://arxiv.org/abs/2508.08615)  
**代码**: 无  
**领域**: 3D视觉  
**关键词**: 网格自适应, 偏微分方程, 无监督学习, 网格移动, 零样本泛化

## 一句话总结
提出 UGM2N，通过局部化节点 patch 特征学习和 M-Uniform 损失函数实现无监督网格移动，无需预适应网格数据即可实现跨方程和跨几何的零样本泛化。

## 研究背景与动机
**领域现状**：PDE 数值求解依赖网格质量，网格移动（r-adaptation）通过重新定位节点到快速变化区域来提升精度和效率

**现有痛点**：传统 Monge-Ampère 方法计算成本高；监督学习方法（M2N/UM2N）依赖预适应网格数据，泛化能力受限

**核心 idea**：借鉴 Vision Transformer 的 patch 概念，将每个节点及其一阶邻居定义为 "node patch"，并行独立计算适应位置；M-Uniform 损失在节点级别强制等分布性质

## 方法详解

### 关键设计
1. **Node Patch 表示**：局部归一化的节点+一阶邻居作为输入，尺度不变，支持任意网格拓扑
2. **M-Uniform 损失**：在 patch 级别编码局部等分布属性（网格移动方法的核心目标），无需监督标签
3. **并行处理**：每个 patch 独立计算适应位置，简化学习目标并实现并行化

## 实验关键数据

### 主实验
| 方法 | 泛化能力 | 需要预适应数据 | 网格缠绕 |
|------|---------|-------------|---------|
| M2N | 需重训练 | 是 | 有风险 |
| UM2N | 有限泛化 | 是 | 有风险 |
| **UGM2N** | **跨方程+跨几何** | **否** | **无** |

### 关键发现
- 在未见过的 PDE 和网格几何上保持鲁棒，误差一致降低且无网格缠绕
- 可扩展到多尺度分辨率

## 亮点与洞察
- **无监督+零样本泛化**的组合在网格自适应领域是第一个
- Node patch 概念优雅地将局部几何与全局自适应解耦

## 局限性
- 仅处理2D网格，3D体网格扩展待验证
