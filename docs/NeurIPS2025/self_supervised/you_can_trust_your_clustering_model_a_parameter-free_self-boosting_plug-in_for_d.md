# You Can Trust Your Clustering Model: A Parameter-free Self-Boosting Plug-in for Deep Clustering

**会议**: NeurIPS 2025  
**arXiv**: [2511.21193](https://arxiv.org/abs/2511.21193)  
**代码**: [有](https://github.com/l-h-y168/DCBoost)  
**领域**: 深度聚类 / 无监督学习  
**关键词**: 深度聚类, 自增强插件, 局部结构, 全局结构优化, 无参数

## 一句话总结

提出 DCBoost，一个无需额外超参数的即插即用模块，通过自适应 k-NN 筛选高置信样本并利用可靠的局部结构信息引导全局特征空间优化，显著提升现有深度聚类模型的性能。

## 研究背景与动机

深度聚类旨在利用深度网络在无标签数据中发现内在结构。近年来，自监督学习（对比学习、非对比学习）与聚类的结合取得了很大进展，代表方法包括 ProPos、CC、SCAN 等。

然而，作者观察到现有方法存在一个被忽视的共性问题：**全局结构与局部结构之间存在显著差异**。

- **局部结构质量高**：k-NN 准确率往往很高（甚至超过聚类准确率），说明语义相似的样本在特征空间中确实被局部地聚集在一起。
- **全局结构质量差**：类内相似度低、类间相似度高、轮廓系数（Silhouette Coefficient）极低（如 ProPos 在 CIFAR-10 上仅 0.10），说明不同类别的决策边界模糊、特征空间纠缠严重。

t-SNE 可视化也证实了这一点：虽然能观察到粗略的簇结构，但类间存在大量重叠。这一观察启发了作者：**能否利用可靠的局部结构来修复不可靠的全局结构？**

## 方法详解

### 整体框架

DCBoost 的核心思路是"用局部带全局"：

1. 以任意预训练的深度聚类模型作为**目标网络** $f_t(\cdot)$，复制一份作为**在线网络** $f_o(\cdot)$，再附加一个随机初始化的非线性预测器 $g(\cdot)$
2. 对每个样本施加两种弱增强生成两个视图，分别经过在线网络和目标网络获得 L2 归一化特征 $z^o$ 和 $z^t$
3. 每个 batch 中，通过自适应 k-NN 筛选高置信样本（局部结构挖掘）
4. 利用高置信样本构建判别性损失微调模型（局部引导全局优化）
5. 每个 epoch 结束后对目标网络的所有样本输出做 k-means 获取伪标签
6. 目标网络通过 EMA（指数移动平均）从在线网络更新参数

### 关键设计

**自适应 k-NN 高置信样本筛选**：

核心思想是选择那些 k 个最近邻全部共享相同伪标签的样本作为高置信样本。关键挑战在于 k 值的选取——k 太小则纳入噪声样本，k 太大则高置信样本过少。

作者提出了一个无参数的自适应选择策略：

- 设候选集 $\mathbb{K} = \{1, 2, \ldots, 50\}$
- 对每个候选 $k_s$，计算得分：$\text{score}_{k_s} = k_s \times \frac{n_s}{n_B}$，其中 $n_s$ 是满足条件的高置信样本数，$n_B$ 是 batch 大小
- 选取得分最高的 $k_s$ 作为当前 batch 的 k 值

这个得分可以几何地理解为以 k 为宽、高置信比例为高的矩形面积，在质量和数量之间自动取得平衡。训练早期 k 较小（纳入更多样本），后期 k 增大（聚焦高质量样本）。

### 损失函数 / 训练策略

总损失由三项组成：$L = L_{pos} + L_{neg} + L_{ins}$

**1. 正样本损失 $L_{pos}$（类内紧凑性）**：

$$L_{pos} = \frac{1}{2c_B} \sum_{c=1}^{c_B} \sum_{i,j \in \mathbb{X}_c} w_c d_{ij}^2$$

拉近同一伪标签的高置信样本对。其中 $w_c$ 是类别平衡权重，用于缓解高置信样本的类别不平衡问题：

$$w_c = \frac{1}{\|\sum_{x \in \mathbb{X}_c} z^o\|_2 \cdot \|\sum_{x \in \mathbb{X}_c} z^t\|_2}$$

样本数多的类别聚合特征范数大，权重自然降低，避免主导训练。

**2. 负样本损失 $L_{neg}$（类间分离性）**：

$$L_{neg} = -\sum_{c_1 \neq c_2} \|v_{c_1}^o - v_{c_2}^t\|_2^2$$

通过最大化不同类别原型之间的欧氏距离来增强类间分离，原型由高置信样本的归一化聚合特征计算得到。

**3. 实例一致性损失 $L_{ins}$**：

$$L_{ins} = \|g(f_o(\mathcal{T}^1(x)) + \sigma\varepsilon) - f_t(\mathcal{T}^2(x))\|_2^2$$

对所有样本（不仅是高置信样本）施加，保持实例级一致性，防止表征坍塌。其中加入微量高斯噪声（$\sigma=0.001$）增强鲁棒性。

## 实验关键数据

### 主实验

在 5 个基准数据集、6 个基线模型上验证，DCBoost 作为插件一致提升所有模型性能：

| 方法 | CIFAR-10 ACC | CIFAR-20 ACC | STL-10 ACC | ImageNet-10 ACC | ImageNet-Dogs ACC | 平均提升 |
|------|-------------|-------------|-----------|----------------|------------------|---------|
| CC | 85.2 | 41.7 | 80.0 | 89.9 | 69.6 | - |
| CC+DCBoost | 88.2 | 48.9 | 80.8 | 90.7 | 70.6 | +3.1% |
| BYOL | 87.5 | 52.3 | 86.1 | 94.7 | 72.9 | - |
| BYOL+DCBoost | 91.5 | 54.7 | 90.2 | 95.7 | 77.1 | +4.6% |
| ProPos | 94.4 | 61.6 | 91.6 | 95.8 | 76.9 | - |
| ProPos+DCBoost | **96.0** | **63.9** | **93.6** | **97.1** | **79.7** | +3.0% |
| CDC | 94.7 | 61.6 | 93.0 | 97.3 | 79.2 | - |
| CDC+DCBoost | 95.1 | 62.7 | 93.4 | 97.3 | 79.7 | +0.8% |

核心发现：基线越弱的模型提升越大（BYOL +4.6%），强基线 ProPos 也能提升 3.0%，轮廓系数从 0.10 提升至 0.74（7× 以上）。

### 消融实验

在 ProPos 基础上的消融结果（CIFAR-10 / CIFAR-20 / STL-10 平均）：

| 自适应筛选 | $L_{ins}$ | $L_{pos}$ | $L_{neg}$ | CIFAR-10 ACC | CIFAR-20 ACC | STL-10 ACC | 平均 |
|-----------|-----------|-----------|-----------|-------------|-------------|-----------|------|
| - | - | - | - | 94.4 | 61.6 | 91.6 | 77.2 |
| ✗ | ✓ | ✓ | ✗ | 94.2 | 57.7 | 90.6 | 75.7 |
| ✓ | ✓ | ✓ | ✗ | 95.6 | 61.8 | 92.2 | 78.7 |
| ✓ | ✓ | ✗ | ✓ | 94.4 | 59.1 | 92.0 | 76.5 |
| ✓ | ✗ | ✓ | ✓ | 91.1 | 62.5 | 20.5 | 51.9 |
| ✗ | ✓ | ✓ | ✓ | 94.9 | 62.5 | 92.0 | 78.3 |
| ✓ | ✓ | ✓ | ✓ | **96.0** | **63.9** | **93.6** | **80.4** |

### 关键发现

1. **所有损失项缺一不可**：移除 $L_{ins}$ 后 STL-10 ACC 从 93.6% 暴跌至 20.5%，说明实例一致性对防止坍塌至关重要；移除 $L_{pos}$ 则几乎无性能提升；移除 $L_{neg}$ 也有明显下降
2. **自适应筛选至关重要**：不做筛选直接用所有损失，平均性能从 80.4% 降至 78.3%；加入筛选后即使损失不完整也能显著提升（75.7% → 78.7%）
3. **类别平衡权重有效**：无权重 $w_0$ 最差，带权重但不参与梯度的 $w_1$ 显著提升，完整的 $w_{ours}$ 最优
4. **对 CLIP 模型同样有效**：在 SIC 和 TAC 上也带来一致提升（如 TAC 在 CIFAR-20 上 ACC 从 61.5% 提升到 62.9%）
5. **优于同类方法 CoNR**：与 CoNR 相比，DCBoost 的提升更大且更稳定，因为 CoNR 仅利用小范围正样本对，而 DCBoost 更充分地利用局部结构信息
6. **自适应 k 选择优于手动设定**：自适应方法在 CIFAR-10 达到 96.0%（手动最优 k=30 为 95.9%），且不存在选错 k 导致性能大幅下降的风险

## 亮点与洞察

- **洞察深刻**：首次明确指出深度聚类中"局部结构可靠、全局结构不可靠"这一被忽视的现象，并将其转化为可行的改进方案
- **设计优雅**：无需引入任何新的超参数，自适应 k-NN 的得分函数（k × 比例）简洁而有效，几何直觉清晰
- **通用性强**：适用于表征式和聚类头式两大类深度聚类方法，甚至对 CLIP-based 方法也有效
- **零额外成本**：仅在 mini-batch 内做 k-NN 搜索，避免了全局 k-NN 的开销

## 局限与展望

1. **假设簇数已知**：实际场景中簇数通常未知，需要额外的簇数估计机制
2. **类别不平衡鲁棒性不足**：虽然引入了平衡权重，但高度不平衡或非均匀分布的数据可能仍有挑战
3. **大规模数据扩展**：目前主要在中等规模数据集上验证（最大 10 万样本），更大规模数据的效果待验证
4. **仅用弱增强**：两个视图都用弱增强，是否引入强增强或混合增强策略能进一步提升值得探索
5. **高置信样本的边界效应**：靠近类边界的样本始终被排除，可能错失有用的难样本信息

## 相关工作与启发

- **ProPos**（TPAMI 2023）：当前 SOTA 深度聚类方法之一，通过原型散射和正采样学习表征，是本文的主要增强目标
- **BYOL**（NeurIPS 2020）：非对比自监督学习，使用 EMA 更新目标网络，DCBoost 借鉴了类似的双网络结构
- **CoNR**（与本文有相似动机）：利用上下文线索增强类内紧凑性，但提升幅度不如 DCBoost
- **启发**：该方法的"局部→全局"思路可推广到其他无监督/半监督任务，如无监督域适应中也常存在局部特征可靠但全局分布不对齐的问题
