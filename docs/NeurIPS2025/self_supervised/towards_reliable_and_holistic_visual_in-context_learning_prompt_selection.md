# Towards Reliable and Holistic Visual In-Context Learning Prompt Selection

**会议**: NeurIPS 2025  
**arXiv**: [2509.25989](https://arxiv.org/abs/2509.25989)  
**代码**: [GitHub](https://github.com/Wu-Wenxiao/RH-Partial2Global)  
**领域**: 视觉上下文学习, 提示选择  
**关键词**: 视觉上下文学习, 共形预测, 覆盖设计, 提示选择, 全局排序

## 一句话总结

本文提出RH-Partial2Global方法，通过Jackknife共形预测构建可靠的候选集，并利用覆盖设计实现全面均匀的成对偏好采样，提升视觉上下文学习中的提示选择质量。

## 研究背景与动机

- 视觉上下文学习（VICL）通过条件化少量上下文示例来适配视觉基础模型
- 核心挑战：如何为给定查询选择最合适的上下文示例
- 现有方法（VPR、Partial2Global）依赖**相似性优先假设**：视觉上更相似的图像是更好的上下文示例
- 本文发现：虽然相似性与质量存在统计显著单调关系，但**相关系数极低**（平均Spearman ρ̄ ≈ 0.03-0.05）
- Partial2Global的随机采样策略导致**覆盖不完整与冗余采样**，损害全局排序精度

## 方法详解

### 整体框架

RH-Partial2Global在Partial2Global基础上引入两个增强：
1. **可靠性策略**（R）：基于Jackknife共形预测筛选可靠候选
2. **全面性策略**（H）：基于覆盖设计的结构化采样

### 关键设计

**1. Jackknife共形预测引导的候选选择**

- 对训练集中每个样本x_i^trn，作为提示应用于其他样本，得到质量分数集Q和相似性分数集S
- 计算一致性分数 ℓ(x_i) = f(Q(x_i), S(x_i))，其中f为负KL散度或Spearman相关
- 在所有一致性分数上计算(1-α)分位数 q_{1-α}
- 构建可靠集 Y_α = {x_i | ℓ(x_i) > q_{1-α}}
- 对查询x_q：初始候选集Y_q（按相似性top-K）∩ 可靠集Y_α = 精炼候选集Y_q*

**2. 覆盖设计引导的全面采样**

- 问题：K=50个候选、k=5的局部排序器，需要C(50,5,2)覆盖设计保证所有成对关系被至少一个子序列覆盖
- Schonheim下界：C(50,5,2) ≥ 130个不同子序列（原方法仅50个随机子序列）
- 方案：使用预计算的最优覆盖设计C*(K',k,t)引导从随机打乱的候选集中采样k长度序列
- 覆盖设计天然保证：(1) 所有成对关系的穷尽覆盖；(2) 最小化采样迭代次数

### 损失函数 / 训练策略

- 与Partial2Global相同的元学习训练策略
- α = 0.85（85%置信度），适用于所有任务
- 一致性函数采用负KL散度
- 特征提取器：DINOv2；相似性计算：CLIP视觉编码器

### 关键发现

- 相似性优先假设虽然统计显著（78-88%样本p<0.05），但**相关强度极弱**（ρ̄ ≈ 0.03-0.05）
- 精炼后丢弃约15%候选，但高质量示例的性能上界几乎不变
- RH-Partial2Global在所有任务和所有fold上**一致性改进**，无额外模型训练
- 可视化显示：RH选择的示例在姿态、场景上下文等细粒度属性上与查询更对齐

## 亮点与洞察

- 首次对VICL中相似性优先假设提供**统计证据**证明其不够稳健
- 共形预测为构建可靠候选集提供了理论保证，且与具体任务无关
- 覆盖设计是解决"如何系统性采样"的优雅数学工具
- 方法无需额外模型训练，仅修改推理阶段的候选选择和采样策略

## 局限性 / 可改进方向

- 改进幅度虽一致但较小（平均~0.6%），在小fold上改进更有限
- 共形预测的效果依赖于校准集大小，小数据集上可能不稳健
- α=0.85的选择可能需要针对不同任务调优
- 覆盖设计的预计算对大规模候选集可能有额外开销
- 未探索自适应α或自适应候选集大小的策略

## 相关工作与启发

- 对VPR和Partial2Global的自然改进，核心贡献在于挑战了基础假设
- 共形预测在ML中的应用趋势的一个新颖实例
- 覆盖设计从组合数学到ML的有趣迁移
