# Text-to-Image Models Leave Identifiable Signatures: Implications for Leaderboard Security

**会议**: NeurIPS 2025
**arXiv**: [2510.06525](https://arxiv.org/abs/2510.06525)
**代码**: 无
**领域**: AI安全, 图像生成
**关键词**: 文生图模型, 排行榜安全, 模型去匿名化, CLIP嵌入, 对抗性操纵

## 一句话总结

本文揭示了文生图（T2I）模型在生成图像中留下可识别的"签名"，使得攻击者可以在投票式排行榜中通过简单的CLIP嵌入空间分类实现模型去匿名化，从而操纵模型排名。

## 研究背景与动机

- 生成式AI排行榜（如Chatbot Arena的图像版本）已成为评估模型能力的核心平台，但面临**排名操纵**攻击的威胁
- 排名操纵的关键步骤是**模型去匿名化**——识别排行榜上匿名展示输出背后的具体模型
- 此前研究主要针对LLM排行榜的去匿名化，通常需要控制输入提示或历史数据来训练分类器
- 本文假设：T2I模型由于训练数据、架构和规模的差异，会产生系统性的风格差异，导致不同模型的生成图像在嵌入空间中形成可区分的聚类
- 与LLM相比，T2I排行榜的去匿名化可能**更容易**，因为视觉输出的"指纹"更为显著

## 方法详解

### 整体框架

提出基于质心的去匿名化算法（Algorithm 1）：

1. 对每个排行榜提示p，向每个候选模型M_i发送并生成k张图像
2. 将所有图像和排行榜提供的图像映射到CLIP嵌入空间
3. 计算每个模型k个嵌入的质心c_i
4. 计算排行榜图像嵌入与各质心的距离
5. 距离最小的模型即为预测的生成模型

### 关键设计

**可区分性度量**（Distinguishability Metric）：

- **模型级可分离性**：对每个提示p和模型M_j，检查其k个嵌入的最近邻是否来自同一模型，计算正确聚类比例 frac(i,j)
- **提示级可区分性**：D(i) = 所有模型中形成可分离聚类的比例，高D(i)意味着该提示下模型容易被区分

### 实验设置

- **数据规模**：19个多样化T2I模型（涵盖OpenAI、Midjourney、Stability AI等多个组织），280个提示，总共生成超过15万张图像
- **评估指标**：Top-k准确率（k∈[1,5]）
- **嵌入**：CLIP嵌入，图像统一缩放到224×224

## 实验关键数据

### 主实验

| 指标 | 数值 |
|------|------|
| Top-1准确率 | ~87% |
| Top-3准确率 | ~95% |
| 随机猜测基线 | ~5.26% (1/19) |
| k=1时（单张生成）准确率 | ~57% |
| 提示可控攻击Top-1准确率 | ~99% |
| 一对多分类准确率 | ~99% |

### 消融实验

| 分析维度 | 发现 |
|----------|------|
| 生成数k的影响 | k=1时57%，k增大性能显著提升，k>10-15后收敛 |
| 架构/尺寸影响 | SD 3.5 large vs medium误分类率仅~3%，两个Flux变体~3.8% |
| 可区分性分数分布 | 部分提示达到完美分数1.0（所有模型完全可分），部分仅0.21 |
| 最佳模型识别 | HiDream和SDXL Turbo在一对多分类中达100%准确率 |

### 关键发现

- 不同模型对**同一提示**生成的图像具有低模内多样性和高模间差异性（Fig.1直观展示）
- 即使**无需控制提示**，简单的实时嵌入空间分类即可高精度去匿名化
- 高可区分性分数与高去匿名化准确率正相关
- 攻击者只需少量生成样本即可建立有效的模型指纹库

## 亮点与洞察

- **根本性张力**：赋予模型竞争力的独特视觉签名，恰恰也使去匿名化攻击成为可能
- 方法极其简单——仅用CLIP嵌入和最近质心分类，无需训练任何分类器
- 揭示了T2I排行榜比LLM排行榜面临**更严重**的安全威胁
- 去匿名化还可作为下游攻击的前置步骤（如选择代理模型进行提示优化攻击）

## 局限性 / 可改进方向

- 当前防御措施（如轮换提示）效果有限，因为模型在未见过的提示上仍高度可区分
- 论文未深入探讨有效的防御方案
- 实验基于静态模型集合，未考虑模型持续更新的动态场景
- 可能的防御方向：分析投票模式异常检测、限制每个提示展示的生成数量

## 相关工作与启发

- 与LLM去匿名化工作（LLMmap等）形成对比——T2I模型的去匿名化更简单、更实时
- GAN指纹归因研究的自然扩展到扩散模型时代
- 对排行榜安全领域提出了需要更强防御机制的警示

## 评分

⭐⭐⭐ — 研究问题重要且实用，方法简洁有效，但方法创新性有限，主要是实验驱动的发现型工作。
