# Training-Free Efficient Video Generation via Dynamic Token Carving (Jenga)

**会议**: NeurIPS 2025  
**arXiv**: [2505.16864](https://arxiv.org/abs/2505.16864)  
**代码**: [GitHub](https://github.com/dvlab-research/Jenga)  
**领域**: 图像/视频生成  
**关键词**: Video DiT, Token剪裁, 推理加速, 空间填充曲线, 渐进分辨率

## 一句话总结

提出 Jenga，一种即插即用的视频扩散 Transformer 推理加速方案，通过动态注意力剪裁和渐进分辨率生成实现 8.83× 加速，VBench 性能仅下降 0.01%。

## 研究背景与动机

视频扩散 Transformer (Video DiT) 模型虽然生成质量出色，但推理速度极慢，主要瓶颈来自：

**自注意力的二次复杂度**: token 数量与视频帧数×分辨率成正比，导致注意力计算量巨大

**多步扩散**: 扩散模型需要多步去噪，每步都要进行完整的前向计算

作者观察到两个关键现象：
- 早期去噪步骤不需要高分辨率潜表示（粗粒度结构先确定）
- 后期步骤不需要密集的全局注意力（细节调整主要是局部的）

## 方法详解

### 整体框架

Jenga 推理管线包含两个核心组件：
1. **动态注意力剪裁 (Dynamic Attention Carving)**: 基于 3D 空间填充曲线选择相关 token 交互
2. **渐进分辨率生成 (Progressive Resolution Generation)**: 逐步提升潜表示的空间分辨率

### 关键设计

1. **块级注意力机制 (Block-wise Attention)**:

    - 将视频 token 沿时空维度划分为局部块
    - 利用 3D 空间填充曲线（如 Hilbert 曲线）对 token 排序
    - 每个 token 仅与同一曲线段内的临近 token 做注意力
    - 关键优势：保持了时空局部性，同时引入跨帧交互

2. **动态稀疏度调度**:

    - 早期步骤：使用全局稀疏注意力（捕获粗粒度结构）
    - 后期步骤：使用更稀疏的局部注意力（细化细节）
    - 稀疏度随去噪步骤动态调整

3. **渐进分辨率策略**:

    - 前 N 步在低分辨率潜空间进行去噪
    - 逐步上采样到目标分辨率
    - 最后几步在全分辨率进行精细化

### 损失函数 / 训练策略

**完全无需训练 (Training-Free)**—— Jenga 只修改推理流程，不改变模型参数，可直接应用于任何预训练的 Video DiT。

## 实验关键数据

### 主实验（VBench 评测）

| 方法 | VBench 总分 ↑ | 加速比 ↑ | GPU 内存 (GB) ↓ | 训练需求 |
|------|-------------|---------|---------------|---------|
| CogVideoX (基线) | 82.14 | 1.0× | 48.2 | — |
| TokenMerge | 81.53 | 2.1× | 38.5 | 无 |
| FastVideo | 81.89 | 3.2× | 42.1 | 有 |
| PAB | 81.72 | 2.8× | 40.3 | 无 |
| Delta-DiT | 81.95 | 3.5× | 39.8 | 无 |
| **Jenga** | **82.13** | **8.83×** | **24.6** | **无** |

### 跨模型泛化实验

| 基座模型 | 原始时间 (s) | Jenga 时间 (s) | 加速比 | VBench Δ |
|---------|------------|-------------|-------|---------|
| CogVideoX-5B | 342 | 38.7 | 8.83× | -0.01% |
| Open-Sora | 285 | 41.2 | 6.92× | -0.15% |
| Latte | 198 | 32.5 | 6.09× | -0.23% |
| HunyuanVideo | 425 | 52.1 | 8.16× | -0.08% |

### 消融实验

| 组件 | VBench | 加速比 |
|------|--------|-------|
| 完整 Jenga | 82.13 | 8.83× |
| 仅动态注意力 | 82.08 | 4.12× |
| 仅渐进分辨率 | 82.11 | 3.85× |
| 固定稀疏度(不动态) | 81.65 | 7.21× |
| 随机 token 选择 | 80.32 | 8.56× |

### 关键发现

1. 两个组件协同作用实现最大加速，单独使用各约 4×
2. 3D 空间填充曲线远优于随机 token 选择（82.13 vs 80.32）
3. 推理时间从分钟级降到秒级，使真实部署成为可能
4. 在多个 Video DiT 模型上即插即用有效

## 亮点与洞察

- **惊人加速比**: 8.83× 加速且几乎无损，在推理加速工作中非常突出
- **空间填充曲线**: 利用 Hilbert 曲线等数学工具解决 3D token 排序问题，思路优雅
- **即插即用**: 无需重新训练，实用价值极高
- **有代码**: 开源代码方便复现和应用

## 局限性 / 可改进方向

1. 在极长视频（>100帧）生成中，动态注意力的时空局部性假设可能不成立
2. 渐进分辨率策略的调度超参需要针对不同模型微调
3. 对于需要全局一致性的场景（如长距离物体运动），可能影响质量
4. 理论分析相对薄弱，主要依赖经验观察

## 相关工作与启发

- **TokenMerge (Bolya et al., 2023)**: 通过合并相似 token 加速，但加速比有限
- **PAB**: 渐进注意力广播，与本文渐进思路有共通之处
- **Delta-DiT**: 利用相邻步骤差异减少计算，互补方法
- **FlashAttention**: 底层注意力加速，可与 Jenga 叠加使用
