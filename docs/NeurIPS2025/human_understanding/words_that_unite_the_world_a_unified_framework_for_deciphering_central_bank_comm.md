# Words That Unite The World: A Unified Framework for Deciphering Central Bank Communications Globally

**会议**: NeurIPS 2025 (Main Conference)  
**arXiv**: [2505.17048](https://arxiv.org/abs/2505.17048)  
**代码**: 有（HuggingFace + GitHub, CC-BY-NC-SA 4.0）  
**领域**: NLP / 金融文本分析  
**关键词**: 央行通讯, 货币政策, 立场检测, 不确定性估计, 多语言

## 一句话总结

构建 World Central Banks (WCB) 数据集——迄今最全面的货币政策语料库（25家央行、38万句、28年历史），定义三项任务（立场检测、时间分类、不确定性估计），在 15,075 个基准实验中验证"整体大于部分之和"的跨央行聚合训练优势。

## 研究背景与动机

全球央行的政策沟通对经济稳定至关重要，但：

**数据分散**: 各央行数据格式不同、语言各异、获取渠道碎片化
**分析缺乏统一框架**: 现有研究通常只分析单一央行（如美联储），缺乏跨地区比较
**误读风险**: 误解央行政策信号可能对弱势群体造成不成比例的影响
**模型验证不足**: LLM 在金融政策文本上的能力缺乏系统评估

## 方法详解

### 整体框架

1. **WCB 数据集构建**: 统一采集和标注 25 家央行的政策文本
2. **三项任务定义**: 立场检测、时间分类、不确定性估计
3. **大规模基准实验**: 7 个 PLM + 9 个 LLM 的全面评估
4. **聚合训练验证**: 证明跨央行联合训练的优势

### 关键设计

1. **数据集构建**:

    - 25 家央行：美联储、欧央行、英央行、日央行、中国人民银行等
    - 覆盖多个地理区域：北美、欧洲、亚洲、拉美、非洲
    - 总量：380K+ 句子，跨 28 年
    - 均匀采样：每家央行 1K 句（总 25K），确保均衡

2. **标注流程**:

    - 双标注员独立标注
    - 分歧解决机制
    - 二次专家审核
    - 每个句子标注三项任务

3. **三项任务**:

    - **立场检测 (Stance Detection)**: 鸽派/中立/鹰派
    - **时间分类 (Temporal Classification)**: 陈述的时间指向（过去/当前/未来）
    - **不确定性估计 (Uncertainty Estimation)**: 确定/不确定/模糊

4. **聚合训练策略**:

    - 将所有央行数据混合训练统一模型
    - 对比：各央行单独训练的模型
    - 验证"整体 > 部分之和"

### 损失函数 / 训练策略

PLM 微调：交叉熵损失
LLM 评估：Zero-Shot / Few-Shot / 加标注指南的三种设置

## 实验关键数据

### PLM 基准实验（15,075 实验中的代表性结果）

| 模型 | 立场检测 F1 ↑ | 时间分类 F1 ↑ | 不确定性估计 F1 ↑ | 训练方式 |
|------|-------------|-------------|----------------|---------|
| BERT-base | 62.5 | 58.8 | 55.2 | 单央行 |
| BERT-base | 71.8 | 67.5 | 64.2 | 聚合训练 |
| RoBERTa | 65.2 | 61.5 | 58.5 | 单央行 |
| RoBERTa | 74.5 | 70.2 | 67.8 | 聚合训练 |
| FinBERT | 68.5 | 63.2 | 60.1 | 单央行 |
| FinBERT | 76.2 | 72.5 | 69.5 | 聚合训练 |
| DeBERTa-v3 | 67.8 | 62.8 | 59.2 | 单央行 |
| DeBERTa-v3 | **77.5** | **73.8** | **71.2** | 聚合训练 |

### LLM 评估

| 模型 | 设置 | 立场检测 F1 ↑ | 时间分类 F1 ↑ | 不确定性 F1 ↑ |
|------|------|-------------|-------------|-------------|
| GPT-4 | Zero-Shot | 58.2 | 52.5 | 48.8 |
| GPT-4 | Few-Shot | 65.8 | 60.2 | 55.5 |
| GPT-4 | +指南 | 72.5 | 68.5 | 63.2 |
| Claude-3.5 | Zero-Shot | 55.8 | 50.2 | 46.5 |
| Claude-3.5 | +指南 | 70.2 | 66.8 | 61.5 |
| Llama-3-70B | Zero-Shot | 45.2 | 42.8 | 38.5 |
| Llama-3-70B | +指南 | 58.5 | 55.2 | 50.8 |

### 聚合 vs 单独训练对比

| 央行 | 单独训练 F1 | 聚合模型 F1 | 提升 |
|------|-----------|-----------|-----|
| 美联储 (Fed) | 72.5 | 78.2 | +5.7 |
| 欧央行 (ECB) | 70.8 | 77.5 | +6.7 |
| 英央行 (BoE) | 71.2 | 76.8 | +5.6 |
| 日央行 (BoJ) | 65.5 | 74.2 | +8.7 |
| 中国人民银行 | 58.2 | 72.5 | +14.3 |
| 巴西央行 | 55.8 | 71.8 | +16.0 |
| 印度储备银行 | 60.5 | 73.5 | +13.0 |

### 关键发现

1. **"整体 > 部分"**: 聚合训练显著优于单央行训练（平均提升 9.3 F1）
2. **数据稀缺央行受益最大**: 中国、巴西等数据较少的央行提升超过 14 F1
3. **PLM > LLM (Zero-Shot)**: 微调的小模型在领域特定任务上优于 LLM 零样本
4. **标注指南很重要**: 给 LLM 提供标注指南后性能提升 10-15 F1
5. **金融专用模型优势有限**: FinBERT 仅比通用 DeBERTa 略差

## 亮点与洞察

- **数据集价值**: WCB 是央行通讯分析领域最全面的资源，覆盖 25 家央行和 28 年
- **"整体 > 部分"发现**: 跨央行的知识迁移效果惊人，对全球金融分析有重要意义
- **实验规模**: 15,075 个实验确保了结论的可靠性
- **多任务标注**: 每个句子标注三项任务，最大化数据价值
- **经济验证**: 预测性任务验证了框架的经济实用性

## 局限性 / 可改进方向

1. 均匀采样 1K/央行可能对数据丰富的央行是信息损失
2. 多语言央行文本目前翻译为英文处理，原文分析可能更准确
3. 句子级标注可能遗失段落级/文档级的上下文
4. 三个任务的标注质量依赖专家，主观性难以完全消除
5. 市场预测等下游任务的直接验证可以更深入
