# GreenHyperSpectra: A Multi-Source Hyperspectral Dataset for Global Vegetation Trait Prediction

**会议**: NeurIPS 2025  
**arXiv**: [2507.06806](https://arxiv.org/abs/2507.06806)  
**代码**: [HuggingFace](https://huggingface.co/datasets/Avatarr05/GreenHyperSpectra)  
**领域**: 遥感 / 植被功能性状  
**关键词**: 高光谱遥感, 植被性状预测, 自监督学习, 多源数据集, 表征学习

## 一句话总结

构建了包含14万+多源高光谱植被反射率光谱的GreenHyperSpectra预训练数据集，涵盖近端、机载和星载平台，通过半监督和自监督方法在标签稀缺场景下显著超越全监督基线的多性状预测精度。

## 研究背景与动机

- 植被功能性状（如叶面积指数、叶绿素含量、碳含量等）是生物多样性和气候变化研究的核心变量
- 高光谱遥感可实现大尺度无损观测，但面临三大挑战：
    - **标签稀缺**：性状标注需要实地采样和实验室分析，成本高昂
    - **域偏移严重**：不同传感器、生态系统、采集条件导致光谱分布差异大
    - **现有数据集局限**：大多基于单一传感器或合成数据，缺乏跨域覆盖
- 现有监督方法受限于稀疏标注数据，跨域泛化能力差
- 合成数据（辐射传递模型RTM）与真实数据存在显著差距

## 方法详解

### 整体框架

GreenHyperSpectra包含两部分：（1）14万+无标注高光谱预训练数据，覆盖近端/机载/星载三类平台；（2）7900个有标注光谱（7种植被性状）。通过半/自监督方法在无标注数据上预训练光谱表征，再在少量标注数据上微调/线性探测进行多性状回归。

### 关键设计

1. **多源多平台数据策略**:
    - 功能：从近端（ASD FieldSpec等）、机载（AVIRIS-NG、NEON AOP等）和星载（PRISMA、EnMAP、EMIT等）三类平台收集光谱
    - 核心思路：不同平台引入有价值的信号变异性（空间/光谱分辨率差异、观测几何差异等），帮助模型学习可迁移表征
    - 设计动机：单一平台数据集限制了模型跨传感器泛化，多源数据提供了真实世界多样性

2. **三种半/自监督框架**:
    - SR-GAN：半监督GAN，生成器生成合成光谱，判别器同时做性状回归和特征对比学习
    - RTM-AE：以物理模型PROSAIL-PRO替代解码器的自编码器，隐空间直接对应植被性状
    - MAE：1D Masked Autoencoder，随机遮蔽光谱区域后重建，学习光谱表征

3. **评估协议设计**:
    - 功能：设计覆盖in-distribution和out-of-distribution的全面评估框架
    - 核心思路：分层分割确保预训练子集的来源多样性一致；OOD评估通过跨50个子数据集的交叉验证
    - 设计动机：真实应用中模型需跨传感器和生态系统泛化

### 损失函数 / 训练策略

- MAE：余弦相似度 + MSE混合重建损失，权重系数 $\alpha$ 调控光谱形状和幅度信息
- RTM-AE：余弦相似度 + MAE重建损失 + 性状监督损失
- SR-GAN：生成器损失 + 判别器对比损失 + 标注数据回归损失
- 所有性状经Box-Cox变换提升训练稳定性

## 实验关键数据

### 主实验（表格）

| 方法 | 平均 R² ↑ | 平均 nRMSE ↓ |
|------|-----------|-------------|
| Supervised | 0.587 | 13.697 |
| SR-GAN | 0.592 | 13.589 |
| RTM-AE | 0.592 | 13.557 |
| MAE (Linear Probe) | 0.466 | 15.499 |
| **MAE (Fine-tune)** | **0.641** | **12.777** |

### 消融实验

- **标签比例敏感性**：在20–40%标签量时，半/自监督方法优势最显著，MAE微调在仅20%标签下即超越全量监督基线
- **无标注数据量**：增加预训练数据量带来的提升有限，分层分割协议能在子集上高效利用数据
- **半光谱（400–900nm）场景**：SR-GAN在仅VNIR光谱下表现最好，MAE可跨分辨率迁移

### 关键发现

- MAE微调是整体最优方法，在全光谱场景下平均R²达0.641
- 半/自监督方法在标签稀缺场景提升显著，解决了植被性状预测的核心瓶颈
- 分层分割策略比简单增加数据量更关键

## 亮点与洞察

- 首个面向植被性状预测的大规模多源高光谱预训练数据集，填补了该领域的重要空白
- 系统对比了三类半/自监督范式在1D光谱数据上的表现，为后续研究提供全面基线
- RTM-AE将物理模型嵌入深度学习框架，是physics-aware ML在遥感中的有趣尝试
- 数据集覆盖1992–2024年的全球采集，时空跨度大

## 局限性 / 可改进方向

- 数据集主要覆盖欧美高纬度植被，热带和亚热带覆盖不足
- 标注数据仅7900个样本，性状种类和数量可进一步扩展
- 未探索基于foundation model的方法（如更大的ViT或多模态模型）
- OOD评估受限于现有标注数据集的地理分布

## 相关工作与启发

- 与HySpecNet、SpectralEarth等高光谱基准互补，但独特优势在于多平台+植被专属
- MAE在光谱域的成功启发了后续工作探索更多SSL范式（如对比学习、DINO等）
- 为Sentinel-2/EnMAP等高光谱卫星任务的大规模应用提供预训练基础

## 评分

- ⭐⭐⭐⭐ — 数据集贡献有价值，实验全面，但方法创新性有限，更多是系统工程和基准建设
