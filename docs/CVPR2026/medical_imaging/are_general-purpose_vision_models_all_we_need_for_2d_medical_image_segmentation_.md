# Are General-Purpose Vision Models All We Need for 2D Medical Image Segmentation? A Cross-Dataset Empirical Study

**会议**: CVPR 2026
**arXiv**: [2603.13044](https://arxiv.org/abs/2603.13044)
**作者**: Vanessa Borst, Samuel Kounev
**代码**: GitHub（论文中提及）
**领域**: medical_imaging
**关键词**: 医学图像分割, 通用视觉模型, 专用医学架构, 跨数据集评估, 可解释性分析

## 一句话总结

通过统一实验协议对比 11 个专用医学分割架构（SMAs）与通用视觉模型（GP-VMs），发现 GP-VMs 在三个异构医学数据集上优于大多数 SMAs，且 Grad-CAM 分析表明 GP-VMs 无需领域特定设计即可捕捉临床相关结构。

## 研究背景与动机

医学图像分割（MIS）是计算机辅助诊断和临床决策支持系统的基础组件。过去十年，大量针对医学影像的专用架构涌现，旨在解决低对比度、小解剖结构、标注数据有限等领域挑战。

与此同时，计算机视觉领域的快速发展产生了大量高性能的通用视觉模型（GP-VMs），这些模型最初为自然图像设计。尽管它们在标准视觉基准上表现出色，但其在医学图像分割中的有效性仍未被充分理解。

**核心问题**：专用医学分割架构（SMAs）相对于现代通用视觉模型（GP-VMs）是否在 2D 医学图像分割中具有系统性优势？

**研究动机**：
- 医学影像领域存在大量定制化架构，但缺乏在统一实验条件下的公平对比
- 通用模型的预训练规模和架构创新可能已弥补领域特定设计的优势
- 模型选择对实际临床部署至关重要，需要可靠的实证指导
- 现有比较研究往往使用不同的训练策略、超参数和评估协议，难以得出可靠结论

## 方法详解

### 实验设计

本文采用控制变量的实证研究方法，核心在于**统一训练和评估协议**：

1. **模型选择**：共选取 11 个代表性模型，涵盖专用医学分割架构（SMAs）和通用视觉模型（GP-VMs）两大类
   - **SMAs**：专为医学影像设计的架构，通常包含跳跃连接、多尺度特征融合、注意力门控等医学领域特定模块（如 U-Net 系列、TransUNet、SwinUNet 等）
   - **GP-VMs**：面向自然图像任务设计的分割模型，具备强大的预训练表示能力（如 DeepLabV3+、SegFormer、Mask2Former 等）

2. **统一协议**：所有模型在相同的数据划分、预处理流程、训练超参数、损失函数和评估指标下进行对比，消除实验不公平性

### 数据集

实验覆盖三个异构数据集，具有不同的成像模态、类别结构和数据特性：
- 不同的医学成像模态（如皮肤镜检查、内窥镜、MRI/CT 等）
- 不同的类别数量和分割难度
- 不同的数据规模和标注质量

### 可解释性分析（Grad-CAM）

除了分割精度评估，本文还引入 Grad-CAM 可视化分析模型的**可解释性行为**：
- 观察不同模型关注的图像区域是否与临床相关结构一致
- 比较 SMAs 和 GP-VMs 在特征关注模式上的差异
- 评估 GP-VMs 是否能在无领域特定设计的情况下学习到有意义的医学特征表示

### 评估指标

采用医学图像分割领域的标准指标：
- **Dice Similarity Coefficient (DSC)**：衡量预测与真值的重叠程度
- **Intersection over Union (IoU/Jaccard)**：区域交并比
- 辅以 Grad-CAM 定性可视化分析

## 实验关键数据

### Table 1: SMAs vs GP-VMs 跨数据集分割性能对比（基于论文核心发现）

| 模型类别 | 典型代表 | 跨数据集表现趋势 | 关键特征 |
|---|---|---|---|
| SMA | U-Net | 基线水平，各数据集稳定 | 编码器-解码器 + 跳跃连接 |
| SMA | Attention U-Net | 略优于 U-Net | 注意力门控抑制无关区域 |
| SMA | TransUNet | 中等偏上 | CNN + Transformer 混合编码器 |
| SMA | SwinUNet | 中等，部分数据集偏弱 | 纯 Swin Transformer |
| GP-VM | DeepLabV3+ | 高，跨数据集一致性强 | 空洞卷积 + ASPP 多尺度特征 |
| GP-VM | SegFormer | 高，部分数据集最优 | 轻量 Transformer + MLP 解码 |
| GP-VM | Mask2Former | 整体最优 | 通用分割框架，掩码注意力 |

**核心发现**：GP-VMs 在所有三个数据集上整体优于大多数 SMAs，表明通用预训练的表示能力超过了领域特定的架构设计优势。

### Table 2: Grad-CAM 可解释性分析对比

| 分析维度 | SMAs 表现 | GP-VMs 表现 |
|---|---|---|
| 临床相关区域覆盖 | 聚焦目标区域，但可能依赖局部纹理 | 同样聚焦临床相关结构，覆盖更全面 |
| 注意力分散度 | 部分模型存在注意力分散到背景区域 | 注意力集中度较好 |
| 边界区域关注 | 通过跳跃连接保留边界细节 | 通过多尺度特征同样捕捉边界 |
| 无领域设计下的适应 | 显式设计适配医学任务 | 无需特定设计即可学习有意义的表示 |

**关键洞察**：GP-VMs 能够在没有显式领域特定架构设计的情况下捕捉临床相关结构，挑战了"医学分割必须使用专用架构"的传统认知。

## 亮点与洞察

- **颠覆性结论**：在控制实验下，通用视觉模型不仅不逊于专用医学架构，反而整体更优，对医学影像分割领域的架构选择有重要指导意义
- **统一评估协议**：11 个模型在完全相同的训练和评估条件下对比，消除了因训练策略不同导致的不公平比较
- **可解释性视角**：通过 Grad-CAM 分析模型行为，不仅比较精度还验证 GP-VMs 的特征关注模式与临床相关结构一致
- **实用指导**：临床部署不必执着于医学专用架构，而应关注预训练质量和模型表示能力，有助于降低部署门槛
- **跨模态验证**：三个异构数据集覆盖不同成像模态和任务，增强了结论的泛化性

## 局限性

- **仅限 2D 分割**：研究限于 2D 切片级分割，3D 体积分割中专用架构（如 nnU-Net）的优势可能更显著
- **数据集数量有限**：三个数据集虽然模态各异，但可能无法完全代表所有场景（超声、病理切片等未涉及）
- **预训练数据的潜在泄露**：GP-VMs 在大规模数据上预训练，不排除包含医学影像相关内容
- **未考虑数据效率**：标注稀缺场景下 SMAs 的领域先验可能更具优势，此维度未充分评估
- **缺少基础模型对比**：未纳入 SAM/MedSAM 等近年来表现突出的基础分割模型

## 相关工作

### 医学图像分割专用架构
- **U-Net**（Ronneberger 2015）：编码器-解码器 + 跳跃连接的开创性架构
- **Attention U-Net**（Oktay 2018）：引入注意力门控机制
- **TransUNet**（Chen 2021）：CNN + Transformer 混合编码器
- **nnU-Net**（Isensee 2021）：自动化训练流程的医学分割标杆

### 通用视觉分割模型
- **DeepLabV3+**（Chen 2018）：空洞卷积 + ASPP 多尺度特征提取
- **SegFormer**（Xie 2021）：轻量 Transformer + MLP 解码器
- **Mask2Former**（Cheng 2022）：统一语义/实例/全景分割的通用框架

### 本文定位
填补了系统性实证研究的空白——首次在多模型、多数据集规模下严格控制实验条件，对比 SMAs 与 GP-VMs 在 2D 医学分割中的表现。
