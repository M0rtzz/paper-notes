# Deep Learning Based Estimation of Blood Glucose Levels from Multidirectional Scleral Blood Vessel Imaging

**会议**: CVPR 2026  
**arXiv**: [2603.12715](https://arxiv.org/abs/2603.12715)  
**代码**: 无  
**领域**: 医学图像  
**关键词**: 血糖估计, 巩膜血管成像, 多视角深度学习, 非侵入式监测, Transformer融合

## 一句话总结

提出 ScleraGluNet，通过多方向巩膜血管图像（5 个注视方向）结合多分支 CNN + MRFO 特征优化 + Transformer 跨视角融合，实现三类代谢状态分类（93.8% 准确率）和连续血糖估计（MAE=6.42 mg/dL, r=0.983）。

## 研究背景与动机

糖尿病是全球性健康威胁（2021 年 5.37 亿成人），需要频繁监测血糖。现有方法的局限：

- **实验室检测**（FPG、OGTT、HbA1c）：准确但需要抽血，不便于频繁自测。
- **指尖采血**：有痛感、感染风险，影响依从性。
- **CGM**：仍需皮下传感器植入，成本较高。

**眼部微血管**提供了无创代谢评估的新途径：慢性高血糖会导致微血管重构（直径改变、迂曲度增加、灌注异常），这在视网膜和巩膜血管上都可观察到。相比视网膜成像（需要专业设备），**巩膜/结膜血管成像更简单**——用普通前段相机即可拍摄，适合远程医疗。

已有研究证实了巩膜微血管与糖尿病的关联（血管形态改变、OCTA 灌注变化等），但现有方法的不足：(1) 仅用单一视角拍摄，遗漏了不同区域的空间异质性血管变化；(2) 没有充分利用多视角特征间的互补信息。

ScleraGluNet 的切入：**多方向采集**（5 个注视角度覆盖巩膜各象限）+ **多视角融合架构**（并行 CNN + MRFO 特征筛选 + Transformer 跨视角注意力）。

## 方法详解

### 整体框架

输入是每位受试者的 5 张巩膜照片（正视、上视、下视、鼻侧、颞侧），经预处理（CLAHE + Frangi 血管增强）后，分别送入 5 个独立参数的 CNN 分支提取各方向特征，再经 MRFO 特征精炼 → Transformer 跨视角融合 → 双头输出（分类 + 回归）。

### 关键设计

1. **多方向巩膜采集协议**：每位参与者在正前方注视后，依次拍摄上、下、左、右 4 个方向的巩膜照片。设计动机：糖尿病引起的微血管改变在空间上是异质的——上方巩膜可能显示不同于颞侧巩膜的血管异常。消融实验证实多视角比单视角系统性提升了分类和回归性能。

2. **并行 CNN 特征提取**：5 个方向各用独立 CNN 分支，共享架构但参数独立，允许各分支**专门学习特定区域的血管模式**（如迂曲度、分支复杂度、口径变化）。设计动机：不同注视方向暴露的巩膜区域血管特征差异较大，统一编码器无法捕获方向特异性。

3. **MRFO 特征优化（Manta Ray Foraging Optimization）**：多视角特征拼接后存在冗余和相关特征。MRFO 是一种生物启发优化算法，用于从拼接特征中筛选最相关的子集。设计动机：直接拼接 5 路特征会引入多视角冗余，稀释判别性信号。MRFO 通过特征选择减少噪声维度。

4. **Transformer 跨视角融合**：经 MRFO 精炼的特征送入 Transformer，通过自注意力机制发现跨视角的长程血管模式（如跨象限的细微血管特征、不对称的血管重构等）。最终 Transformer 输出连接到分类头（3 类 softmax）和回归头（FPG 连续值）。

### 损失函数 / 训练策略

联合训练分类和回归：$L = L_{\text{CE}} + L_{\text{MSE}}$，交叉熵用于代谢状态分类，MSE 用于血糖估计。Adam 优化器，subject-wise 5-fold 交叉验证（GroupKFold，确保同一参与者的所有图片在同一 fold）。

图像预处理流程：ROI 提取 → 颜色和亮度归一化 → CLAHE 对比度增强 → Frangi 血管滤波增强管状结构 → 二值化血管掩码（质量控制用）。

## 实验关键数据

### 主实验

**数据集**：445 名参与者（正常 150, 控制型糖尿病 140, 高血糖型 155），共 2225 张巩膜照片。

**三类分类**

| 指标 | 结果 |
|------|------|
| 整体准确率 | **93.8%** (95% CI: 91.8–95.4%) |
| 正常 F1 | 0.937 |
| 控制型糖尿病 F1 | 0.918 |
| 高血糖型 F1 | 0.942 |
| AUC (正常 / 控制 / 高血糖) | 0.971 / 0.956 / 0.982 |

**血糖回归**

| 指标 | 结果 |
|------|------|
| MAE | **6.42 mg/dL** |
| RMSE | 7.91 mg/dL |
| Pearson r | 0.983 |
| R² | 0.966 |
| Bland-Altman 均值偏差 | +1.45 mg/dL |
| 95% 一致限 | -8.33 ~ +11.23 mg/dL |

### 消融实验

| 配置 | MAE (mmol/L) | 说明 |
|------|-------------|------|
| 单视角 CNN baseline | 最高 | 仅用 1 张图 |
| 多视角 CNN (无 MRFO/Transformer) | 中等 | 拼接但未优化 |
| 多视角 CNN + MRFO | 较低 | 特征筛选有效 |
| **ScleraGluNet (full)** | **最低** | MRFO + Transformer 联合最优 |

### 关键发现

- 5-fold 交叉验证的每折精度稳定在 92.8%–94.6%（SD=0.7%），不依赖有利的数据划分。
- Grad-CAM 分析显示：正常组的注意力弱且弥漫，控制型聚焦于轻度血管变化区域，高血糖型对扩张/迂曲血管有强烈且跨方向一致的激活。
- 误分类主要发生在相邻代谢类别之间（正常↔控制型），符合血糖连续谱的临床现实。

## 亮点与洞察

- 从巩膜血管估计血糖是一个新颖的临床应用场景，潜在的远程医疗价值很大。
- 多方向采集的设计有生理学依据（微血管异常在空间上不均匀），且消融实验确认了其有效性。
- 回归性能令人印象深刻：r=0.983, Bland-Altman 分析显示与实验室测量有良好的一致性。
- Grad-CAM 可解释性分析清晰展示了模型确实在关注血管区域而非背景。

## 局限性 / 可改进方向

- **单中心研究**：445 名参与者来自同一家医院，泛化性未验证，设备和人群多样性不足。
- **混淆因素未充分控制**：高血压、吸烟、贫血等可影响巩膜血管形态的因素未被考虑。
- **仅针对空腹血糖**：未纳入餐后血糖和纵向监测，临床适用范围有限。
- **写作质量偏低**：部分段落明显有 LLM 生成痕迹，描述冗余且有不自然表述。
- **数据集规模适中**：445 人对临床验证来说偏少，需要更大规模的多中心试验。

## 相关工作与启发

- 与视网膜图像预测 HbA1c 的工作（如 Google 的 retinal biomarker 研究）互补：巩膜成像设备更简单、更适合远程场景。
- MRFO 算法在特征选择中的应用在医学影像领域较少见，可借鉴到其他多源特征融合任务。
- 多视角学习的思路（并行分支 + 跨视角 Transformer）可迁移到其他需要多角度采集的医学场景（如多角度X光、多切面超声）。

## 评分

- **新颖性**: ⭐⭐⭐⭐ 从巩膜血管到血糖估计是全新的临床场景，多方向采集有创意
- **实验充分度**: ⭐⭐⭐ 单中心 445 人数据集，缺少外部验证和基线模型对比
- **写作质量**: ⭐⭐ 明显的 LLM 辅助写作痕迹，部分段落冗余和不自然
- **价值**: ⭐⭐⭐ 临床应用潜力大但需多中心验证才能确认可行性
