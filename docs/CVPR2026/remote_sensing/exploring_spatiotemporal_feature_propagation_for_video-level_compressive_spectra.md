# Exploring Spatiotemporal Feature Propagation for Video-Level Compressive Spectral Reconstruction

**会议**: CVPR 2026  
**arXiv**: [2603.00611](https://arxiv.org/abs/2603.00611)  
**代码**: https://github.com/nju-cite/DynaSpec (有)  
**领域**: 遥感 / 计算光谱成像  
**关键词**: 光谱压缩成像, 高光谱视频重建, 时空特征传播, Transformer, 数据集

## 一句话总结

首次将光谱压缩成像 (SCI) 从图像级推进到视频级重建，构建了首个高质量动态高光谱数据集 DynaSpec，提出 PG-SVRT Transformer 通过时空特征传播实现高光谱视频高质量、时间一致的重建。

## 研究背景与动机

高光谱图像 (HSI) 能检测不同材料的光谱属性，广泛应用于分类、检测、跟踪等任务。传统高光谱成像需要沿空间或光谱维扫描，无法应对动态场景。光谱压缩成像 (SCI) 通过空间-光谱编码将 3D 数据压缩为 2D 测量实现快照采集，但现有重建方法存在两个根本限制：

1. **重建不确定性**：掩码编码不可避免地导致空间-光谱信息丢失，仅从单帧压缩测量中恢复被遮挡的信息存在固有不确定性
2. **时间不一致性**：逐帧重建范式无法保证时间连续性，表现为频闪强度曲线

关键洞察：时序测量序列中的相邻帧携带互补特征。固定编码模式可在相邻帧间差异化捕获互补信息，从而改善被遮挡信息的传播重建并增强时间一致性。核心 idea：**利用相邻帧的互补特征和时间连续性，从压缩测量序列重建高光谱视频**。

## 方法详解

### 整体框架

PG-SVRT 采用 U-Net 架构，三个核心组件：掩码引导退化感知 (MGDP)、跨域传播注意力 (CDPA) 和多域前馈网络 (MDFFN)。模块数量 (N1, N2, N3) = (4, 8, 8)。

### 关键设计

1. **DynaSpec 数据集**
   - 做什么：构建首个高质量动态高光谱图像数据集
   - 核心思路：使用 GaiaField 推扫式高光谱相机逐帧拍摄可控物体，手动设计平移、旋转、关节运动等模拟真实场景复杂运动
   - 数据规模：30 场景、300 帧 HSI，分辨率 1280x1280，光谱 2nm，400-700nm（151 通道）
   - 设计动机：现有数据集要么图像级（CAVE、KAIST），要么光谱分辨率低。通过长曝光降噪、光谱校正、强度校准确保质量

2. **掩码引导退化感知 (MGDP)**
   - 做什么：主架构前建模压缩退化过程，辅助帧内编码信息解耦
   - 核心思路：将掩码按 SCI 架构压缩，学习原始与压缩掩码间强度分布差异作为权重，与测量特征逐元素加权后拼接原测量
   - 设计动机：退化先验对重建至关重要，显式建模帮助网络理解被遮挡位置

3. **跨域传播注意力 (CDPA)**
   - 做什么：空间-然后-时间渐进式注意力，共享 value 跨域传播特征
   - 核心思路：空间注意力：非重叠窗口 + 桥接 token（池化 Query 生成）间接交互；时间注意力：空间输出作共享 value 直接计算
   - 复杂度：O = 4THWC^2 + 4THW*N_B*C + 2T^2*HWC，桥接 token 有效降低开销
   - 设计动机：联合时空提取消耗大，完全分离限制交互，此方案兼顾

4. **多域前馈网络 (MDFFN)**
   - 做什么：光谱特征分头在空间和时间域独立提取，促进融合
   - 设计动机：传统 FFN 复杂度高且难整合多域信息

### 损失函数 / 训练策略

多阶段 RMSE 损失，Adam (beta1=0.9, beta2=0.999)，lr 3e-4 余弦退火至 1e-6，T=3，80 epochs，batch 2，窗口 8x32，桥接 token N_B=64。

## 实验关键数据

### 主实验

| 方法 | PSNR-K | PSNR-D | SAM-K | ST-RRED-K | GFLOPs |
|------|--------|--------|-------|-----------|--------|
| MST-L (CVPR'22) | 39.99 | 39.58 | 3.82 | 30.99 | 28.23 |
| DPU (CVPR'24) | 40.02 | 41.01 | 5.22 | 25.90 | 31.04 |
| DPU* (加时域) | 40.50 | 41.36 | 5.17 | 26.71 | 77.36 |
| **PG-SVRT (Ours)** | **41.23** | **41.82** | **3.81** | **19.35** | **28.18** |

### 消融实验

| 配置 | PSNR | SSIM | SAM | ST-RRED | GFLOPs |
|------|------|------|-----|---------|--------|
| Baseline (F-MSA+FFN) | 39.97 | 0.9827 | 5.53 | 43.90 | 30.11 |
| + CDPA | 41.30 | 0.9884 | 4.32 | 25.44 | 21.11 |
| + CDPA + MGDP | 41.41 | 0.9886 | 4.25 | 24.63 | 21.31 |
| + CDPA + MGDP + MDFFN | **41.52** | **0.9893** | **3.91** | **23.25** | 28.18 |

### 关键发现

- DD-CASSI 在四种 SCI 架构对比中最优（PSNR 41.52 vs 次优 37.84）
- CDPA 是最大贡献模块：+1.33dB PSNR，复杂度反而降低（30.11->21.11 GFLOPs）
- PG-SVRT 虽为视频模型，单帧 FLOPs 28.18G 比多个图像模型更低
- 空间-然后-时间 + 传播策略最佳（41.52），优于并行处理（41.35）

## 亮点与洞察

- 首次系统性将光谱重建推进到视频级，填补重要空白
- DynaSpec 数据集构建方法论严谨，对社区有长期价值
- 桥接 token 巧妙：池化 query 生成中介 token 实现间接注意力，降低复杂度至近似线性
- 共享 value 的空间-然后-时间策略优雅解决多域特征传播

## 局限性 / 可改进方向

- DynaSpec 仅 30 场景，多样性有限
- 帧数固定 T=3，长序列扩展未验证
- 训练裁剪 256x256，全分辨率推理效率未讨论
- 未探索与光流对齐、变形卷积的组合

## 相关工作与启发

- DPU* 对比有说服力：简单拼接时域帧代价高（77.36G）效果有限
- 桥接 token 可推广到其他高维重建任务
- 跨系统公平对比为 SCI 系统选型提供基准
