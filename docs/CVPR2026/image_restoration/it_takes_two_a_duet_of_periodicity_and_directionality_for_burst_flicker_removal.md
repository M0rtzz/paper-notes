# Flickerformer: A Duet of Periodicity and Directionality for Burst Flicker Removal

**会议**: CVPR 2026
**arXiv**: [2603.22794](https://arxiv.org/abs/2603.22794)
**代码**: 有
**领域**: 图像修复 / 去闪烁
**关键词**: Burst去闪烁, 周期性, 方向性, 小波注意力, 相位融合

## 一句话总结
揭示闪烁伪影的两个内在物理特性——**周期性**(空间-时间重复的亮暗条纹模式)和**方向性**(沿行式曝光方向传播)——提出Flickerformer Transformer架构：基于相位的融合模块(PFM)利用帧间相位相关自适应聚合burst特征、自相关前馈网络(AFFN)利用帧内结构规律、小波方向注意力模块(WDAM)用高频方向信息引导低频暗区修复，在burst闪烁去除上全面超越SOTA。

## 研究背景与动机

1. **领域现状**：闪烁伪影源于不稳定照明(如荧光灯频闪)和行式曝光不一致(rolling shutter)→在短曝光摄影中严重退化图像质量。现有修复框架(去噪/低光增强)将闪烁视为通用退化。
2. **现有痛点**：闪烁不是随机噪声而是**结构化退化**——具有特定的空间-时间模式。通用修复框架不建模这些模式→抑制不充分且引入ghosting伪影。
3. **核心发现**：闪烁具有两个可利用的内在特性——(1) **周期性**：闪烁条纹在空间和时间上呈重复模式(与交流电频率/快门速度相关)；(2) **方向性**：闪烁条纹沿行式曝光方向(行扫描方向)传播→高频信息在特定方向保留。
4. **核心idea**：设计三个互补模块分别利用周期性(PFM+AFFN)和方向性(WDAM)进行针对性去闪烁。

## 方法详解

### 整体框架
Burst输入帧 → Flickerformer编码器(PFM-SA-AFFN-WDAM堆叠块 × N层) → 解码器 → 去闪烁输出。

### 关键设计

1. **PFM(Phase-based Fusion Module, 基于周期性)**:
   - 做什么：利用帧间相位相关自适应聚合burst帧特征
   - 核心思路：将burst帧变换到频域→计算相位相关(phase correlation)→基于相位差异自适应加权聚合→利用闪烁的时间周期性找到稳定的跨帧特征
   - 设计动机：闪烁在不同帧中的相位偏移是可预测的(受交流电频率控制)→相位相关可以有效对齐和融合burst帧中闪烁条纹的互补信息

2. **AFFN(Autocorrelation Feed-Forward Network, 基于周期性)**:
   - 做什么：在帧内利用自相关捕捉空间重复模式
   - 核心思路：计算特征的自相关（而非标准FFN）→捕捉闪烁条纹的空间周期→利用这些周期模式增强网络对结构性退化的感知
   - 设计动机：闪烁条纹在空间维度上呈周期排列→自相关天然适合检测这种规律性

3. **WDAM(Wavelet-based Directional Attention Module, 基于方向性)**:
   - 做什么：利用小波域高频方向信息引导低频暗区恢复
   - 核心思路：对特征做小波变换→分离高频(LH/HL/HH)和低频(LL)分量→高频分量保留了闪烁方向的边缘/纹理信息→用高频引导低频(闪烁暗区)的注意力→精确定位闪烁区域→方向性修复
   - 设计动机：闪烁暗区=低频信息被抑制，但高频方向信息(如水平边缘)可能在视觉上更鲁棒→用高频"告诉"网络哪里是闪烁区域、朝什么方向→引导低频恢复

### PFM与AFFN的互补关系
- PFM利用**帧间**时间周期性→跨帧融合
- AFFN利用**帧内**空间周期性→单帧内结构感知
- 两者共同增强网络对闪烁周期性模式的感知能力

### WDAM的频率域互补设计
- 闪烁→低频受损严重(整体变暗)但高频(边缘方向)相对稳定
- 用稳定的高频引导修复受损的低频→跨频率互补修复

## 实验关键数据

### 主实验

| 方法 | PSNR↑ | SSIM↑ | 视觉质量 |
|------|:---:|:---:|------|
| 通用去噪(NAFNet等) | 基准 | 基准 | 闪烁残留 |
| 通用去模糊 | 更差 | 更差 | 不适配 |
| Burst HDR方法 | 中等 | 中等 | ghosting |
| **Flickerformer** | **SOTA** | **SOTA** | **无闪烁/无ghosting** |

### 消融实验

| 配置 | PSNR |
|------|:---:|
| 无PFM(标准融合) | 下降 |
| 无AFFN(标准FFN) | 下降 |
| 无WDAM(无小波注意力) | 下降 |
| PFM+AFFN(无方向性) | 中等 |
| PFM+WDAM(无空间周期性) | 中等 |
| **完整Flickerformer** | **最优** |

### 关键发现
- 三个模块各自独立贡献且互补——证实了周期性和方向性都是有用的先验
- PFM+AFFN(周期性组合)的贡献比WDAM(方向性)更大→周期性是更强的先验
- 在不同帧数的burst(3-7帧)上都有效→不限于特定burst大小
- Flickerformer在纯去噪场景(无闪烁)上也不退化→周期性/方向性模块在无结构退化时不引入伪影

## 亮点与洞察
- **物理特性驱动的网络设计**：从闪烁的物理成因出发设计每个模块——不是通用修复而是针对性解决。这种"理解退化→设计方法"的范式对其他结构化退化(moiré/banding/rolling shutter)同样适用
- **"高频引导低频"的小波互补**：打破了"低频是主体，高频是细节"的思维定式——在闪烁场景下低频反而是受损最重的部分，高频方向信息成为修复的"罗盘"
- **相位相关的burst融合**：不同于直接帧平均或注意力融合——利用闪烁的时间周期性做相位对齐→消除闪烁的同时避免ghosting

## 局限性 / 可改进方向
- 当前假设闪烁有明确的周期性——对非周期(随机)闪烁可能不适用
- 小波变换固定为Haar小波→可学习的小波基可能更适配
- burst大小是手动设定的→自适应burst大小选择可能更灵活
- 目前仅处理短曝光burst→长曝光视频(如LED频闪)的闪烁处理是扩展方向

## 相关工作与启发
- **vs NAFNet/Restormer通用修复**: 不建模闪烁特有的周期性/方向性→对闪烁效果有限
- **vs Burst HDR方法**: 针对曝光融合设计→不处理闪烁的条纹模式→引入ghosting
- **vs 去moiré方法**: Moiré也有周期性→本文的周期性建模可能有迁移价值
