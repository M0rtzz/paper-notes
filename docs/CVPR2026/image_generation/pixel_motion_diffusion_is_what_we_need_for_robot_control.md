# Pixel Motion Diffusion Is What We Need for Robot Control

**会议**: CVPR 2026  
**arXiv**: [2509.22652](https://arxiv.org/abs/2509.22652)  
**代码**: [有](https://eronguyen.github.io/DAWN)  
**领域**: 图像生成  
**关键词**: 像素运动扩散, 机器人控制, 视觉-语言-动作, 光流表征, 层级扩散策略

## 一句话总结

DAWN 提出两阶段全扩散框架，通过 Motion Director 生成稠密像素运动场作为可解释中间表征，再由 Action Expert 转化为机器人动作序列，在 CALVIN 基准上实现 SOTA（平均长度 4.00）且数据效率极高。

## 研究背景与动机

**领域现状**：VLA 模型通过大规模训练实现泛化，部分方法引入运动感知增强动作生成。路线分稀疏像素轨迹和未来 RGB 帧预测。
**核心问题**：
   - 稀疏像素轨迹仅追踪少量关键点，丢失运动信息
   - RGB 帧预测计算开销大且缺乏显式运动表征
   - 从视频中间接提取轨迹（Gen2Act）增加不必要间接性
**本文方案**：直接预测稠密像素运动而非 RGB 帧，简化学习问题，两阶段都用扩散模型

## 方法详解

### 整体框架

DAWN 由两个扩散模型组成：

1. **Motion Director**（高层）：基于 LDM，接收多视角图像和语言指令，生成稠密像素运动场
2. **Action Expert**（低层）：基于扩散 Transformer，将像素运动+视觉+语言+状态转化为动作序列

两者通过结构化像素运动表征连接，形成可解释模块化控制管道。

### 关键设计

#### 1. Motion Director

- **像素运动定义**：帧间运动 F = [u,v]，编码为三通道图像 [u, v, (u+v)/2] 以利用预训练 RGB 扩散模型
- **架构**：预训练 LDM（U-Net + 文本编码器 + VAE），额外引入视觉编码器编码夹爪视角
- **条件注入**：当前帧 VAE 编码与噪声拼接（不加噪）作为 U-Net 输入，语言/夹爪/时间偏移通过交叉注意力
- **训练**：仅更新 U-Net，GT 由 RAFT 光流提取

#### 2. Action Expert

- **多模态编码**：DINOv3 ConvNeXt-S 编码像素运动和观测，T5-small 编码语言，2层MLP编码状态
- **去噪 Transformer**：迭代去噪生成动作块，条件通过交叉注意力注入
- 视觉/文本编码器冻结，去噪器和状态编码器从头训练

#### 3. 训练与推理

- 两模块可并行训练（RAFT 光流作 GT），Action Expert 可在 Motion Director 输出上微调
- 推理：25步扩散 -> 像素运动 -> 动作序列 -> 执行 -> 闭环
- 模块可独立升级替换

### 损失函数 / 训练策略

- 两模块均用 MSE 噪声估计损失
- Motion Director 在潜在图像空间预测像素运动
- Action Expert 在动作空间预测动作块

### 消融实验

- Enhanced DP（仅 Action Expert）：Avg Len = 2.78
- DAWN（加 Motion Director）：Avg Len = 4.00（+43.9%），验证像素运动中间表征的关键作用

### 关键发现

- 无外部数据即达 SOTA（4.00 > VPP 3.93）
- 像素运动在语义相似但不同任务上表现佳（open-door vs close-door）
- 稠密像素运动+预训练模型实现极高数据效率
- 真实机器人仅需 minimal finetuning 即可迁移

## 亮点与洞察

1. **设计简洁**：两阶段全扩散，稠密像素运动作为通用中间语言
2. **可解释**：像素运动场直观展示模型预期场景变化
3. **高数据效率**：预训练图像扩散模型大幅降低训练需求
4. **模块化**：两模块可独立训练升级

## 局限性 / 可改进方向

1. 两阶段推理延迟，实时性不足
2. RAFT 光流估计错误会传播
3. 仅支持单步运动预测，长程规划有限
4. 真实世界实验规模有限
5. 遮挡/大形变场景可能失效

## 相关工作与启发

- **Gen2Act**：从视频间接提取运动；DAWN 直接预测
- **Diffusion Policy**：端到端但缺中间运动抽象
- **VPP**：RGB 空间视频扩散特征
- **启发**：图像扩散模型可有效迁移到运动预测
