# SimLBR: Learning to Detect Fake Images by Learning to Detect Real Images

**会议**: CVPR 2026  
**arXiv**: [2602.20412](https://arxiv.org/abs/2602.20412)  
**代码**: 有（将在HuggingFace和GitHub发布）  
**领域**: 图像生成  
**关键词**: 假图检测, AI生成图像, 潜空间混合正则化, 跨生成器泛化, DINOv3

## 一句话总结
本文提出SimLBR，通过在DINOv3潜空间中将少量假图信息混入真图嵌入作为正则化手段，迫使检测器学习真实图像分布的紧致决策边界，从而实现对未知生成器的强泛化能力，在GenImage上平均准确率达94.54%，在硬测试集Chameleon上比AIDE提升25%准确率和70%召回率。

## 研究背景与动机

**领域现状**：AI生成图像检测是关键的媒体取证任务。现有方法通常在一个生成器的假图上训练，期望泛化到其他生成器。UnivFD使用CLIP特征+近邻分类，AIDE结合DCT和CLIP特征。

**现有痛点**：几乎所有SOTA检测器都过拟合到训练数据的生成器特定指纹上。当遇到未见过的生成器时，性能灾难性下降。AIDE论文提出的Chameleon硬测试集上，所有现有方法严重失效——假图被误判为真图（召回率低至5%），说明这些方法实际上学习的是围绕假图的紧致边界，把"真"当成了兜底类（sink class）。

**核心矛盾**：假图分布不断演变（新生成器层出不穷），围绕假图学边界必然会遗漏分布外的新假图。而真实图像分布相对稳定——真图就是真图，任何不是真图的东西都应该被判为假。

**本文要解决什么？** 让检测器学习围绕真实图像分布的紧致决策边界，将"假"作为兜底类，实现对任意未知生成器的泛化。

**切入角度**：通过潜空间混合正则化（LBR）让真图的分类变得"更难"——在训练时向真图嵌入中混入少量假图信息并标为假，迫使模型只将完全未受污染的真图判为真。

**核心idea一句话**：在DINOv3语义潜空间中用简单的线性插值混合真假图嵌入，让模型学会围绕纯真图分布画一个紧致圈。

## 方法详解

### 整体框架

SimLBR的pipeline极简，分三步：

1. **嵌入预计算**：用DINOv3对所有训练图像（真图集合 $\mathbb{R}=\{R_1,...,R_n\}$ 和假图集合 $\mathbb{F}=\{F_1,...,F_n\}$，来自单一训练生成器 $G_s$）预计算嵌入向量
2. **LBR训练**：对每张真图 $R_i$ 随机采样标签 $y_i \in \{0,1\}$。若 $y_i=0$，直接使用真图嵌入 $L_i = I(R_i)$；若 $y_i=1$，用LBR混合假图信息并标为假
3. **轻量MLP分类**：2层ReLU MLP + BCE损失，batch size 500，Adam (lr=1e-4, wd=1e-2)，最多10 epochs

整个训练在单卡H100上仅需**3分钟**（嵌入预计算除外，GenImage约22分钟、AIGC约50分钟）。对比：AIDE需要8卡A100训练约2小时。

### 核心模块：Latent Blending Regularization (LBR)

**做什么**：在预训练DINOv3的潜空间中，将真图嵌入与假图嵌入进行线性插值，生成"被污染的真图"表示，并将其标记为假。

**具体步骤**：

1. 对真图 $R_i$（标签 $y_i=1$），随机选取假图 $F_i$
2. 分别提取嵌入：$L_i^R = I(R_i)$，$L_i^F = I(F_i)$
3. 线性插值：$L_i = \alpha \cdot L_i^R + (1-\alpha) \cdot L_i^F$
4. 将 $L_i$ 标为假类，送入MLP训练

**混合系数 $\alpha$ 的采样**：

$$\alpha \sim \text{Uniform}(0.5, B), \quad B=0.8$$

- $\alpha > 0.5$ 保证混合结果保留**大部分**真图信息，但含有足够假图信息被标为假
- $\alpha$ 太低（如0.1-0.3）：假信息占主导，分类太简单，不能有效正则化
- $\alpha$ 太高（如0.95-0.99）：混合结果几乎等同纯真图但标假，训练不稳定
- 消融实验验证 $0.7 < B < 0.9$ 区间均稳定有效

**设计动机**：训练时模型必须将含有**任何程度**假信息的样本判为假，因此不得不学会精确地识别"什么是完全纯净的真图"——即围绕真图分布形成紧致决策边界。推理时，任何不在这个紧致边界内的样本（无论来自何种生成器）都被判为假。

### 为什么在潜空间而非像素空间混合？

先前的人脸伪造检测方法（Face X-ray等）在像素空间做混合，但存在三个限制：

1. **低级伪影泄露**：像素混合直接复制假图的低级伪影（如频域特征），模型可以"偷懒"靠检测这些伪影做判断，绕过了学习真图分布的目标
2. **领域特定预处理**：像素混合通常需要人脸关键点检测等任务特定操作，限制了方法的通用性
3. **操控粒度粗**：像素混合难以精确控制假信息注入量

在DINOv3的高级语义空间中混合的优势：
- 低级线索在语义嵌入中被抽象化，模型被迫学习**语义级**的真假区分
- 简单的向量线性运算即可完成，高效且鲁棒
- 可精细控制混合比例，生成几乎无限多的正则化样本

### DINOv3潜空间的关键作用

LBR的有效性**强依赖**嵌入空间的几何性质。线性插值的前提是嵌入流形足够光滑——两个嵌入之间的线性路径应落在有效流形上。

实验关键发现：
- **DINOv3 + LBR**：平均准确率从78.05%提升至88.26%（+10.21%）
- **DINOv2 + LBR**：平均准确率从68.92%微变至68.51%（几乎无效！）

说明DINOv3的嵌入流形满足光滑性要求，而DINOv2的流形结构可能存在局限。作者推测这与弱嵌入模型的流形结构缺陷有关。

### 训练目标

标准二元交叉熵损失：

$$\mathcal{L}_{\text{BCE}} = -\frac{1}{N}\sum_{i=1}^{N} [y_i \log(F_\theta(L_i)) + (1-y_i)\log(1-F_\theta(L_i))]$$

### 评估指标创新

**1. 可靠性得分（Reliability Score）**：

$$\text{Reliability} = \frac{\mu_{\text{acc}} - A_{\text{base}}}{\sigma_{\text{acc}}}$$

借鉴金融领域Sharpe比率，$\mu_{\text{acc}}$ 为跨生成器平均准确率，$\sigma_{\text{acc}}$ 为标准差，$A_{\text{base}}=50\%$。高可靠性 = 高准确率 + 低方差。

**2. 最差场景估计（Worst-Case Estimate, WCE）**：检测器在所有评估生成器中的最低准确率，作为面对未来最难生成器时的性能上界。

### 关键发现
- LBR在DINOv3上带来+10.21%平均提升，但在DINOv2上几乎无效→嵌入空间几何性质是LBR成功的前提
- 假图召回率提升惊人：Chameleon-ProGAN上从7.24%→75.80%，AIGC从54.07%→82.07%，RSFake从48.28%→78.87%
- MLP深度：0-4层效果好，>4层严重过拟合
- $\alpha$ 上界 $B$ 在0.7~0.9区间均稳定
- 最差场景：SimLBR在GenImage/AIGC上分别保持88%/75.54%，远超其他方法~50%

## 亮点与洞察
- **"检测真图而非假图"的范式转换**极具洞察力：将问题从不断追赶新生成器转变为建模相对稳定的真图分布，是方法论层面的重要贡献
- **LBR的极简设计**令人印象深刻——仅用嵌入空间的线性插值就实现了强正则化效果，训练仅需3分钟
- **Reliability Score**借鉴金融风控的Sharpe比率，为检测器评估提供了更全面视角，可推广到所有分类鲁棒性评估场景
- t-SNE可视化直观展示了LBR前后决策边界的质变：无LBR时未见生成器的假图混入真图聚类；有LBR后真图形成紧致簇，任何偏离都被判为假

## 局限性 / 可改进方向
- 强依赖DINOv3嵌入质量，DINOv2上完全无效，未来新backbone需重新验证
- LBR仅用线性插值，可探索流形上更复杂的混合策略（如球面插值slerp）
- 真图分布"相对不变"的假设长期来看可能被打破（相机传感器演进、后处理pipeline变化）
- MLP容量有限，面对精心设计的对抗攻击可能不够

## 相关工作与启发
- **vs UnivFD**：同样利用预训练特征空间，但UnivFD用nearest neighbor，SimLBR用LBR正则化MLP。LBR显式学习真图边界是关键区别
- **vs AIDE**：AIDE用DCT+CLIP特征，8卡A100训练2小时；SimLBR仅用DINOv3嵌入+单卡3分钟，在硬测试集上反而大幅领先→特征空间选择和训练范式比模型复杂度更重要
- **vs 像素混合方法**（Face X-ray等）：像素空间混合依赖领域特定处理且泄露低级伪影；潜空间混合域无关、强制语义级区分
- **vs 异常检测方法**：纯异常检测（仅用真图建模，无假图引导）在高维自然图像空间中假设不切实际。SimLBR用假图作为引导信号构建决策边界，比纯无监督方法更有效
