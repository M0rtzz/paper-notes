# Editing Physiological Signals in Videos Using Latent Representations

**会议**: CVPR 2026  
**arXiv**: [2509.25348](https://arxiv.org/abs/2509.25348)  
**代码**: 有  
**领域**: AI安全 / 隐私保护  
**关键词**: 心率编辑, rPPG隐私, 视频生理信号, 3D VAE, 生物特征匿名化

## 一句话总结
提出面向视频生理信号(心率)编辑的学习框架——在3D VAE潜空间中融合视频编码和目标心率文本嵌入，通过可训练时空融合层(AdaLN)捕捉rPPG时间相干性+FiLM解码器调制实现精确心率修改，视觉质量保持PSNR 38.96dB/SSIM 0.98的同时心率调制误差10 bpm MAE，可用于面部视频的心率匿名化或合成带指定心率的视频。

## 研究背景与动机

**领域现状**：摄像头远程光电容积脉搏波(rPPG)技术可从面部视频无接触提取心率→泄露健康/情绪/压力等敏感信息→隐私威胁。
**现有痛点**：(1) 现有视觉隐私方法(模糊/替换人脸)可能意外破坏或保留PPG信号→不专门处理生理维度；(2) 面部替换同时引入新的身份线索→在隐私保护场景中不理想；(3) 无法选择性修改心率到目标值(如统一为60 bpm做匿名化)。
**核心需求**："看不见的隐私"保护——心率信号嵌入在微妙的皮肤颜色变化中→肉眼不可见但可被算法提取→需要在不影响视觉外观的情况下精确修改底层生理信号。
**核心idea**：在3D VAE潜空间做心率编辑→利用潜空间的压缩性和结构性实现精确调制+保持视觉质量。

## 方法详解

### 整体框架
输入视频 → 预训练3D VAE编码潜表示 + 冻结文本编码器嵌入目标HR prompt → **时空融合层**(AdaLN时间自注意力)融合 → FiLM调制解码器+微调输出层 → 输出视频(视觉不变,HR已修改)。

### 关键设计

1. **3D VAE潜空间**:

    - 选择3D VAE(而非2D)：rPPG信号本质是时间序列→需要3D表示捕捉时空相干性
    - 与Latent Diffusion/Video Diffusion兼容→可作为视频生成后处理

2. **时空融合层(AdaLN)**:

    - 目标HR prompt嵌入→时间步嵌入(暗含HR频率)→AdaLN注入每层时间自注意力
    - 时间自注意力跨帧传播→捕捉rPPG的强时间相干性(心率是周期信号)→精确、长程条件化
    - 设计动机：rPPG变化极微妙(亚像素级颜色波动)→需要精细的时间调制

3. **FiLM解码器+微调输出层**:

    - FiLM(Feature-wise Linear Modulation)在解码器中调制→确保解码时生理信号不被重建损失抵消
    - 仅微调输出层(而非整个解码器)→保持视觉重建质量
    - 设计动机：标准3D VAE解码器可能在重建时"丢失"微妙的rPPG调制→FiLM显式保持信号

4. **可控心率编辑**:

    - 匿名化模式：映射到固定HR(如60 bpm)→所有人看起来心率相同
    - 合成模式：指定目标HR→生成带特定心率的训练/测试视频
    - 多样化模式：随机HR→增加rPPG估计数据集多样性

## 实验关键数据

### 主实验

| 指标 | 数值 |
|------|:---:|
| 平均PSNR | **38.96 dB** |
| 平均SSIM | **0.98** |
| HR调制误差(MAE) | **10.00 bpm** |
| HR调制误差(MAPE) | **10.09%** |

视觉质量极高(PSNR>38)的同时实现可控心率编辑。

### 消融实验

| 配置 | PSNR | HR MAE |
|------|:---:|:---:|
| 无时间自注意力 | 高 | 差(准确度低) |
| 无FiLM解码器 | 高 | 差(信号被抵消) |
| 无微调输出层 | 降低 | 中等 |
| **完整框架** | **38.96** | **10.00** |

### 关键发现
- 时间自注意力是HR准确调制的关键——rPPG是时间周期信号→需要跨帧一致的调制
- FiLM解码器防止了解码器重建过程"擦除"微妙的rPPG调制
- PSNR>38说明对人眼完全不可见的修改→成功的"看不见的隐私保护"
- 在SOTA rPPG估计器(EfficientPhys等)上验证→编辑后的HR确实被提取为目标值

## 亮点与洞察
- **"看不见的隐私"问题定义**：首次专门针对视频中不可见的生理信号进行可控编辑——这是面部模糊/替换无法解决的隐私维度
- **3D VAE兼容视频生成管线**：在Latent Diffusion/Video Diffusion的潜空间工作→可直接作为现有视频生成框架的后处理模块
- **匿名化+合成的双重价值**：匿名化→保护隐私；合成→为rPPG研究生成带标注的训练数据
- **微妙信号编辑的工程方案**：rPPG变化量级<1%像素值→需要极精细的调制→FiLM+微调输出层的组合巧妙保持了这一精度

## 局限性 / 可改进方向
- HR MAE 10 bpm虽可接受但对精确健康监测应用仍嫌大
- 当前仅处理心率→呼吸率/血压等其他生理信号的编辑待探索
- 3D VAE的重建本身可能引入微妙的视觉变化(PSNR=39不是无限)→在极敏感场景需评估
- 不同肤色/光照条件下rPPG编辑的鲁棒性需更多验证
