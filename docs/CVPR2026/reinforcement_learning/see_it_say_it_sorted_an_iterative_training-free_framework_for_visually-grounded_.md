# See It, Say It, Sorted: An Iterative Training-Free Framework for Visually-Grounded Multimodal Reasoning in LVLMs

**会议**: CVPR 2026  
**arXiv**: [2602.21497](https://arxiv.org/abs/2602.21497)  
**代码**: 待确认  
**领域**: 强化学习  
**关键词**: 视觉定位, 多模态推理, 免训练, 迭代框架, LVLM

## 一句话总结

提出一个无需训练的迭代框架，通过"看到-说出-排序"的循环过程增强大视觉语言模型的视觉定位多模态推理能力，在多个推理任务上显著提升性能。

## 研究背景与动机

大视觉语言模型（LVLM）在多模态推理中表现出色，但在需要精确视觉定位的任务上仍有不足。模型往往依赖语言先验而忽略关键视觉证据。

核心痛点在于模型无法有效地将语言推理与精确的视觉区域关联起来。本文提出无训练的迭代方案，通过交替的视觉定位和语言推理实现更精确的视觉-语言对齐。

## 方法详解

### 整体框架

迭代式三阶段流程：See It（视觉定位关键区域）→ Say It（基于定位区域生成推理描述）→ Sorted（排序整合多轮推理结果），循环迭代直到收敛。

### 关键设计

1. **See It（视觉定位）**：引导 LVLM 聚焦于问题相关的视觉区域，通过注意力引导或区域裁剪实现精确视觉信息提取。

2. **Say It（推理描述）**：基于定位的视觉区域，让模型生成结构化推理链，将视觉观察转化为语言推理步骤。

3. **Sorted（排序整合）**：对多轮迭代的推理结果进行一致性排序和整合，选出最可靠的推理路径作为最终答案。

### 损失函数 / 训练策略

完全免训练，利用预训练 LVLM 的现有能力，通过提示工程和迭代策略实现推理增强。

## 实验关键数据

### 主实验

| 方法 | 主要指标 | 对比 |
|------|----------|------|
| 直接推理 | 基线 | - |
| CoT 推理 | 稍好 | 小幅提升 |
| **本文(迭代)** | **最优** | **显著提升** |

### 消融实验

| 配置 | 效果 | 说明 |
|------|------|------|
| 单轮推理 | 中 | 缺少迭代精化 |
| 多轮迭代 | 最优 | 逐步聚焦关键区域 |
| 无排序整合 | 不稳定 | 需要一致性筛选 |

### 关键发现

- 迭代视觉定位显著提升推理准确性
- 免训练方法可即插即用到任何 LVLM
- 在空间推理、计数等任务上提升最显著

## 亮点与洞察

- **免训练即插即用**：无需额外训练即可增强任何 LVLM
- **迭代精化思想**：通过多轮"看-说"循环逐步逼近正确答案
- **实用性强**：可直接应用于现有部署的 LVLM

## 局限性 / 可改进方向

- 迭代增加推理延迟
- 依赖模型本身的视觉定位能力
- 收敛轮次需要启发式设定

## 相关工作与启发

- 与 CoT 推理互补，增加了视觉定位维度
- 可与其他推理增强方法（如 ToT）结合
