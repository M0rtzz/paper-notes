# DiffFNO: Diffusion Fourier Neural Operator

**会议**: CVPR 2025  
**arXiv**: [2411.09911](https://arxiv.org/abs/2411.09911)  
**代码**: 无（未提及）  
**领域**: 图像修复 / 扩散模型  
**关键词**: 任意尺度超分, 傅里叶神经算子, 扩散模型, 模式重平衡, 自适应步长

## 一句话总结
提出 DiffFNO，将加权傅里叶神经算子（WFNO，通过模式重平衡保留高频傅里叶模式）与扩散框架结合，配合门控融合机制和自适应时间步 ODE 求解器，实现任意尺度图像超分辨率，在 DIV2K 多尺度上比 SRNO/LMI 提升 0.3-1.0 dB PSNR。

## 研究背景与动机

**领域现状**：任意尺度图像超分需要模型在连续尺度因子上工作，傅里叶神经算子（FNO）天然适合此任务因为它在频域操作。但标准 FNO 截断高频模式导致超分时细节丢失。

**现有痛点**：(1) SRNO 等方法直接截断高于阈值的傅里叶模式，丢失关键高频信息（影响纹理/边缘）；(2) 纯 FNO 方法缺乏空间局部性，对图像超分不够高效；(3) 扩散超分方法步数固定，某些简单样本浪费计算。

**本文要解决什么？** 在任意尺度超分中保留高频细节，同时高效推理。

**切入角度**：重新加权（而非截断）傅里叶模式让高频不被忽视 + 门控融合频域和空间域特征 + 自适应步长加速扩散采样。

**核心idea一句话**：WFNO 的模式重平衡保留高频 + GFM 融合频域/空间域特征 + ATS 动态步长 ODE 加速扩散采样。

## 方法详解

### 关键设计

1. **加权傅里叶神经算子 (WFNO) + 模式重平衡**：对傅里叶系数加权而非截断，高频模式获得更大权重以增强细节恢复。保留完整频谱信息

2. **门控融合机制 (GFM)**：自适应融合 WFNO 的频域全局特征和 Attention Neural Operator 的空间局部特征，结合两者优势

3. **自适应时间步 (ATS) ODE 求解器**：根据去噪过程中信号变化量动态调整积分步长——变化大时小步、变化小时大步，减少总步数

## 实验关键数据（DIV2K, EDSR 编码器）

| 尺度 | DiffFNO PSNR↑ | 最优基线 PSNR | 提升 |
|------|-------------|------------|------|
| ×2 | **35.72** | 35.40 (LMI) | +0.32 |
| ×4 | **30.88** | 30.40 (LMI) | +0.48 |
| ×8 | **26.78** | 26.16 (LMI) | +0.62 |
| ×12 | **26.48** | 25.91 (SRNO) | +0.57 |

极端尺度（×8, ×12）提升更大，说明高频保留对大尺度超分尤为关键。
