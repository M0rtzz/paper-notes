# Split Adaptation for Pre-trained Vision Transformers

**会议**: CVPR 2025  
**arXiv**: [2503.00441](https://arxiv.org/abs/2503.00441)  
**代码**: https://github.com/conditionWang/Split_Adaptation (有)  
**领域**: AI安全 / 隐私保护  
**关键词**: 模型分割, 低比特量化, 隐私保护, 少样本适应, OOD增强

## 一句话总结

提出 Split Adaptation，将预训练 ViT 分为客户端前端（量化+噪声）和服务端后端，通过 OOD 增强量化、Hilbert 变换增强和 patch 级检索增强实现少样本适应，在保护数据隐私和模型知识产权的同时在 CIFAR-100 5-shot 上达到 81.98%，计算量仅为线性探测的 4%。

## 研究背景与动机

**领域现状**：预训练 ViT 的部署面临双重隐私挑战——用户不想把数据传到服务器（数据隐私），服务提供商不想把模型暴露给用户（模型IP）。模型分割（split learning）是一种解法，但直接分割后少样本适应效果差。

**现有痛点**：（1）分割后的前端特征可能被攻击恢复原始图像（FORA 攻击可达 SSIM 0.80）；（2）量化降低了模型表达能力；（3）少样本场景下适应数据极少。

**核心矛盾**：隐私保护（量化+噪声）降低特征质量 vs 少样本适应需要高质量特征。

**切入角度**：用 OOD 数据增强+Hilbert 变换+patch 级检索来补偿量化和噪声带来的信息损失。

**核心idea一句话**：模型分割 + 8位量化 + 双层噪声 + OOD/HT/PR 三重增强 = 隐私安全的少样本适应。

## 方法详解

### 关键设计

1. **OOD 增强量化（OOD QAT）**：用公开 OOD 数据集的 representation-mixup 进行量化感知微调，减少量化误差

2. **Hilbert 变换增强**：将特征的频率信息通过 Hilbert 变换扩展，提供额外的训练信号

3. **Patch 级检索增强（PR）**：从支持集中检索相似 patch 特征来增强少样本的特征多样性

### 损失函数 / 训练策略

OOD QAT 用 Beta(0.75, 0.75) 混合增强 + Hessian 引导重建损失。最终适应用交叉熵。

## 实验关键数据

| 设置 | Split Adapt | Fine-tune | Linear Probe |
|------|------------|-----------|-------------|
| CIFAR-100 5-shot | **81.98%** | 42.37% | 74.05% |
| 计算量 | **2.5min** | 34min | 66min |
| 内存 | **2233MB** | 10302MB | 6979MB |
| FORA攻击 SSIM | **0.23** | - | 0.80 |

### 关键发现
- 三重增强各贡献 1.6-3.0%——OOD QAT 最重要
- 双层噪声将 FORA 攻击 SSIM 从 0.80 降到 0.23——有效保护隐私
- 计算量仅为线性探测的 4%（2.5min vs 66min）

## 亮点与洞察
- **同时保护数据和模型的实用方案**——量化保护模型IP，噪声保护数据隐私
- **OOD 数据就能补偿量化损失**——不需要目标域数据做量化校准

## 局限性 / 可改进方向
- 仅限少样本（3-10 shot）
- 需要服务端公开数据集做 OOD 增强
- 8位量化仍限制模型表达能力
