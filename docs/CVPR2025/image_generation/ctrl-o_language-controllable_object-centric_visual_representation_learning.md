# CTRL-O: Language-Controllable Object-Centric Visual Representation Learning

**会议**: CVPR 2025  
**arXiv**: [2503.21747](https://arxiv.org/abs/2503.21747)  
**代码**: https://ctrl-o-paper.github.io  
**领域**: 扩散模型 / 自监督学习  
**关键词**: 物体中心表示, 语言控制, Slot Attention, 视觉接地, 对比损失

## 一句话总结
提出 CTRL-O，通过语言查询条件化 Slot Attention 的 slot 表示，引入控制对比损失将 slot 的注意力加权 DINO 特征与语言查询对齐，实现可控的物体中心表示学习，无需掩码监督即可进行视觉接地和指代分割。

## 研究背景与动机

**领域现状**：物体中心表示学习（Object-Centric Learning）旨在将图像分解为独立的物体 slot 表示。现有方法（Dinosaur 等）无监督地发现物体，但 slot 与语义概念之间缺乏对应关系——不知道哪个 slot 对应"狗"还是"猫"。

**现有痛点**：(1) 无控制性——无法指定关注哪个物体；(2) slot 与语义的绑定（binding）是开放问题，传统方法的绑定率极低（8.1%）；(3) 无法直接用于下游任务如指代表达分割。

**本文要解决什么？** 让物体中心表示可以被语言控制——指定"红色汽车"就让对应 slot 表示该物体。

**切入角度**：在 Slot Attention 中引入语言查询条件化（查询初始化+解码器条件化），配合对比损失将 slot 注意力图内的视觉特征与语言对齐。

**核心idea一句话**：语言查询条件化 Slot Attention + 控制对比损失（slot 注意力加权 DINO 特征对齐语言嵌入）。

## 实验关键数据

| 任务/数据集 | CTRL-O | 最优基线 |
|----------|--------|---------|
| 物体发现 COCO FG-ARI | **47.5** | 40.5 (Dinosaur) |
| 绑定率 Binding Hits | **61.3%** | 8.1% (无控制) |
| 指代分割 RefCOCO mIoU | **28.2** | 21.8 (Shatter&Gather) |

控制对比损失 + 解码器条件化组合使绑定率从 8.1% 提升到 61.3%。
