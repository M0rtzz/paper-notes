## 研究背景与动机

动物的3D姿态和形状估计是计算机视觉中一个极具挑战性的研究方向。与人体姿态估计不同，动物界存在极其丰富的物种多样性：哺乳动物、鸟类、爬行动物等在形态学上差异巨大，这使得设计一个通用的模型来估计所有动物的姿态和形状变得异常困难。

**核心挑战：**

**形态多样性**：不同动物家族（如猫科、犬科、牛科、马科等）的体型比例、关节结构和运动模式差异显著。一只猫的身体结构与一匹马有本质区别
**数据稀缺性**：获取动物的3D标注数据非常困难，不像人体数据那样有大规模的MoCap数据集。野外动物很难进行精确的3D扫描
**遮挡与外观变化**：动物的毛发、羽毛等使得精确的体表几何估计更加困难，同时野外场景中的遮挡也很常见
**现有模型的局限性**：SMAL（Skinned Multi-Animal Linear）模型虽然提供了可变形的动物体型模板，但早期工作通常只针对特定类别（如狗、马）训练独立模型，无法跨物种泛化

本文的核心动机是设计一个能够感知动物家族关系的统一模型，利用同一家族内物种之间的形态学相似性来辅助姿态和形状估计。

## 方法详解

### 整体架构

AniMer的核心架构是一个家族感知Transformer（Family-Aware Transformer），将动物按照形态学相似性分组，并为每个家族设计专门的特征处理分支。

### 动物家族分组策略

模型根据形态学相似性将动物分为若干家族组：

| 家族组 | 典型物种 | 形态特征 |
|--------|----------|----------|
| 猫科 | 猫、豹、虎、狮 | 柔软体型，灵活脊柱 |
| 犬科 | 狗、狼、狐 | 修长四肢，直立尾巴 |
| 牛科/鹿科 | 牛、羊、鹿 | 粗壮体型，蹄类足部 |
| 马科 | 马、斑马、驴 | 高大体型，长颈 |
| 其他 | 熊、兔等 | 各种独特体型 |

这种分组策略使得同一组内的物种可以共享更多的形态先验信息，同时保留跨组的通用几何知识。

### Family-Aware Transformer模块

Transformer编码器从输入图像中提取全局特征，然后通过家族感知注意力（Family-Aware Attention）机制，根据检测到的动物类别动态调整特征表示：

- **共享主干**：所有物种共享相同的视觉特征提取器，学习通用的动物外观表示
- **家族特定头部**：每个家族组拥有独立的预测头，负责输出该家族特有的形状参数
- **跨家族注意力**：通过注意力机制在不同家族之间传递有用的几何信息

### SMAL模型参数估计

模型输出SMAL模型的参数，包括：

- **姿态参数 $\theta$**：控制关节角度，描述动物的姿态
- **形状参数 $\beta$**：控制体型变化，描述物种特异性的体型差异
- **全局朝向 $R$**：动物在3D空间中的旋转
- **平移参数 $t$**：动物在3D空间中的位置

### 损失函数设计

模型的训练采用多任务损失：

$$L = \lambda_\text{kp} L_\text{keypoint} + \lambda_\text{shape} L_\text{shape} + \lambda_\text{pose} L_\text{pose} + \lambda_\text{reg} L_\text{reg}$$

其中关键点损失衡量2D/3D关键点的投影误差，形状和姿态损失则直接监督SMAL参数。

## 实验结果

### 主要评估

AniMer在多个动物姿态估计基准上进行了评估：

- 在跨物种泛化能力方面表现优异，特别是对于训练集中未出现过的物种
- 家族感知设计有效提升了同家族内的估计精度
- 与现有的单物种方法相比，实现了更好的效率-精度平衡

### 消融实验

| 组件 | 作用 |
|------|------|
| 家族分组 | 提供了形态学先验，显著提升跨物种性能 |
| Family-Aware Attention | 动态调整特征，比固定分支更灵活 |
| 跨家族信息传递 | 防止家族间信息隔离，保持通用性 |

## 总结与展望

AniMer通过引入家族感知的设计理念，有效解决了动物姿态和形状估计中的物种多样性挑战。该方法展示了利用生物学分类知识来指导计算机视觉模型设计的潜力。未来可以考虑扩展到更多动物类别，以及结合视频序列进行时序一致的动物运动估计。
