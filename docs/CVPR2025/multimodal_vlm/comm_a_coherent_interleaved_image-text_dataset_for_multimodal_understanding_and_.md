# CoMM: A Coherent Interleaved Image-Text Dataset for Multimodal Understanding and Generation

## 论文信息

| 主题 | 多模态数据集 / 交错图文理解与生成 |

## 研究背景与动机

多模态大语言模型（MLLMs）在视觉-语言任务上取得了巨大进展，但现有模型大多在独立的图文对上训练，缺乏对交错图文内容（interleaved image-text）的深入理解能力。现实世界中的多模态内容往往是图文交错排列的，比如教程网页、新闻文章、科学论文等，图片和文字之间存在复杂的语义关联。

**现有数据集的核心问题：**

**缺乏连贯性**：现有的交错图文数据集（如从网页爬取的数据）通常图文之间的语义关联较弱，存在大量噪声。图片可能是广告、装饰图，与正文内容关系不大
**叙事结构缺失**：简单地将图片插入文本中并不能形成连贯的叙事（narrative）。一个好的交错内容应该像讲故事一样，图文之间有逻辑递进关系
**规模与质量的矛盾**：手动标注高质量的交错图文数据成本极高，而自动爬取的大规模数据质量参差不齐
**评估标准不统一**：缺乏针对交错图文理解能力的系统性评估基准

本文旨在构建一个大规模、高质量、具有连贯叙事结构的交错图文数据集CoMM，以提升MLLMs在交错内容上的理解和生成能力。

## 方法详解

### 数据集构建流程

CoMM的构建包含以下关键步骤：

**Step 1: 源数据收集与筛选**

从多种高质量来源收集原始图文内容，并通过严格的筛选机制过滤低质量数据。

**Step 2: 连贯性评估与增强**

设计自动化的连贯性评估模型，评估图文之间的语义关联强度，并对弱关联进行增强处理。

**Step 3: 叙事结构标注**

通过大语言模型辅助的标注流程，为每个文档标注叙事结构信息，使得图文之间形成逻辑通顺的叙事链条。

### 数据集特征

| 维度 | CoMM | 现有数据集 |
|------|------|------------|
| 图文连贯性 | 高（经过筛选和增强） | 低（原始爬取） |
| 叙事结构 | 有完整叙事标注 | 无或弱 |
| 内容多样性 | 覆盖多领域多主题 | 通常领域单一 |
| 质量控制 | 多轮自动+人工审核 | 有限或无 |
| 适用任务 | 理解+生成双任务 | 通常仅理解 |

### 连贯性度量

本文提出了多维度的连贯性度量指标：

- **局部连贯性**：相邻图文元素之间的语义关联强度
- **全局连贯性**：整个文档层面的叙事逻辑一致性
- **主题一致性**：图文内容是否围绕统一主题展开

### 训练策略

基于CoMM训练MLLMs时，采用了专门设计的训练策略：

- **交错注意力掩码**：允许模型同时关注前后文的图片和文本信息
- **课程学习**：从简单的图文对逐步过渡到复杂的多图文交错序列
- **生成-理解联合训练**：在同一框架中同时优化理解和生成目标

## 实验结果

### 数据集规模与统计

CoMM包含大规模的高质量交错图文样本，覆盖了多种领域和主题。数据集经过严格的质量控制流程筛选。

### 下游任务表现

在CoMM上预训练的模型在多项多模态任务上取得了显著提升：

- **交错图文理解**：在需要结合多张图片和上下文文字进行推理的任务上表现突出
- **连贯内容生成**：能够生成图文交错且叙事连贯的多模态内容
- **少样本学习**：得益于丰富的交错上下文，模型的少样本能力得到增强

| 能力维度 | 提升效果 |
|----------|----------|
| 多图推理 | 显著提升，尤其是需要跨图关联的场景 |
| 上下文理解 | 更好地利用前文图文信息 |
| 内容生成 | 生成内容的连贯性明显改善 |
| 少样本能力 | 交错上下文提供了更好的示例 |

## 总结与展望

CoMM通过提供大规模、高质量、具有连贯叙事结构的交错图文数据集，为多模态大语言模型的训练提供了重要的数据基础。该数据集的设计理念强调了图文内容之间的语义连贯性和叙事逻辑性，弥补了现有数据集在交错内容方面的不足。未来工作可以进一步扩展数据集的领域覆盖范围，并探索更复杂的多模态交互模式。
