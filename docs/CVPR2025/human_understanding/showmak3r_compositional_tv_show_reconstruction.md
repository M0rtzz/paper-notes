# ShowMak3r: Compositional TV Show Reconstruction

**会议**: CVPR 2025  
**arXiv**: [2504.19584](https://arxiv.org/abs/2504.19584)  
**代码**: https://nstar1125.github.io/showmak3r/ (有)  
**领域**: 人体理解 / 4D重建  
**关键词**: 视频4D重建, 3DGS舞台, SMPL时空定位, 镜头匹配, 表情拟合

## 一句话总结

提出 ShowMak3r，首个从网络视频（芭蕾/跑酷/电影片段）中进行组合式 4D 重建的完整流水线：3DGS 舞台重建 + 深度引导的 SMPL 时空定位 + 跨镜头演员匹配 + 隐式 MLP 面部表情精化，舞台 PSNR 19.65，面部 PSNR 24.34（比 ExAvatar +4.17）。

## 研究背景与动机

**领域现状**：4D 人体重建通常需要多视角同步摄像或单视角长时间连续拍摄。但娱乐视频（舞蹈/电影/体育）天然是多镜头切换的，每个镜头只有短暂的单视角片段。

**现有痛点**：（1）娱乐视频有大量镜头切换，传统方法无法跨镜头关联同一演员；（2）背景可能包含遮挡物需要先去除再重建；（3）演员在 3D 舞台中的精确定位需要考虑深度/穿透/平滑约束；（4）面部表情需要精细拟合但SMPL只有粗糙的FLAME参数。

**核心矛盾**：娱乐视频的信息是碎片化的（短镜头+切换+遮挡），但重建需要连续一致的 4D 表示。

**切入角度**：将问题分解为四个独立模块——舞台/定位/匹配/面部——每个模块解决一个子问题。

**核心idea一句话**：3DGS 舞台 + 深度对齐 SMPL + 镜头匹配 + 面部 MLP = 从网络视频到 4D 重建。

## 方法详解

### 关键设计

1. **3DGS 舞台重建**：深度引导优化 + 物体移除修复。$\mathcal{L}_{depth}$ 对齐渲染深度和单目估计深度，$\mathcal{L}_{TV}$ 平滑深度

2. **时空 SMPL 定位**：用深度/2D 关键点/轨迹 jerk（三阶差分最小化）/穿透约束联合优化 SMPL 的缩放、平移和朝向

3. **ShotMatcher 跨镜头匹配**：用欧氏距离在特征空间中关联不同镜头中的同一演员

4. **面部表情精化**：在 3DGS 上加隐式 MLP 残差调整颜色和不透明度

### 损失函数 / 训练策略

舞台：$\mathcal{L} = (1-\lambda_{SSIM})\mathcal{L}_{color} + \lambda_{SSIM}\mathcal{L}_{SSIM} + \lambda_{depth}\mathcal{L}_{depth} + \lambda_{smooth}\mathcal{L}_{TV}$。定位用 Geman-McClure 鲁棒关键点损失+穿透损失。面部用 L1+VGG 损失。

## 实验关键数据

| 指标 | ShowMak3r | 最佳基线 |
|------|-----------|---------|
| 舞台 PSNR | **19.65** | 19.34 (FSGS) |
| 面部 PSNR | **24.34** | 20.17 (ExAvatar) |
| MTED (定位误差) | **1.19** | 1.99 (无时空) |

### 关键发现
- 面部精化提升最大（+4.17 PSNR）——隐式 MLP 补偿了 SMPL 粗糙的面部参数
- 穿透约束改善 PSNR 0.58——防止演员与舞台交叉
- 可处理芭蕾/跑酷/电影等多种娱乐视频类型

## 亮点与洞察
- **首个从网络娱乐视频到4D重建的完整流水线**——连接了计算机视觉多个子任务
- **组合式而非端到端**——每个模块可独立替换/升级

## 局限性 / 可改进方向
- 离线处理（30分钟舞台+3小时演员@A6000）
- Web 视频只能重建已观测的背景
- 对深度估计敏感
