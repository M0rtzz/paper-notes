# DualPM: Dual Posed-Canonical Point Maps for 3D Shape and Pose Reconstruction

## 一句话总结

提出 Dual Point Maps (DualPM) 表示——从单张图像预测一对点图（相机空间 P + 规范空间 Q），将可变形物体的 3D 形状和姿态重建简化为点图预测问题，并引入分层 amodal 点图实现完整形状恢复（含自遮挡部分），仅用 1-2 个合成 3D 模型训练即可泛化到真实图像。

## 研究背景与动机

3D 计算机视觉中，数据表示的选择对深度学习的成功至关重要。DUSt3R 近期证明了点图(point maps)的强大——将像素映射到 3D 点，可以将静态场景重建中的匹配、相机估计、三角化等问题统一归约为点图预测。

**本文提出的核心问题**：能否将类似的思想应用于一个截然不同的问题——**可变形物体**的单目 3D 形状和姿态重建？

关键挑战：
- 单一点图 $P$ 仅给出 3D 位置，无法编码物体的变形/姿态信息
- 需要知道每个点在"中性姿态"下的位置才能恢复变形场
- 现有方法(MagicPony、3D-Fauna、Farm3D)依赖弱监督学习类别级模板或大量真实图像数据

**核心洞察**：如果预测**两个**点图——一个是相机空间的 3D 位置 $P$，另一个是规范空间(中性姿态)下的位置 $Q$——那么变形场就是两者的简单差 $P - Q$，姿态恢复变为平凡操作。

## 方法详解

### 整体框架

给定输入图像 $I$ 和物体掩码 $M$，先用预训练特征提取器 $\Psi$（如 DINOv2）提取特征 $F$，然后：(1) 从 $F$ 预测规范点图 $Q = \Phi_Q(\Psi(I))$；(2) 以 $Q$ 为条件预测姿态点图 $P = \Phi_P(Q)$。同时扩展为分层 amodal 表示，通过多层点图恢复自遮挡区域，实现完整 3D 重建。

### 关键设计

#### 1. 双点图表示 (Dual Point Maps)

- **功能**：统一编码可变形物体的 3D 形状和姿态信息，使姿态恢复和跨图像匹配变为平凡操作
- **核心思路**：每个像素 $\boldsymbol{u}$ 映射到两个 3D 点：$\boldsymbol{p} = P(\boldsymbol{u})$（相机空间当前姿态下的位置）和 $\boldsymbol{q} = Q(\boldsymbol{u})$（规范空间中性姿态下的位置）。变形场 = $P - Q$；跨图像匹配通过比较 $Q$ 实现（因为 $Q$ 对视角和姿态不变）
- **设计动机**：单一点图无法区分刚性运动和变形，双点图通过 canonical 参照系将两者解耦。类似于 NOCS (Normalized Object Coordinates) 但扩展到可变形物体

#### 2. 分层 Amodal 点图

- **功能**：重建物体完整 3D 形状，包括自遮挡不可见的部分
- **核心思路**：将每个像素与沿射线穿过物体表面的**多个**交叉点关联，形成有序序列 $(\boldsymbol{p}_1, \boldsymbol{p}_2, \ldots)$。使用分层表示，第一层编码可见点，后续层编码被前层遮挡的点，同时预测不透明度 $\sigma$。每个像素输出 $2K$ 层点（射线进出成对），总共 $3 \times 2 + (3+1) \times 2 \times (K-1)$ 个标量
- **设计动机**：标准点图只重建可见部分，不完整的重建限制了下游应用（如动画、骨架拟合）。分层表示类似于计算机图形学中的 depth peeling 技术，自然且高效

#### 3. 级联预测架构 (Q → P)

- **功能**：提升模型泛化能力，特别是在分布外图像上的表现
- **核心思路**：先预测规范点图 $Q$（利用预训练的 DINOv2 特征），再以 $Q$ 作为条件预测姿态点图 $P$。而非两者都从原始图像特征预测
- **设计动机**：$Q$ 的预测本质上是一个像素标注问题（因为 $Q$ 对姿态不变），相对简单且易泛化。用 $Q$ 作为 $P$ 的条件而非原始图像特征，可以减少对外观的过拟合，提升对新姿态和新外观的泛化。消融实验验证了此设计对分布外泛化的关键作用

### 损失函数

使用 self-calibrated L2 损失（来自 DUSt3R）：

$$\mathcal{L}_P = \frac{1}{|M|}\sum_{\boldsymbol{u} \in M} c_P(\boldsymbol{u})\|\hat{P}(\boldsymbol{u}) - P(\boldsymbol{u})\|^2 - \alpha \log c_P(\boldsymbol{u})$$

其中 $c_P(\boldsymbol{u}) > 0$ 是逐像素置信度。$\mathcal{L}_Q$ 类似定义。额外添加 $\mathcal{L}_\sigma$ 监督 amodal 层的不透明度预测。

## 实验关键数据

### 主实验表 (PASCAL VOC + Animodel)

| 方法 | PCK@0.1 (Horse) ↑ | Chamfer (Horse, cm) ↓ | Chamfer (Cow) ↓ | Chamfer (Sheep) ↓ |
|------|-------------------|----------------------|-----------------|-------------------|
| A-CSM | 32.9 | 11.75 | 9.52 | 9.24 |
| MagicPony | 42.9 | 11.19 | 10.29 | — |
| Farm3D | 49.1 | 11.34 | 9.63 | 11.01 |
| 3D-Fauna | 53.9 | 11.86 | 10.54 | 9.61 |
| Trellis | — | 6.93 | 6.80 | 5.91 |
| **DualPM (ours)** | **73.2** | **4.30** | **3.18** | **3.30** |

### 消融表

| 设计选择 | PCK (Horse) |
|---------|-------------|
| P 从图像特征预测 (不经过 Q) | 下降显著 |
| P 以 Q 为条件预测 (完整模型) | **73.2** |
| 无 amodal 扩展 | 仅重建可见部分 |

### 关键发现

- **PCK 提升 20 个百分点**：相比最强 baseline 3D-Fauna (53.9%) 提升至 73.2%
- **Chamfer 距离减半**：Horse 从 6.93cm (Trellis) 大幅降至 4.30cm
- **极少合成数据训练**：每个类别仅用 1-2 个 3D 模型渲染的合成数据训练，即可泛化到真实图像和分布外样本（小马驹、驴）
- **级联预测关键**：Q → P 的预测顺序比直接从图像预测两者显著提升分布外泛化能力
- **骨架拟合**：DualPM 的变形场可直接用于拟合 3D 关节骨架，实现动作迁移和动画

## 亮点与洞察

1. **概念上的优雅**：将 DUSt3R 的点图思想推广到可变形物体重建，对偶点图的设计极其自然——姿态就是两个空间的差异
2. **问题归约的力量**：将复杂的「3D 重建 + 姿态估计 + 匹配 + 骨架拟合」统统归约为「预测两个密集图」，网络友好且统一
3. **Amodal 扩展**：分层 depth peeling 的方式优雅地解决了自遮挡问题，且与训练流程无缝集成
4. **数据效率极高**：1-2 个合成模型的训练数据即可超越使用大规模真实数据训练的方法，证明了表示选择比数据量更重要
5. **canonical 点图作为特征**：发现 $Q$ 本身就是一个优秀的特征图，比 DINOv2 特征更适合作为 $P$ 预测的条件

## 局限性/可改进方向

1. **类别特定模型**：每个动物类别需要单独训练，虽有初步的零样本跨类能力但不完善
2. **仅验证四足动物**：未扩展到人体、鸟类等其他可变形物体类别
3. **依赖掩码输入**：需要预先提供物体分割掩码
4. **合成-真实域差距**：虽然泛化表现好，但极端情况（如遮挡严重的场景）仍可能失败
5. **Amodal 层数固定**：预设 $K$ 值限制了对复杂拓扑物体的建模能力

## 相关工作与启发

- **DUSt3R/MASt3R**：点图概念的起源，证明了点图预测可以简化 SfM 各环节
- **NOCS**：规范空间坐标的前身，但仅处理刚性物体
- **3D-Fauna/Farm3D/MagicPony**：可变形物体重建的先前 SOTA，依赖弱监督学习或大量真实数据
- **启发**：好的表示设计可以将困难问题变简单——DualPM 通过对偶点图让网络学习的任务变得更容易，因此即使用极少的合成数据也能超越复杂的弱监督方法
