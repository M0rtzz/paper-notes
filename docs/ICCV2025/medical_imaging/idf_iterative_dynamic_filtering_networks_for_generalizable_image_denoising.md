# IDF: Iterative Dynamic Filtering Networks for Generalizable Image Denoising

**会议**: ICCV 2025  
**arXiv**: [2508.19649](https://arxiv.org/abs/2508.19649)  
**代码**: [dongjinkim9.github.io/projects/idf](https://dongjinkim9.github.io/projects/idf)  
**领域**: 图像去噪  
**关键词**: 图像去噪, 动态滤波, 泛化能力, 迭代精炼, 轻量级模型

## 一句话总结

提出迭代动态滤波网络 (IDF)，仅用约 0.04M 参数，通过逐像素动态核预测 + 自适应迭代精炼策略，仅在单一级别高斯噪声上训练即可泛化到各种未见噪声类型（高斯/泊松/椒盐/蒙特卡洛渲染/真实噪声），实现出色的 OOD 去噪性能。

## 研究背景与动机

- 深度学习去噪方法（DnCNN、SwinIR、Restormer）在训练噪声分布上表现优异，但对未见噪声类型和级别的**泛化能力**严重不足
- 自监督方法需要耗时的测试时适应；数据增强方法受限于训练分布；基于预训练先验的方法需要大规模预训练
- MaskedDenoising 通过掩码训练正则化学习，但在细粒度纹理区域表现不佳
- **核心洞察**：动态核预测 + 迭代精炼可以避免过拟合特定噪声模式——
  - 动态核逐像素生成，适应局部图像上下文和噪声模式
  - 权重求和归一化约束使核充当加权平均算子，防止记忆训练噪声
  - 迭代策略渐进去噪，兼顾效率和质量

## 方法详解

### 整体框架

IDF 采用迭代去噪方案：输入噪声图像 $\mathbf{I}_{Noisy}$ 经过 T 个 Dynamic Image Denoising (DID) 块迭代去噪，所有 DID 块**共享权重**以减少参数并避免过拟合。推理时支持固定迭代和自适应迭代两种策略。

### 关键设计

1. **Dynamic Image Denoising (DID) 块**: 每次迭代 t，将上一步输出 $\hat{\mathbf{I}}_{Clean}^{(t-1)}$ 通过 unfold 操作提取 $K \times K$ 大小的重叠 patch，DID 块预测逐像素变化的去噪核 $\mathbf{w}^{(t)} \in \mathbb{R}^{1 \times K^2 \times (H \cdot W)}$，对 patch 执行卷积 $\mathbf{x}_i^{(t)} = \mathbf{w}_i^{(t)} \circledast \mathbf{y}_i^{(t)}$ 后折叠回图像。交替使用不同 dilation rate（如 2 和 1）鼓励学习多样特征。

2. **Feature Extraction Module (FEM)**: 通过 **RMS 归一化** 对输入进行样本级归一化 $\text{Norm}(\mathbf{a}) = \mathbf{a} / \sqrt{\frac{1}{N}\sum_i \mathbf{a}_i^2 + \epsilon}$，使特征对全局噪声级别变化保持不变。归一化后接两层 3×3 卷积 + ReLU，保持浅层和高效。

3. **Global Statistics Module (GSM) + Local Correlation Module (LCM)**:
   - GSM 利用迭代策略的优势：计算相邻迭代的残差 $\mathbf{I}_{Res}^{(t)} = \hat{\mathbf{I}}_{Clean}^{(t-1)} - \hat{\mathbf{I}}_{Clean}^{(t-2)}$，提取均值和标准差作为全局噪声统计量，类似 ISO 信息
   - LCM 计算 patch 内像素间的 Pearson 相关系数，高相关区域（平坦区）适用均匀核，低相关区域（边缘）使用选择性权重
   - 两者互补：GSM 提供全局噪声级别先验，LCM 提供局部结构先验

4. **Kernel Prediction Module (KPM)**: 将 FEM 特征与 GSM 特征进行通道注意力式相乘（$\mathbf{F}_{FE}^{(t)} \odot \mathbf{F}_{GS}^{(t)}$），拼接 LCM 特征后归一化，经 3×3 卷积生成核权重。使用 **Power Normalization**（$p=3$）代替 softmax 进行核归一化的求和归一约束，对异常值更鲁棒。

5. **Dynamic Iteration Control (DIC)**: 自适应决定迭代次数。核心观察：去噪完成时，核中心值收敛到 1。通过置信度图 $\mathbf{C}^{(t)} = \mathbf{w}^{(t)}(c_x,c_y) - \mathbf{w}^{(t-1)}(c_x,c_y)$ 监测连续迭代核心值变化，当空间平均 $\frac{1}{HW}|\sum_i \mathbf{C}_i^{(t)}| < \kappa$（$\kappa=0.015$）时提前终止。

### 损失函数 / 训练策略

- 损失函数：$\mathcal{L}_1$ 距离，$|\hat{\mathbf{I}}_{Clean}^{(T)} - \mathbf{I}_{Clean}|$
- 训练数据：CBSD432 数据集，仅加 **σ=15 的高斯噪声**
- 训练配置：AdamW 优化器，学习率 1e-4，50k 迭代，128×128 patch，batch=8
- 最大迭代 T=10，DIC 阈值 κ=0.015

### 关键发现

- **Power Normalization 是最关键的组件**：去除后 PSNR 下降超过 4 dB，说明求和归一约束是防止过拟合的核心
- DIC 自适应策略在保持性能的同时减少约 30% 迭代次数
- 仅 0.04M 参数（比 DnCNN 小 17 倍），在各种 OOD 噪声上全面超越大容量模型（Restormer 26.1M）
- 蒙特卡洛渲染噪声（64/128 spp）上同样表现优异，PSNR 分别达到 28.09 和 31.13

## 亮点与洞察

- **极简但有效**：0.04M 参数 + 单一级别训练数据 → 泛化到所有类型噪声，颠覆了"更大更好"的惯性思维
- **求和归一约束**的设计理念深刻：使每个核成为加权平均算子，从本质上防止网络记忆训练噪声模式
- **迭代策略**与动态核预测的天然协同：迭代产生的残差信息（GSM）和核收敛信息（DIC）都是"免费"的
- RMS 归一化实现噪声级别无关性，简单有效
- Power Normalization（p=3）替代 softmax 的选择很有洞察力——对异常值更鲁棒

## 局限性 / 可改进方向

- 推理速度受 PyTorch unfold 操作限制，优化这一操作可显著提升速度
- 在 PolyU 和 Nam 真实数据集上略逊于 CODE，说明极端轻量化有其天花板
- 仅在灰度/RGB 去噪上验证，未扩展到其他低级视觉任务（超分、去模糊）
- 可以探索核大小自适应或多尺度核预测

## 相关工作与启发

- 动态核预测思路源自 Kernel-Predicting CNN (Bako et al.) 和 KPN (Mildenhall et al.)，本文创新在于结合迭代精炼
- 与 MaskedDenoising 的掩码正则化思路正交，可以尝试结合
- RMS 归一化借鉴自 NLP 领域（LLM 的 RMSNorm），跨领域迁移成功
- 迭代精炼策略与 DBPN（超分）、IDR（无监督去噪）的思路一脉相承
