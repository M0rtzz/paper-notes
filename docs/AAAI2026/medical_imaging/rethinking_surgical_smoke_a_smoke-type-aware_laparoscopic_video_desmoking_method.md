# Rethinking Surgical Smoke: A Smoke-Type-Aware Laparoscopic Video Desmoking Method and Dataset

**会议**: AAAI 2026  
**arXiv**: [2512.02780](https://arxiv.org/abs/2512.02780)  
**代码**: [GitHub (数据集)](https://simon-leong.github.io/STSVD/)  
**领域**: 医学图像 / 视频修复  
**关键词**: 腹腔镜视频去烟, 烟雾类型感知, 烟雾遮罩分割, 烟雾解耦, 手术视频

## 一句话总结

本文首次将手术烟雾分为扩散烟（Diffusion Smoke）和环境烟（Ambient Smoke）两种类型，提出了第一个烟雾类型感知的腹腔镜视频去烟网络 STANet，包含语义软分割、粗到精解耦和双分支重建三个子网络，并构建了首个包含烟雾类型标注的大规模合成视频去烟数据集 STSVD。

## 研究背景与动机

腹腔镜手术视频能为外科医生提供实时视觉反馈，但电烧灼或激光等手术工具在灼烧组织时会不可避免地产生手术烟雾，严重降低视频可见度，遮挡解剖结构并影响临床决策。因此，腹腔镜视频去烟是一个重要的临床需求。

**现有方法的局限**：

**去雾方法**（DVD, SGDN, DehazeFormer 等）：虽然在自然场景中表现良好，但未考虑手术烟雾的时空特性，特别是烟雾的运动模式，因此在手术场景中效果不佳。

**去烟方法**（CycleGAN-based, AALIDNet, SelfSVD 等）：均将烟雾视为单一类型，忽略了烟雾运动模式的多样性。

**关键观察**：作者发现手术烟雾根据其运动模式可以被分为两种截然不同的类型：

- **扩散烟（Diffusion Smoke）**：出现在灼烧的早期阶段（碰撞前），具有**局部性和方向性**，烟雾从工具尖端沿特定方向扩散。
- **环境烟（Ambient Smoke）**：出现在灼烧的后期阶段（碰撞后），当烟雾与手术腔壁碰撞后产生湍流并重新分布，具有**全局性和无方向性**。

这两种烟雾具有完全不同的时空特征，因此需要针对性的去除策略。更复杂的是，多次灼烧操作会导致前次产生的环境烟与后续产生的扩散烟**相互交缠**，增加了分割和去除的难度。

## 方法详解

### 整体框架

STANet 是一个端到端的腹腔镜视频去烟网络，由三个子网络组成：

1. **烟雾特征感知子网络**（粉色区域）：提取和细化烟雾视频特征
2. **烟雾遮罩分割子网络**（蓝色区域）：联合预测烟雾遮罩和烟雾类型
3. **无烟视频重建子网络**（橙色区域）：在两种烟雾遮罩引导下进行双分支去烟

### 关键设计

1. **轻量级非刚性轨迹注意力**：考虑到手术烟雾的非刚性运动特征，作者借鉴 SODA 的时间-空间反转公式，引入了轻量级的非刚性轨迹注意力模块。与原始 SODA 相比，采用共享投影层、基于窗口的注意力方案，并减少注意力头数和向量维度，以压缩时间注意力和空间变形注意力的计算量。这使得网络能够捕获烟雾的可变形运动特征。

2. **语义软分割模块（S3M）**：这是烟雾遮罩分割子网络的核心组件。将不同类型烟雾的软分割建模为**集合预测范式**：

    - 使用 $N$ 个可学习查询 $q_i$，每个查询预测一个局部烟雾遮罩 $m_i$ 和烟雾类型 $t_i$
    - 查询通过三层级联分割块（含掩码交叉注意力、自注意力和前馈网络）与多尺度时空特征交互
    - 通过**注意力加权遮罩聚合机制**将局部遮罩汇聚为两个烟雾类型特定的全局遮罩：

   $$M^*_{typ} = \sum_i \frac{w_i}{\sum_i w_i} \cdot m_i, \quad \{i \mid t_i = typ\}$$

   其中 $typ$ 为 $diff$（扩散烟）或 $amb$（环境烟）。

3. **粗到精解耦模块（C2FDM）**：这是处理两种烟雾交缠问题的关键模块，嵌入在 S3M 之后进行遮罩细化。

    - **遮罩区域选择子模块**：通过二值化和集合运算从 $M^*_{diff}$ 和 $M^*_{amb}$ 生成三个互斥区域遮罩，分别提取扩散烟区域 $R^*_{diff}$、环境烟区域 $R^*_{amb}$ 和交缠区域 $R^*_{ent}$ 的特征
    - **烟雾类型感知交叉注意力子模块**：三种区域特征经 patch embedding 和线性投影产生烟雾类型特定的查询 $Q_{typ}$、键 $K_{typ}$ 和共享的交缠值 $V_{ent}$，然后通过两个交叉注意力将交缠区域分离为两种烟雾类型：

   $$M'_{typ} = \text{FFN}\left(\text{Softmax}\left(\frac{Q_{typ} K_{typ}^\top}{\sqrt{d}}\right) V_{ent}\right) + M^*_{typ} \cdot B_{typ}$$

   - **迭代细化块**：包含四头自注意力层、归一化层和前馈网络，逐步细化遮罩得到最终的 $M_{diff}$ 和 $M_{amb}$

4. **双分支无烟重建子网络**：根据两种烟雾的不同特性采用差异化策略：

    - **扩散烟分支**（可变形卷积）：利用相邻帧的扩散烟遮罩拼接为时间复合遮罩 $\bar{M}_{diff}$，通过 CoordConv 层和通道缩减注意力生成 18 通道偏移场，再用 $3 \times 3$ 可变形卷积沿烟雾扩散轨迹提取对齐特征：$F_{diff} = \text{DeformConv}(F, \Delta_{offset})$
    - **环境烟分支**（自适应膨胀卷积）：利用时间复合遮罩 $\bar{M}_{amb}$ 估计 $K=3$ 个自适应膨胀采样位置图，以膨胀率 1、2、3 进行并行膨胀卷积并融合：$F_{amb} = \sum_{k=1}^{3} \text{DilatConv}(F, rate_k, map_k)$
    - 两分支的特征送入 U-Net 解码器重建无烟视频帧。当视频中仅有一种烟雾时，只激活对应分支以节省计算。

### 损失函数 / 训练策略

采用多任务损失 $\mathcal{L}_{mul}$ 联合监督烟雾遮罩分割、烟雾类型分类和无烟视频重建。此外引入**烟雾高频翼损失（SHWL）** $\mathcal{L}_{shwl}$：

- 用 $3 \times 3$ 高通滤波器提取 GT 遮罩和预测遮罩的高频成分
- 用 Wing Loss 优化高频误差，更好地关注被忽视的小误差
- 引入指数调制因子 $\phi = 1 + \lambda_g(e^{M_{GT}} - 1)$，在浓烟区域增大惩罚，稀烟区域减少过拟合

总损失：$\mathcal{L}_{total} = \mathcal{L}_{mul} + \phi \mathcal{L}_{shwl}$

### 主实验

**三个测试集综合比较**（合成 STSVD + 配对真实 Vivo + 无配对真实 STSVD-R）：

| SGDN | 去雾 | 32.66 | 0.9674 | 23.15 | 0.8631 | 0.3056 | 40.19 |
| SelfSVD | 去烟 | 29.51 | 0.9588 | 22.13 | 0.8376 | 0.3154 | 38.42 |

### 关键发现

1. **STSVD 数据集的重要性**：M1→M2 的训练数据集切换带来了 PSNR 1.89dB 的巨大提升，证明高质量、多样化的烟雾类型标注数据集的关键作用。
2. **S3M 的有效性**：M2→M3 中 S3M 带来 0.71dB 提升，证明烟雾类型感知的遮罩预测对去烟质量有显著贡献。
3. **C2FDM + SHWL 的协同效果**：M3→M5 中 C2FDM 和 SHWL 共同带来 1.19dB 提升，解决了两种烟雾交缠区域的准确分割问题。
4. **下游任务泛化**：去烟处理显著提升了息肉检测和器械分割的性能，本文方法在所有下游任务上均为最优。

## 亮点与洞察

- **问题定义的创新**：首次将手术烟雾分为两种类型进行分析处理，这一分类基于物理现实（烟雾与腔壁碰撞产生湍流），具有很强的合理性。
- **端到端的类型感知设计**：从遮罩分割到视频重建全流程都考虑了烟雾类型的差异，使用可变形卷积处理方向性的扩散烟、膨胀卷积处理全局性的环境烟，设计与物理特性高度匹配。
- **解耦机制的巧妙设计**：C2FDM 利用非交缠区域的信息引导交缠区域的分离，通过交叉注意力实现类型感知的解耦，是整个方法的关键技术贡献。
- **完整的数据闭环**：不仅提出了方法，还构建了首个包含烟雾类型标注的大规模数据集（120 个视频、12000 帧、28 种手术类型），极大地推动了该领域的研究基础。

## 局限性 / 可改进方向

1. **合成-真实域差距**：虽然在真实数据集上也表现优异，但训练数据仍为合成数据，合成烟雾与真实烟雾之间可能存在差距。
2. **烟雾类型的简化假设**：将烟雾严格分为两类可能无法覆盖所有真实场景，实际中可能存在更多过渡状态。
3. **计算效率**：总参数 25.15M，FLOPs 174.13G，虽不是最高但对于实时手术应用可能仍需优化。
4. **时序建模深度有限**：当前仅利用相邻帧的时间信息，对于长时间的烟雾变化模式建模可能不足。
5. **无真实烟雾类型标注**：STSVD-R 真实数据集仅有场景级别标注，缺乏像素级真实烟雾类型标注用于更严格的验证。

## 相关工作与启发

- 烟雾类型分类的思路可以扩展到其他视频退化问题，如将雾霾按密度/运动状态分类进行差异化处理。
- C2FDM 的解耦思想对多类型退化混合场景（如雨+雾、噪声+模糊）具有借鉴意义。
- 构建任务专用的大规模合成数据集并包含细粒度标注（如类型标注），是数据驱动方法持续进步的重要基础建设。
- 在下游任务上的评估展示了去烟方法作为前处理对整个手术辅助系统的影响，这种评估方式值得提倡。
