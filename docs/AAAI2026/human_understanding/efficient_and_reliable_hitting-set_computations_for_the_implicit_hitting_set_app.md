# Efficient and Reliable Hitting-Set Computations for the Implicit Hitting Set Approach

**会议**: AAAI 2026  
**arXiv**: [2508.07015](https://arxiv.org/abs/2508.07015)  
**领域**: 组合优化  
**关键词**: 隐式打击集, PB推理, IP求解器, 计算证书

## 一句话总结
探索IHS框架中打击集计算的替代方案（PB推理+随机局部搜索），发现商业IP最高效但有数值不稳定性，PB推理可提供竞争性效率+正确性保证+计算证书。

## 方法详解
IHS迭代于冲突提取和打击集优化。探索PB推理和随机局部搜索作为替代优化器。基于VeriPB证明格式实现首个IHS计算证书。
