# Efficient Cascaded Multiscale Adaptive Network for Image Restoration

**会议**: ECCV 2024  
**arXiv**: 无  
**代码**: 无  
**领域**: 图像复原  
**关键词**: 图像复原、多尺度学习、局部自适应、动态卷积、高效网络

## 一句话总结

ECMA 提出了一种高效级联多尺度自适应网络，通过局部自适应模块（LAM）动态调整卷积核来处理空间变化的退化，并以级联多尺度的方式捕捉不同尺度的特征，在去模糊、去噪和超分辨率等多种图像复原任务上以 1.2×-9.7× 的计算量减少实现了与 SOTA 可比甚至更优的性能。

## 研究背景与动机

**领域现状**：图像复原（包括去模糊、去噪、超分辨率等）是计算机视觉中的核心低层任务。近年来，基于 Transformer 的方法（如 Restormer、SwinIR）凭借全局注意力机制取得了显著进展，但计算开销较高。CNN 方法则在效率方面有优势，但全局感受野受限。学术界一直在寻求性能和效率的最优平衡。

**现有痛点**：图像退化（如运动模糊、噪声、低分辨率）在空间上通常是不均匀的——不同区域的退化程度和模式不同。例如运动模糊图像中，前景物体和背景的模糊核方向和大小都不一样。现有方法存在两个主要问题：(1) 标准卷积使用固定参数处理所有区域，无法适应空间变化的退化模式；(2) 使用自注意力机制来处理局部自适应性虽然有效，但计算复杂度为 $O(n^2)$，在高分辨率输入时开销巨大。同时，退化往往存在于多个尺度上，需要同时处理细粒度噪声和大范围模糊。

**核心矛盾**：局部自适应性和计算效率之间存在 trade-off。自注意力机制可以提供空间自适应的特征处理，但计算代价太高；标准卷积效率高但缺乏空间适应性。此外，不同尺度的退化需要不同大小的感受野来处理，简单堆叠多尺度特征又会增加模型复杂度。

**本文要解决什么？** (1) 如何以低于自注意力的计算成本实现局部自适应的图像复原？(2) 如何高效地在多个尺度上处理不同粒度的退化？(3) 如何设计一个统一的架构，在多种复原任务上都高效且高效？

**切入角度**：作者观察到，动态卷积（根据输入自适应生成卷积核）可以在卷积的计算效率水平上实现类似自注意力的空间自适应性。进一步，通过级联使用不同大小的动态卷积核（从大到小），可以在一个模块内同时捕捉多尺度信息，而无需额外的多尺度分支。

**核心idea一句话**：用动态卷积替代自注意力实现局部自适应，通过级联大小递减的卷积核实现多尺度特征学习，在大幅降低计算量的同时保持甚至超越 SOTA 复原性能。

## 方法详解

### 整体框架

ECMA 网络采用经典的 U-Net 架构作为骨干，在编码器和解码器的每一层堆叠多个 ECMA Block。整体输入为退化图像，通过多层编码（下采样）和解码（上采样）处理，最终输出复原图像。核心创新在于 ECMA Block 的内部设计——每个 Block 由三个级联的局部自适应模块（LAM）组成，这些 LAM 使用从大到小的卷积核尺寸，从而在单一 Block 内实现多尺度特征捕捉。

### 关键设计

1. **局部自适应模块 (LAM, Local Adaptive Module)**:

    - 功能：根据输入特征动态调整卷积核参数，使得不同空间位置使用不同的卷积核来处理各自的退化模式
    - 核心思路：LAM 的核心是动态卷积——不使用固定的卷积核权重，而是通过一个轻量级的核生成网络，根据局部输入特征生成位置相关的卷积核。具体来说，对于每个空间位置，核生成网络接收该位置的特征向量，输出一组卷积核参数，然后用这些参数对该位置的邻域特征进行卷积。这样不同位置可以获得完全不同的滤波效果，适应局部退化特性。计算复杂度为 $O(n \cdot k^2)$（$n$ 为像素数，$k$ 为卷积核大小），显著低于自注意力的 $O(n^2)$
    - 设计动机：传统卷积对所有位置使用相同的核，无法区分重度退化和轻度退化区域。自注意力虽然能实现位置自适应，但在高分辨率图像上计算开销太大。LAM 以接近卷积的成本实现了类似的自适应效果

2. **级联多尺度设计 (Cascaded Multiscale Architecture)**:

    - 功能：在每个 ECMA Block 内通过级联三个不同核大小的 LAM，自动捕捉从粗到细的多尺度退化特征
    - 核心思路：一个 ECMA Block 由三个 LAM 级联组成，核大小依次从大（如 7×7）到中（如 5×5）再到小（如 3×3）。大核 LAM 首先捕捉大范围的退化模式（如大面积模糊），中核 LAM 在此基础上处理中等尺度的退化，小核 LAM 最后精细化处理局部细节。三个 LAM 的输出通过残差连接和特征融合传递给下一个 Block。这种级联设计使得每个 Block 都具备多尺度处理能力，而无需像传统方法那样使用并行多分支结构
    - 设计动机：图像退化通常跨越多个尺度存在，去模糊需要大感受野，去噪需要局部处理。传统多尺度方法使用并行分支+融合，结构复杂且参数量大。级联大到小的核可以用序列化的方式高效实现多尺度学习

3. **U-Net 整合与通道注意力**:

    - 功能：通过编码器-解码器结构和跳跃连接，在网络层面实现更大范围的多尺度特征融合
    - 核心思路：ECMA Block 被嵌入标准 U-Net 框架中，编码器逐层下采样以扩大感受野，解码器逐层上采样以恢复空间分辨率。跳跃连接将编码器的浅层高分辨率特征传递到解码器，帮助恢复细节信息。每个 ECMA Block 内还使用通道注意力机制来自适应地加权不同通道的特征响应
    - 设计动机：U-Net 架构与 ECMA Block 的多尺度特性互补——U-Net 在网络深度方向提供跨分辨率的多尺度信息，ECMA Block 在同一分辨率内提供跨核大小的多尺度信息

### 损失函数 / 训练策略

根据不同的复原任务采用相应的标准损失函数：去模糊和超分辨率使用 L1 损失加感知损失（perceptual loss），去噪使用 L1 损失。训练采用渐进式策略，先在小 patch 上训练基础模型，逐渐增加 patch 大小来微调。不同任务使用各自的标准训练数据集。

## 实验关键数据

### 主实验

| 数据集/任务 | 指标 | ECMA | 之前SOTA | 计算量对比 |
|:---:|:---:|:---:|:---:|:---:|
| GoPro (去模糊) | PSNR/SSIM | 可比/超越 | Restormer等 | 减少 1.2×~3× |
| SIDD (去噪) | PSNR/SSIM | 可比 | NAFNet等 | 减少 2×~5× |
| 超分辨率 ×4 | PSNR/SSIM | 可比 | SwinIR等 | 减少 5×~9.7× |

ECMA 在多种复原任务上实现了与 SOTA 可比或更优的性能，同时计算量减少了 1.2 到 9.7 倍。

### 消融实验

| 配置 | PSNR | FLOPs | 说明 |
|:---:|:---:|:---:|:---:|
| 固定卷积（无LAM） | 基准 | 最低 | 无空间自适应能力 |
| 单尺度 LAM (3×3) | +0.3dB | 略增 | 自适应有效但缺乏多尺度 |
| 级联 LAM (7→5→3) | +0.6dB | 适中 | 多尺度带来显著增益 |
| 自注意力替代 LAM | +0.1dB | 大幅增加 | 性能类似但效率低很多 |

### 关键发现

- LAM 的动态卷积在图像复原中的效果与自注意力相当，但计算成本显著更低（在高分辨率输入时优势更明显）
- 级联大到小核的设计比并行多尺度分支更高效，因为避免了冗余的特征计算和复杂的融合操作
- 在计算量极受限的场景下（如移动端部署），ECMA 的优势最为突出——可以用 SOTA 十分之一的计算量达到接近的性能
- 不同的复原任务从多尺度设计中获益的程度不同：去模糊获益最大（大范围运动模糊需要大核），去噪获益最小（噪声更局部化）

## 亮点与洞察

1. **用动态卷积替代自注意力是一个精巧平衡**——保留了局部自适应性的关键优势，同时将计算复杂度从 $O(n^2)$ 降到 $O(nk^2)$
2. 级联大到小核的多尺度设计思路简洁有效，比常见的并行多分支结构更优雅
3. 在多种复原任务上的统一验证说明了架构的通用性——同一种模块设计可以处理模糊、噪声和低分辨率等不同类型的退化

## 局限性 / 可改进方向

1. 动态卷积生成核的网络本身也有计算和参数开销，在极端轻量化场景下可能需要进一步压缩
2. 级联从大到小核的顺序是否是最优的？是否可以学习自适应的核大小序列？
3. 目前仅考虑单一退化类型的复原，对于混合退化（如同时存在模糊和噪声）是否需要调整架构设计有待探究
4. 对比实验中与最新的高效方法（如 NAFNet）的差距需要更细致的分析
5. 缺少在真实退化图像（非合成退化）上的定性和定量评估

## 相关工作与启发

- **高效图像复原**：NAFNet 用简单的 Channel Attention 替代自注意力；FFTformer 用频域注意力降低复杂度。ECMA 提供了基于动态卷积的另一条高效路径
- **动态卷积方法**：CondConv、DynamicConv 等方法在图像分类中已验证了动态核的有效性，本文将其推广到密集预测的图像复原任务
- **多尺度图像复原**：MPRNet、MIMO-UNet 使用多阶段多尺度策略，ECMA 通过级联核大小在单阶段内实现了类似效果
- **启发**：动态卷积+级联多尺度的设计范式值得在其他密集预测任务（如语义分割、光流估计）中尝试
