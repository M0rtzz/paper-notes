# Exploring Pre-trained Text-to-Video Diffusion Models for Referring Video Object Segmentation

**会议**: ECCV 2024  
**arXiv**: [2403.12042](https://arxiv.org/abs/2403.12042)  
**代码**: [GitHub](https://github.com/buxiangzhiren/VD-IT)  
**领域**: 图像分割 (Referring Video Object Segmentation)  
**关键词**: 扩散模型, 文本到视频, 指代视频对象分割, 时序一致性, 特征表示

## 一句话总结

首次探索**预训练文本到视频(T2V)扩散模型**的特征表示用于视频理解任务，提出 VD-IT 框架，通过文本引导图像投影和视频专属噪声预测两个设计，在 R-VOS 任务上实现了优于判别式视频骨干网络的时序一致性和分割性能。

## 研究背景与动机

### 问题背景

Referring Video Object Segmentation (R-VOS) 任务要求根据自然语言描述，在视频序列中分割出特定目标物体，并保持时序上的一致性。现有方法（如 SgMg、ReferFormer 等）主要依赖**判别式预训练的视频骨干网络**（如 Video Swin Transformer），这些骨干网络通过分类等判别式预训练任务学习特征，再与文本驱动的 mask decoder 联合微调。

### 核心动机

作者提出一个大胆的假设：**预训练的生成式 T2V 扩散模型**天然地学习了丰富的语义信息和连贯的时序对应关系，因此可以比判别式模型更好地服务于视频理解任务。这一假设基于以下直觉：

**时序一致性优势**：T2V 扩散模型在生成过程中使用全局文本提示作为条件输入，指导生成语义一致的视频帧，天然具有跨帧语义对齐能力
**去噪鲁棒性**：扩散模型的去噪机制赋予其对光照变化、相机运动等外部干扰的鲁棒性
**丰富的视觉先验**：T2V 模型在大规模视频数据上训练，编码了丰富的时空视觉先验

通过 K-Means 聚类可视化特征发现，V-Swin 特征会因光照变化等因素产生显著波动，而 T2V 扩散模型的特征保持了更强的时序一致性。

### 关键挑战

直接使用 T2V 扩散模型并不能达到 SOTA 性能，需要专门的设计来充分挖掘其潜力：
- 仅用文本条件能保证时序一致但缺乏细节
- 仅用图像条件能保留细节但容易混淆不同实例
- 标准高斯噪声会模糊关键细节

## 方法详解

### 整体框架

VD-IT 基于开源的 ModelScopeT2V 扩散模型构建，包含两个核心部分：
1. **视觉特征提取**：利用固定的预训练 T2V 扩散模型提取视频特征
2. **Mask 预测头**：基于查询的分割模型，融合文本和视觉特征生成最终分割 mask

特征提取需要两个输入：条件提示嵌入（由 Text-Guided Image Projection 生成）和加噪视频（由 Noise Prediction 模块处理）。

### 关键设计

#### 1. Text-Guided Image Projection (文本引导图像投影)

**做什么**：生成融合文本和图像信息的条件提示嵌入，用于引导 T2V 模型提取特征。

**核心思路**：作者系统性地探索了三种条件设置：
- **VD-T**（仅文本条件）：直接用 referring text 作为提示，但产生的 mask 缺乏细粒度细节，因为文本只是泛化的条件，无法提供逐帧的实例级信息
- **VD-I**（仅图像条件）：用 CLIP 视觉模型提取每帧的视觉 token 作为隐式文本嵌入，能保留细节但会在高层特征中混淆不同实例（如错误包含背景中的其他物体）
- **VD-IT**（文本+图像条件）：同时使用 referring text 和 visual tokens，通过交叉注意力融合二者

融合公式为：

$$p_{ve,t} = \text{MLP}\left(p_e + \text{SoftMax}\left(\frac{p_e W^Q (p_{v,t} W^K)^T}{\sqrt{d_k}}\right) p_{v,t} W^V\right)$$

其中文本 token $p_e$ 作为 query，图像 token $p_{v,t}$ 作为 key 和 value，确保最终提示兼具时序语义一致性和细节精确性。

**设计动机**：低层特征分析表明 VD-T 缺乏细节而 VD-I 高层语义混乱，VD-IT 在两个层级都取得了最优效果。

#### 2. Noise Prediction (视频专属噪声预测)

**做什么**：预测与视频内容相关的噪声，替代标准高斯噪声，在扩散前向过程中更好地保留特征保真度。

**核心思路**：首先将视频的潜在向量 $\mathcal{F}_o$ 通过卷积层，然后进行归一化得到预测噪声：

$$n_t = \frac{f_{n,t} W^N - \mu(f_{n,t} W^N)}{\sigma(f_{n,t} W^N) + \epsilon}$$

其中 $W^N \in \mathbb{R}^{4 \times 4}$ 是可训练权重，$\epsilon = 1e\text{-}5$ 防止除零。

**设计动机**：以往方法（如用于图像分割的扩散方法）添加独立的高斯噪声会模糊关键细节。本文方法通过预测与视频内容相关的噪声，能更好地保留原始视觉信号的保真度，进而提升最终分割质量。

#### 3. Mask Prediction Head (Mask 预测头)

**做什么**：融合文本和视觉特征，生成最终的分割 mask。

**核心思路**：
- **Text-Instance Matching**：使用 Q 个可学习的查询向量与 RoBERTa 编码的文本特征做交叉注意力，得到实例查询
- **Vision-Language Fusion**：将视觉特征通过 Deformable Transformer 编码器处理多尺度特征，然后在解码器中与实例查询融合
- **Segmentation Head**：三个子头（bbox、分类、mask），使用 Hungarian 算法匹配最优实例

特征从 U-Net 提取 3 级（8× 到 32× 下采样），加上自编码器中间层的 4× 下采样特征，构成 4 级层次化视觉特征。

### 损失函数 / 训练策略

- **Mask 损失**：Dice loss + Focal loss
- **分类损失**：Focal loss（二分类器判断每帧是否包含目标）
- **边框损失**：L1 loss + GIoU loss
- 使用与 SgMg、ReferFormer 等方法相同的损失权重以确保公平比较
- 训练使用 2 块 A100 GPU，每个 clip 5 帧，训练 9 个 epoch
- 帧尺寸调整为最长边 640 像素

## 实验关键数据

### 主实验

**Ref-YouTube-VOS & Ref-DAVIS17** (不使用 RefCOCO 预训练)：

| SgMg | Video-Swin-T | 58.9 | 56.7 | 65 |
| SgMg | Video-Swin-B | 61.6 | - | 40 |
| **VD-IT** | **Video Diffusion** | **64.8** | **63.0** | 21 |

### 关键发现

1. **单独使用 VD-I 不一定优于判别式骨干**：VD-I (59.7 J&F) 低于 SgMg (61.6)，说明 SOTA 性能并非来自扩散模型的更大容量或更多数据
2. **文本引导是关键**：加入 referring text 后时序一致性大幅提升，帧间 IoU 差异降低约 2 个点
3. **VD-IT 在光照噪声下更鲁棒**：对亮度变化的抵抗力明显强于 V-Swin 方法
4. **在 RefCOCO 图像分割上与 SgMg 性能相近**：证明优势来自时序一致性而非单帧质量

## 亮点与洞察

1. **首次验证 T2V 扩散模型的时序先验可用于视频理解**：这是一个范式级的洞察，打破了"生成模型只能生成、判别模型只能理解"的固有认知
2. **系统性的条件设计探索**：VD-T、VD-I、VD-IT 的逐步对比非常有说服力，每一步改进都有清晰的可视化支撑
3. **噪声预测模块设计精巧**：用可学习的视频相关噪声替代高斯噪声，思路简单但有效

## 局限性 / 可改进方向

1. **推理速度较慢**：FPS 仅 21（SgMg 为 65），扩散模型的计算开销限制了实时应用
2. **仅基于 ModelScopeT2V**：如果使用更强的 T2V 模型（如 Stable Video Diffusion）是否能进一步提升？
3. **T2V 模型参数固定**：探索部分微调 T2V 模型是否能进一步提升性能
4. **对更长视频的扩展性**：当前实验以短 clip 为单位，长视频场景下的表现未知

## 相关工作与启发

- **生成式 → 判别式的趋势**：与 DiffusionDet（扩散模型做检测）等工作思路一脉相承，进一步拓展到视频领域
- **ODISE、VPD 等工作**：在图像理解中已验证扩散特征的有效性，本文是其视频版延伸
- **R-VOS 方法演进**：从 MTTR → ReferFormer → SgMg → VD-IT，backbone 从判别式走向生成式

## 评分

- **新颖性**: ⭐⭐⭐⭐⭐ — 首次探索 T2V 扩散模型用于视频理解，视角新颖且具有启发性
- **实验充分度**: ⭐⭐⭐⭐ — 4 个基准、详细消融和特征分析，但缺少与更多 T2V 模型的对比
- **写作质量**: ⭐⭐⭐⭐ — 动机清晰，逐步推导条件设计的逻辑链条完整
- **价值**: ⭐⭐⭐⭐ — 为视频理解提供了全新的特征提取范式，但推理效率待优化
