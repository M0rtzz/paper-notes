# ProMerge: Prompt and Merge for Unsupervised Instance Segmentation

**会议**: ECCV 2024  
**arXiv**: [2409.18961](https://arxiv.org/abs/2409.18961)  
**代码**: 无  
**领域**: Segmentation  
**关键词**: 无监督实例分割, 自监督特征, patch合并, 背景剪枝, DINO

## 一句话总结

提出ProMerge方法，利用自监督视觉特征进行patch初始分组和策略性合并，结合背景mask剪枝技术，实现高效的无监督实例分割。

## 研究背景与动机

无监督实例分割（Unsupervised Instance Segmentation）旨在无需人工标签的情况下分割图像中不同的物体实例。这个任务的实际意义重大——消除了对昂贵实例级标注的依赖，使模型能在开放世界场景中工作。

近年来，自监督学习模型（如DINO）学到的视觉特征展示了强大的局部对应能力，为无监督分割提供了基础。当前的SOTA方法通常将图像表示为图结构，并通过求解广义特征值系统（即normalized-cut）来生成前景mask。虽然效果较好，但这类方法存在一个关键瓶颈——**计算开销大**，导致推理速度慢，限制了实际应用。

本文的核心动机是：能否在保持甚至超越normalized-cut方法性能的同时，大幅降低计算复杂度？ProMerge给出了肯定的答案——通过简单的patch分组和合并策略，配合精巧的背景处理技术，实现了更快速且有竞争力的无监督实例分割。

## 方法详解

### 整体框架

ProMerge的pipeline包含四个步骤：(1) 使用自监督模型（DINO/DINOv2）提取patch级视觉特征；(2) 基于特征相似性进行初始patch分组（prompting）；(3) 对初始分组进行策略性合并（merging）以形成完整的实例mask；(4) 应用基于背景模型的mask剪枝，去除错误的背景分割。

### 关键设计

1. **Prompt-based Initial Grouping（基于提示的初始分组）**:
    - 功能：将图像patch快速聚类为初步的区域组
    - 核心思路：利用DINO特征的局部对应性质，以每个patch为种子点（prompt），通过特征相似度阈值收集邻近的相似patch形成初始组。这些初始组可能过度分割（一个物体被分成多个组），但保证了高精度
    - 设计动机：避免了normalized-cut方法中的大规模矩阵运算，用简单的相似度计算替代

2. **Strategic Merging（策略性合并）**:
    - 功能：将过度分割的初始组合并为完整的实例mask
    - 核心思路：设计多级合并策略：首先计算初始组之间的特征相似度和空间邻接关系；然后按照相似度从高到低逐步合并相邻的组。合并过程中引入大小约束和紧凑性约束，避免将不同物体的组错误合并
    - 设计动机：初始分组追求高精度（precision），合并步骤追求高召回率（recall），两者互补

3. **Background-based Mask Pruning（基于背景的mask剪枝）**:
    - 功能：去除错误的背景区域分割
    - 核心思路：利用DINO特征的CLS token和空间注意力图来估计背景区域。构建一个背景模型，然后将与背景高度相似的分割结果（即错误识别为前景的背景区域）进行剪枝。具体通过阈值化背景得分来实现
    - 设计动机：无监督方法常将复杂纹理的背景区域（如天空中的云、地面纹理）误识别为前景物体，背景剪枝直接解决这一问题

### 损失函数 / 训练策略

ProMerge是一种纯推理时方法，不涉及训练。关键超参数包括：
- 初始分组的特征相似度阈值
- 合并的距离阈值和大小约束
- 背景剪枝的置信度阈值
此外，ProMerge生成的mask可以作为伪标签训练目标检测器，进一步提升性能。

## 实验关键数据

### 主实验

| 数据集 | 指标 | 本文 | 之前SOTA | 提升 |
|--------|------|------|----------|------|
| VOC2012 | AP50 | 有竞争力 | CutLER(NCut) | 接近或超越 |
| COCO val | AP | SOTA | CutLER | 超越 |
| 推理速度 | FPS | 显著更快 | NCut基线 | 5-10x |
| 伪标签训练检测器 | AP | SOTA | 无监督检测器 | 超越 |

### 消融实验

| 配置 | 关键指标 | 说明 |
|------|---------|------|
| 仅初始分组无合并 | 过度分割 | 每个物体被分成多个碎片 |
| 合并无背景剪枝 | 假阳性高 | 背景区域被错误分割 |
| 不同DINO版本 | DINOv2更优 | 更强的特征带来更好的分割 |
| 完整ProMerge | 最优 | 三个步骤互相配合 |

### 关键发现

- 简单的分组+合并策略在性能上可以匹敌甚至超越复杂的normalized-cut方法
- 推理速度提升5-10倍，使方法具有实际可部署性
- 背景剪枝显著减少了假阳性分割结果
- 使用ProMerge的伪标签训练目标检测器后，检测器性能超越了直接使用NCut伪标签的方法

## 亮点与洞察

- 方法哲学：用简单高效的方法替代复杂的优化过程，同时保持性能
- "先分后合"的策略在设计上很直觉，precision-first然后recover recall
- 背景处理是无监督分割中经常被忽视但至关重要的问题
- 将分割mask作为伪标签的应用展示了方法的实际价值

## 局限性 / 可改进方向

- 方法的效果强依赖于底层自监督特征的质量
- 合并策略的超参数需要调节，不同场景可能需要不同的阈值
- 对于重度遮挡的场景，空间邻接关系可能不够准确
- 可以探索可学习的合并策略替代基于规则的合并
- 扩展到视频实例分割是有意义的方向

## 相关工作与启发

- **CutLER**: 使用normalized-cut进行无监督实例分割的代表性工作
- **DINO/DINOv2**: 提供强大视觉特征的自监督模型
- **TokenCut**: 将DINO特征和图切割结合用于无监督对象发现
- 启发：自监督特征的质量不断提升，使得更简单的下游方法成为可能
